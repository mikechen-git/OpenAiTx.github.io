🌟 چارچوب کلی

آموزش: یک طرح آموزش دو مرحله‌ای پویا به‌طور متناوب بین بهینه‌سازی انسجام زمانی (مرحله‌ی سازگاری) و بهبود جزئیات مکانی با فرکانس بالا (مرحله‌ی ارتقاء) با درون‌یابی روان ضرر برای اطمینان از پایداری جابجا می‌شود.

استنتاج: در مرحله استنتاج، هر دو C-LoRA و D-LoRA در مدل UNet انتشار منجمد ادغام می‌شوند و ارتقاء یک‌مرحله‌ای ورودی‌های کم‌کیفیت به خروجی‌های با کیفیت بالا را ممکن می‌سازند.

python src/test_DLoRAL.py \ --pretrained_model_path /path/to/stable-diffusion-2-1-base \ --ram_ft_path /path/to/DAPE.pth \ --ram_path '/path/to/ram_swin_large_14m.pth' \ --merge_and_unload_lora False \ --process_size 512 \ --pretrained_model_name_or_path '/path/to/stable-diffusion-2-1-base' \ --vae_encoder_tiled_size 4096 \ --load_cfr \ --pretrained_path /path/to/model_checkpoint.pkl \ --stages 1 \ -i /path/to/input_videos/ \ -o /path/to/results
