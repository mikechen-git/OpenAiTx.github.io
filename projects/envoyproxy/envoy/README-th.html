envoy

⭐ 26105 stars Thai by envoyproxy

พร็อกซี่สำหรับ edge/middle/service ประสิทธิภาพสูงแบบ Cloud-native

Envoy ได้รับการดูแลโดย Cloud Native Computing Foundation (CNCF) หากคุณเป็นบริษัทที่ต้องการมีส่วนร่วมในการพัฒนาเทคโนโลยีที่บรรจุในคอนเทนเนอร์, ถูกจัดการแบบไดนามิก และเน้นไมโครเซอร์วิส โปรดพิจารณาเข้าร่วม CNCF สำหรับรายละเอียดเกี่ยวกับผู้ที่มีส่วนร่วมและบทบาทของ Envoy โปรดอ่าน ประกาศ ของ CNCF

  • envoy-perf: เฟรมเวิร์คสำหรับทดสอบประสิทธิภาพ
  • envoy-filter-example: ตัวอย่างการเพิ่มฟิลเตอร์ใหม่และลิงก์ไปยัง repository หลัก

การมีส่วนร่วม

การมีส่วนร่วมกับ Envoy นั้นสนุกและ C++ สมัยใหม่ก็ไม่ได้ยากอย่างที่คิด แม้ไม่มีประสบการณ์มาก่อน ในการเริ่มต้น:

  • คู่มือการมีส่วนร่วม
  • ปัญหาสำหรับผู้เริ่มต้น
  • เริ่มต้น build/test อย่างรวดเร็วโดยใช้ docker
  • คู่มือนักพัฒนา
  • พิจารณาติดตั้ง development support toolchain ของ Envoy ซึ่งช่วยอัตโนมัติบางส่วนของกระบวนการพัฒนา โดยเฉพาะอย่างยิ่งที่เกี่ยวกับการรีวิวโค้ด
  • โปรดแจ้งให้เราทราบหากคุณกำลังทำงานกับปัญหาใด เพื่อป้องกันการทำงานซ้ำซ้อน!

การประชุมชุมชน

ทีม Envoy มีกำหนดประชุมเดือนละสองครั้งในวันอังคาร เวลา 9am PT ปฏิทิน Google สาธารณะดูได้ ที่นี่ การประชุมจะจัดขึ้นเฉพาะเมื่อมีหัวข้อใน บันทึกการประชุม สมาชิกชุมชนสามารถเสนอหัวข้อวาระได้โดยเพิ่มในบันทึกการประชุม ผู้ดูแลจะยืนยันหัวข้อ หรือยกเลิกการประชุมภายใน 24 ชั่วโมงก่อนเวลานัด หากไม่มีหัวข้อที่ได้รับการยืนยัน

ความปลอดภัย

การตรวจสอบความปลอดภัย

มีการว่าจ้างบุคคลที่สามเพื่อตรวจสอบความปลอดภัยของ Envoy หลายครั้ง:

  • ในปี 2018 Cure53 ได้ทำการตรวจสอบความปลอดภัย, รายงานฉบับเต็ม
  • ในปี 2021 Ada Logics ได้ตรวจสอบโครงสร้าง fuzzing พร้อมข้อเสนอแนะปรับปรุง, รายงานฉบับเต็ม

การรายงานช่องโหว่ด้านความปลอดภัย

หากคุณพบช่องโหว่หรืออาจเป็นช่องโหว่ใน Envoy โปรดแจ้งเราที่ envoy-security เราจะส่งอีเมลยืนยันการรับรายงาน และจะส่งอีเมลเพิ่มเติมเมื่อเราตรวจสอบประเด็นดังกล่าวแล้ว

ดูรายละเอียดเพิ่มเติมได้ที่ กระบวนการออกประกาศด้านความปลอดภัยฉบับเต็ม

การ build สำหรับ ppc64le

การ build สำหรับสถาปัตยกรรม ppc64le หรือการใช้ aws-lc ไม่อยู่ภายใต้นโยบายความปลอดภัยของ envoy สถาปัตยกรรม ppc64le ขณะนี้อยู่ในขั้นทดสอบและไม่ได้รับการดูแลโดยทีม Envoy

การออกเวอร์ชัน

สำหรับรายละเอียดเพิ่มเติม โปรดดู กระบวนการออกเวอร์ชัน
