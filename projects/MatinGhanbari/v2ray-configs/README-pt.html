♾️ Configurações V2ray Gratuitas Para Todos

فارسی | 中文

🧩 Repositório mantido pela comunidade oferecendo configurações V2ray gratuitas para ajudar você a navegar com segurança e privacidade.

✨ Principais Características

  • Compatibilidade multi-protocolo (VMess, VLESS, Trojan, Shadowsocks, SSR, TUIC, Hysteria2, Warp)
  • Configurações atualizadas automaticamente a cada 15 minutos
  • Organizado por tipo de protocolo
  • Codificado em Base64 para conveniência
  • Fontes atualizadas frequentemente

🚀 Primeiros Passos

Escolha um dos métodos:

  1. Use as URLs de assinatura fornecidas
  2. Baixe manualmente as configurações da pasta subscriptions
