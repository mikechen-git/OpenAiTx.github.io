فارسی | 中文

🧩 커뮤니티가 관리하는 저장소로, 안전하고 개인적으로 인터넷을 탐색할 수 있도록 무료 V2ray 구성을 제공합니다.

✨ 주요 특징

  • 다중 프로토콜 호환성 (VMess, VLESS, Trojan, Shadowsocks, SSR, TUIC, Hysteria2, Warp)
  • 15분마다 자동으로 구성 파일 업데이트
  • 프로토콜 유형별로 정리됨
  • 편리함을 위한 Base64 인코딩 지원
  • 자주 갱신되는 소스

🚀 시작하기

아래 방법 중 하나를 선택하세요:
  • 제공된 구독 URL을 사용
  • subscriptions 폴더에서 직접 구성 파일 다운로드
