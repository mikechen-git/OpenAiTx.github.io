♾️ Бесплатные V2ray конфигурации для всех

فارسی | 中文

🧩 Репозиторий, поддерживаемый сообществом, предлагает бесплатные V2ray-конфиги для безопасного и приватного серфинга.

✨ Основные возможности

  • Мультипротокольная совместимость (VMess, VLESS, Trojan, Shadowsocks, SSR, TUIC, Hysteria2, Warp)
  • Автоматическое обновление конфигураций каждые 15 минут
  • Организовано по типу протокола
  • Кодирование в Base64 для удобства
  • Часто обновляемые источники

🚀 Начало работы

Выберите любой из способов:

  1. Используйте предоставленные ссылки на подписки
  2. Скачайте конфиги вручную из папки subscriptions
