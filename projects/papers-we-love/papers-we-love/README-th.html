Papers We Love (PWL) คือชุมชนที่สร้างขึ้นเพื่อการอ่าน พูดคุย และเรียนรู้เพิ่มเติมเกี่ยวกับบทความวิชาการด้านวิทยาการคอมพิวเตอร์ รีโพสิตอรีนี้ทำหน้าที่เป็นไดเรกทอรีรวบรวมบทความที่ดีที่สุดที่ชุมชนสามารถหาได้ นำเอกสารที่กระจัดกระจายอยู่บนอินเทอร์เน็ตมารวมไว้ด้วยกัน คุณยังสามารถเยี่ยมชม เว็บไซต์ Papers We Love เพื่อดูข้อมูลเพิ่มเติม

เนื่องจาก ลิขสิทธิ์ เราไม่สามารถโฮสต์บทความได้เสมอไป (หากโฮสต์ได้ คุณจะเห็นอีโมจิ :scroll: ถัดจากชื่อบทความใน README ของไดเรกทอรี) แต่เราสามารถให้ลิงก์ไปยังแหล่งที่มาของบทความเหล่านั้นได้

หากคุณชื่นชอบบทความเหล่านี้ อาจแวะเข้าร่วมงานพบปะของบทท้องถิ่นและเข้าร่วมการอภิปรายอย่างมีชีวิตชีวาเกี่ยวกับบทความเหล่านั้น คุณยังสามารถพูดคุยเกี่ยวกับกิจกรรม PWL เนื้อหาในรีโพสิตอรีนี้ หรือเรื่องอื่นๆ ที่เกี่ยวข้องกับ PWL ได้ที่ Discord ของเรา

ดาวน์โหลดบทความ

เปิดเทอร์มินัลที่คุณชื่นชอบแล้วรันคำสั่ง:

$ ./scripts/download.sh

คำสั่งนี้จะดึงลิงก์ PDF จากไฟล์ markdown และดาวน์โหลดบทความไปยังไดเรกทอรีที่เกี่ยวข้อง
