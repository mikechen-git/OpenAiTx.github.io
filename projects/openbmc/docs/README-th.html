เอกสาร OpenBMC

โครงการ OpenBMC เป็นโครงการของ Linux Foundation ซึ่งมีเป้าหมายในการสร้างเฟิร์มแวร์แบบโอเพ่นซอร์สที่สามารถปรับแต่งได้ สำหรับ Baseboard Management Controllers (BMCs) ที่เก็บนี้ประกอบด้วยเอกสาร สำหรับ OpenBMC ทั้งหมด อาจมีเอกสารเฉพาะแต่ละส่วนประกอบอยู่ในที่เก็บของแต่ละส่วนประกอบด้วย

เอกสาร คุณสมบัติ แสดงรายการคุณสมบัติหลักของโครงการพร้อม ลิงก์ไปยังข้อมูลเพิ่มเติม
