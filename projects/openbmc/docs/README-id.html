Dokumentasi OpenBMC

Proyek OpenBMC adalah proyek dari Linux Foundation yang bertujuan untuk menghasilkan tumpukan firmware sumber terbuka yang dapat dikustomisasi untuk Baseboard Management Controllers (BMC). Repositori ini berisi dokumentasi untuk OpenBMC secara keseluruhan. Mungkin terdapat dokumentasi spesifik komponen di repositori untuk setiap komponen.

Dokumen fitur mencantumkan fitur utama proyek dengan tautan ke informasi lebih lanjut.

Kontak

  • Arsip Daftar: https://lore.kernel.org/openbmc/

Pengembangan OpenBMC

Dokumen-dokumen berikut berisi detail tentang pengembangan kode OpenBMC itu sendiri

  • cheatsheet.md: Referensi cepat untuk beberapa tugas pengembangan umum

  • CONTRIBUTING.md: Panduan untuk berkontribusi ke OpenBMC

  • tutorial pengembangan: Tutorial untuk memulai pengembangan OpenBMC

  • kernel-development.md: Referensi untuk tugas-tugas pengembangan kernel umum

Penggunaan OpenBMC

Dokumen-dokumen berikut menjelaskan cara menggunakan OpenBMC, termasuk menggunakan antarmuka programatik ke sistem OpenBMC.

  • code-update: Memperbarui firmware OpenBMC dan platform host

  • console.md: Menggunakan konsol host

  • host-management.md: Melakukan tugas manajemen host dengan OpenBMC

  • rest-api.md: Pengantar penggunaan OpenBMC REST API

  • REDFISH-cheatsheet.md: Referensi cepat untuk beberapa perintah OpenBMC Redfish yang umum

  • REST-cheatsheet.md: Referensi cepat untuk beberapa perintah OpenBMC REST API yang umum
