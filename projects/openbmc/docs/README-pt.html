Documentação do OpenBMC

O projeto OpenBMC é um projeto da Linux Foundation cujo objetivo é produzir uma pilha de firmware personalizável e de código aberto para Controladores de Gerenciamento de Placa-mãe (BMCs). Este repositório contém a documentação do OpenBMC como um todo. Pode haver documentação específica de componentes no repositório de cada componente.

O documento de funcionalidades lista as principais funcionalidades do projeto com links para mais informações.
