Dokumentacja OpenBMC

Projekt OpenBMC to projekt Fundacji Linux, którego celem jest stworzenie konfigurowalnego, otwartoźródłowego stosu firmware dla kontrolerów zarządzania płytą główną (BMC). To repozytorium zawiera dokumentację dla całego projektu OpenBMC. Dokumentacja specyficzna dla komponentów może znajdować się w repozytorium każdego z komponentów.

Dokument funkcje zawiera listę głównych funkcji projektu wraz z odnośnikami do dodatkowych informacji.

Rozwój OpenBMC

Te dokumenty zawierają szczegóły dotyczące rozwoju samego kodu OpenBMC

  • cheatsheet.md: Szybka referencja dla niektórych typowych zadań deweloperskich

  • CONTRIBUTING.md: Wytyczne dotyczące współtworzenia OpenBMC

  • samouczki rozwoju: Samouczki pomagające rozpocząć pracę z rozwojem OpenBMC

Użytkowanie OpenBMC

Te dokumenty opisują, jak używać OpenBMC, w tym korzystać z interfejsów programistycznych systemu OpenBMC.

  • code-update: Aktualizacja OpenBMC oraz firmware platformy głównej

  • console.md: Korzystanie z konsoli hosta

  • host-management.md: Wykonywanie zadań zarządzania hostem za pomocą OpenBMC

  • rest-api.md: Wprowadzenie do korzystania z OpenBMC REST API

  • REDFISH-cheatsheet.md: Szybka referencja dla typowych poleceń OpenBMC Redfish

  • REST-cheatsheet.md: Szybka referencja dla typowych poleceń OpenBMC REST API
