ComfyUI FLOAT

本專案提供 FLOAT 的 ComfyUI 包裝器，用於音頻驅動的說話人像生成運動潛在流匹配。

如需更先進且持續維護的版本，請參考：ComfyUI-FLOAT_Optimized

⭐ 支持

如果你喜歡我的專案並希望看到更多更新與新功能，請考慮支持我。這對我幫助很大！

🚀 安裝

git clone https://github.com/yuvraj108c/ComfyUI-FLOAT.git
cd ./ComfyUI-FLOAT
pip install -r requirements.txt

☀️ 使用方式

  • 載入 範例工作流
  • 上傳驅動圖像及音頻，點擊 queue
  • 模型 會自動下載到 /ComfyUI/models/float
  • 模型目錄結構如下：
``.bash |-- float.pth # 主模型 |-- wav2vec2-base-960h/ # 音頻編碼器 | |-- config.json | |-- model.safetensors | |-- preprocessor_config.json |-- wav2vec-english-speech-emotion-recognition/ # 情緒編碼器 |-- config.json |-- preprocessor_config.json |-- pytorch_model.bin

🛠️ 參數

  • ref_image: 參考臉部圖像（必須為 batch size 1）
  • ref_audio: 參考音頻（針對較長音頻（例如 3 分鐘以上），請確保你有足夠的 ram/vram）
  • a_cfg_scale: 音頻分類器自由引導比例（預設：2）
  • r_cfg_scale: 參考分類器自由引導比例（預設：1）
  • emotion: none, angry, disgust, fear, happy, neutral, sad, surprise（預設：none）
  • e_cfg_scale: 情緒強度（預設：1）。若需更強烈的情緒表現，請嘗試設置 5 至 10 的較大值
  • crop: 僅當參考圖像未置中臉部時啟用
  • fps`: 輸出影片的幀率（預設：25）
