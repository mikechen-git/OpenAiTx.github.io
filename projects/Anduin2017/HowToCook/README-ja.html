プログラマーのための料理ガイド

最近は家で料理をすることが多く、プログラマーとして時々ネットでレシピや作り方を探しています。しかし、これらのレシピは書き方がバラバラで、途中で突然材料が出てきたりします。形式言語に慣れたプログラマーにとっては非常に分かりづらいです。

そこで、自分でレシピを探し、実際に料理した経験と組み合わせて、より明確で正確な記述で一般的な料理の作り方をまとめ、プログラマーが家で料理しやすいようにしようと思います。

同時に、これはコミュニティ主導・維持のオープンソースプロジェクトにしたいと考えています。多くの人が一緒に面白いリポジトリを作れるようにしたいので、ぜひ貢献してください！

ローカルデプロイ

レシピのWebサービスをローカルでデプロイしたい場合は、Dockerをインストール後、以下のコマンドを実行してください。

docker pull ghcr.io/anduin2017/how-to-cook:latest
docker run -d -p 5000:5000 ghcr.io/anduin2017/how-to-cook:latest

PDFバージョンをダウンロードしたい場合は、ブラウザで /document.pdf にアクセスしてください。

貢献方法

問題点を発見した場合は、直接修正しPull requestを送ってください。

新しいレシピを書く際は、既存のレシピテンプレートをコピー・修正して作成してください: サンプル料理。
