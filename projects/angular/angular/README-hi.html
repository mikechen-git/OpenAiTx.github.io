विकास सेटअप

आवश्यकताएँ

  • Node.js स्थापित करें जिसमें Node Package Manager शामिल है

प्रोजेक्ट सेटअप करना

Angular CLI को ग्लोबली इंस्टॉल करें:

npm install -g @angular/cli

वर्कस्पेस बनाएं:

ng new [PROJECT NAME]

एप्लिकेशन चलाएं:

cd [PROJECT NAME]
ng serve

Angular क्रॉस-प्लेटफ़ॉर्म, तेज़, स्केलेबल है, इसमें शानदार टूलिंग है, और इसे लाखों लोग पसंद करते हैं।

चेंजलॉग

नवीनतम सुधारों के बारे में जानें।

अपग्रेड करना

अपने प्रोजेक्ट को अपग्रेड करने का सबसे अच्छा तरीका जानने के लिए हमारा अपग्रेड गाइड देखें।

योगदान

योगदान दिशानिर्देश

हमारे योगदान दिशानिर्देश पढ़ें ताकि आप हमारे सबमिशन प्रक्रिया, कोडिंग नियमों और अन्य बातों के बारे में जान सकें।

मदद करना चाहते हैं?

क्या आप कोई बग रिपोर्ट करना चाहते हैं, कुछ कोड में योगदान देना चाहते हैं, या दस्तावेज़ में सुधार करना चाहते हैं? उत्कृष्ट! योगदान करने के लिए हमारे दिशानिर्देश पढ़ें और फिर हमारे मदद चाहिए या पहला अच्छा मुद्दा लेबल वाले मुद्दों में से किसी एक को देखें।
