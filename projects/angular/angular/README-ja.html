開発環境セットアップ

前提条件

  • Node.js（Node Package Managerを含む）をインストールしてください。

プロジェクトのセットアップ

Angular CLIをグローバルにインストールします:

npm install -g @angular/cli

ワークスペースの作成:

ng new [PROJECT NAME]

アプリケーションの実行:

cd [PROJECT NAME]
ng serve
