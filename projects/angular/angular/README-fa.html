راه‌اندازی توسعه

پیش‌نیازها

  • نصب Node.js که شامل Node Package Manager نیز می‌شود

راه‌اندازی یک پروژه

Angular CLI را به صورت سراسری نصب کنید:

npm install -g @angular/cli

ایجاد workspace:

ng new [PROJECT NAME]

اجرای برنامه:

cd [PROJECT NAME]
ng serve

Angular چندسکویی، سریع، مقیاس‌پذیر، دارای ابزارهای فوق‌العاده و محبوب میلیون‌ها نفر است.
