开发环境搭建

前置条件

  • 安装 Node.js（包含 Node 包管理器）

项目搭建

全局安装 Angular CLI：

npm install -g @angular/cli

创建工作区：

ng new [PROJECT NAME]

运行应用：

cd [PROJECT NAME]
ng serve
