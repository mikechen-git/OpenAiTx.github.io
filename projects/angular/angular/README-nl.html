Vereisten

Een project opzetten

Installeer de Angular CLI globaal:

npm install -g @angular/cli

Maak een workspace aan:

ng new [PROJECTNAAM]

Start de applicatie:

cd [PROJECTNAAM]
ng serve

Angular is cross-platform, snel, schaalbaar, heeft geweldige tooling, en wordt geliefd door miljoenen.
