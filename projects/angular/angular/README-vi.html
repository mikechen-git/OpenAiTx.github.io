Thiết lập phát triển

Yêu cầu

  • Cài đặt Node.js, bao gồm cả Node Package Manager

Thiết lập một dự án

Cài đặt Angular CLI toàn cục:

npm install -g @angular/cli

Tạo workspace:

ng new [TÊN DỰ ÁN]

Chạy ứng dụng:

cd [TÊN DỰ ÁN]
ng serve
