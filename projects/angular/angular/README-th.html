Angular - แพลตฟอร์มสำหรับนักพัฒนาเว็บยุคใหม่


Angular เป็นแพลตฟอร์มสำหรับการพัฒนาแอปพลิเคชันเว็บทั้งบนมือถือและเดสก์ท็อป
โดยใช้ TypeScript/JavaScript และภาษาอื่นๆ

angular.dev

แนวทางการมีส่วนร่วม · แจ้งปัญหา · บล็อก

เอกสารประกอบ

เริ่มต้นใช้งาน Angular, ศึกษาพื้นฐาน และสำรวจหัวข้อขั้นสูงได้ที่เว็บไซต์เอกสารของเรา

  • เริ่มต้นใช้งาน
  • สถาปัตยกรรม
  • คอมโพเนนต์และเทมเพลต
  • ฟอร์ม
  • API

ขั้นสูง

  • Angular Elements
  • Server Side Rendering
  • Schematics
  • Lazy Loading
  • แอนิเมชัน

การพัฒนาในเครื่อง

หากต้องการร่วมพัฒนาเอกสาร Angular ดูข้อมูลที่ Angular.dev README

การตั้งค่าสำหรับการพัฒนา

ข้อกำหนดเบื้องต้น

  • ติดตั้ง Node.js ซึ่งรวม Node Package Manager มาแล้ว

การตั้งค่าโปรเจกต์

ติดตั้ง Angular CLI ทั่วไปในระบบ:

npm install -g @angular/cli

สร้าง workspace:

ng new [PROJECT NAME]

รันแอปพลิเคชัน:

cd [PROJECT NAME]
ng serve

Angular รองรับข้ามแพลตฟอร์ม, รวดเร็ว, ขยายขนาดได้, มีเครื่องมือที่ยอดเยี่ยม และเป็นที่ชื่นชอบของผู้ใช้นับล้าน

เริ่มต้นอย่างรวดเร็ว

เริ่มต้นใช้งานใน 5 นาที.

ระบบนิเวศ

  • Angular Material

บันทึกการเปลี่ยนแปลง

เรียนรู้เกี่ยวกับการพัฒนาใหม่ล่าสุด.

การอัปเกรด

ดู คู่มือการอัปเกรด เพื่อค้นหาวิธีที่ดีที่สุดในการอัปเกรดโปรเจกต์ของคุณ
