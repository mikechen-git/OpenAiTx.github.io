CentrifugalDustSeparator

Centrifugal Dust Separator - MK2 - Berkas 3D untuk pencetakan (Rilis Beta)
Dukung saya di: https://www.patreon.com/CapturingDust

Pemasok tabung dapat berbeda tergantung pada lokasi geografis Anda, begitu juga dengan dimensi tabung.
Berkas telah disesuaikan, lihat file README di versi build spesifik untuk info pemasok:

*catatan: Cetak terlebih dahulu cincin skala untuk memeriksa dimensi tabung Anda. Lihat: https://www.youtube.com/shorts/JZXLTLeB2oE
**catatan: Untuk bagian yang tidak tercantum di subfolder (seperti Y-pieces) lihat versi MK2 biasa.

Semua bagian dirancang untuk dicetak pada volume build bambulab (WxDxH) 256 x 256 x 256 mm³.
Hanya Y-pieces yang tidak kompatibel dengan build plate Prusa.
