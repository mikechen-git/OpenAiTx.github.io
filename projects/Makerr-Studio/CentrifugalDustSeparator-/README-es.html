CentrifugalDustSeparator

Separador de Polvo Centrífugo - MK2 - Archivos 3D para impresión (Lanzamiento Beta)
Apóyame en: https://www.patreon.com/CapturingDust

Los proveedores de tubos pueden variar según tu ubicación geográfica, al igual que las dimensiones del tubo.
Los archivos están ajustados en consecuencia, consulta los archivos README en las versiones de compilación específicas para información sobre proveedores:

*nota: Primero imprime el anillo de escala para comprobar las dimensiones de tu tubo. Ver: https://www.youtube.com/shorts/JZXLTLeB2oE
**nota: Para las piezas que no están listadas en las subcarpetas (como las piezas en Y), consulta la versión normal de MK2.

Todas las piezas están diseñadas para imprimirse en el volumen de construcción de bambulab (AnxPrxAl) 256 x 256 x 256 mm³.
Solo las piezas en Y no son compatibles con la placa de construcción de Prusa.
