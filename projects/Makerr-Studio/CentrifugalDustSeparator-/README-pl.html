CentrifugalDustSeparator

Separator Pyłu Odśrodkowego - MK2 - Pliki 3D do druku (Wersja beta)
Wesprzyj mnie na: https://www.patreon.com/CapturingDust

Dostawcy rur mogą się różnić w zależności od Twojej lokalizacji geograficznej, podobnie jak wymiary rur.
Pliki są odpowiednio dostosowane, zobacz pliki README w odpowiednich wersjach projektowych, aby uzyskać informacje o dostawcach:

*uwaga: Najpierw wydrukuj pierścień skalujący, aby sprawdzić wymiary swojej rury. Zobacz: https://www.youtube.com/shorts/JZXLTLeB2oE
**uwaga: Części, które nie są wymienione w podfolderach (takie jak elementy Y), znajdują się w standardowej wersji MK2.

Wszystkie części są zaprojektowane do druku na polu roboczym bambulab (WxDxH) 256 x 256 x 256 mm³.
Tylko elementy Y nie są kompatybilne z platformą roboczą Prusa.
