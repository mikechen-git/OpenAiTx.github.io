CentrifugalDustSeparator

원심 분진 분리기 - MK2 - 3D 프린팅 파일 (베타 릴리스)
후원하기: https://www.patreon.com/CapturingDust

튜브 공급업체는 지역에 따라 다를 수 있으며, 튜브의 치수 또한 다를 수 있습니다.
파일은 이에 맞게 조정되어 있으니, 특정 빌드 버전의 README 파일에서 공급업체 정보를 확인하세요:

*참고: 스케일링 링을 먼저 출력하여 튜브 치수를 확인하세요. 참고: https://www.youtube.com/shorts/JZXLTLeB2oE
**참고: 하위 폴더에 없는 부품(예: Y-피스)은 일반 MK2 버전을 참고하세요.

모든 부품은 bambulab 빌드 볼륨(WxDxH) 256 x 256 x 256 mm³ 내에서 출력할 수 있도록 설계되었습니다.
Y-피스만 Prusa 빌드 플레이트와 호환되지 않습니다.
