CentrifugalDustSeparator

Separatore di Polveri Centrifugo - MK2 - File 3D per la stampa (Beta release)
Sostienimi su: https://www.patreon.com/CapturingDust

I fornitori dei tubi possono variare in base alla tua posizione geografica, così come le dimensioni dei tubi.
I file sono adattati di conseguenza, consulta i file README nelle versioni specifiche di build per informazioni sui fornitori:

*nota: Prima stampa l’anello di calibrazione per controllare le dimensioni del tuo tubo. Vedi: https://www.youtube.com/shorts/JZXLTLeB2oE
**nota: Per le parti non elencate nelle sottocartelle (come i pezzi a Y) consulta la versione normale MK2.

Tutte le parti sono progettate per essere stampate su un volume di stampa bambulab (LxPxA) 256 x 256 x 256 mm³.
Solo i pezzi a Y non sono compatibili con il piano di stampa Prusa.
