Tome

Tome es una aplicación de escritorio que permite que cualquiera aproveche la magia de los LLMs y MCP. Descarga Tome, conecta cualquier LLM local o remoto y enlázalo con miles de servidores MCP para crear tu propio libro de hechizos impulsado por IA.

🫥 ¿Quieres que sea 100% local, 100% privado? Usa Ollama y Qwen3 solo con servidores MCP locales para lanzar hechizos en tu propio universo de bolsillo. ⚡ ¿Quieres modelos en la nube de última generación con los servidores MCP remotos más recientes? También puedes tener eso. ¡Todo depende de ti!

🏗️ Esta es una Vista Previa Técnica, así que ten en cuenta que habrá aspectos por pulir. Únete a nosotros en Discord para compartir consejos, trucos y problemas que encuentres. ¡Dale una estrella a este repositorio para estar al tanto de actualizaciones y lanzamientos de nuevas funciones!

🪄 Características

  • 🧙 Experiencia simplificada y apta para principiantes
    • Simplemente descarga e instala Tome y conecta el LLM de tu elección
    • Sin complicaciones con JSON, Docker, python o node
  • 🤖 Soporte para modelos de IA
    • Remoto: Google Gemini, OpenAI, cualquier endpoint compatible con la API de OpenAI
    • Local: Ollama, LM Studio, Cortex, cualquier endpoint compatible con la API de OpenAI
  • 🔮 Soporte mejorado para MCP
    • Interfaz para instalar, eliminar, activar/desactivar servidores MCP
    • Soporte inmediato para servidores MCP basados en npm, uvx, node y python
  • 🏪 Integración en el registro de Smithery.ai
    • Miles de servidores MCP disponibles mediante instalación con un solo clic
  • ✏️ Personalización de ventanas de contexto y temperatura
  • 🧰 Soporte nativo para llamadas a herramientas y modelos de razonamiento
    • Mejoras en la interfaz que distinguen claramente las llamadas a herramientas y los mensajes de razonamiento

Primeros Pasos

Requisitos

  • MacOS o Windows (¡Linux próximamente!)
  • Proveedor de LLM de tu elección: Ollama o clave de API de Gemini son fáciles/gratis
  • Descarga la última versión de Tome

Inicio Rápido

  1. Instala Tome
  2. Conecta tu proveedor LLM preferido: OpenAI, Ollama y Gemini ya están preconfigurados, pero también puedes añadir proveedores como LM Studio usando http://localhost:1234/v1 como URL
  3. Abre la pestaña MCP en Tome e instala tu primer servidor MCP (Fetch es uno fácil para empezar, solo pega uvx mcp-server-fetch en el campo del servidor).
  4. ¡Chatea con tu modelo potenciado por MCP! Pídele que obtenga la noticia principal en Hacker News.

Visión

Queremos hacer que los LLMs locales y MCP sean accesibles para todos. Estamos construyendo una herramienta que te permite ser creativo con los LLMs, sin importar si eres ingeniero, aficionado, entusiasta o cualquier persona intermedia.

Principios Fundamentales

  • Tome es primero local: Tú tienes el control de a dónde va tu información.
  • Tome es para todos: No deberías tener que manejar lenguajes de programación, gestores de paquetes o archivos de configuración json.

Próximos Pasos

Hemos recibido muchos comentarios increíbles en las últimas semanas desde que lanzamos Tome, pero tenemos grandes planes para el futuro. Queremos sacar a los LLMs de su caja de chat, y tenemos muchas funciones en camino para ayudarles a hacer eso.

  • Tareas programadas: Los LLMs deberían hacer cosas útiles incluso cuando no estés frente a la computadora.
  • Integraciones nativas: Los servidores MCP son una excelente forma de acceder a herramientas e información, pero queremos añadir integraciones más potentes para interactuar con los LLMs de formas únicas.
  • Constructor de aplicaciones: creemos que a largo plazo las mejores experiencias no estarán en una interfaz de chat. Tenemos planes para añadir herramientas adicionales que te permitirán crear aplicaciones y flujos de trabajo potentes.
