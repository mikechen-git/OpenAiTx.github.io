शुरुआती के लिए वेब डेवलपमेंट - एक पाठ्यक्रम

Microsoft Cloud Advocates द्वारा हमारे 12-सप्ताह के व्यापक पाठ्यक्रम के साथ वेब डेवलपमेंट के मूल सिद्धांत सीखें। प्रत्येक 24 पाठ JavaScript, CSS, और HTML को प्रैक्टिकल प्रोजेक्ट्स जैसे टेरारियम, ब्राउज़र एक्सटेंशन, और स्पेस गेम्स के माध्यम से समझाता है। क्विज़, चर्चाओं, और प्रैक्टिकल असाइनमेंट्स के साथ जुड़ें। अपनी स्किल्स बढ़ाएं और हमारे प्रभावी प्रोजेक्ट-आधारित शिक्षण के साथ ज्ञान को बेहतर बनाएं। आज ही अपनी कोडिंग यात्रा शुरू करें!

🧑‍🎓 क्या आप एक छात्र हैं?

Student Hub पेज पर जाएं जहाँ आपको शुरुआती संसाधन, Student packs और यहां तक कि फ्री प्रमाणपत्र वाउचर प्राप्त करने के तरीके मिलेंगे। इस पेज को बुकमार्क करें और समय-समय पर चेक करते रहें क्योंकि हम हर महीने सामग्री बदलते हैं।

  • मजेदार और आकर्षक कहानी, आप समय यात्रा करेंगे!

🌱 प्रारंभ करना

शिक्षकों के लिए, हमने कुछ सुझाव शामिल किए हैं कि इस पाठ्यक्रम का उपयोग कैसे करें। हमें आपकी प्रतिक्रिया का इंतजार रहेगा हमारे चर्चा फोरम में!

शिक्षार्थियों, प्रत्येक पाठ के लिए, एक प्री-लेक्चर क्विज़ से शुरू करें और व्याख्यान सामग्री पढ़ें, विभिन्न गतिविधियाँ पूरी करें और पोस्ट-लेक्चर क्विज़ के साथ अपनी समझ जांचें।

अपने सीखने के अनुभव को बेहतर बनाने के लिए, अपने साथियों से जुड़ें और प्रोजेक्ट्स पर एक साथ काम करें! चर्चाओं के लिए हमारे चर्चा फोरम में भाग लें, जहाँ हमारी मॉडरेशन टीम आपके सवालों के जवाब देगी।

अपनी शिक्षा को आगे बढ़ाने के लिए, हम Microsoft Learn में अतिरिक्त अध्ययन सामग्री देखने की सलाह देते हैं।

📋 अपना वातावरण सेटअप करें

यह पाठ्यक्रम एक तैयार डेवलपमेंट वातावरण के साथ आता है! आप चाहें तो इसे Codespace (ब्राउज़र-आधारित, कोई इंस्टॉल आवश्यकता नहीं) या अपने कंप्यूटर पर जैसे Visual Studio Code के टेक्स्ट एडिटर के साथ चला सकते हैं।

अपनी रिपॉजिटरी बनाएं

अपने कार्य को आसानी से सहेजने के लिए, अनुशंसा की जाती है कि आप इस रिपॉजिटरी की अपनी प्रति बनाएं। ऐसा करने के लिए पेज के शीर्ष पर Use this template बटन पर क्लिक करें। इससे आपके GitHub अकाउंट में एक नई रिपॉजिटरी बन जाएगी जिसमें पाठ्यक्रम की कॉपी होगी।

इन चरणों का पालन करें:

  1. रिपॉजिटरी को फोर्क करें: इस पेज के ऊपर दाईं ओर "Fork" बटन पर क्लिक करें।
  2. रिपॉजिटरी को क्लोन करें: git clone https://github.com/microsoft/Web-Dev-For-Beginners.git

Codespace में पाठ्यक्रम चलाएं

अपनी बनाई गई रिपॉजिटरी में Code बटन पर क्लिक करें और Open with Codespaces चुनें। इससे आपके लिए एक नया Codespace बन जाएगा।

अपने कंप्यूटर पर पाठ्यक्रम चलाएं

अपने कंप्यूटर पर पाठ्यक्रम चलाने के लिए, आपको एक टेक्स्ट एडिटर, एक ब्राउज़र और एक कमांड लाइन टूल की आवश्यकता होगी। हमारा पहला पाठ, प्रोग्रामिंग भाषाएँ और टूल्स का परिचय, आपको इन टूल्स के विकल्पों के बारे में बताएगा।

हमारी सिफारिश है कि आप Visual Studio Code का उपयोग करें, जिसमें एक बिल्ट-इन Terminal भी है। Visual Studio Code यहाँ डाउनलोड करें।

  1. अपनी रिपॉजिटरी को अपने कंप्यूटर पर क्लोन करें। Code बटन पर क्लिक करके URL कॉपी करें:

    फिर, Visual Studio Code के अंदर Terminal खोलें और निम्न कमांड चलाएँ, <your-repository-url> को अपने URL से बदलें:

    git clone <your-repository-url>

  2. Visual Studio Code में फ़ोल्डर खोलें। इसके लिए File > Open Folder पर क्लिक करें और जो फ़ोल्डर आपने क्लोन किया है उसे चुनें।

अनुशंसित Visual Studio Code एक्सटेंशन:

  • Live Server - Visual Studio Code के भीतर HTML पेज का प्रीव्यू देखने के लिए
  • Copilot - कोड जल्दी लिखने में मदद के लिए

  • ज्ञान जांच

यह प्रोग्राम JavaScript, HTML, और CSS की मूल बातें सिखाता है, साथ ही आज के वेब डेवलपर्स द्वारा उपयोग किए जाने वाले नवीनतम टूल्स और तकनीकों को भी। विद्यार्थी टाइपिंग गेम, वर्चुअल टेरारियम, इको-फ्रेंडली ब्राउज़र एक्सटेंशन, स्पेस-इनवेडर-स्टाइल गेम, और व्यवसायों के लिए बैंकिंग ऐप बनाकर व्यावहारिक अनुभव प्राप्त करेंगे। श्रृंखला के अंत तक, विद्यार्थियों को वेब डेवलपमेंट की मजबूत समझ मिल जाएगी।

🎓 आप इस पाठ्यक्रम के शुरूआती कुछ पाठों को Learn Path के रूप में Microsoft Learn पर ले सकते हैं!

सुनिश्चित करके कि सामग्री प्रोजेक्ट्स के साथ मेल खाती है, प्रक्रिया विद्यार्थियों के लिए अधिक आकर्षक बनती है और अवधारणाओं की पुनरावृत्ति बढ़ती है। हमने JavaScript बेसिक्स में कई स्टार्टर पाठ भी लिखे हैं, जिन्हें "Beginners Series to: JavaScript" वीडियो ट्यूटोरियल संग्रह के वीडियो के साथ जोड़ा गया है, जिसके कुछ लेखक इस पाठ्यक्रम के योगदानकर्ता हैं।

इसके अलावा, कक्षा से पहले एक कम जोखिम वाला क्विज़ विद्यार्थी के इरादे को सेट करता है, जबकि कक्षा के बाद दूसरा क्विज़ आगे की पुनरावृत्ति सुनिश्चित करता है। यह पाठ्यक्रम लचीला और मजेदार होने के लिए डिज़ाइन किया गया है और इसे आंशिक या पूरी तरह से लिया जा सकता है। प्रोजेक्ट्स छोटे से शुरू होते हैं और 12-सप्ताह के चक्र के अंत तक जटिल हो जाते हैं।

हमने जान-बूझकर JavaScript फ्रेमवर्क्स को शामिल नहीं किया है ताकि एक वेब डेवलपर बनने के लिए आवश्यक बुनियादी कौशलों पर ध्यान केंद्रित किया जा सके, फ्रेमवर्क अपनाने से पहले। इस पाठ्यक्रम को पूरा करने के बाद, एक अच्छा अगला कदम Node.js के बारे में सीखना होगा: "Beginner Series to: Node.js"।

हमारे आचार संहिता और योगदान दिशानिर्देश देखें। हम आपकी रचनात्मक प्रतिक्रिया का स्वागत करते हैं!
