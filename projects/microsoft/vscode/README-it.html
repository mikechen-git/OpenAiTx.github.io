Visual Studio Code - Open Source ("Code - OSS")

Il Repository

Questo repository ("Code - OSS") è il luogo dove noi (Microsoft) sviluppiamo il prodotto Visual Studio Code insieme alla comunità. Qui non solo lavoriamo sul codice e sulle segnalazioni, ma pubblichiamo anche la nostra roadmap, i piani di iterazione mensili e i nostri piani di rilascio finale. Questo codice sorgente è disponibile per tutti sotto la normale licenza MIT.

Visual Studio Code

Visual Studio Code è una distribuzione del repository Code - OSS con personalizzazioni specifiche di Microsoft rilasciata sotto una tradizionale licenza prodotto Microsoft.

Visual Studio Code unisce la semplicità di un editor di codice a ciò di cui gli sviluppatori hanno bisogno per il loro ciclo fondamentale di modifica-compilazione-debug. Offre un supporto completo per l'editing del codice, la navigazione e la comprensione del codice, oltre a un debug leggero, un ricco modello di estensibilità e un'integrazione leggera con gli strumenti esistenti.

Visual Studio Code viene aggiornato mensilmente con nuove funzionalità e correzioni di bug. Puoi scaricarlo per Windows, macOS e Linux sul sito di Visual Studio Code. Per ricevere le versioni più recenti ogni giorno, installa la build Insiders.

Contribuire

Ci sono molti modi in cui puoi partecipare a questo progetto, ad esempio:

  • Segnala bug e richieste di funzionalità, e aiutaci a verificarli man mano che vengono risolti
  • Revisiona le modifiche al codice sorgente
  • Revisiona la documentazione e invia pull request per qualsiasi cosa, da refusi a contenuti aggiuntivi o nuovi

Se sei interessato a risolvere problemi e contribuire direttamente alla base di codice, consulta il documento Come contribuire, che tratta i seguenti argomenti:

  • Come compilare ed eseguire dal sorgente
  • Il flusso di lavoro di sviluppo, compreso il debug e l'esecuzione dei test
  • Linee guida per la scrittura del codice
  • Invio di pull request
  • Come trovare un problema su cui lavorare
  • Contributi alle traduzioni

Progetti Collegati

Molti dei componenti principali e delle estensioni di VS Code hanno repository propri su GitHub. Ad esempio, i repository del node debug adapter e del mono debug adapter sono separati tra loro. Per un elenco completo, visita la pagina Progetti Collegati sul nostro wiki.

Estensioni Incluse

VS Code include una serie di estensioni integrate situate nella cartella extensions, incluse grammatiche e snippet per molti linguaggi. Le estensioni che forniscono un supporto linguistico avanzato (completamento del codice, Vai alla definizione) per un linguaggio hanno il suffisso language-features. Ad esempio, l'estensione json fornisce la colorazione per JSON e l'estensione json-language-features fornisce un supporto linguistico avanzato per JSON.

Container di Sviluppo

Questo repository include un container di sviluppo per Visual Studio Code Dev Containers / GitHub Codespaces.

  • Per i Dev Containers, usa il comando Dev Containers: Clone Repository in Container Volume... che crea un volume Docker per migliori prestazioni disco su macOS e Windows.

    • Se hai già installato VS Code e Docker, puoi anche cliccare qui per iniziare. Questo farà sì che VS Code installi automaticamente l'estensione Dev Containers se necessario, cloni il codice sorgente in un volume del container e avvii un container di sviluppo pronto all'uso.

  • Per Codespaces, installa l'estensione GitHub Codespaces in VS Code e utilizza il comando Codespaces: Create New Codespace.

Docker / il Codespace dovrebbe avere almeno 4 Core e 6 GB di RAM (8 GB consigliati) per eseguire una build completa. Consulta il README del container di sviluppo per maggiori informazioni.
