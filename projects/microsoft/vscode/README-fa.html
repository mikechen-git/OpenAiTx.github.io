ویژوال استودیو کد - متن‌باز ("Code - OSS")

مخزن

این مخزن ("Code - OSS") جایی است که ما (مایکروسافت) محصول ویژوال استودیو کد را به همراه جامعه توسعه می‌دهیم. نه تنها روی کد و مسائل اینجا کار می‌کنیم، بلکه نقشه راه، برنامه‌های تکرار ماهانه و برنامه‌های مرحله نهایی خود را نیز منتشر می‌کنیم. این کد منبع برای همه تحت مجوز MIT استاندارد در دسترس است.

ویژوال استودیو کد

ویژوال استودیو کد توزیعی از مخزن Code - OSS با سفارشی‌سازی‌های خاص مایکروسافت است که تحت مجوز محصول سنتی مایکروسافت منتشر شده است.

ویژوال استودیو کد سادگی یک ویرایشگر کد را با آنچه توسعه‌دهندگان برای چرخه اصلی ویرایش-ساخت-اشکال‌زدایی نیاز دارند، ترکیب می‌کند. این ابزار امکانات جامع ویرایش، پیمایش و درک کد به همراه اشکال‌زدایی سبک، مدل توسعه‌پذیری غنی و یکپارچگی سبک با ابزارهای موجود را ارائه می‌دهد.

ویژوال استودیو کد هر ماه با ویژگی‌های جدید و رفع اشکال به‌روزرسانی می‌شود. می‌توانید آن را برای ویندوز، مک و لینوکس از وب‌سایت ویژوال استودیو کد دانلود کنید. برای دریافت جدیدترین نسخه‌ها به صورت روزانه، نسخه Insiders را نصب کنید.

مشارکت

راه‌های زیادی برای مشارکت در این پروژه وجود دارد، برای مثال:

  • ثبت باگ‌ها و درخواست ویژگی‌ها، و کمک به ما برای بررسی آن‌ها هنگام اضافه شدن
  • بازبینی تغییرات کد منبع
  • بازبینی مستندات و ارسال pull request برای هر چیزی از غلط‌ املایی تا افزودن و محتوای جدید

اگر علاقه‌مند به رفع مشکلات و مشارکت مستقیم در پایگاه کد هستید، لطفاً سند چگونه مشارکت کنیم را ببینید که شامل موارد زیر است:

  • چگونه کد را بسازیم و از منبع اجرا کنیم
  • فرآیند توسعه، شامل اشکال‌زدایی و اجرای تست‌ها
  • راهنمای کدنویسی
  • ارسال pull request
  • یافتن مسئله‌ای برای کار
  • مشارکت در ترجمه‌ها

بازخورد

  • پرسش خود را در Stack Overflow مطرح کنید
  • درخواست ویژگی جدید
  • درخواست‌های ویژگی محبوب را رای دهید
  • ثبت یک مشکل
  • ارتباط با جامعه نویسندگان افزونه‌ها در GitHub Discussions یا Slack
  • @code را دنبال کنید و نظر خود را با ما در میان بگذارید!

برای توضیحات هر یک از این کانال‌ها و اطلاعات بیشتر درباره سایر کانال‌های جامعه‌محور، ویکی ما را ببینید.

پروژه‌های مرتبط

بسیاری از اجزای اصلی و افزونه‌های VS Code در مخازن جداگانه‌ای در گیت‌هاب نگهداری می‌شوند. به عنوان مثال، مخزن‌های افزونه اشکال‌زدایی node و افزونه اشکال‌زدایی mono مجزا هستند. برای فهرست کامل، لطفاً به صفحه پروژه‌های مرتبط در ویکی ما مراجعه کنید.

افزونه‌های بسته‌بندی‌شده

VS Code مجموعه‌ای از افزونه‌های داخلی را که در پوشه extensions قرار دارند، شامل می‌شود؛ از جمله گرامرها و قطعه‌کدها برای زبان‌های مختلف. افزونه‌هایی که پشتیبانی کامل از زبان (تکمیل کد، رفتن به تعریف) را فراهم می‌کنند پسوند language-features دارند. به عنوان مثال، افزونه json رنگ‌بندی برای JSON را فراهم می‌کند و افزونه json-language-features پشتیبانی کامل زبانی برای JSON ارائه می‌دهد.

کانتینر توسعه

این مخزن شامل یک کانتینر توسعه Visual Studio Code Dev Containers / GitHub Codespaces است.

  • برای Dev Containers، از دستور Dev Containers: Clone Repository in Container Volume... استفاده کنید که یک حجم Docker برای عملکرد بهتر دیسک در macOS و ویندوز ایجاد می‌کند.

    • اگر VS Code و Docker را نصب کرده‌اید، می‌توانید برای شروع اینجا کلیک کنید. این کار باعث می‌شود VS Code به طور خودکار افزونه Dev Containers را نصب کند (در صورت نیاز)، کد منبع را در یک حجم کانتینر شبیه‌سازی کند و یک کانتینر توسعه برای استفاده ایجاد نماید.

  • برای Codespaces، افزونه GitHub Codespaces را در VS Code نصب کرده و از دستور Codespaces: Create New Codespace استفاده کنید.

Docker / Codespace باید حداقل دارای ۴ هسته و ۶ گیگابایت رم (۸ گیگابایت توصیه می‌شود) باشد تا ساخت کامل اجرا شود. برای اطلاعات بیشتر به README کانتینر توسعه مراجعه کنید.
