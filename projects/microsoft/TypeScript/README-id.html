TypeScript

TypeScript adalah bahasa untuk JavaScript skala aplikasi. TypeScript menambahkan tipe opsional ke JavaScript yang mendukung alat untuk aplikasi JavaScript berskala besar di browser manapun, untuk host manapun, di sistem operasi manapun. TypeScript dikompilasi menjadi JavaScript yang dapat dibaca dan berbasis standar. Coba di playground, dan tetap dapatkan informasi terbaru melalui blog kami dan akun Twitter.

Temukan pengguna TypeScript lainnya di halaman komunitas kami.

Instalasi

Untuk versi stabil terbaru:

npm install -D typescript

Untuk build nightly kami:

npm install -D typescript@next

Kontribusi

Ada banyak cara untuk berkontribusi ke TypeScript.

  • Laporkan bug dan bantu kami memverifikasi perbaikan saat sudah diperiksa.
  • Tinjau perubahan kode sumber.
  • Berinteraksi dengan pengguna dan pengembang TypeScript lainnya di StackOverflow.
  • Saling membantu di TypeScript Community Discord.
  • Bergabung dalam diskusi #typescript di Twitter.
  • Kontribusi perbaikan bug.

Proyek ini mengadopsi Microsoft Open Source Code of Conduct. Untuk informasi lebih lanjut, lihat FAQ Code of Conduct atau hubungi opencode@microsoft.com untuk pertanyaan atau komentar tambahan.
