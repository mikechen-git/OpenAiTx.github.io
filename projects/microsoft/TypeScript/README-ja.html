TypeScript

TypeScript は、アプリケーション規模のJavaScript向けの言語です。TypeScriptは、JavaScriptにオプションの型を追加し、あらゆるブラウザ、あらゆるホスト、あらゆるOS上で大規模なJavaScriptアプリケーションのためのツールをサポートします。TypeScriptは、可読性が高く標準に基づいたJavaScriptにコンパイルされます。プレイグラウンドで試したり、公式ブログやTwitterアカウントで最新情報を入手してください。

TypeScriptを利用している他のユーザーは、コミュニティページで見つけることができます。

インストール

最新の安定版バージョンの場合:

npm install -D typescript

ナイトリービルドの場合:

npm install -D typescript@next

コントリビュート
