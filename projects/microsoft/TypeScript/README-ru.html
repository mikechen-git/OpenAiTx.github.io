TypeScript

TypeScript — это язык для масштабируемого JavaScript. TypeScript добавляет необязательные типы в JavaScript, которые поддерживают инструменты для разработки крупных JavaScript-приложений для любого браузера, на любом хосте, под любой ОС. TypeScript компилируется в читаемый, основанный на стандартах JavaScript. Попробуйте его в песочнице и следите за новостями в нашем блоге и Twitter-аккаунте.

Найдите других пользователей TypeScript на нашей странице сообщества.

Установка

Для последней стабильной версии:

npm install -D typescript

Для наших ночных сборок:

npm install -D typescript@next

  • Сообщайте об ошибках и помогайте нам проверять исправления по мере их внедрения.
  • Просматривайте изменения в исходном коде.
  • Общайтесь с другими пользователями и разработчиками TypeScript на StackOverflow.
