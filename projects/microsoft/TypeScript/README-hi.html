टाइपस्क्रिप्ट

टाइपस्क्रिप्ट एप्लिकेशन-स्केल जावास्क्रिप्ट के लिए एक भाषा है। टाइपस्क्रिप्ट, जावास्क्रिप्ट में वैकल्पिक प्रकार जोड़ता है, जो किसी भी ब्राउज़र, किसी भी होस्ट, किसी भी OS के लिए बड़े पैमाने पर जावास्क्रिप्ट एप्लिकेशन के लिए टूल्स को सपोर्ट करता है। टाइपस्क्रिप्ट पठनीय, मानक-आधारित जावास्क्रिप्ट में संकलित होता है। इसे playground पर आज़माएँ, और हमारे ब्लॉग व Twitter खाते के माध्यम से अपडेट रहें।

हमारे समुदाय पृष्ठ पर अन्य टाइपस्क्रिप्ट उपयोगकर्ताओं से मिलें।

इंस्टालेशन

नवीनतम स्थिर संस्करण के लिए:

npm install -D typescript

हमारे नाइटली बिल्ड के लिए:

npm install -D typescript@next

योगदान करें

टाइपस्क्रिप्ट में योगदान देने के कई तरीके हैं।

  • बग सबमिट करें और जब फिक्स चेक-इन हो जाए तो उनकी पुष्टि करने में मदद करें।
  • सोर्स कोड परिवर्तनों की समीक्षा करें।
  • StackOverflow पर अन्य टाइपस्क्रिप्ट उपयोगकर्ताओं और डेवलपर्स से संवाद करें।
  • टाइपस्क्रिप्ट कम्युनिटी डिस्कॉर्ड में एक-दूसरे की मदद करें।
  • ट्विटर पर #typescript चर्चा में शामिल हों।
  • बग फिक्स में योगदान करें।

इस परियोजना ने Microsoft ओपन सोर्स आचार संहिता को अपनाया है। अधिक जानकारी के लिए आचार संहिता FAQ देखें या opencode@microsoft.com पर अतिरिक्त प्रश्न या टिप्पणियों के लिए संपर्क करें।
