TypeScript

TypeScript é uma linguagem para JavaScript em escala de aplicação. TypeScript adiciona tipos opcionais ao JavaScript que suportam ferramentas para aplicações JavaScript de grande escala para qualquer navegador, qualquer host, em qualquer sistema operacional. TypeScript é compilado para JavaScript legível e baseado em padrões. Experimente no playground, e mantenha-se atualizado através do nosso blog e da conta no Twitter.

Encontre outras pessoas que estão usando TypeScript em nossa página da comunidade.

Instalação

Para a versão estável mais recente:

npm install -D typescript

Para nossos builds noturnos:

npm install -D typescript@next

Contribua

Existem muitas maneiras de contribuir com o TypeScript.

  • Envie bugs e nos ajude a verificar correções assim que forem implementadas.
  • Revise as mudanças no código-fonte.
  • Interaja com outros usuários e desenvolvedores de TypeScript no StackOverflow.
  • Ajude uns aos outros no Discord da Comunidade TypeScript.
  • Participe da discussão #typescript no Twitter.
  • Contribua com correções de bugs.
