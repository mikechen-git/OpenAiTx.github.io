TypeScript

TypeScript là một ngôn ngữ dành cho JavaScript quy mô ứng dụng. TypeScript bổ sung các kiểu dữ liệu tuỳ chọn cho JavaScript, hỗ trợ các công cụ phát triển ứng dụng JavaScript quy mô lớn trên mọi trình duyệt, mọi máy chủ, trên mọi hệ điều hành. TypeScript được biên dịch thành JavaScript dễ đọc, dựa trên tiêu chuẩn. Hãy thử sử dụng tại playground, và cập nhật tin tức mới nhất qua blog của chúng tôi và tài khoản Twitter.

Tìm những người khác đang sử dụng TypeScript tại trang cộng đồng của chúng tôi.

Cài đặt

Để cài đặt phiên bản ổn định mới nhất:

npm install -D typescript

Để cài đặt bản build hằng đêm:

npm install -D typescript@next

Đóng góp

Có nhiều cách để đóng góp cho TypeScript.

  • Gửi lỗi và giúp chúng tôi xác minh các bản vá khi chúng được cập nhật.
  • Xem xét các thay đổi mã nguồn.
  • Trao đổi với người dùng và nhà phát triển TypeScript khác trên StackOverflow.
  • Hỗ trợ lẫn nhau trong TypeScript Community Discord.
  • Tham gia thảo luận #typescript trên Twitter.
  • Đóng góp sửa lỗi.
