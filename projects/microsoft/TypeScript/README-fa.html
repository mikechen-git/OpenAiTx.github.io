تایپ‌اسکریپت

تایپ‌اسکریپت یک زبان برای جاوااسکریپت در مقیاس برنامه‌های کاربردی است. تایپ‌اسکریپت انواع اختیاری را به جاوااسکریپت اضافه می‌کند که از ابزارها برای برنامه‌های بزرگ جاوااسکریپت روی هر مرورگر، هر میزبان و هر سیستم‌عاملی پشتیبانی می‌کند. تایپ‌اسکریپت به جاوااسکریپت قابل خواندن و مبتنی بر استانداردها کامپایل می‌شود. آن را در زمین‌بازی امتحان کنید و از طریق وبلاگ ما و حساب توییتر به‌روز بمانید.

افراد دیگری که از تایپ‌اسکریپت استفاده می‌کنند را در صفحه جامعه ما بیابید.

نصب

برای آخرین نسخه پایدار:

npm install -D typescript

برای نسخه‌های شبانه ما:

npm install -D typescript@next

مشارکت

راه‌های زیادی برای مشارکت در تایپ‌اسکریپت وجود دارد.

  • گزارش باگ‌ها و کمک به ما در تأیید رفع مشکلات هنگام ثبت آن‌ها.
  • بازبینی تغییرات کد منبع.
  • تعامل با دیگر کاربران و توسعه‌دهندگان تایپ‌اسکریپت در StackOverflow.
  • کمک به یکدیگر در دیسکورد جامعه تایپ‌اسکریپت.
  • پیوستن به بحث #typescript در توییتر.
  • مشارکت در رفع باگ‌ها.

این پروژه از آیین‌نامه رفتاری کد منبع باز مایکروسافت پیروی می‌کند. برای اطلاعات بیشتر به پرسش‌های متداول آیین‌نامه رفتاری مراجعه کنید یا با opencode@microsoft.com تماس بگیرید.
