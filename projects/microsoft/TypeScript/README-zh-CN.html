TypeScript

TypeScript 是一门面向应用规模 JavaScript 的语言。TypeScript 为 JavaScript 添加了可选类型，支持为任何浏览器、任何主机、任何操作系统上的大型 JavaScript 应用程序提供工具。TypeScript 会编译为可读、基于标准的 JavaScript。可以在 playground 上试用，并通过 我们的博客 和 Twitter 账号 了解最新动态。

在 我们的社区页面 找到其他正在使用 TypeScript 的用户。

安装

获取最新稳定版本：

npm install -D typescript

获取我们的每夜构建版本：

npm install -D typescript@next

参与贡献

有很多方式可以为 TypeScript 做出贡献。

  • 提交 bug，并帮助我们在修复时进行验证。
  • 查看源代码变更。
  • 在 StackOverflow 上与其他 TypeScript 用户和开发者互动。
  • 在 TypeScript 社区 Discord 中互相帮助。
  • 参与 Twitter 上的 #typescript 讨论。
  • 贡献 bug 修复。

本项目采用了 Microsoft 开源行为准则。更多信息请参见 行为准则常见问题，或通过 opencode@microsoft.com 联系我们，提出任何其他问题或意见。
