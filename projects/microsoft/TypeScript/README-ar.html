تايب سكريبت

تايب سكريبت هي لغة موجهة لجافاسكريبت على نطاق التطبيقات الكبيرة. تضيف تايب سكريبت أنواع اختيارية إلى جافاسكريبت تدعم الأدوات الخاصة بتطبيقات جافاسكريبت واسعة النطاق لأي متصفح، على أي مضيف، وعلى أي نظام تشغيل. تقوم تايب سكريبت بالترجمة إلى جافاسكريبت قابلة للقراءة ومعتمدة على المعايير. جربها في ساحة اللعب، وابقَ على اطلاع من خلال مدونتنا وحساب تويتر.

ابحث عن آخرين يستخدمون تايب سكريبت في صفحة المجتمع الخاصة بنا.

التثبيت

للحصول على أحدث إصدار مستقر:

npm install -D typescript

للحصول على إصداراتنا الليلية:

npm install -D typescript@next
