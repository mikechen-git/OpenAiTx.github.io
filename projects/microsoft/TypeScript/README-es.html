TypeScript

TypeScript es un lenguaje para JavaScript a escala de aplicaciones. TypeScript agrega tipos opcionales a JavaScript que soportan herramientas para aplicaciones JavaScript de gran escala en cualquier navegador, en cualquier host y en cualquier sistema operativo. TypeScript se compila a JavaScript legible y basado en estándares. Pruébalo en el playground y mantente actualizado a través de nuestro blog y cuenta de Twitter.

Encuentra a otros que están usando TypeScript en nuestra página de comunidad.

Instalación

Para la última versión estable:

npm install -D typescript

Para nuestras versiones nocturnas:

npm install -D typescript@next

Contribuir

Existen muchas formas de contribuir a TypeScript.

  • Envía errores y ayúdanos a verificar las correcciones a medida que se realizan.
  • Revisa los cambios en el código fuente.
  • Interactúa con otros usuarios y desarrolladores de TypeScript en StackOverflow.
  • Ayúdense mutuamente en el Discord de la Comunidad TypeScript.
  • Únete a la discusión de #typescript en Twitter.
  • Contribuye con correcciones de errores.
