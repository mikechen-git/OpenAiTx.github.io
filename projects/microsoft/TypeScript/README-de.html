TypeScript

TypeScript ist eine Sprache für JavaScript in Anwendungsgröße. TypeScript fügt JavaScript optionale Typen hinzu, die Werkzeuge für groß angelegte JavaScript-Anwendungen für jeden Browser, jeden Host und jedes Betriebssystem unterstützen. TypeScript wird zu lesbarem, standardbasiertem JavaScript kompiliert. Probiere es im Playground aus und bleibe über unseren Blog und unseren Twitter-Account auf dem Laufenden.

Finde andere, die TypeScript nutzen, auf unserer Community-Seite.

Installation

Für die neueste stabile Version:

npm install -D typescript

Für unsere Nightly-Builds:

npm install -D typescript@next

Mitwirken

Es gibt viele Möglichkeiten, zu TypeScript beizutragen.

  • Fehler melden und uns helfen, Korrekturen zu überprüfen, sobald sie eingecheckt werden.
  • Die Quellcode-Änderungen prüfen.
  • Mit anderen TypeScript-Anwendern und Entwicklern auf StackOverflow interagieren.
  • Sich gegenseitig im TypeScript Community Discord unterstützen.
  • An der Diskussion #typescript auf Twitter teilnehmen.
  • Fehlerkorrekturen beitragen.
