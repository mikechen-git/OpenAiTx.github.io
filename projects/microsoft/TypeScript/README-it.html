TypeScript

TypeScript è un linguaggio per JavaScript su scala applicativa. TypeScript aggiunge tipi opzionali a JavaScript che supportano strumenti per applicazioni JavaScript di grandi dimensioni, per qualsiasi browser, su qualsiasi host, su qualsiasi sistema operativo. TypeScript compila in JavaScript leggibile e basato sugli standard. Provalo nel playground, e rimani aggiornato tramite il nostro blog e l'account Twitter.

Trova altri utenti che utilizzano TypeScript alla nostra pagina della community.

Installazione

Per la versione stabile più recente:

npm install -D typescript

Per le nostre build nightly:

npm install -D typescript@next

Contribuire

Ci sono molti modi per contribuire a TypeScript.

  • Segnala bug e aiutaci a verificare le correzioni man mano che vengono inserite.
  • Revisiona i cambiamenti del codice sorgente.
  • Partecipa con altri utenti e sviluppatori TypeScript su StackOverflow.
  • Aiuta gli altri nel TypeScript Community Discord.
  • Unisciti alla discussione #typescript su Twitter.
  • Contribuisci con correzioni di bug.
