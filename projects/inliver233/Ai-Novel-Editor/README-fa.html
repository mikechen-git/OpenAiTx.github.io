ویرایشگر رمان هوشمند (AI Novel Editor)

ویرایشگر رمان با کمک هوش مصنوعی مبتنی بر PyQt6 که ابزارهای حرفه‌ای نویسندگی مانند تکمیل هوشمند، مدیریت پروژه و سازماندهی مفاهیم را ارائه می‌دهد.

نصب

  • کلون کردن پروژه
git clone https://github.com/inliver233/Ai-Novel-Editor.git
cd ai-novel-editor

  • ایجاد محیط مجازی
python -m venv venv
venv\Scripts\activate     # ویندوز

  • نصب وابستگی‌ها
pip install -r requirements.txt
  • اجرای برنامه
python src/main.py

قابلیت‌های اصلی

مدیریت پروژه

  • ساختار سلسله‌مراتبی اسناد (اثر > فصل > صحنه)
  • واردات و صادرات پروژه (پشتیبانی از فرمت‌های TEXT، MARKDOWN، DOCX، PDF، HTML)
  • پشتیبان‌گیری خودکار و کنترل نسخه
  • مدیریت چندین پروژه

قابلیت تکمیل AI

  • سه حالت تکمیل: تکمیل خودکار، تکمیل دستی (توصیه می‌شود با یک بار فشردن tab فعال شود و با بار دیگر اعمال شود)، غیرفعال‌سازی تکمیل
  • سه حالت زمینه:
  • حالت سریع: زمینه سبک، پاسخگویی سریع
  • حالت متعادل: زمینه متوسط، تعادل بین کیفیت و سرعت
  • حالت کلی: زمینه کامل پروژه، بهترین نتیجه
  • پشتیبانی از سرویس‌های مختلف AI: OpenAI، Claude، Tongyi Qianwen، Zhipu AI، DeepSeek، Groq و غیره

مدیریت ساختار کلی (Outline)

  • نمایش ساختار درختی به صورت بصری
  • مرتب‌سازی و تنظیم سطوح با کشیدن و رها کردن
  • ناوبری سریع در اسناد
  • تحلیل و ارائه پیشنهاد برای بهینه‌سازی ساختار

پیکربندی API

  • مرکز پیکربندی AI یکپارچه
  • پیکربندی‌های پیش‌فرض برای چندین ارائه‌دهنده خدمات
  • قابلیت تست اتصال
  • ذخیره‌سازی و وارد/خارج‌سازی پیکربندی‌ها

تنظیمات تکمیل هوشمند

  • قابلیت تنظیم تاخیر فعال‌سازی
  • محدودیت طول تکمیل
  • پیکربندی طول متن زمینه
  • پشتیبانی از پاسخ‌دهی جریانی (streaming)

ویرایش کلیدواژه‌های راهنما (Prompt)

  • قالب‌های نگارشی متنوع داخلی
  • قالب‌های قابل تنظیم کلیدواژه‌های راهنما
  • مدیریت دسته‌بندی قالب‌ها
  • پیکربندی قالب مخصوص هر حالت

قابلیت‌های رابط کاربری

  • دو تم روشن و تیره
  • چینش سه ستونه (درخت پروژه، ویرایشگر، پنل مفاهیم)
  • نوار کناری قابل جمع شدن
  • حالت نوشتن تمام‌صفحه
  • حالت تمرکز (جمله، پاراگراف، ماشین تحریر، حالت ذن، حالت غرق‌شدن)

جستجو و جایگزینی

  • جستجوی متن در کل پروژه
  • پشتیبانی از عبارات با قاعده (Regex)
  • قابلیت جایگزینی گروهی
  • گزینه‌های جستجوی پیشرفته

مدیریت مفاهیم

  • شناسایی خودکار شخصیت‌ها، مکان‌ها و اشیاء
  • مدیریت روابط مفهومی
  • سیستم برچسب‌گذاری و دسته‌بندی
  • درج سریع مفهوم

کلیدهای میانبر

  • Ctrl+N: ایجاد پروژه جدید
  • Ctrl+O: باز کردن پروژه
  • Ctrl+S: ذخیره سند
  • F11: حالت تمام‌صفحه
  • Tab: فعال‌سازی دستی تکمیل خودکار AI (در حالت دستی)
  • Ctrl+F: جستجو
  • Ctrl+H: جستجو و جایگزینی
  • Ctrl+Shift+H: جستجو و جایگزینی پیشرفته

الزامات سیستم

  • Python 3.8+
  • PyQt6
  • سیستم عامل: Windows، macOS، Linux
