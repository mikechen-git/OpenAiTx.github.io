۳۰ ثانیه کد

مقالات برنامه‌نویسی برای ارتقاء مهارت‌های توسعه‌ی شما

  • برای مشاهده تمام مقالات، به وب‌سایت مراجعه کنید.
  • با استفاده از نام، برچسب‌ها، زبان یا توضیحات، مقالات و مجموعه‌هایی که مناسب نیاز شما هستند را جستجو کنید.
  • همه‌ی مقالات یا مجموعه‌های مربوط به هر موضوع را مرور کنید.
  • برای مشاهده کامل هر مقاله شامل قطعه‌کدها، توضیحات و مثال‌ها، روی هر کارت کلیک کنید.
  • اگر از پروژه خوشتان آمد، به آن ستاره بدهید. این کار اهمیت زیادی دارد.

مایل به همکاری هستید؟

  • در حال حاضر مشارکت محتوای جدید پذیرفته نمی‌شود.
  • با این حال می‌توانید مشکلات وب‌سایت را گزارش دهید یا پیشنهاد بهبود ارائه کنید، با ایجاد یک ایشو.

اعتبارها

  • این مخزن توسط آنجلس چالاریس نگهداری می‌شود.
  • تمام قطعه‌کدها تحت لایسنس CC-BY-4.0 منتشر شده‌اند، مگر اینکه به‌طور صریح خلاف آن ذکر شده باشد.
  • هرگونه محتوای دیگر (شامل متن، تصاویر، کد منبع وب‌سایت، لوگوها، نام‌ها و علائم تجاری) بدون اجازه صریح صاحب اثر قابل استفاده نیست.
  • این وب‌سایت با استفاده از Netlify، Astro و GitHub راه‌اندازی شده است.
  • تمامی عکس‌ها توسط Unsplash ارائه شده‌اند.
