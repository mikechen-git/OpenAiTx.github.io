Sora 意為「穹」，象徵著無限的可能性和廣闊的創造力

My Blog

誰在使用

歡迎提交一個 Issue 來展示自己的网站。

開始

  1. 安裝主題

    • 安裝 pnpm

    • pnpm create astro@latest --template Liksone/astro-theme-sora

    • 進入專案目錄，運行 pnpm dev 啟動專案

  2. 配置主題

    修改配置檔 theme.config.ts 來自訂主題。詳見配置。

  3. 編輯文章

    在 src/content/ 目錄中編輯，支援 Markdown 格式。

  4. 部署網站

    運行 pnpm build 建置網站，建置產物在 dist/ 目錄。可運行 pnpm preview 在本地預覽網站。

    自行部署網站至伺服器，或者參考 Astro 文件部署網站至 Vercel、Netlify、GitHub Pages 等託管平台。

配置

配置檔為 theme.config.ts。

網站資訊

site: {
  // 網站地址
  url: "https://blog.liks.space",
  // 網站標題
  title: "Sora",
  // 站長名稱
  author: "Liks",
  // 網站簡介
  description: "A blog theme built with Astro",
  // 網站圖示
  // 支援 SVG、PNG、ICO 格式
  // 位於 public/ 目錄內的本地檔案路徑
  favicon: "/images/favicon.ico",
}

global: {

  // 啟用多語言支援（開發中）
  i18n: true,
