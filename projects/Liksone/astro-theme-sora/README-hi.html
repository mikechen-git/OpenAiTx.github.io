पूर्वावलोकन

My Blog

कौन उपयोग कर रहा है

अपनी वेबसाइट दिखाने के लिए एक Issue प्रस्तुत करने के लिए आपका स्वागत है।

प्रारंभ करें

  1. थीम स्थापित करें

    • pnpm स्थापित करें

    • pnpm create astro@latest --template Liksone/astro-theme-sora

    • प्रोजेक्ट डायरेक्ट्री में जाएं, और pnpm dev चलाकर प्रोजेक्ट शुरू करें

  2. थीम कॉन्फ़िगर करें

    थीम को कस्टमाइज़ करने के लिए theme.config.ts कॉन्फ़िगरेशन फ़ाइल को संशोधित करें। विवरण के लिए कॉन्फ़िगरेशन देखें।

  3. लेख संपादित करें

    src/content/ डायरेक्ट्री में संपादित करें, Markdown प्रारूप समर्थित है।

  4. वेबसाइट डिप्लॉय करें

    pnpm build चलाकर वेबसाइट बनाएँ, निर्मित फाइलें dist/ डायरेक्ट्री में होंगी। लोकल पूर्वावलोकन के लिए pnpm preview चला सकते हैं।

    वेबसाइट को स्वयं सर्वर पर डिप्लॉय करें, या Astro डाक्यूमेंटेशन के अनुसार Vercel, Netlify, GitHub Pages आदि होस्टिंग प्लेटफार्म पर डिप्लॉय करें।

कॉन्फ़िगरेशन

कॉन्फ़िगरेशन फ़ाइल है theme.config.ts।

साइट जानकारी

site: {
  // वेबसाइट पता
  url: "https://blog.liks.space",
  // वेबसाइट शीर्षक
  title: "Sora",
  // साइट ओनर का नाम
  author: "Liks",
  // वेबसाइट परिचय
  description: "A blog theme built with Astro",
  // वेबसाइट आइकन
  // SVG, PNG, ICO प्रारूप समर्थित
  // public/ डायरेक्ट्री में स्थानीय फाइल पथ
  favicon: "/images/favicon.ico",
}
