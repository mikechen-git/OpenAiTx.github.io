Introducción

HelloGitHub comparte proyectos de código abierto interesantes y para principiantes en GitHub. El día 28 de cada mes se publica una nueva edición en formato de boletín mensual, incluyendo: proyectos de código abierto interesantes y para principiantes, libros de código abierto, proyectos prácticos, proyectos a nivel empresarial, etc. ¡Permitiéndote experimentar el encanto del open source en poco tiempo y enamorarte de él!
