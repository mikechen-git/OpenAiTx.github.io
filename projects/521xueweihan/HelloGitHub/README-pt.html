Introdução

HelloGitHub compartilha projetos open source interessantes e para iniciantes no GitHub. Todo dia 28 de cada mês, é publicado em formato de revista mensal atualizado com conteúdos como: projetos open source interessantes e para iniciantes, livros open source, projetos práticos, projetos de nível empresarial, etc. Tudo para que você sinta o charme do open source em pouco tempo e se apaixone por ele!
