GGbengSite

  • Astro で開発された個人サイト
  • algolia を使用して検索機能を実現 🔍
  • giscus を使用してコメントやディスカッションを実現 🌈
  • mailerlite を使用してメール購読・配信を実現 📮
  • 本プロジェクトは netlify 上でホスティングされており、crawler を利用してドキュメントをクロールし、algolia の検索に必要な schema を保存

ローカル開発


pnpm run install

pnpm run start
