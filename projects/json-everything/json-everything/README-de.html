Was ist json-everything?

Die Hauptphilosophie hinter json-everything ist es, sicherzustellen, dass gängige JSON-Funktionalitäten im System.Text.Json-Umfeld gut unterstützt werden.

Der erste Schritt besteht darin zu prüfen, ob es bereits andere Projekte gibt, die eine bestimmte Funktion unterstützen.

Falls ja, wird dieser Aufwand hier wahrscheinlich nicht dupliziert. Etwas weiter unten findest du eine Liste anderer Projekte, die einige interessante Dinge mit JSON machen.

Andernfalls wird die Aufnahme in dieses Projekt in Erwägung gezogen.

Was ist im Paket enthalten?

Tatsächlich gibt es mehrere Pakete. Jede Funktionalität wurde in eine eigene Bibliothek ausgelagert, sodass du genau diejenige(n) auswählen kannst, die du benötigst.

Alle Bibliotheken sind für .Net Standard 2.0 und .Net 8 gebaut.

JsonSchema.Net
 JSON Schema, Drafts 6 und höher
JsonSchema.Net.Generation
 Generierung von Schemata aus .Net-Typen
JsonSchema.Net.Generation.DataAnnotations
 Schemagenerierung für System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 Zufällige Instanzdatengenerierung (basiert auf Bogus)
JsonSchema.Net.Data
 Ein Vokabular für den Zugriff auf Instanz- und externe Daten
JsonSchema.Net.ArrayExt
 Ein Vokabular zur Erweiterung der Array-Validierung
JsonSchema.Net.OpenApi
 OpenApi 3.1 Vokabular-Erweiterung (verwendet von Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) und Relative JSON Pointer (Spezifikation)
JsonLogic
 JsonLogic (Website)
JsonE.Net
 JSON-e (Website)
Json.More.Net
 Allgemeine Erweiterungen, die eigentlich in System.Text.Json[.Nodes] enthalten sein sollten, es aber nicht sind
Yaml2JsonNode
 Konvertierungen zwischen dem YAML-Dokumentmodell in YamlDotNet und JsonNode (in beide Richtungen).

Alle oben genannten Bibliotheken bieten vollständige Unterstützung für ihre zugehörigen Spezifikationen.

HINWEIS Jedes Sprachpaket wird als eigenes Nuget-Paket bereitgestellt.

Sie können auch die Ergebnisse der JSON Schema Test Suite auf Bowtie anzeigen, die die Test Suite gegen mehrere Implementierungen auf verschiedenen Plattformen ausführt.

HAFTUNGSAUSSCHLUSS Meine Bibliothek, JsonSchema.Net, steht in keiner Beziehung zu oder Verbindung mit der Website https://jsonschema.net, außer dass beide hervorragende JSON Schema Werkzeuge sind.

Veraltete Pakete

Diese Pakete waren ebenfalls Teil dieses Pakets, werden aber nicht mehr unterstützt.

JsonSchema.Net.CodeGeneration

Wenn Sie JSON für etwas verwenden, das von keiner Bibliothek in diesem Paket oder einem der oben genannten Projekte abgedeckt wird, können Sie gerne ein Feature-Issue erstellen.

Wenn Sie ein weiteres Projekt pflegen oder kennen, das System.Text.Json auf interessante Weise erweitert, würde ich es gerne hier auflisten. Bitte erstellen Sie ein allgemeines Issue, um mich darüber zu informieren.
