Cos'è json-everything?

La filosofia principale dietro json-everything è garantire che le funzionalità comuni di JSON abbiano un buon supporto nello spazio System.Text.Json.

Il primo passo in questa direzione è verificare se esistano altri progetti che supportano una determinata funzione.

In tal caso, probabilmente tale sforzo non verrà duplicato qui. Più in basso troverai un elenco di altri progetti che stanno facendo cose interessanti con JSON.

Altrimenti, la funzionalità viene presa in considerazione per questo progetto.

Cosa c'è nella scatola?

In realtà ci sono più scatole. Ogni funzionalità è stata separata nella propria libreria, così puoi scegliere solo quella(e) di cui hai bisogno.

Tutte le librerie sono costruite per .Net Standard 2.0 e .Net 8.

JsonSchema.Net
 JSON Schema, draft 6 e successivi
JsonSchema.Net.Generation
 Generazione di schemi da tipi .Net
JsonSchema.Net.Generation.DataAnnotations
 Supporto per la generazione di schemi per System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 Generazione di dati casuali per le istanze (basato su Bogus)
JsonSchema.Net.Data
 Un vocabolario per accedere ai dati dell'istanza e ai dati esterni
JsonSchema.Net.ArrayExt
 Un vocabolario per estendere la validazione degli array
JsonSchema.Net.OpenApi
 Estensione del vocabolario OpenApi 3.1 (utilizzato da Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) e Relative JSON Pointer (Specifica)
JsonLogic
 JsonLogic (Sito web)
JsonE.Net
 JSON-e (Sito web)
Json.More.Net
 Estensioni di uso generale che probabilmente avrebbero dovuto essere incluse in System.Text.Json[.Nodes] ma non lo sono state
Yaml2JsonNode
 Conversioni tra il modello di documento YAML in YamlDotNet e JsonNode (in entrambe le direzioni).

Tutte le librerie sopra elencate offrono supporto completo per le specifiche a cui sono associate.

Le traduzioni dei messaggi di errore per JsonSchema.Net sono disponibili in:

  • Italiano it
  • Norvegese nb-NO
  • Polacco pl-PL
  • Russo ru
  • Spagnolo es
  • Svedese sv-SE
  • Turco tr-TR

NOTA Ogni pacchetto lingua è fornito come pacchetto Nuget separato.

Puoi anche consultare i risultati della JSON Schema Test Suite su Bowtie, che esegue la test suite su molteplici implementazioni su diverse piattaforme.

DISCLAIMER La mia libreria, JsonSchema.Net, non è collegata né associata al sito https://jsonschema.net, se non per il fatto che entrambi sono ottimi strumenti per JSON Schema.

Pacchetti deprecati

Anche questi pacchetti hanno fatto parte di questa suite, ma non sono più supportati.

JsonSchema.Net.CodeGeneration
 Generazione di codice C# a partire da schemi Deprecato a favore di Corvus.JsonSchema

Supporto System.Text.Json da altri progetti

Se non trovi quello che cerchi qui, prova uno di questi eccellenti progetti:

  • JsonCons.Net di @danielaparker
    • JSON Pointer
    • JSON Patch
    • JSON Merge Patch
    • JSON Path
    • JMES Path
  • Corvus.JsonSchema di @mwadams
    • Validazione JSON Schema con particolare attenzione alla generazione di codice C#
  • (altri in arrivo)

Se utilizzi JSON per fare qualcosa che non è coperto da una libreria in questa suite o da uno dei progetti sopra, sentiti libero di creare una richiesta di funzionalità.

Se mantieni o conosci un altro progetto che estende System.Text.Json per fare cose interessanti, mi piacerebbe elencarlo qui, quindi per favore crea una segnalazione generica per farmelo sapere.
