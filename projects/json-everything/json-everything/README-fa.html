json-everything چیست؟

فلسفه اصلی پشت json-everything این است که اطمینان حاصل شود قابلیت‌های رایج JSON پشتیبانی مناسبی در فضای System.Text.Json دارند.

اولین گام در این راستا بررسی این است که آیا پروژه‌های دیگری وجود دارند که یک قابلیت خاص را پشتیبانی می‌کنند یا خیر.

اگر چنین باشد، احتمالاً آن تلاش در اینجا تکرار نخواهد شد. کمی پایین‌تر لیستی از پروژه‌های دیگری را خواهید یافت که کارهای جالبی با JSON انجام می‌دهند.

در غیر این صورت، بررسی آن قابلیت برای این پروژه آزاد است.

چه چیزی در جعبه است؟

در واقع چندین جعبه وجود دارد. هر بخش از قابلیت‌ها به صورت یک کتابخانه جداگانه ارائه شده است تا بتوانید هر کدام را که لازم دارید انتخاب کنید.

تمام کتابخانه‌ها برای .Net Standard 2.0 و .Net 8 ساخته شده‌اند.

JsonSchema.Net
 JSON Schema، پیش‌نویس ۶ و بالاتر
JsonSchema.Net.Generation
 تولید شماتیک از انواع .Net
JsonSchema.Net.Generation.DataAnnotations
 پشتیبانی از تولید اسکیمای System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 تولید داده نمونه تصادفی (با استفاده از Bogus)
JsonSchema.Net.Data
 واژگان برای دسترسی به داده نمونه و داده خارجی
JsonSchema.Net.ArrayExt
 واژگان برای گسترش اعتبارسنجی آرایه
JsonSchema.Net.OpenApi
 افزونه واژگان OpenApi 3.1 (استفاده شده توسط Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) و Relative JSON Pointer (مشخصات)
JsonLogic
 JsonLogic (وب‌سایت)
JsonE.Net
 JSON-e (وب‌سایت)
Json.More.Net
 افزونه‌های کاربردی عمومی که احتمالاً باید در System.Text.Json[.Nodes] وجود می‌داشتند اما قرار نگرفته‌اند
Yaml2JsonNode
 تبدیل بین مدل سند YAML در YamlDotNet و JsonNode (در هر دو جهت).

تمام کتابخانه‌های فوق پشتیبانی کامل از مشخصات مربوط به خود را ارائه می‌دهند.

ترجمه پیام‌های خطا برای JsonSchema.Net در زبان‌های زیر موجود است:

  • ایتالیایی it
  • نروژی nb-NO
  • لهستانی pl-PL
  • روسی ru
  • اسپانیایی es
  • سوئدی sv-SE
  • ترکی tr-TR

توجه هر بسته زبان توسط بسته Nuget جداگانه‌ای ارائه می‌شود.

همچنین می‌توانید نتایج مجموعه تست JSON Schema را در Bowtie مشاهده کنید، که مجموعه تست را روی چندین پیاده‌سازی در پلتفرم‌های مختلف اجرا می‌کند.

سلب مسئولیت کتابخانه من، JsonSchema.Net، هیچ ارتباطی با وب‌سایت https://jsonschema.net ندارد، جز اینکه هر دو ابزار عالی برای JSON Schema هستند.

بسته‌های منسوخ شده

این بسته‌ها نیز قبلاً بخشی از این مجموعه بودند اما دیگر پشتیبانی نمی‌شوند.

JsonSchema.Net.CodeGeneration
 تولید کد C# از اسکیمای‌ها در حال حاضر به جای آن از Corvus.JsonSchema استفاده کنید

پشتیبانی از System.Text.Json توسط پروژه‌های دیگر

اگر آنچه را که به دنبال آن هستید اینجا پیدا نکردید، لطفاً یکی از این پروژه‌های عالی را امتحان کنید:

  • JsonCons.Net توسط @danielaparker
    • JSON Pointer
    • JSON Patch
    • JSON Merge Patch
    • JSON Path
    • JMES Path
  • Corvus.JsonSchema توسط @mwadams
    • اعتبارسنجی JSON Schema با تمرکز بر تولید کد C#
  • (موارد بیشتر به زودی)

اگر شما با JSON کاری انجام می‌دهید که هیچ یک از کتابخانه‌های این مجموعه یا پروژه‌های فوق پوشش نمی‌دهند، آزاد هستید که درخواست ویژگی جدیدی ثبت کنید.

اگر شما نگهدارنده یا آگاه به پروژه دیگری هستید که System.Text.Json را توسعه می‌دهد و قابلیت‌های جالبی ارائه می‌کند، خوشحال می‌شوم که آن را اینجا فهرست کنم. لطفاً یک Issue عمومی ایجاد کنید تا من را در جریان قرار دهید.
