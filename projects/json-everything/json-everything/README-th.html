json-everything คืออะไร?

ปรัชญาหลักเบื้องหลัง json-everything คือการรับรองว่าฟังก์ชันการทำงานทั่วไปของ JSON ได้รับการสนับสนุนที่ดีในพื้นที่ System.Text.Json

ขั้นตอนแรกของสิ่งนี้คือการตรวจสอบว่ามีโปรเจกต์อื่นที่รองรับฟังก์ชันนั้นหรือไม่

ถ้ามี ก็มีแนวโน้มว่าจะไม่ทำซ้ำความพยายามนั้นที่นี่ ด้านล่างนี้คุณจะพบรายการของโปรเจกต์อื่น ๆ ที่กำลังทำสิ่งเจ๋ง ๆ กับ JSON

มิฉะนั้น ก็เปิดโอกาสให้พิจารณาสำหรับโปรเจกต์นี้

มีอะไรอยู่ในกล่อง?

จริง ๆ แล้วมีกล่องหลายกล่อง โดยแต่ละฟังก์ชันได้ถูกแยกออกเป็นไลบรารีของตัวเอง คุณจึงสามารถเลือกใช้เฉพาะอันที่คุณต้องการได้

ไลบรารีทั้งหมดถูกสร้างขึ้นสำหรับ .Net Standard 2.0 และ .Net 8

JsonSchema.Net
 JSON Schema, drafts 6 และสูงกว่า
JsonSchema.Net.Generation
 การสร้าง schema จาก .Net types
JsonSchema.Net.Generation.DataAnnotations
 รองรับการสร้างสคีมาสำหรับ System.ComponentModel.DataAnnotations
JsonSchema.Net.DataGeneration
 การสร้างข้อมูลอินสแตนซ์แบบสุ่ม (ขับเคลื่อนโดย Bogus)
JsonSchema.Net.Data
 คำศัพท์สำหรับเข้าถึงข้อมูลอินสแตนซ์และข้อมูลภายนอก
JsonSchema.Net.ArrayExt
 คำศัพท์สำหรับขยายการตรวจสอบความถูกต้องของอาเรย์
JsonSchema.Net.OpenApi
 ส่วนขยายคำศัพท์ของ OpenApi 3.1 (ใช้โดย Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) และ Relative JSON Pointer (Specification)
JsonLogic
 JsonLogic (เว็บไซต์)
JsonE.Net
 JSON-e (เว็บไซต์)
Json.More.Net
 ส่วนขยายการใช้งานทั่วไปที่น่าจะถูกบรรจุอยู่ใน System.Text.Json[.Nodes] แต่ยังไม่ได้ถูกเพิ่มเข้าไป
Yaml2JsonNode
 การแปลงระหว่างโมเดลเอกสาร YAML ใน YamlDotNet และ JsonNode (ทั้งสองทิศทาง)

ไลบรารีทั้งหมดข้างต้นรองรับสเปกที่เกี่ยวข้องของตนอย่างสมบูรณ์

การแปลข้อความข้อผิดพลาดสำหรับ JsonSchema.Net มีให้ใน:

  • อิตาเลียน it
  • นอร์เวย์ nb-NO
  • โปแลนด์ pl-PL
  • รัสเซีย ru
  • สเปน es
  • สวีเดน sv-SE
  • ตุรกี tr-TR

หมายเหตุ แต่ละชุดภาษาแยกเป็นแพ็กเกจของ Nuget เอง

คุณยังสามารถดูผลลัพธ์ชุดทดสอบ JSON Schema ได้ที่ Bowtie ซึ่งรันชุดทดสอบกับหลายการใช้งานบนแพลตฟอร์มที่แตกต่างกัน

ข้อสงวนสิทธิ์ ไลบรารีของฉัน JsonSchema.Net ไม่ได้เกี่ยวข้องหรือสัมพันธ์กับเว็บไซต์ https://jsonschema.net ยกเว้นเพียงว่าทั้งสองเป็นเครื่องมือ JSON Schema ที่ยอดเยี่ยมเหมือนกัน

แพ็กเกจที่เลิกใช้แล้ว

แพ็กเกจเหล่านี้เคยเป็นส่วนหนึ่งของชุดนี้แต่ไม่ได้รับการสนับสนุนอีกต่อไป

JsonSchema.Net.CodeGeneration
 การสร้างโค้ด C# จากสคีมา เลิกใช้แล้ว แนะนำให้ใช้ Corvus.JsonSchema

การรองรับ System.Text.Json โดยโปรเจกต์อื่น

หากคุณไม่พบสิ่งที่ต้องการที่นี่ กรุณาลองโปรเจกต์ที่ยอดเยี่ยมเหล่านี้:

  • JsonCons.Net โดย @danielaparker
    • JSON Pointer
    • JSON Patch
    • JSON Merge Patch
    • JSON Path
    • JMES Path
  • Corvus.JsonSchema โดย @mwadams
    • ตรวจสอบ JSON Schema โดยเน้นการสร้างโค้ด C#
  • (จะมีเพิ่มเติม)

หากคุณใช้ JSON เพื่อทำสิ่งใดก็ตามที่ยังไม่มีไลบรารีในชุดนี้หรือโปรเจกต์ข้างต้นรองรับ สามารถ สร้าง issue ขอเพิ่มฟีเจอร์ ได้เลย

หากคุณดูแลหรือทราบโปรเจกต์อื่นที่ขยาย System.Text.Json ให้ทำสิ่งเจ๋ง ๆ ได้ ผมต้องการจะนำมาแสดงไว้ที่นี่ กรุณา สร้าง issue ทั่วไป เพื่อแจ้งให้ทราบ

การมีส่วนร่วม

โปรดดูที่ ข้อปฏิบัติในการมีส่วนร่วม และไฟล์ CONTRIBUTING สำหรับข้อมูลเพิ่มเติม

การสนับสนุน

หากคุณพบว่าห้องสมุดเหล่านี้มีประโยชน์และต้องการส่งเสริมการพัฒนาต่อไป โปรดพิจารณา สนับสนุนผู้ดูแลโครงการ

.NET Foundation

โครงการนี้ได้รับการสนับสนุนโดย .NET Foundation
