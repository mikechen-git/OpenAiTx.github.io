json-everything क्या है?

json-everything के पीछे मुख्य विचारधारा यह है कि सामान्य JSON कार्यक्षमता के लिए System.Text.Json क्षेत्र में अच्छा समर्थन सुनिश्चित किया जाए।

इसका पहला कदम यह जांचना है कि क्या कोई अन्य परियोजनाएँ पहले से ही किसी दिए गए फ़ंक्शन का समर्थन करती हैं।

यदि हाँ, तो संभवतः यहाँ उस प्रयास की पुनरावृत्ति नहीं की जाएगी। नीचे आपको अन्य परियोजनाओं की एक सूची मिलेगी, जो JSON के साथ कुछ शानदार कार्य कर रही हैं।

अन्यथा, इस परियोजना के लिए इसे विचारार्थ खुला रखा गया है।

बॉक्स में क्या है?

वास्तव में कई बॉक्स हैं। प्रत्येक कार्यक्षमता को उसकी अपनी लाइब्रेरी में विभाजित किया गया है, ताकि आप अपनी आवश्यकता अनुसार एक या अधिक चुन सकें।

सभी लाइब्रेरी .Net Standard 2.0 और .Net 8 के लिए बनाई गई हैं।

JsonSchema.Net
 JSON Schema, ड्राफ्ट 6 और ऊपर
JsonSchema.Net.Generation
 .Net टाइप्स से स्कीमा का निर्माण
JsonSchema.Net.Generation.DataAnnotations
 System.ComponentModel.DataAnnotations के लिए स्कीमा जनरेशन समर्थन।
JsonSchema.Net.DataGeneration
 रैंडम इंस्टेंस डेटा जनरेशन (Bogus द्वारा संचालित)
JsonSchema.Net.Data
 इंस्टेंस और बाहरी डेटा तक पहुँचने के लिए एक शब्दावली
JsonSchema.Net.ArrayExt
 एरे वेलिडेशन को बढ़ाने के लिए एक शब्दावली
JsonSchema.Net.OpenApi
 OpenApi 3.1 शब्दावली विस्तार (Graeae द्वारा उपयोग किया गया)
JsonPath.Net
 JSON पाथ (RFC 9535)
JsonPatch.Net
 JSON पैच (RFC 6902)
JsonPointer.Net
 JSON पॉइंटर (RFC 6901) और रिलेटिव JSON पॉइंटर (स्पेसिफिकेशन)
JsonLogic
 JsonLogic (वेबसाइट)
JsonE.Net
 JSON-e (वेबसाइट)
Json.More.Net
 सामान्य-उपयोग एक्सटेंशन जो संभवतः System.Text.Json[.Nodes] में शामिल होने चाहिए थे लेकिन नहीं हुए
Yaml2JsonNode
 YamlDotNet में YAML डाक्यूमेंट मॉडल और JsonNode के बीच (दोनों दिशाओं में) रूपांतरण।

ऊपर दी गई सभी लाइब्रेरीज़ अपने संबंधित विनिर्देशों के लिए पूर्ण समर्थन प्रदान करती हैं।

JsonSchema.Net के लिए त्रुटि संदेश अनुवाद निम्नलिखित भाषाओं में उपलब्ध हैं:

  • इतालवी it
  • नॉर्वेजियन nb-NO
  • पोलिश pl-PL
  • रूसी ru
  • स्पेनिश es
  • स्वीडिश sv-SE
  • तुर्की tr-TR

नोट प्रत्येक भाषा पैक अपनी स्वयं की Nuget पैकेज के रूप में उपलब्ध है।

आप Bowtie पर JSON Schema Test Suite के परिणाम भी देख सकते हैं, जो विभिन्न प्लेटफार्मों पर कई इम्प्लीमेंटेशन के साथ टेस्ट सूट चलाता है।

अस्वीकरण मेरी लाइब्रेरी, JsonSchema.Net, का https://jsonschema.net वेबसाइट से कोई संबंध या एसोसिएशन नहीं है, सिवाय इसके कि दोनों ही बेहतरीन JSON Schema टूल्स हैं।

अप्रचलित पैकेज

ये पैकेज भी इस सुइट का हिस्सा रहे हैं, लेकिन अब समर्थित नहीं हैं।

JsonSchema.Net.CodeGeneration
 स्कीमाओं से C# कोड जनरेट करना Corvus.JsonSchema के पक्ष में अप्रचलित

System.Text.Json का समर्थन अन्य प्रोजेक्ट्स द्वारा

यदि आपको यहाँ वह नहीं मिलता जिसकी आपको तलाश है, तो कृपया इन शानदार प्रोजेक्ट्स में से किसी एक को आज़माएँ:

  • JsonCons.Net द्वारा @danielaparker
    • JSON पॉइंटर
    • JSON पैच
    • JSON मर्ज पैच
    • JSON पाथ
    • JMES पाथ
  • Corvus.JsonSchema द्वारा @mwadams
    • C# कोड जनरेशन फोकस के साथ JSON स्कीमा वेलिडेशन
  • (और भी आ सकते हैं)

यदि आप JSON का उपयोग किसी ऐसे कार्य के लिए करते हैं जो इस सुइट की किसी लाइब्रेरी या ऊपर दिए गए प्रोजेक्ट्स द्वारा कवर नहीं किया गया है, तो बेझिझक फीचर इश्यू बनाएँ।

यदि आप कोई अन्य प्रोजेक्ट बनाते हैं या जानते हैं, जो System.Text.Json को एक्सटेंड करता है और कुछ शानदार करता है, तो मैं उसे यहाँ सूचीबद्ध करना चाहूँगा, कृपया जनरल इश्यू बनाएँ और मुझे उसके बारे में बताएं।
