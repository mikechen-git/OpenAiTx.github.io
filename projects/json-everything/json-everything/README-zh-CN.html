什么是 json-everything？

json-everything 背后的主要理念是确保在 System.Text.Json 生态中，对常见 JSON 功能提供良好的支持。

第一步是检查是否已经存在支持某项功能的其他项目。

如果有，这里可能不会重复实现该功能。稍后你会看到一个正在用 JSON 做一些很酷事情的其他项目列表。

否则，该功能会被纳入本项目的考虑范围。

有哪些内容？

实际上有多个“盒子”。每一项功能都被拆分为独立的库，因此你可以按需选择所需的库。

所有库都基于 .Net Standard 2.0 和 .Net 8 构建。

JsonSchema.Net
 JSON Schema，draft 6 及以上
JsonSchema.Net.Generation
 基于 .Net 类型自动生成 Schema
JsonSchema.Net.Generation.DataAnnotations
 支持 System.ComponentModel.DataAnnotations 的 Schema 生成。
JsonSchema.Net.DataGeneration
 随机实例数据生成（由 Bogus 提供支持）
JsonSchema.Net.Data
 访问实例及外部数据的词汇表
JsonSchema.Net.ArrayExt
 扩展数组校验的词汇表
JsonSchema.Net.OpenApi
 OpenApi 3.1 词汇表扩展（被 Graeae 使用）
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) 及 Relative JSON Pointer (规范)
JsonLogic
 JsonLogic (官方网站)
JsonE.Net
 JSON-e (官方网站)
Json.More.Net
 通用扩展，本应包含在 System.Text.Json[.Nodes] 中但未被包含
Yaml2JsonNode
 在 YamlDotNet 的 YAML 文档模型和 JsonNode 之间的转换（双向支持）。

上述所有库均对其相关规范提供完整支持。

JsonSchema.Net 的错误信息翻译支持如下语言：

  • 意大利语 it
  • 挪威语 nb-NO
  • 波兰语 pl-PL
  • 俄语 ru
  • 西班牙语 es
  • 瑞典语 sv-SE
  • 土耳其语 tr-TR

注意：每个语言包都由独立的 Nuget 包提供。

你还可以在 Bowtie 查看 JSON Schema 测试套件的结果，该网站会在不同平台的多种实现上运行测试套件。

免责声明：本库 JsonSchema.Net 与网站 https://jsonschema.net 无任何关联，仅同为优秀的 JSON Schema 工具。

已废弃的包

以下包也曾属于本套件，但现已不再维护。

JsonSchema.Net.CodeGeneration
 从 Schema 生成 C# 代码 已废弃，推荐使用 Corvus.JsonSchema

其他项目对 System.Text.Json 的支持

如果你没有在这里找到所需的功能，可以尝试以下优秀项目：

  • JsonCons.Net by @danielaparker
    • JSON Pointer
    • JSON Patch
    • JSON Merge Patch
    • JSON Path
    • JMES Path
  • Corvus.JsonSchema by @mwadams
    • 以 C# 代码生成为重点的 JSON Schema 校验
  • （更多项目即将推出）

如果你用 JSON 做了本套件或上述项目未涵盖的事情，欢迎创建新功能需求。

如果你维护或了解其他扩展 System.Text.Json 的项目，也欢迎提交通用 issue 通知我们，以便将其列出。
