简介

本列表最初是 StackOverflow - List of Freely Available Programming Books 的克隆版本，由 Karan Bhangui 和 George Stocker 贡献内容。

该列表由 Victor Felder 迁移至 GitHub，便于协作更新和维护。现已成长为 GitHub 最受欢迎的项目之一。

   
   

该仓库现由 Free Ebook Foundation 管理，这是一个致力于促进免费电子书的创作、分发、归档和可持续发展的非营利组织。向 Free Ebook Foundation 捐赠 在美国可享受税收减免。
