कई भाषाओं में मुफ्त लर्निंग संसाधनों की सूची

   

सूची में खोजें https://ebookfoundation.github.io/free-programming-books-search/ ।

यह पृष्ठ एक आसानी से पढ़े जाने वाले वेबसाइट के रूप में भी उपलब्ध है। इसे एक्सेस करने के लिए क्लिक करें ।

परिचय

यह सूची मूल रूप से StackOverflow - List of Freely Available Programming Books की एक क्लोन थी, जिसमें करन भांगी और जॉर्ज स्टॉकर का योगदान था।

इस सूची को सहयोगात्मक अद्यतन और रखरखाव के लिए विक्टर फेल्डर द्वारा GitHub पर स्थानांतरित किया गया था। यह GitHub के सबसे लोकप्रिय रिपोजिटरीज़ में से एक बन गई है।

   
   

अब Free Ebook Foundation इस रिपोजिटरी का संचालन करती है, जो मुक्त ईबुक्स के निर्माण, वितरण, संग्रहण और स्थिरता को बढ़ावा देने के लिए समर्पित एक गैर-लाभकारी संगठन है। अमेरिका में Free Ebook Foundation को दान कर टैक्स में छूट मिलती है।

योगदान कैसे करें

कृपया CONTRIBUTING पढ़ें। यदि आप GitHub में नए हैं, तो स्वागत है! कृपया हमारे आचार संहिता का भी पालन करें (अनुवाद भी उपलब्ध हैं)।
