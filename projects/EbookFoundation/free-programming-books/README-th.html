รายชื่อแหล่งเรียนรู้ฟรีในหลายภาษา

   

ค้นหารายการได้ที่ https://ebookfoundation.github.io/free-programming-books-search/ .

หน้านี้มีให้ใช้งานในรูปแบบเว็บไซต์ที่อ่านง่าย เข้าใช้งานได้โดยคลิกที่ .

แนะนำ

รายการนี้เริ่มต้นจากการโคลน StackOverflow - รายชื่อหนังสือโปรแกรมมิ่งที่มีให้ใช้ฟรี พร้อมการมีส่วนร่วมจาก Karan Bhangui และ George Stocker

รายการนี้ถูกย้ายมายัง GitHub โดย Victor Felder เพื่อให้สามารถอัปเดตและดูแลร่วมกันได้ ปัจจุบันได้เติบโตขึ้นเป็นหนึ่งใน คลังข้อมูลที่ได้รับความนิยมมากที่สุดบน GitHub

   
   

Free Ebook Foundation เป็นผู้ดูแลคลังข้อมูลนี้ในปัจจุบัน โดยเป็นองค์กรไม่แสวงหาผลกำไรที่มุ่งส่งเสริมการสร้าง เผยแพร่ เก็บถาวร และความยั่งยืนของหนังสืออิเล็กทรอนิกส์ฟรี การบริจาค ให้กับ Free Ebook Foundation สามารถหักลดหย่อนภาษีได้ในสหรัฐอเมริกา
