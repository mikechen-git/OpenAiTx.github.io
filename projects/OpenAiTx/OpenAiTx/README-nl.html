Introductie

OpenAiTx 20-talen automatische vertaaltool voor de readme & wiki van je GitHub-project.

Waarom?

  • Eenmalige vertaling met premium-kwaliteit LLM en maak het resultaat vrij toegankelijk voor de open-source community.
  • Ondersteunt Google/Bing meertalige SEO-zoekopdrachten, een client-vertaaltool kan dit niet.
  • Altijd gratis & open source
  • Eén keer indienen en automatisch bijwerken in de toekomst, het bespaart je tijd als je een projectbeheerder bent.

Aan de slag

  • Vervang de URL GitHub door OpenAiTx, bijv. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • Klik en kopieer de stijlbadges of tekst die je mooi vindt.

of

  • Ga naar https://openaitx.github.io/
  • Dien de link van je project in.

Bijdrage aan Computing

Als je wilt bijdragen aan het project, hoef je alleen het volgende te doen: Project forken → Project clonen → Kies een script in jouw taal → Vul je AI-token in → Voer het uit → Commit & push & maak een PR aan

Opmerking: Upload je tokens alstublieft niet!

Notities

  • Alleen ondersteuning voor github markdown.
  • Als je projecteigenaar of core developer bent, kun je je github-account autoriseren om automatisch een PR aan te maken nadat deze is geïndexeerd.

Overige

  • 10~30 min/per project
  • 200~400 projecten/één console per dag.
  • Elke 3~7 dagen worden bestaande projecten bijgewerkt (afhankelijk van toekomstige computercapaciteit)
