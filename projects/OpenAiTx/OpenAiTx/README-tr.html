Giriş

OpenAiTx 20 dilde otomatik çeviri aracı, GitHub projenizin readme ve wiki’si için.

Neden?

  • Premium seviyede LLM kullanarak tek seferlik çeviri yapın ve sonucu açık kaynak topluluğuna ücretsiz sunun.
  • Google/Bing çok dilli SEO aramalarını destekler, istemci çeviri aracı bunu yapamaz.
  • Sonsuza kadar Ücretsiz & Açık Kaynak
  • Bir kez gönderin ve gelecekte otomatik olarak güncellensin, bir proje yöneticisiyseniz zaman kazandırır.

Başlarken

  • URL’deki GitHub kısmını OpenAiTx ile değiştirin, ör. https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx

Katkı Sağlama

Projeye katkıda bulunmak isterseniz, yapmanız gerekenler: Projeyi çatallayın → Projeyi klonlayın → Kendi dilinizde bir betik seçin → AI tokenınızı doldurun → Çalıştırın → Commit & push & PR oluşturun

Not: Lütfen tokenlarınızı yüklemeyin!

Notlar

  • Sadece github markdown desteklenmektedir.
  • Eğer proje sahibi veya çekirdek geliştiriciyseniz, indekslendikten sonra otomatik PR oluşturmak için github hesabınızı doğrulayabilirsiniz.

Diğer

  • 10~30 dakika/proje başına
  • 200~400 proje/günlük bir konsol başına.
  • Her 3~7 günde bir mevcut projeler güncellenecek (gelecekteki hesaplama kaynaklarına bağlı olarak)

Teşekkürler

  • Microsoft MVP ekibi
