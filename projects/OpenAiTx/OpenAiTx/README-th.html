บทนำ

OpenAiTx เครื่องมือแปลภาษาอัตโนมัติ 20 ภาษา สำหรับ readme & wiki ของโปรเจกต์ GitHub ของคุณ

ทำไม?

  1. แปลเพียงครั้งเดียวโดยใช้ LLM ระดับพรีเมียม และเปิดผลลัพธ์ให้ชุมชนโอเพ่นซอร์สเข้าถึงได้ฟรี
  2. รองรับ SEO ค้นหาหลายภาษา Google/Bing ซึ่งเครื่องมือแปลฝั่งลูกค้าไม่สามารถทำได้
  3. ฟรี & โอเพ่นซอร์สตลอดไป
  4. ส่งเพียงครั้งเดียวและอัปเดตอัตโนมัติในอนาคต ช่วยประหยัดเวลาหากคุณเป็นผู้ดูแลโปรเจกต์

  1. เปลี่ยน URL GitHub เป็น OpenAiTx เช่น https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx

วิดีโอสาธิต

วิธีมีส่วนร่วมในการประมวลผล

หากคุณต้องการมีส่วนร่วมในโปรเจกต์นี้ สิ่งที่คุณต้องทำมีดังนี้: Fork โปรเจกต์ → Clone โปรเจกต์ → เลือกสคริปต์ในภาษาของคุณ → ใส่โทเคน AI ของคุณ → รันสคริปต์ → Commit & push & สร้าง PR

หมายเหตุ: โปรดอย่าอัปโหลดโทเคนของคุณ!

หมายเหตุ

  • รองรับเฉพาะ github markdown เท่านั้น
  • หากคุณเป็นเจ้าของโปรเจกต์หรือผู้พัฒนาหลัก คุณสามารถยืนยันบัญชี github เพื่อสร้าง PR อัตโนมัติหลังจากที่ได้รับการจัดทำดัชนีแล้ว

อื่นๆ

  • 10~30 นาที/ต่อโปรเจกต์
  • 200~400 โปรเจกต์/คอนโซลหนึ่งเครื่องต่อวัน
  • ทุก ๆ 3~7 วัน จะอัปเดตโปรเจกต์ที่มีอยู่ (ขึ้นอยู่กับทรัพยากรการประมวลผลในอนาคต)
