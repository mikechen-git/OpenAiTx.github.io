Введение

OpenAiTx — инструмент для автоматического перевода на 20 языков для вашего readme и wiki проекта на GitHub.

Почему?

  • Одноразовый перевод с использованием передовой LLM-платформы, и результат свободно доступен для сообщества с открытым исходным кодом.
  • Поддержка SEO-поиска Google/Bing на нескольких языках, чего не могут клиентские инструменты перевода.
  • Бесплатно и с открытым исходным кодом навсегда
  • Однократная отправка и автоматическое обновление в будущем, что экономит ваше время, если вы являетесь сопровождающим проекта.

Как начать

  • Замените URL GitHub на OpenAiTx, например: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • Кликните и скопируйте стиль бейджа или текст, который вам нравится.

или

  • Перейдите по адресу https://openaitx.github.io/
  • Отправьте ссылку на ваш проект.

Демонстрационное видео

Внесение вклада

Если вы хотите внести вклад в проект, вам нужно сделать следующее: Сделайте форк проекта → Клонируйте проект → Выберите скрипт на вашем языке → Вставьте свой AI-токен → Запустите скрипт → Сделайте commit & push и создайте PR

Примечание: Пожалуйста, не загружайте свои токены!

Примечания

  • Поддерживается только markdown от github.
  • Если вы владелец проекта или основной разработчик, вы можете авторизовать github-аккаунт для автоматического создания PR после индексации.

Другое

  • 10~30 минут/на проект
  • 200~400 проектов/одна консоль в день.
  • Каждые 3~7 дней будет обновляться существующие проекты (в зависимости от будущих вычислительных ресурсов)
