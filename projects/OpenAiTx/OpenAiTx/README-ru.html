Введение

OpenAiTx автоматически создает readme-файлы с переводом на 20 языков для вашего проекта на GitHub, вам просто нужно заменить URL проекта GitHub на OpenAiTx, например: https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

Поддержка/Вклад

Если вы хотите внести свой вклад в проект, все, что вам нужно сделать: Сделать форк проекта => Клонировать проект => Выбрать скрипт на вашем языке => Вставить ваш AI-токен => Запустить => Commit & push & создать PR

Примечание: Пожалуйста, не загружайте и не раскрывайте ваши токены!

Примечания

  • Сейчас поддерживается только markdown readme.
  • Рекомендуется использовать полные URL-адреса для изображений.

Другое

  • Для readme размером 50 КБ требуется 2 минуты/на язык с помощью GPT4.1.
  • 200-400 проектов/одна консоль в день.
  • Каждые 3-N дней обновляются существующие проекты (в зависимости от будущих вычислительных ресурсов)
