## Giới thiệu

OpenAiTx là công cụ tự động dịch 20 ngôn ngữ cho tài liệu readme & wiki của dự án GitHub của bạn.

Tại sao?

  1. Dịch một lần bằng LLM chất lượng cao và làm cho kết quả được truy cập miễn phí cho cộng đồng mã nguồn mở.
  2. Hỗ trợ tìm kiếm đa ngôn ngữ Google/Bing SEO, công cụ dịch phía khách hàng không thể làm được điều này.
  3. Miễn phí & Mã nguồn mở mãi mãi
  4. Gửi một lần và tự động cập nhật về sau, giúp bạn tiết kiệm thời gian nếu bạn là người duy trì dự án.

Bắt đầu

  1. Thay thế URL GitHub bằng OpenAiTx, ví dụ https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  2. Nhấn và sao chép biểu tượng hoặc đoạn văn bản bạn thích.

hoặc

  1. Truy cập https://openaitx.github.io/
  2. Gửi liên kết dự án của bạn.

Tính Toán Đóng Góp

Nếu bạn muốn đóng góp cho dự án, tất cả những gì bạn cần làm là: Fork dự án → Clone dự án → Chọn một script bằng ngôn ngữ của bạn → Điền token AI của bạn → Chạy script → Commit & push & tạo PR

Lưu ý: Vui lòng không tải lên token của bạn!

Ghi chú

  • Chỉ hỗ trợ markdown của github.
  • Nếu bạn là chủ dự án hoặc lập trình viên cốt lõi, bạn có thể xác thực tài khoản github để tự động tạo PR sau khi đã lập chỉ mục.

Khác

  • 10~30 phút/ mỗi dự án
  • 200~400 dự án/một bảng điều khiển mỗi ngày.
  • Cứ mỗi 3~7 ngày sẽ cập nhật các dự án hiện có (tùy thuộc vào tài nguyên tính toán trong tương lai)
