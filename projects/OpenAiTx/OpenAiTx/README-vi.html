Chia sẻ đóng góp

Nếu bạn có thể giúp bằng cách gửi PR đến dự án yêu thích của mình, điều đó sẽ giúp quảng bá dự án của chúng tôi rất nhiều.

Ví dụ:

Tiêu đề:
docs: thêm liên kết README 20 ngôn ngữ

Nội dung:
PR thêm liên kết 20 ngôn ngữ vào README giúp người dùng dễ dàng truy cập README đã dịch, hỗ trợ tìm kiếm SEO đa ngôn ngữ google/bing.

Đóng góp tính toán

Nếu bạn muốn đóng góp vào dự án, tất cả những gì bạn cần làm là: Fork dự án → Clone dự án → Chọn script ngôn ngữ của bạn → Điền token AI → Chạy script → Commit & push & tạo PR

Lưu ý: Vui lòng không tải lên token của bạn!

Lưu ý

  • Chỉ hỗ trợ markdown readme của github.

Khác

  • 10~30 phút/ mỗi dự án
  • 200~400 dự án/mỗi console mỗi ngày.
  • Cứ 3~7 ngày sẽ cập nhật lại các dự án đã có (phụ thuộc vào tài nguyên tính toán trong tương lai)
