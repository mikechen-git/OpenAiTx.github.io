Introduction

OpenAiTx est un outil de traduction automatique en 20 langues pour le readme et le wiki de votre projet GitHub.

Pourquoi ?

  • Traduction unique utilisant un LLM de qualité premium et mise à disposition gratuite du résultat pour la communauté open source.
  • Prise en charge du référencement multilingue Google/Bing, ce que les outils de traduction côté client ne peuvent pas faire.
  • Gratuit & Open Source pour toujours
  • Soumettez une seule fois et mettez à jour automatiquement à l'avenir, cela peut vous faire gagner du temps si vous êtes mainteneur de projet.

Pour commencer

  • Remplacez l’URL GitHub par OpenAiTx, par exemple https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • Cliquez et copiez les badges de style ou le texte qui vous plaît.

ou

  • Accédez à https://openaitx.github.io/
  • Soumettez le lien de votre projet.

Vidéo de démonstration

Calcul de la contribution

Si vous souhaitez contribuer au projet, il vous suffit de : Forker le projet → Cloner le projet → Choisir un script dans votre langue → Renseigner votre jeton IA → L’exécuter → Commit & push & créer une PR

Remarque : Veuillez ne pas télécharger vos jetons !

Notes

  • Prend uniquement en charge le markdown de Github.
  • Si vous êtes le propriétaire du projet ou un développeur principal, vous pouvez authentifier votre compte Github pour créer automatiquement une PR après l’indexation.

Autre

  • 10~30 minutes / par projet
  • 200~400 projets / une console par jour.
  • Tous les 3~7 jours, mise à jour des projets existants (en fonction des ressources informatiques futures)
