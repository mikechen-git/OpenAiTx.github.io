Introduction

OpenAiTx crée automatiquement des readmes traduits en 20 langues pour votre projet GitHub, vous n’avez qu’à remplacer l’URL du projet GitHub par OpenAiTx, par exemple https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx .

Support/Contribution

Si vous souhaitez contribuer au projet, il vous suffit de : Forker le projet => Cloner le projet => Choisir un script dans votre langue => Remplir votre token AI => Lancer => Commit & push & créer une PR

Note : Merci de ne pas uploader ou divulguer vos tokens !
