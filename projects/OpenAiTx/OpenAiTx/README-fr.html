Introduction

OpenAiTx est un outil de traduction automatique en 20 langues pour le fichier readme de votre projet GitHub.

Pourquoi ?

  1. Traduction unique en utilisant un LLM premium et rendre le résultat librement accessible à la communauté open-source.
  2. Prise en charge du référencement multilingue Google/Bing, ce que ne peuvent pas faire les outils de traduction côté client.
  3. Gratuit & Open Source pour toujours
  4. Soumettez une fois et mettez à jour automatiquement à l’avenir, cela peut vous faire gagner du temps si vous êtes mainteneur de projet.

Pour Commencer

  1. Remplacez l’URL GitHub par OpenAiTx, par exemple https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  2. Cliquez et copiez le badge ou le texte de style que vous aimez, le système créera automatiquement une PR pour le projet après indexation.

Contribution informatique

Si vous souhaitez contribuer au projet, tout ce que vous avez à faire est : Fork du projet → Cloner le projet → Choisir un script dans votre langue → Renseigner votre jeton IA → Lancer le script → Commit & push & créer une PR

Remarque : Veuillez ne pas télécharger vos jetons !

Remarques

  • Prend en charge uniquement le readme markdown de github.

Autre

  • 10~30 min / par projet
  • 200~400 projets / une console par jour
  • Tous les 3~7 jours, mise à jour des projets existants (en fonction des ressources informatiques futures)

Remerciements

  • L’équipe Microsoft MVP fournit un budget de plateforme Azure open source de 150 USD/mois à la communauté.

Plan futur

  • Prendre en charge la traduction automatique du wiki GitHub
