Introduction

OpenAiTx est un outil de traduction automatique en 20 langues pour le fichier readme de votre projet GitHub.

Pourquoi ?

  1. Traduction unique utilisant un LLM de qualité premium et rend le résultat librement accessible à la communauté open source.
  2. Prend en charge la recherche SEO multilingue Google/Bing, ce qu'un outil de traduction côté client ne peut pas faire.
  3. Gratuit et open source pour toujours
  4. Soumettez une fois et mise à jour automatique à l’avenir, cela peut vous faire gagner du temps si vous êtes mainteneur de projet.

Pour Commencer

  1. Remplacez l’URL GitHub par OpenAiTx, par exemple https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx

  2. Cliquez et copiez le badge de style ou le texte qui vous plaît.

Contribution informatique

Si vous souhaitez contribuer au projet, il vous suffit de : Forker le projet → Cloner le projet → Choisir un script dans votre langue → Renseigner votre jeton IA → L'exécuter → Commit & push & créer une PR

Remarque : Veuillez ne pas télécharger vos jetons !

Notes

  • Prise en charge uniquement du readme markdown GitHub.
  • Si vous êtes propriétaire du projet ou développeur principal, vous pouvez authentifier votre compte GitHub pour créer automatiquement des PR après l'indexation.

Autre

  • 10 à 30 minutes / par projet
  • 200 à 400 projets / une console par jour
  • Mise à jour des projets existants tous les 3 à 7 jours (selon les ressources informatiques futures)
