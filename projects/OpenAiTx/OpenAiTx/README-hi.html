परिचय

OpenAiTx 20 भाषाओं के लिए ऑटो-ट्रांसलेट टूल आपके GitHub प्रोजेक्ट की readme और wiki के लिए।

क्यों?

  • प्रीमियम-ग्रेड LLM का उपयोग करके एक बार अनुवाद करें और परिणाम को ओपन-सोर्स समुदाय के लिए स्वतंत्र रूप से उपलब्ध कराएँ।
  • Google/Bing की कई भाषाओं में SEO सर्च को सपोर्ट करता है, क्लाइंट ट्रांसलेट टूल ऐसा नहीं कर सकता।
  • हमेशा के लिए मुफ्त और ओपन सोर्स
  • एक बार सबमिट करें और भविष्य में ऑटो-अपडेट हो जाए, यदि आप प्रोजेक्ट मेंटेनर हैं तो यह आपका समय बचा सकता है।

शुरू करें

  • क्लिक करें और अपने पसंदीदा स्टाइल बैज या टेक्स्ट को कॉपी करें।

योगदान की गणना

यदि आप इस प्रोजेक्ट में योगदान करना चाहते हैं, तो आपको बस यह करना है: प्रोजेक्ट को फोर्क करें → प्रोजेक्ट को क्लोन करें → अपनी भाषा में एक स्क्रिप्ट चुनें → अपना AI टोकन भरें → इसे चलाएं → कमिट करें & पुश करें & PR बनाएं

नोट: कृपया अपने टोकन अपलोड न करें!

नोट्स

  • केवल GitHub मार्कडाउन को सपोर्ट करता है।
  • यदि आप प्रोजेक्ट ओनर या कोर डेवलपर हैं, तो आप GitHub अकाउंट को प्रमाणित करके इंडेक्सिंग के बाद ऑटो-PR बना सकते हैं।

अन्य

  • 10~30 मिनट/ प्रति परियोजना
  • 200~400 परियोजनाएँ/एक कंसोल प्रतिदिन।
  • हर 3~7 दिनों में मौजूदा परियोजनाओं को अपडेट किया जाएगा (भविष्य के कंप्यूटिंग संसाधनों पर निर्भर करता है)
