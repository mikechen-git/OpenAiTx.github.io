कंप्यूटिंग योगदान

यदि आप प्रोजेक्ट में योगदान देना चाहते हैं, तो आपको केवल यह करना है: प्रोजेक्ट को Fork करें → प्रोजेक्ट को Clone करें → अपनी भाषा में स्क्रिप्ट चुनें → अपना AI टोकन भरें → रन करें → Commit करें & push करें & PR बनाएं

नोट: कृपया अपने टोकन अपलोड न करें!

नोट्स

  • केवल GitHub मार्कडाउन रीडमी का समर्थन करता है।

अन्य

  • 10~30 मिनट/प्रति प्रोजेक्ट
  • 200~400 प्रोजेक्ट्स/एक कंसोल प्रति दिन।
  • हर 3~7 दिन में मौजूदा प्रोजेक्ट्स अपडेट होंगे (आगे की कंप्यूटिंग संसाधनों पर निर्भर करता है)
