소개

OpenAiTx는 GitHub 프로젝트의 readme 및 위키를 위한 20개 언어 자동 번역 도구입니다.

왜 사용하나요?

  • 프리미엄 등급 LLM을 사용한 일회성 번역으로 결과를 오픈소스 커뮤니티에 자유롭게 공개합니다.
  • Google/Bing 다국어 SEO 검색을 지원하며, 클라이언트 번역 도구는 이를 할 수 없습니다.
  • 영원히 무료 & 오픈소스
  • 한 번 제출하면 자동 업데이트가 지원되어, 프로젝트 관리자의 시간을 절약할 수 있습니다.

시작하기

  • URL의 GitHub를 OpenAiTx로 교체하세요. 예: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • 원하는 스타일 배지나 텍스트를 클릭하여 복사하세요.

또는

  • https://openaitx.github.io/ 에 접속하세요.
  • 프로젝트 링크를 제출하세요.

기여 방법

프로젝트에 기여하고 싶으시다면, 아래의 절차를 따라주세요: 프로젝트 포크 → 프로젝트 클론 → 자신의 언어로 된 스크립트 선택 → AI 토큰 입력 → 실행 → 커밋 & 푸시 & PR 생성

참고: 절대로 자신의 토큰을 업로드하지 마세요!

참고 사항

  • Github 마크다운만 지원합니다.
  • 프로젝트 소유자나 핵심 개발자인 경우, 깃허브 계정을 인증하여 인덱싱 후 자동으로 PR을 생성할 수 있습니다.

기타

  • 프로젝트당 10~30분
  • 하루에 콘솔당 200~400개 프로젝트
  • 3~7일마다 기존 프로젝트 업데이트 (향후 컴퓨팅 자원에 따라 다름)

감사 인사
