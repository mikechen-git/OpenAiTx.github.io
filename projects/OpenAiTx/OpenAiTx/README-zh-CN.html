介绍

OpenAiTx 为您的 GitHub 项目自述文件和 Wiki 提供 20 种语言的自动翻译工具。

为什么选择它？

  • 一次性使用高级 LLM 翻译，并让开源社区免费访问翻译结果。
  • 支持 Google/Bing 多语言 SEO 搜索，客户端翻译工具无法做到。
  • 永久免费且开源
  • 一次提交并自动更新后续内容，如果您是项目维护者，这可以为您节省时间。

快速开始

  • 将 URL 中的 GitHub 替换为 OpenAiTx，例如 https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • 点击并复制您喜欢的样式徽章或文本。

或者

  • 访问 https://openaitx.github.io/
  • 提交您的项目链接。

计算贡献

如果您想为该项目做出贡献，只需按照以下步骤操作： Fork 项目 → 克隆项目 → 选择您的语言脚本 → 填写您的 AI Token → 运行脚本 → 提交 & 推送 & 创建 PR

注意：请不要上传您的 Token！

注意事项

  • 仅支持 GitHub Markdown。
  • 如果您是项目所有者或核心开发者，可以通过 GitHub 账号授权，在索引后自动创建 PR。

其他

  • 10~30 分钟/每个项目
  • 200~400 个项目/每台控制台每天。
  • 每隔 3~7 天会更新现有项目（取决于未来的计算资源）
