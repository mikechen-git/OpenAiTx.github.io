مقدمه

OpenAiTx ابزار ترجمه خودکار به ۲۰ زبان برای مستندات readme و ویکی پروژه‌های GitHub شما.

چرا؟

  • ترجمه یک‌باره با استفاده از مدل‌های زبانی پیشرفته و ارائه رایگان نتیجه به جامعه متن‌باز.
  • پشتیبانی از جستجوی چندزبانه گوگل/بینگ برای سئو؛ ابزارهای ترجمه سمت کاربر این قابلیت را ندارند.
  • رایگان و متن‌باز برای همیشه
  • ارسال یک‌باره و به‌روزرسانی خودکار در آینده، در صورت نگهداری پروژه موجب صرفه‌جویی در زمان شما می‌شود.

شروع کنید

  • آدرس GitHub را با OpenAiTx جایگزین کنید، مثلاً https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • نشان‌ها یا متنی را که دوست دارید کلیک و کپی کنید.

یا

  • به https://openaitx.github.io/ مراجعه کنید.
  • لینک پروژه خود را ارسال کنید.

نحوه مشارکت محاسباتی

اگر مایل به مشارکت در پروژه هستید، کافیست مراحل زیر را انجام دهید: فورک کردن پروژه → کلون کردن پروژه → انتخاب یک اسکریپت به زبان خودتان → وارد کردن توکن هوش مصنوعی خود → اجرای آن → کامیت و پوش و ایجاد PR

توجه: لطفاً توکن‌های خود را آپلود نکنید!

نکات

  • تنها از مارک‌داون گیت‌هاب پشتیبانی می‌شود.
  • اگر مالک پروژه یا توسعه‌دهنده اصلی هستید، می‌توانید حساب گیت‌هاب خود را احراز هویت کنید تا پس از ایندکس شدن، PR به صورت خودکار ایجاد شود.

سایر

  • ۱۰ تا ۳۰ دقیقه برای هر پروژه
