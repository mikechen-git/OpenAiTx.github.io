介紹

OpenAiTx 能自動為您的 GitHub 專案建立 20 種語言的翻譯版 readme，您只需將專案網址的 GitHub 替換為 OpenAiTx，例如：https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx。

支援/貢獻

如果您希望對本專案做出貢獻，只需： Fork 專案 => Clone 專案 => 選擇您語言的腳本 => 填入您的 AI token => 執行 => Commit & push & 建立 PR

注意：請勿上傳或洩漏您的 token！

其他

  • 50 KB 的 readme，GPT4.1 需要「每種語言 2 分鐘」。
  • 每天每個控制台可處理 200-400 個專案。
  • 每 3-N 天會更新現有專案（根據未來的運算資源而定）
