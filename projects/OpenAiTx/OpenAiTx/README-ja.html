はじめに

OpenAiTxは、あなたのGitHubプロジェクトのために20言語の翻訳READMEを自動生成します。プロジェクトのURLのGitHubをOpenAiTxに置き換えるだけで利用できます。例: https://github.com/OpenAiTx/OpenAiTx => https://openaitx.com/OpenAiTx/OpenAiTx 。

サポート/貢献

プロジェクトに貢献したい場合、必要なのは以下の手順だけです: プロジェクトをフォーク => プロジェクトをクローン => あなたの言語のスクリプトを選択 => AIトークンを入力 => 実行 => コミット＆プッシュ＆PRを作成

注意: トークンをアップロードしたり漏洩させたりしないでください！

その他

  • 50KBのREADMEはGPT4.1で1言語あたり2分かかります。
  • 1日あたり1コンソールで200～400プロジェクト対応。
  • 既存プロジェクトは3～N日ごとに更新されます（今後の計算リソース次第😎）
