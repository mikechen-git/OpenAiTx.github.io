Introducción

OpenAiTx es una herramienta de traducción automática a 20 idiomas para el archivo readme y wiki de tu proyecto en GitHub.

¿Por qué?

  • Traducción única utilizando un modelo LLM de calidad premium y hacer el resultado accesible libremente a la comunidad de código abierto.
  • Soporte para SEO multilingüe en Google/Bing, lo que no pueden hacer las herramientas de traducción del cliente.
  • Gratis y de código abierto para siempre
  • Enviar una sola vez y actualizar automáticamente en el futuro, puede ahorrarte tiempo si eres un mantenedor del proyecto.

Comenzar

  • Reemplaza la URL de GitHub por OpenAiTx, por ejemplo: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • Haz clic y copia las insignias de estilo o el texto que prefieras.

o

  • Accede a https://openaitx.github.io/
  • Envía el enlace de tu proyecto.

Contribución de Cómputo

Si deseas contribuir al proyecto, todo lo que necesitas hacer es: Bifurcar el proyecto → Clonar el proyecto → Elegir un script en tu idioma → Rellenar tu token de IA → Ejecutarlo → Hacer commit & push & crear un PR

Nota: ¡Por favor no subas tus tokens!

Notas

  • Solo se admite markdown de GitHub.
  • Si eres el propietario del proyecto o desarrollador principal, puedes autenticar tu cuenta de GitHub para crear PR automáticamente después de indexar.

Otros

  • 10~30 minutos/ por proyecto
  • 200~400 proyectos/una consola por día.
  • Cada 3~7 días se actualizarán los proyectos existentes (dependiendo de los recursos de computación futuros)
