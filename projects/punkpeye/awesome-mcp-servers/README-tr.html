MCP nedir?

MCP açık bir protokoldür; AI modellerinin standartlaştırılmış sunucu uygulamaları üzerinden yerel ve uzak kaynaklarla güvenli bir şekilde etkileşime girmesini sağlar. Bu liste, dosya erişimi, veritabanı bağlantıları, API entegrasyonları ve diğer bağlamsal hizmetler yoluyla AI yeteneklerini genişleten üretime hazır ve deneysel MCP sunucularına odaklanmaktadır.
