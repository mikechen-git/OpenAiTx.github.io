什麼是 MCP？

MCP 是一個開放協議，使 AI 模型能夠透過標準化的伺服器實作，安全地與本地及遠端資源互動。此清單聚焦於生產就緒與實驗性的 MCP 伺服器，這些伺服器可透過檔案存取、資料庫連線、API 整合及其他情境服務來擴展 AI 能力。

[!NOTE] 分不清本地 🏠 與雲端 ☁️？

  • 當 MCP 伺服器與本機安裝的軟體（如控制 Chrome 瀏覽器）互動時，請使用本地。
  • 當 MCP 伺服器與遠端 API（如天氣 API）互動時，請使用網路。
