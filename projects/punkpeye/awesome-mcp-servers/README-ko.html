MCP란?

MCP는 AI 모델이 표준화된 서버 구현을 통해 로컬 및 원격 리소스와 안전하게 상호작용할 수 있게 해주는 오픈 프로토콜입니다. 이 목록은 파일 접근, 데이터베이스 연결, API 통합, 기타 컨텍스트 서비스 등 AI 기능을 확장하는 프로덕션 및 실험적 MCP 서버에 초점을 맞추고 있습니다.
