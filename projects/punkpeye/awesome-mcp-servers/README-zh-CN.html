什么是MCP？

MCP 是一个开放协议，使AI模型能够通过标准化的服务器实现安全地与本地和远程资源交互。此列表关注于生产可用和实验性的MCP服务器，通过文件访问、数据库连接、API集成和其他上下文服务扩展AI能力。
