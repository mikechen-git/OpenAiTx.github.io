MCP چیست؟

MCP یک پروتکل باز است که به مدل‌های هوش مصنوعی امکان می‌دهد تا از طریق پیاده‌سازی‌های استانداردشده سرور، به صورت امن با منابع محلی و راه دور تعامل داشته باشند. این لیست بر روی سرورهای MCP آماده تولید و آزمایشی تمرکز دارد که قابلیت‌های هوش مصنوعی را از طریق دسترسی به فایل، اتصال به پایگاه داده، یکپارچه‌سازی با API و سایر سرویس‌های زمینه‌ای گسترش می‌دهند.
