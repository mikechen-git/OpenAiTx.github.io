Wat is MCP?

MCP is een open protocol waarmee AI-modellen veilig kunnen communiceren met lokale en externe bronnen via gestandaardiseerde server-implementaties. Deze lijst richt zich op productieklare en experimentele MCP-clients die AI-mogelijkheden uitbreiden via bestands­toegang, database­verbindingen, API-integraties en andere contextuele diensten.
