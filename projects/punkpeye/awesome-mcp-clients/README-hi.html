MCP क्या है?

MCP एक खुला प्रोटोकॉल है जो AI मॉडलों को स्थानीय और दूरस्थ संसाधनों के साथ मानकीकृत सर्वर इम्प्लीमेंटेशन के माध्यम से सुरक्षित रूप से इंटरैक्ट करने की सुविधा देता है। यह सूची उन्हीं MCP क्लाइंट्स पर केंद्रित है जो प्रोडक्शन के लिए तैयार हैं या प्रयोगात्मक हैं, और फाइल एक्सेस, डाटाबेस कनेक्शन, API इंटीग्रेशन तथा अन्य कॉन्टेक्स्टुअल सेवाओं के माध्यम से AI की क्षमताओं को बढ़ाते हैं।

  • MCP टूल्स का समर्थन।
  • एजेंट्स को व्यक्तिगत प्रॉम्प्ट, मॉडल, नॉलेज और MCP टूल्स के साथ प्रबंधित करें।
