Что такое MCP?

MCP — это открытый протокол, который позволяет AI-моделям безопасно взаимодействовать с локальными и удалёнными ресурсами через стандартизированные серверные реализации. Этот список включает готовые к использованию и экспериментальные MCP-клиенты, расширяющие возможности искусственного интеллекта с помощью доступа к файлам, подключения к базам данных, интеграции с API и других контекстуальных сервисов.
