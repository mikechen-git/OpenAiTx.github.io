O que é MCP?

MCP é um protocolo aberto que permite que modelos de IA interajam de forma segura com recursos locais e remotos por meio de implementações de servidor padronizadas. Esta lista foca em clientes MCP prontos para produção e experimentais que expandem as capacidades de IA através de acesso a arquivos, conexões com bancos de dados, integrações de API e outros serviços contextuais.

Tome é um aplicativo desktop open source multiplataforma projetado para trabalhar com LLMs locais e servidores MCP. O Tome gerencia seus servidores MCP para que você não precise mexer com uv/npm ou arquivos json — conecte ao Ollama, copie/cole alguns servidores MCP e converse com um modelo alimentado por MCP em segundos.
