MCP là gì?

MCP là một giao thức mở cho phép các mô hình AI tương tác an toàn với các tài nguyên cục bộ và từ xa thông qua các triển khai máy chủ tiêu chuẩn hóa. Danh sách này tập trung vào các client MCP sẵn sàng sử dụng trong sản xuất và thử nghiệm, mở rộng khả năng của AI thông qua truy cập tệp, kết nối cơ sở dữ liệu, tích hợp API và các dịch vụ ngữ cảnh khác.
