Visage: библиотека UI встречает креативное программирование

Visage — это кроссплатформенная C++ библиотека с аппаратным ускорением на GPU для нативного UI и 2D графики. Она объединяет структуру UI-фреймворка с возможностями креативных графических библиотек.

Полный базовый пример

#include <https://raw.githubusercontent.com/VitalAudio/visage/main/visage_app/application_window.h>

int main() {
  visage::ApplicationWindow app;

  app.onDraw() = [&app](visage::Canvas& canvas) {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };

  app.show(800, 600); // Открывается окно размером 800 x 600 пикселей
  app.runEventLoop(); // Запускает обработку событий окна. Возвращается после закрытия окна.
  return 0;
}

Особенности UI

  • Нормализация событий
       Кроссплатформенная поддержка нормализации ввода с клавиатуры и мыши.

  • Нормализация окон
       Кроссплатформенная поддержка открытия и управления окнами.

  • Ввод текста
       Ввод текста в формате Unicode с поддержкой многострочного редактирования.

  • ✨ Эмодзи ✨
       Если вам это интересно 🤷

  • Частичная отрисовка
       Перерисовываются только изменённые области для оптимальной производительности.

Особенности графики

  • Плавное движение
       Новые кадры отображаются с частотой обновления монитора, а анимации плавные.

  • Автоматическая группировка фигур
       Автоматически группирует фигуры для эффективного рендеринга на GPU.

  • Режимы смешивания
       Поддержка смешивания слоёв с аддитивным, вычитательным режимами или с использованием масок для пропуска UI.

  • Шейдеры
       Пишите шейдеры один раз и транспилируйте их для Direct3D, Metal и OpenGL.

  • Встроенные эффекты
       Эффекты в реальном времени, такие как сильное размытие и свечения.

  • Точность по пикселям
       Доступ к размеру пикселя устройства обеспечивает точную отрисовку без размытия.

Поддерживаемые платформы

  • Windows: Direct3D11
  • MacOS: Metal
  • Linux: Vulkan
  • Web/Emscripten: WebGL
