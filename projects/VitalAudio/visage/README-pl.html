Visage: biblioteka UI spotyka kreatywne kodowanie

Visage to akcelerowana przez GPU, wieloplatformowa biblioteka C++ do natywnego UI i grafiki 2D. Łączy strukturę frameworka UI z funkcjami kreatywnych bibliotek graficznych.

Pełny podstawowy przykład

#include <https://raw.githubusercontent.com/VitalAudio/visage/main/visage_app/application_window.h>

int main() {
  visage::ApplicationWindow app;

  app.onDraw() = [&app](visage::Canvas& canvas) {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };

  app.show(800, 600); // Otwiera okno o rozmiarze 800 x 600 pikseli
  app.runEventLoop(); // Uruchamia pętlę zdarzeń okna. Zwraca po zamknięciu okna.
  return 0;
}

Funkcje UI

  • Normalizacja zdarzeń
       Wieloplatformowe wsparcie dla normalizacji wejścia z klawiatury i myszy.

  • Normalizacja okna
       Wieloplatformowe wsparcie dla otwierania i obsługi okien.

  • Wprowadzanie tekstu
       Wprowadzanie tekstu Unicode z edycją wielowierszową

  • ✨ Emoji ✨
       Jeśli lubisz takie rzeczy 🤷

  • Częściowe renderowanie
       Przerysowuje tylko zmienione obszary dla optymalnej wydajności.

Funkcje graficzne

  • Płynny ruch
       Nowe klatki wyświetlane są z częstotliwością odświeżania monitora, animacje są płynne

  • Automatyczne grupowanie kształtów
       Automatycznie grupuje kształty dla efektywnego renderowania na GPU.

  • Tryby mieszania
       Obsługuje mieszanie warstw poprzez dodawanie, odejmowanie lub rysowanie niestandardowej maski, przez którą UI może przechodzić

  • Shadery
       Napisz shader raz i transpiluj go dla Direct3D, Metal i OpenGL

  • Dołączone efekty
       Efekty w czasie rzeczywistym takie jak duże rozmycie i bloom

  • Dokładność pikseli
       Dostęp do rozmiaru piksela urządzenia zapewnia precyzyjne renderowanie bez rozmycia.

Obsługiwane platformy

  • Windows: Direct3D11
  • MacOS: Metal
  • Linux: Vulkan
  • Web/Emscripten: WebGL
