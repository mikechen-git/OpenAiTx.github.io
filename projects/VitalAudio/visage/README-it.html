Visage: la libreria UI incontra il creative coding

Visage è una libreria C++ nativa, multipiattaforma e accelerata GPU per UI e grafica 2D. Unisce la struttura di un framework UI con le funzionalità di librerie grafiche creative.

Esempio Base Completo

#include <https://raw.githubusercontent.com/VitalAudio/visage/main/visage_app/application_window.h>

int main() {
  visage::ApplicationWindow app;

  app.onDraw() = [&app](visage::Canvas& canvas) {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };

  app.show(800, 600); // Si apre come finestra di 800 x 600 pixel
  app.runEventLoop(); // Gestisce gli eventi della finestra. Ritorna quando la finestra viene chiusa.
  return 0;
}

Caratteristiche Grafiche

  • Movimento Fluido
       I nuovi frame vengono mostrati al refresh rate del monitor e le animazioni sono fluide.

  • Batching Automatico delle Forme
       Raggruppa automaticamente le forme per un rendering GPU efficiente.

  • Modalità di Fusione
       Supporta la fusione dei layer con modalità additive, sottrattive o disegnando una maschera personalizzata per il passaggio UI.

  • Shader
       Scrivi shader una volta e transpila per Direct3D, Metal e OpenGL.

  • Effetti Inclusi
       Effetti in tempo reale come sfocatura intensa e bloom.

  • Precisione Pixel
       Accesso alla dimensione pixel del dispositivo per un rendering preciso senza sfocature.

Piattaforme Supportate

  • Windows: Direct3D11
  • MacOS: Metal
  • Linux: Vulkan
  • Web/Emscripten: WebGL
