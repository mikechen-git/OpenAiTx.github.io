Visage: UI kütüphanesi yaratıcı kodlama ile buluşuyor

Visage, yerel UI ve 2D grafikler için GPU hızlandırmalı, çapraz platform C++ kütüphanesidir. Bir UI çerçevesinin yapısını yaratıcı grafik kütüphanelerinin özellikleriyle birleştirir.

Tam Temel Örnek

#include <https://raw.githubusercontent.com/VitalAudio/visage/main/visage_app/application_window.h>

int main() {
  visage::ApplicationWindow app;

  app.onDraw() = [&app](visage::Canvas& canvas) {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };

  app.show(800, 600); // 800 x 600 piksel pencere olarak açılır
  app.runEventLoop(); // Pencere olaylarını çalıştırır. Pencere kapandığında döner.
  return 0;
}

UI Özellikleri

  • Olay Normalizasyonu

  • Pencere Normalizasyonu
       Pencerelerin açılması ve yönetilmesi için çapraz platform destek.

  • Metin Girişi
       Çok satırlı metin düzenlemeli Unicode metin girişi

  • ✨ Emojiler ✨
       Eğer böyle şeyleri seviyorsanız 🤷

  • Kısmi Çizim
       Optimum performans için sadece değişen bölgeleri yeniden çizer.

Grafik Özellikleri

  • Akıcı hareket
       Yeni kareler monitörün yenileme hızında gösterilir ve animasyonlar pürüzsüzdür.

  • Otomatik Şekil Gruplama
       Şekilleri GPU verimli çizimi için otomatik olarak gruplar.

  • Karıştırma Modları
       Katmanları ekleyerek, çıkararak veya UI'nin geçiş yapması için özel bir maske çizerek karıştırmayı destekler.

  • Shaderlar
       Shaderları bir kez yazın ve Direct3D, Metal ve OpenGL için dönüştürün.

  • Dahil Edilen Efektler
       Büyük bulanıklık ve bloom gibi gerçek zamanlı efektler.

  • Piksel Doğruluğu
       Cihaz piksel boyutuna erişim, bulanıklık olmadan hassas çizim sağlar.

Desteklenen Platformlar

  • Windows: Direct3D11

  • Linux: Vulkan
