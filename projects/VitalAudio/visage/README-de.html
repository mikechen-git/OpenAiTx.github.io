Visage: UI-Bibliothek trifft kreatives Coding

Visage ist eine GPU-beschleunigte, plattformübergreifende C++-Bibliothek für native UI und 2D-Grafiken. Sie verbindet die Struktur eines UI-Frameworks mit den Funktionen kreativer Grafikbibliotheken.

Vollständiges Basisbeispiel

#include <visage_app/application_window.h>

int main() {
  visage::ApplicationWindow app;

  app.onDraw() = [&app](visage::Canvas& canvas) {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };

  app.show(800, 600); // Öffnet ein Fenster mit 800 x 600 Pixeln
  app.runEventLoop(); // Führt Fensterevents aus. Rückkehr bei Fenster-Schließung.
  return 0;
}

UI-Funktionen

  • Event-Normalisierung
       Plattformübergreifende Unterstützung zur Normalisierung von Tastatur- und Mauseingaben.

  • Fenster-Normalisierung
       Plattformübergreifende Unterstützung zum Öffnen und Verwalten von Fenstern.

  • Texteingabe
       Unicode-Texteingabe mit mehrzeiliger Textbearbeitung.

  • ✨ Emojis ✨
       Falls du darauf stehst 🤷

  • Teilweise Darstellung
       Zeichnet nur die veränderten Bereiche neu für optimale Leistung.

Grafik-Funktionen

  • Flüssige Bewegung
       Neue Frames werden mit der Bildwiederholfrequenz des Monitors angezeigt, Animationen laufen flüssig.

  • Automatisches Shape-Batching
       Gruppiert Formen automatisch für effizientes GPU-Rendering.

  • Blend-Modi
       Unterstützt das Mischen von Ebenen mit additiver, subtraktiver oder durch das Zeichnen einer benutzerdefinierten Maske für das UI-Durchscheinen.

  • Shader
       Schreibe Shader einmal und transpiliere sie für Direct3D, Metal und OpenGL.

  • Inklusive Effekte
       Echtzeiteffekte wie starker Blur und Bloom.

  • Pixelgenauigkeit
       Zugriff auf die Gerätepixelgröße garantiert präzise Darstellung ohne Unschärfe.

Unterstützte Plattformen

  • Windows: Direct3D11
  • MacOS: Metal
  • Linux: Vulkan
  • Web/Emscripten: WebGL
