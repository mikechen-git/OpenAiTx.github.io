D3：数据驱动文档

D3（或 D3.js）是一个用于数据可视化的免费开源 JavaScript 库。其基于 Web 标准的底层方法为创作动态、数据驱动的图形提供了无与伦比的灵活性。十多年来，D3 支持了许多开创性和获奖的可视化项目，成为更高级图表库的基础构建模块，并在全球范围内促进了一个充满活力的数据从业者社区的发展。
