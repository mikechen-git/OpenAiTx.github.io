Bekijk de demo!

Productoverzicht

AnythingLLM is een full-stack applicatie waarmee je commerciële kant-en-klare LLM's of populaire open source LLM's en vectorDB-oplossingen kunt gebruiken om een privé ChatGPT te bouwen zonder compromissen, die je zowel lokaal als op afstand kunt hosten en intelligent kunt chatten met alle documenten die je toevoegt.

AnythingLLM verdeelt je documenten in objecten genaamd workspaces. Een workspace functioneert veel als een thread, maar met de toevoeging van containerisatie van je documenten. Workspaces kunnen documenten delen, maar communiceren niet met elkaar zodat je per workspace een schone context behoudt.

Technisch Overzicht

Deze monorepo bestaat uit zes hoofdsecties:

  • frontend: Een viteJS + React frontend waarmee je eenvoudig alle content kunt creëren en beheren die de LLM kan gebruiken.
  • server: Een NodeJS express server die alle interacties afhandelt en het vectorDB-beheer en LLM-interacties uitvoert.
  • collector: NodeJS express server die documenten van de UI verwerkt en parseert.
  • docker: Docker-instructies en buildproces + informatie voor bouwen vanuit de bron.
  • embed: Submodule voor generatie & creatie van de web embed widget.
  • browser-extension: Submodule voor de chrome browserextensie.

Hoe instellen voor ontwikkeling

  • yarn setup Vul de vereiste .env bestanden in die je nodig hebt in elk van de applicatiesecties (vanuit root van de repo).
    • Vul deze eerst in voordat je doorgaat. Zorg dat server/.env.development is ingevuld, anders werkt het niet goed.
  • yarn dev:server Start de server lokaal (vanuit root van de repo).
  • yarn dev:frontend Start de frontend lokaal (vanuit root van de repo).
  • yarn dev:collector Start vervolgens de document collector (vanuit root van de repo).

Telemetrie & Privacy

AnythingLLM door Mintplex Labs Inc bevat een telemetrie-functie die anonieme gebruiksinformatie verzamelt.

Meer over Telemetrie & Privacy voor AnythingLLM

Waarom?

We gebruiken deze informatie om te begrijpen hoe AnythingLLM wordt gebruikt, om ons te helpen prioriteit te geven aan nieuwe functies en bugfixes, en om de prestaties en stabiliteit van AnythingLLM te verbeteren.

Uitschrijven

Stel DISABLE_TELEMETRY in op je server of in de docker .env-instellingen in op "true" om je af te melden voor telemetrie. Je kunt dit ook in de app doen door naar de zijbalk te gaan > Privacy en telemetrie uit te schakelen.

Wat wordt er expliciet bijgehouden?

We houden alleen gebruiksgegevens bij die ons helpen bij het nemen van product- en roadmapbeslissingen, specifiek:

  • Type installatie (Docker of Desktop)

  • Wanneer een document wordt toegevoegd of verwijderd. Geen informatie over het document. Alleen dat het evenement heeft plaatsgevonden. Dit geeft ons een idee van het gebruik.

  • Type gebruikte vector-database. Dit helpt ons prioriteiten te stellen wanneer er updates zijn voor die provider.

  • Type LLM-provider & model-tag in gebruik. Dit helpt ons prioriteiten te stellen wanneer er updates zijn voor die provider of model, of combinatie daarvan. Bijvoorbeeld: redeneren versus regulier, multimodale modellen, etc.

  • Wanneer een chat wordt verzonden. Dit is het meest voorkomende "event" en geeft ons een idee van de dagelijkse activiteit van dit project over alle installaties heen. Opnieuw: alleen het event wordt verzonden - we hebben geen informatie over de aard of inhoud van de chat zelf.

Je kunt deze beweringen verifiëren door alle locaties te vinden waar Telemetry.sendTelemetry wordt aangeroepen. Daarnaast worden deze events naar het uitvoerlogboek geschreven, zodat je ook precies kunt zien welke gegevens zijn verzonden - indien ingeschakeld. Er worden geen IP- of andere identificerende gegevens verzameld. De telemetrieprovider is PostHog - een open-source telemetrieverzameldienst.

Wij nemen privacy zeer serieus, en we hopen dat je begrijpt dat we willen leren hoe onze tool wordt gebruikt, zonder vervelende pop-up enquêtes te gebruiken, zodat we iets waardevols kunnen bouwen. De anonieme data wordt nooit gedeeld met derden, nooit.
