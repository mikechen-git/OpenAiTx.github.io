Deno

Deno (/ˈdiːnoʊ/, pronunciato dee-no) è un runtime per JavaScript, TypeScript e WebAssembly con impostazioni di sicurezza predefinite e un'eccellente esperienza per gli sviluppatori. È costruito su V8, Rust e Tokio.

Scopri di più sul runtime Deno nella documentazione.

Installazione

Installa il runtime Deno sul tuo sistema utilizzando uno dei comandi qui sotto. Nota che esistono diversi modi per installare Deno: una lista completa delle opzioni di installazione è disponibile qui.

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

Compilare e installare dal sorgente

Le istruzioni complete per compilare Deno dal sorgente sono disponibili qui.

Il tuo primo programma Deno

Deno può essere utilizzato per molte applicazioni diverse, ma viene più comunemente usato per costruire web server. Crea un file chiamato server.ts e inserisci il seguente codice TypeScript:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Esegui il tuo server con il seguente comando:

deno run --allow-net server.ts

Questo avvierà un server web locale su http://localhost:8000.
