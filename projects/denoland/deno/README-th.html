Deno

Deno (/ˈdiːnoʊ/, อ่านว่า ดี-โน) คือรันไทม์สำหรับ JavaScript, TypeScript และ WebAssembly ที่มีค่าเริ่มต้นความปลอดภัยสูงและประสบการณ์ที่ดีสำหรับนักพัฒนา พัฒนาบนพื้นฐานของ V8, Rust และ Tokio

เรียนรู้เพิ่มเติมเกี่ยวกับ Deno runtime ในเอกสารประกอบ

การติดตั้ง

ติดตั้ง Deno runtime บนระบบของคุณโดยใช้คำสั่งใดคำสั่งหนึ่งด้านล่าง โปรดทราบว่ามีหลายวิธีในการติดตั้ง Deno - คุณสามารถดูรายการตัวเลือกทั้งหมดได้ ที่นี่

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

การคอมไพล์และติดตั้งจากซอร์ส

คำแนะนำแบบสมบูรณ์สำหรับการคอมไพล์ Deno จากซอร์ส สามารถดูได้ ที่นี่

โปรแกรม Deno แรกของคุณ

Deno สามารถใช้งานได้หลากหลายรูปแบบ แต่โดยทั่วไปนิยมใช้สร้างเว็บเซิร์ฟเวอร์ สร้างไฟล์ชื่อ server.ts และเพิ่มโค้ด TypeScript ดังนี้:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

รันเซิร์ฟเวอร์ของคุณด้วยคำสั่งนี้:

deno run --allow-net server.ts

คำสั่งนี้จะเริ่มต้นเว็บเซิร์ฟเวอร์บนเครื่องที่ http://localhost:8000
