Deno

Deno (/ˈdiːnoʊ/, произносится ди-но) — это среда выполнения JavaScript, TypeScript и WebAssembly с безопасными настройками по умолчанию и отличным опытом для разработчиков. Основана на V8, Rust и Tokio.

Узнайте больше о среде выполнения Deno в документации.

Установка

Установите среду выполнения Deno на свою систему с помощью одной из команд ниже. Обратите внимание, что существует несколько способов установки Deno — полный список вариантов установки доступен здесь.

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

Сборка и установка из исходников

Полные инструкции по сборке Deno из исходников можно найти здесь.

Ваша первая программа на Deno

Deno может использоваться для различных приложений, но чаще всего применяется для создания веб-серверов. Создайте файл с названием server.ts и добавьте следующий код на TypeScript:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Запустите ваш сервер следующей командой:

deno run --allow-net server.ts

Это запустит локальный веб-сервер по адресу http://localhost:8000.

Узнайте больше о написании и запуске программ на Deno в документации.
