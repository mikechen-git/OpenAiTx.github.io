دنو

دنو (/ˈdiːnoʊ/، تلفظ دی-نو) یک محیط اجرای جاوااسکریپت، تایپ‌اسکریپت و WebAssembly با پیش‌فرض‌های امنیتی و تجربه توسعه‌دهنده عالی است. این محیط بر پایه V8، Rust و Tokio ساخته شده است.

برای کسب اطلاعات بیشتر درباره محیط اجرای دنو به مستندات مراجعه کنید.

نصب

برای نصب محیط اجرای دنو روی سیستم خود از یکی از دستورهای زیر استفاده کنید. توجه داشته باشید که روش‌های مختلفی برای نصب دنو وجود دارد. فهرست کامل گزینه‌های نصب را می‌توانید اینجا بیابید.

شل (مک، لینوکس):

curl -fsSL https://deno.land/install.sh | sh

پاورشل (ویندوز):

irm https://deno.land/install.ps1 | iex

Homebrew (مک):

brew install deno

Chocolatey (ویندوز):

choco install deno

WinGet (ویندوز):

winget install --id=DenoLand.Deno

ساخت و نصب از سورس

دستورالعمل کامل برای ساخت دنو از سورس را می‌توانید اینجا پیدا کنید.

اولین برنامه دنو شما

دنو می‌تواند برای کاربردهای مختلفی استفاده شود اما عمدتاً برای ساخت سرورهای وب استفاده می‌شود. یک فایل به نام server.ts ایجاد کرده و کد تایپ‌اسکریپت زیر را در آن قرار دهید:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

سرور خود را با دستور زیر اجرا کنید:

deno run --allow-net server.ts

این دستور باید یک سرور وب محلی در http://localhost:8000 راه‌اندازی کند.

برای یادگیری بیشتر در مورد نوشتن و اجرای برنامه‌های دنو به مستندات مراجعه کنید.

منابع بیشتر

  • مستندات دنو: راهنماها و مستندات مرجع رسمی برای محیط اجرای دنو، Deno Deploy و موارد دیگر.
  • کتابخانه استاندارد دنو: ابزارهای رایج و پشتیبانی‌شده رسمی برای برنامه‌های دنو.
  • JSR: رجیستری بسته متن‌باز برای جاوااسکریپت و تایپ‌اسکریپت مدرن.
  • وبلاگ توسعه‌دهندگان: به‌روزرسانی‌های محصول، آموزش‌ها و مطالب بیشتر از تیم دنو.
