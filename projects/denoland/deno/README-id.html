Deno

Deno (/ˈdiːnoʊ/, diucapkan dee-no) adalah runtime JavaScript, TypeScript, dan WebAssembly dengan pengaturan default yang aman dan pengalaman pengembang yang luar biasa. Ini dibangun di atas V8, Rust, dan Tokio.

Pelajari lebih lanjut tentang runtime Deno di dokumentasi.

Instalasi

Pasang runtime Deno di sistem Anda menggunakan salah satu perintah di bawah ini. Perlu dicatat bahwa ada beberapa cara untuk menginstal Deno - daftar lengkap opsi instalasi dapat ditemukan di sini.

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

Build dan instal dari sumber

Instruksi lengkap untuk membangun Deno dari sumber dapat ditemukan di sini.

Program Deno Pertama Anda

Deno dapat digunakan untuk berbagai aplikasi, namun paling umum digunakan untuk membuat web server. Buat file bernama server.ts dan masukkan kode TypeScript berikut:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Jalankan server Anda dengan perintah berikut:

deno run --allow-net server.ts

Ini akan memulai web server lokal pada http://localhost:8000.
