Deno

Deno (/ˈdiːnoʊ/, prononcé dee-no) est un environnement d’exécution JavaScript, TypeScript et WebAssembly avec des paramètres de sécurité par défaut et une excellente expérience développeur. Il est construit sur V8, Rust et Tokio.

Pour en savoir plus sur l'environnement d’exécution Deno, consultez la documentation.

Installation

Installez l’environnement d’exécution Deno sur votre système en utilisant l’une des commandes ci-dessous. Notez qu’il existe plusieurs méthodes pour installer Deno – une liste complète des options d’installation est disponible ici.

Shell (Mac, Linux) :

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows) :

irm https://deno.land/install.ps1 | iex

Homebrew (Mac) :

brew install deno

Chocolatey (Windows) :

choco install deno

WinGet (Windows) :

winget install --id=DenoLand.Deno

Votre premier programme Deno

Deno peut être utilisé pour de nombreuses applications différentes, mais il est le plus souvent utilisé pour créer des serveurs web. Créez un fichier appelé server.ts et ajoutez le code TypeScript suivant :

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Exécutez votre serveur avec la commande suivante :

deno run --allow-net server.ts

Cela devrait lancer un serveur web local sur http://localhost:8000.

Pour en savoir plus sur l’écriture et l’exécution de programmes Deno, consultez la documentation.
