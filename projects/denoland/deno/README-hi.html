Deno

Deno (/ˈdiːnoʊ/, उच्चारण डी-नो) एक JavaScript, TypeScript, और WebAssembly रनटाइम है जिसमें डिफ़ॉल्ट रूप से सुरक्षा होती है और यह डेवलपर्स के लिए शानदार अनुभव प्रदान करता है। यह V8, Rust, और Tokio पर आधारित है।

Deno रनटाइम के बारे में अधिक जानें डॉक्युमेंटेशन में।

स्थापना

अपने सिस्टम पर Deno रनटाइम स्थापित करने के लिए नीचे दिए गए किसी एक कमांड का उपयोग करें। ध्यान दें कि Deno को स्थापित करने के कई तरीके हैं - स्थापना विकल्पों की पूरी सूची यहाँ पाई जा सकती है।

शेल (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

स्रोत से निर्माण और स्थापना

स्रोत से Deno बनाने के लिए पूरी निर्देशावली यहाँ पाई जा सकती है।

आपका पहला Deno प्रोग्राम

Deno का उपयोग कई विभिन्न अनुप्रयोगों के लिए किया जा सकता है, लेकिन आमतौर पर इसका उपयोग वेब सर्वर बनाने के लिए किया जाता है। एक फ़ाइल बनाएं जिसका नाम server.ts हो और उसमें निम्नलिखित TypeScript कोड जोड़ें:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

अपने सर्वर को निम्नलिखित कमांड से चलाएँ:

deno run --allow-net server.ts

इससे आपके लोकल सिस्टम पर एक वेब सर्वर http://localhost:8000 पर शुरू हो जाना चाहिए।

Deno प्रोग्राम लिखने और चलाने के बारे में और जानें डॉक्स में।
