Deno

Deno (/ˈdiːnoʊ/, pronunciado dee-no) é um ambiente de execução para JavaScript, TypeScript e WebAssembly com padrões de segurança e uma ótima experiência para desenvolvedores. Ele é construído sobre V8, Rust e Tokio.

Saiba mais sobre o runtime do Deno na documentação.

Instalação

Instale o runtime Deno em seu sistema usando um dos comandos abaixo. Note que existem diversas formas de instalar o Deno - uma lista abrangente de opções de instalação pode ser encontrada aqui.

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

Compilar e instalar a partir do código-fonte

Instruções completas para compilar o Deno a partir do código-fonte podem ser encontradas aqui.

Seu primeiro programa em Deno

O Deno pode ser utilizado para diversas aplicações, mas é mais comumente usado para construir servidores web. Crie um arquivo chamado server.ts e inclua o seguinte código TypeScript:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Execute seu servidor com o seguinte comando:

deno run --allow-net server.ts

Isso deve iniciar um servidor web local em http://localhost:8000.

Saiba mais sobre como escrever e executar programas Deno na documentação.
