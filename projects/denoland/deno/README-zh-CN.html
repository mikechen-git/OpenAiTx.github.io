Deno

Deno (/ˈdiːnoʊ/，发音为 dee-no) 是一个默认安全、开发体验极佳的 JavaScript、TypeScript 和 WebAssembly 运行时。它基于 V8、Rust 和 Tokio 构建。

在文档中了解更多关于 Deno 运行时的信息。

安装

使用以下命令之一在您的系统上安装 Deno 运行时。请注意，安装 Deno 的方式有多种——完整的安装选项列表请见 这里。

Shell（Mac，Linux）:

curl -fsSL https://deno.land/install.sh | sh

PowerShell（Windows）:

irm https://deno.land/install.ps1 | iex

Homebrew（Mac）:

brew install deno

Chocolatey（Windows）:

choco install deno

WinGet（Windows）:

winget install --id=DenoLand.Deno

从源码构建与安装

关于从源码构建 Deno 的完整说明请见 这里。

你的第一个 Deno 程序

Deno 可用于多种不同的应用，但最常见的是用于构建 Web 服务器。创建一个名为 server.ts 的文件，并包含以下 TypeScript 代码：

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

使用以下命令运行你的服务器：

deno run --allow-net server.ts

这将在本地启动一个 Web 服务器，访问 http://localhost:8000。
