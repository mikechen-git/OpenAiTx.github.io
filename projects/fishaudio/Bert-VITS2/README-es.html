【Presentación del proyecto】

El nuevo TTS autorregresivo bajo FishAudio Fish-Speech ya está disponible, con resultados al nivel del estado del arte abierto actual, y en mantenimiento continuo, se recomienda usar este proyecto como sustituto de BV2/GSV. Este proyecto no recibirá mantenimiento a corto plazo.

Está estrictamente prohibido usarlo para cualquier propósito político.
