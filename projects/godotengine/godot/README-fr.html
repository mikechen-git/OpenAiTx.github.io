Godot Engine

Moteur de jeu multiplateforme 2D et 3D

Godot Engine est un moteur de jeu multiplateforme riche en fonctionnalités, permettant de créer des jeux 2D et 3D depuis une interface unifiée. Il fournit un ensemble complet d'outils courants, afin que les utilisateurs puissent se concentrer sur la création de jeux sans avoir à réinventer la roue. Les jeux peuvent être exportés en un clic vers de nombreuses plateformes, y compris les principales plateformes de bureau (Linux, macOS, Windows), les plateformes mobiles (Android, iOS), ainsi que les plateformes Web et les consoles.

Libre, open source et dirigé par la communauté

Godot est totalement gratuit et open source sous la licence très permissive MIT. Aucune condition, aucune redevance, rien. Les jeux des utilisateurs leur appartiennent, jusqu'à la dernière ligne de code du moteur. Le développement de Godot est entièrement indépendant et dirigé par la communauté, permettant aux utilisateurs de contribuer à façonner leur moteur selon leurs attentes. Il est soutenu par la Godot Foundation, une organisation à but non lucratif.

Avant d’être open source en février 2014, Godot avait été développé pendant plusieurs années par Juan Linietsky et Ariel Manzur (tous deux assurant encore la maintenance du projet) en tant que moteur interne, utilisé pour publier plusieurs titres en prestation de service.

Communauté et contribution

Godot n’est pas seulement un moteur, c’est aussi une communauté croissante d’utilisateurs et de développeurs du moteur. Les principaux canaux communautaires sont listés sur la page d’accueil.

La meilleure façon de contacter les développeurs principaux du moteur est de rejoindre le Godot Contributors Chat.

Pour commencer à contribuer au projet, consultez le guide de contribution. Ce document inclut également des directives pour signaler les bugs.

Documentation et démonstrations

La documentation officielle est hébergée sur Read the Docs. Elle est maintenue par la communauté Godot dans son propre dépôt GitHub.

La référence des classes est également accessible depuis l’éditeur Godot.

Nous maintenons également des démonstrations officielles dans leur propre dépôt GitHub ainsi qu’une liste de ressources communautaires Godot remarquables.

Il existe également de nombreuses ressources d’apprentissage proposées par la communauté, telles que des tutoriels écrits et vidéo, des démonstrations, etc. Consultez les canaux communautaires pour plus d’informations.
