موتور Godot

موتور بازی‌سازی بین‌پلتفرمی برای ۲بعدی و ۳بعدی

موتور Godot یک موتور بازی‌سازی بین‌پلتفرمی و مجهز به امکانات فراوان برای ساخت بازی‌های ۲بعدی و ۳بعدی از طریق یک رابط کاربری یکپارچه است. این موتور مجموعه‌ای کامل از ابزارهای رایج را فراهم می‌کند تا کاربران بتوانند روی ساخت بازی تمرکز کنند، بدون اینکه نیاز به اختراع دوباره چرخ داشته باشند. بازی‌ها می‌توانند تنها با یک کلیک به چندین پلتفرم مختلف، از جمله پلتفرم‌های دسکتاپ اصلی (لینوکس، مک‌اواس، ویندوز)، پلتفرم‌های موبایل (اندروید، iOS)، همچنین پلتفرم‌های مبتنی بر وب و کنسول‌ها صادر شوند.

رایگان، متن‌باز و مبتنی بر جامعه

Godot کاملاً رایگان و متن‌باز تحت مجوز بسیار آزاد MIT است. هیچ شرطی وجود ندارد، بدون حق امتیاز، هیچ محدودیتی. بازی‌های کاربران کاملاً متعلق به آن‌هاست، حتی تا آخرین خط کد موتور. توسعه Godot کاملاً مستقل و مبتنی بر جامعه است و به کاربران این قدرت را می‌دهد که موتور خود را مطابق با انتظاراتشان شکل دهند. این پروژه توسط بنیاد Godot غیرانتفاعی پشتیبانی می‌شود.

قبل از متن‌باز شدن در فوریه ۲۰۱۴، Godot توسط خوان لینیتسکی و آریل مانزور (که هر دو هنوز پروژه را نگهداری می‌کنند) برای چندین سال به عنوان یک موتور داخلی توسعه یافته بود و برای انتشار چندین عنوان سفارشی مورد استفاده قرار گرفته بود.

راهنمای رسمی را برای دستورالعمل‌های کامپایل روی هر پلتفرم پشتیبانی‌شده مشاهده کنید.

جامعه و مشارکت

Godot تنها یک موتور نیست، بلکه یک جامعه در حال رشد از کاربران و توسعه‌دهندگان موتور است. کانال‌های اصلی جامعه در صفحه اصلی فهرست شده‌اند.

بهترین راه ارتباط با توسعه‌دهندگان اصلی موتور، پیوستن به گپ مشارکت‌کنندگان Godot است.

برای شروع مشارکت در پروژه، راهنمای مشارکت را ببینید. این سند همچنین شامل دستورالعمل‌هایی برای گزارش اشکال‌ها است.

مستندات و نمونه‌ها

مستندات رسمی در Read the Docs میزبانی می‌شود. این مستندات توسط جامعه Godot در مخزن GitHub مخصوص خود نگهداری می‌شود.

مرجع کلاس‌ها نیز از داخل ویرایشگر Godot قابل دسترسی است.

ما همچنین نمونه‌های رسمی را در مخزن GitHub مخصوص آن‌ها و همچنین لیستی از منابع عالی جامعه Godot نگهداری می‌کنیم.

همچنین تعداد زیادی منابع آموزشی دیگر توسط جامعه ارائه شده است، مانند آموزش‌های متنی و ویدیویی، نمونه‌ها و غیره. برای اطلاعات بیشتر به کانال‌های جامعه مراجعه کنید.
