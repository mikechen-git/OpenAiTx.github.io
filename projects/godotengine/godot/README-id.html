Godot Engine

Mesin permainan lintas platform 2D dan 3D

Godot Engine adalah mesin permainan lintas platform yang kaya fitur untuk membuat game 2D dan 3D dari satu antarmuka terpadu. Mesin ini menyediakan seperangkat alat umum yang komprehensif, sehingga pengguna dapat fokus membuat game tanpa harus menciptakan ulang roda. Game dapat diekspor dengan satu klik ke berbagai platform, termasuk platform desktop utama (Linux, macOS, Windows), platform mobile (Android, iOS), serta platform berbasis web dan konsol.

Gratis, sumber terbuka, dan didorong oleh komunitas

Godot sepenuhnya gratis dan sumber terbuka di bawah lisensi MIT yang sangat permisif. Tanpa syarat, tanpa royalti, tidak ada apa-apa. Game milik pengguna sepenuhnya, hingga baris kode mesin terakhir. Pengembangan Godot sepenuhnya independen dan didorong oleh komunitas, memberdayakan pengguna untuk membantu membentuk mesin sesuai harapan mereka. Proyek ini didukung oleh Godot Foundation yang nirlaba.

Sebelum menjadi sumber terbuka pada Februari 2014, Godot telah dikembangkan oleh Juan Linietsky dan Ariel Manzur (keduanya masih menjadi pemelihara proyek) selama beberapa tahun sebagai mesin internal, yang digunakan untuk menerbitkan beberapa judul kerja-untuk-sewa.

Komunitas dan kontribusi

Godot bukan hanya sebuah mesin, tetapi juga komunitas pengguna dan pengembang mesin yang terus berkembang. Kanal komunitas utama terdaftar di halaman utama.

Cara terbaik untuk berkomunikasi dengan pengembang inti mesin adalah dengan bergabung di Godot Contributors Chat.

Untuk mulai berkontribusi pada proyek ini, lihat panduan kontribusi. Dokumen ini juga mencakup pedoman untuk melaporkan bug.

Dokumentasi dan demo

Dokumentasi resmi dihosting di Read the Docs. Dokumentasi ini dikelola oleh komunitas Godot di repository GitHub.

Referensi kelas juga dapat diakses dari editor Godot.

Kami juga memelihara demo resmi di repository GitHub serta daftar sumber daya komunitas Godot yang keren.

Ada juga sejumlah sumber belajar lain yang disediakan oleh komunitas, seperti tutorial teks dan video, demo, dll. Konsultasikan kanal komunitas untuk informasi lebih lanjut.
