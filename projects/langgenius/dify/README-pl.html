Dify Cloud · Samodzielna instalacja · Dokumentacja · Przegląd edycji Dify

Dify to otwartoźródłowa platforma do tworzenia aplikacji LLM. Jej intuicyjny interfejs łączy agentowe przepływy pracy AI, pipeline RAG, możliwości agentów, zarządzanie modelami, funkcje obserwowalności i wiele więcej, umożliwiając szybkie przechodzenie od prototypu do produkcji.

Szybki start

Przed instalacją Dify upewnij się, że Twój komputer spełnia poniższe minimalne wymagania systemowe:

  • CPU >= 2 rdzenie
  • RAM >= 4 GiB

Najłatwiejszym sposobem uruchomienia serwera Dify jest użycie docker compose. Przed uruchomieniem Dify poniższymi poleceniami upewnij się, że masz zainstalowane Docker oraz Docker Compose:

cd dify
cd docker
cp .env.example .env
docker compose up -d

Po uruchomieniu możesz uzyskać dostęp do panelu Dify w przeglądarce pod adresem http://localhost/install i rozpocząć proces inicjalizacji.

Kluczowe funkcje

1. Przepływ pracy (Workflow): Buduj i testuj zaawansowane przepływy pracy AI na wizualnym płótnie, wykorzystując wszystkie poniższe funkcje i nie tylko.

2. Kompleksowe wsparcie modeli: Bezproblemowa integracja z setkami zamkniętych i otwartoźródłowych modeli LLM od dziesiątek dostawców inferencji oraz rozwiązań samohostowanych, obejmująca GPT, Mistral, Llama3 i wszelkie modele zgodne z API OpenAI. Pełna lista obsługiwanych dostawców modeli dostępna jest tutaj.

3. Prompt IDE: Intuicyjny interfejs do tworzenia promptów, porównywania wydajności modeli oraz dodawania dodatkowych funkcji, takich jak zamiana tekstu na mowę w aplikacji czatowej.

4. Pipeline RAG: Rozbudowane możliwości RAG, obejmujące wszystko od wczytywania dokumentów po ich wyszukiwanie, z gotowym wsparciem dla ekstrakcji tekstu z PDF, PPT i innych popularnych formatów dokumentów.

5. Możliwości agentów: Możesz definiować agentów na bazie LLM Function Calling lub ReAct oraz dodawać gotowe lub własne narzędzia dla agenta. Dify oferuje ponad 50 wbudowanych narzędzi dla agentów AI, takich jak Google Search, DALL·E, Stable Diffusion i WolframAlpha.

6. LLMOps: Monitoruj i analizuj logi oraz wydajność aplikacji w czasie. Możesz stale ulepszać prompt'y, zbiory danych i modele na podstawie danych produkcyjnych i adnotacji.

7. Backend-as-a-Service: Wszystkie funkcje Dify posiadają odpowiadające im API, dzięki czemu możesz łatwo zintegrować Dify ze swoją logiką biznesową.

Korzystanie z Dify

  • Chmura
    Udostępniamy Dify Cloud dla każdego, kto chce wypróbować bez żadnej konfiguracji. Oferuje wszystkie możliwości wersji samodzielnie wdrażanej oraz zawiera 200 darmowych wywołań GPT-4 w planie sandbox.

  • Samodzielne wdrażanie Dify Community Edition
     Szybko uruchom Dify w swoim środowisku korzystając z tego przewodnika startowego. Skorzystaj z naszej dokumentacji po dalsze instrukcje i szczegółowe informacje.

  • Dify dla przedsiębiorstw/organizacji
     Oferujemy dodatkowe funkcje ukierunkowane na potrzeby firm. Zadaj nam pytanie przez tego chatbota lub wyślij do nas e-mail, aby omówić potrzeby Twojej organizacji.

    Dla startupów i małych firm korzystających z AWS, sprawdź Dify Premium na AWS Marketplace i wdrażaj do własnego AWS VPC jednym kliknięciem. To przystępna cenowo oferta AMI z opcją tworzenia aplikacji z własnym logo i brandingiem.

Zaawansowana konfiguracja

Jeśli chcesz dostosować konfigurację, zapoznaj się z komentarzami w naszym pliku .env.example i zaktualizuj odpowiednie wartości w swoim pliku .env. Dodatkowo możesz potrzebować dokonać zmian w samym pliku docker-compose.yaml, np. zmienić wersje obrazów, mapowania portów lub montowania wolumenów, w zależności od środowiska wdrożeniowego i wymagań. Po wprowadzeniu zmian ponownie uruchom docker-compose up -d. Pełną listę dostępnych zmiennych środowiskowych znajdziesz tutaj.

Jeśli chcesz skonfigurować środowisko wysokiej dostępności, społeczność przygotowała Helm Chart'y i pliki YAML umożliwiające wdrożenie Dify na Kubernetes.
