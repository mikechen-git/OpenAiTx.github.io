ClickHouse® ist ein Open-Source spaltenorientiertes Datenbankmanagementsystem, mit dem analytische Datenberichte in Echtzeit erstellt werden können.

So installieren Sie (Linux, macOS, FreeBSD)

curl https://clickhouse.com/ | sh
