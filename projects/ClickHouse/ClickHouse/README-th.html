ClickHouse® คือระบบจัดการฐานข้อมูลเชิงคอลัมน์แบบโอเพ่นซอร์สที่ช่วยให้สามารถสร้างรายงานข้อมูลเชิงวิเคราะห์แบบเรียลไทม์ได้

วิธีการติดตั้ง (Linux, macOS, FreeBSD)

curl https://clickhouse.com/ | sh

ลิงก์ที่มีประโยชน์

  • เว็บไซต์ทางการ มีภาพรวมแบบรวดเร็วเกี่ยวกับ ClickHouse บนหน้าแรก
  • ClickHouse Cloud ClickHouse ในรูปแบบบริการ คลาวด์ โดยทีมผู้สร้างและผู้ดูแล
  • บทเรียน สอนการตั้งค่าและคิวรีคลัสเตอร์ ClickHouse ขนาดเล็ก
  • เอกสารประกอบ ให้ข้อมูลเชิงลึกเพิ่มเติม
  • ช่อง YouTube มีเนื้อหาเกี่ยวกับ ClickHouse ในรูปแบบวิดีโอจำนวนมาก
  • Slack และ Telegram สำหรับพูดคุยกับผู้ใช้ ClickHouse แบบเรียลไทม์
  • บล็อก มีบทความเกี่ยวกับ ClickHouse, ข่าวสาร และรายงานกิจกรรมต่าง ๆ
  • Bluesky และ X สำหรับข่าวสั้น
  • Code Browser (github.dev) พร้อมไฮไลต์ซินแทกซ์ ขับเคลื่อนโดย github.dev
  • ติดต่อเรา สำหรับสอบถามหรือขอความช่วยเหลือ
