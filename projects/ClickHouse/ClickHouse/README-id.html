ClickHouse® adalah sistem manajemen basis data kolom sumber terbuka yang memungkinkan pembuatan laporan data analitik secara real-time.

Cara Instalasi (Linux, macOS, FreeBSD)

curl https://clickhouse.com/ | sh

  • ClickHouse Cloud ClickHouse sebagai layanan, dibuat oleh para pencipta dan pengelola.
  • Tutorial menunjukkan cara mengatur dan melakukan query pada cluster ClickHouse kecil.
