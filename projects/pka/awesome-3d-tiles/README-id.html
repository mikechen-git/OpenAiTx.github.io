Awesome 3D Tiles

Daftar pilihan perangkat lunak sumber terbuka, pustaka, alat, dan sumber daya terbaik untuk 3D tiles.

Kontribusi sangat diterima. Tambahkan tautan melalui pull requests atau buat issue untuk memulai diskusi.

3D Tiles adalah spesifikasi terbuka untuk berbagi, memvisualisasikan, menggabungkan, dan berinteraksi dengan konten geospasial 3D heterogen dalam jumlah besar di aplikasi desktop, web, dan mobile.

3D Tiles adalah standar komunitas OGC.

Penampil (Viewers)

  • CesiumJS - Pustaka JavaScript untuk membuat globe 3D dan peta 2D di peramban web.
  • 3DCityDB-Web-Map-Client - Penampil berbasis Cesium untuk CityGML dan 3D Tiles.
  • loaders.gl - Parser dan encoder untuk banyak format 3D, geospasial, dan tabular utama.
  • deck.gl - Framework bertenaga WebGL untuk analisis data eksploratif visual pada dataset besar.
  • 3d-tiles-renderer - Renderer berbasis Three.js untuk 3D Tiles.
  • three-loader-3dtiles - Modul loader Three.js untuk menangani OGC 3D Tiles.
  • threedtiles - Penampil 3DTiles untuk three.js.
  • mapbox-3dtiles - Layer kustom Mapbox GL JS untuk 3D Tiles.
  • A-Frame component - Komponen A-Frame yang menggunakan 3D-Tiles.
  • iTowns - Framework JS/WebGL berbasis Three.js.
  • giro3d - Framework JS/WebGL berbasis Three.js (penerus iTown).
  • UD-Viz - Visualisasi Data Perkotaan. Framework untuk membuat aplikasi web guna memvisualisasikan dan berinteraksi dengan data kota 3D geospasial (berbasis iTowns/Tree.js).
  • vsgCs - pustaka untuk menggunakan 3D Tiles dan konten geospasial lain dalam aplikasi Vulkan Scene Graph (VSG).

Integrasi engine game, desktop GIS, dan alat pemodelan 3D (Sumber Terbuka)

  • QGIS - Sistem informasi geografis desktop multi-platform (sejak versi 3.34).
  • 3D Tiles for Godot - Ekstensi Godot 4 yang mengintegrasikan kemampuan Cesium 3D Tiles ke dalam Godot Engine.
  • Open 3D Engine (O3DE) - Cesium untuk O3DE.
  • blender-3d-tiler - Alat Blender untuk membagi model 3D menjadi tile.

Pembuatan Tile (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - Pustaka C++ untuk streaming 3D Tiles, pemrosesan glTF.
  • 3D Tiles Tools - Alat dan utilitas untuk mengonversi, mengoptimalkan, memproses, dan menganalisis data 3D Tiles.
  • glTF Pipeline - Alat Javascript untuk konversi dan optimasi glTF / GLB.

Pembuatan Tile (Komunitas)

  • py3dtiles - Membaca dan menulis pnts dan b3dm melalui API atau CLI.
  • Python 3DTiles Tilers - Alat dan pustaka Python yang memungkinkan pembuatan tileset 3D Tiles dari berbagai format geometris seperti OBJ, GeoJSON, IFC atau CityGML melalui database 3dCityDB.
  • pg2b3dm - Konversi geometri 3D PostGIS ke tile b3dm.
  • mago 3DTiler - mago 3DTiler mengonversi berbagai data spasial ke OGC 3D Tiles.
  • PLATEAU GIS Converter - Konversi model kota 3D (CityGML) ke berbagai format geospasial, termasuk 3D Tiles, MVT, dan GeoPackage.
  • Obj2Tiles - Mengonversi file OBJ ke OGC 3D tiles.
  • 3dtiles - Mengonversi Osgb (OpenSceneGraph Binary), Esri Shapefile, dan file Fbx ke 3D Tiles.
  • gltf-to-3d-tiles - Mengonversi model glTF ke format Glb, b3dm, atau 3d tiles.
  • citygml-to-3dtiles - Konversi dari CityGML ke Cesium 3D Tiles.
  • vts23dtiles - Mengonversi tileset VTS ke 3D Tileset.
  • 3D City Database - Basis data geo untuk menyimpan, merepresentasikan, dan mengelola model kota virtual 3D.
  • Cesium Point Cloud Generator - XYZ → 3D Point Cloud Tiles.
  • loaders.gl tile-converter - Mengonversi I3S ke 3D Tiles.
  • tyler - Membagi model kota 3D yang dikodekan dalam CityJSON menjadi tile.

Alat glTF

  • glTF-Transform - glTF-Transform mendukung pembacaan, pengeditan, dan penulisan model 3D dalam format glTF 2.0.
  • Open Asset Import Library (assimp) - Pustaka dan konverter untuk 40+ format file 3D ke glTF dan lainnya (format file).
  • Blender exporter - Eksporter Blender glTF 2.0.

Terrain

Quantized Mesh adalah format untuk mengkode mesh terrain agar rendering terrain di sisi klien lebih efisien. File semacam ini didukung di Cesium dan deck.gl.

Lihat awesome-quantized-mesh-tiles untuk sumber daya lainnya.
