Awesome 3D Tiles

Danh sách tuyển chọn các phần mềm mã nguồn mở tuyệt vời, thư viện, công cụ và tài nguyên cho 3D tiles.

Hoan nghênh đóng góp. Thêm liên kết thông qua pull requests hoặc tạo một issue để bắt đầu thảo luận.

3D Tiles là một đặc tả mở để chia sẻ, trực quan hóa, kết hợp và tương tác với dữ liệu không gian 3D lớn, không đồng nhất trên các ứng dụng máy tính để bàn, web và di động.

3D Tiles là một chuẩn cộng đồng OGC.

Trình xem

  • CesiumJS - Thư viện JavaScript để tạo quả địa cầu 3D và bản đồ 2D trong trình duyệt web.
  • 3DCityDB-Web-Map-Client - Trình xem dựa trên Cesium cho CityGML và 3D Tiles.
  • loaders.gl - Bộ phân tích cú pháp và bộ mã hóa cho nhiều định dạng 3D, địa lý và dạng bảng chính.
  • deck.gl - Khung WebGL cho phân tích dữ liệu khám phá trực quan với tập dữ liệu lớn.
  • 3d-tiles-renderer - Bộ kết xuất dựa trên Three.js cho 3D Tiles.
  • three-loader-3dtiles - Mô-đun tải Three.js để xử lý OGC 3D Tiles.
  • threedtiles - Trình xem 3DTiles cho three.js.
  • mapbox-3dtiles - Lớp tùy chỉnh Mapbox GL JS cho 3D Tiles.
  • A-Frame component - Thành phần A-Frame sử dụng 3D-Tiles.
  • iTowns - Khung JS/WebGL dựa trên Three.js.
  • giro3d - Khung JS/WebGL dựa trên Three.js (kế thừa của iTown).
  • UD-Viz - Trực quan hóa dữ liệu đô thị. Khung tạo ứng dụng web để trực quan hóa và tương tác với dữ liệu không gian 3D đô thị (dựa trên iTowns/Tree.js).
  • vsgCs - Thư viện sử dụng 3D Tiles và nội dung không gian địa lý khác trong ứng dụng Vulkan Scene Graph (VSG)

Tích hợp game engine, desktop GIS và công cụ mô hình hóa 3D (Mã nguồn mở)

  • QGIS - Hệ thống thông tin địa lý trên máy tính đa nền tảng (từ phiên bản 3.34).
  • 3D Tiles for Godot - Phần mở rộng Godot 4 tích hợp khả năng 3D Tiles của Cesium vào Godot Engine
  • Open 3D Engine (O3DE) - Cesium cho O3DE
  • blender-3d-tiler - Công cụ Blender để chia nhỏ mô hình 3D.

Tạo tile (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - Thư viện C++ cho truyền tải 3D Tiles, xử lý glTF.
  • 3D Tiles Tools - Công cụ và tiện ích để chuyển đổi, tối ưu, xử lý và phân tích dữ liệu 3D Tiles.
  • glTF Pipeline - Công cụ Javascript cho chuyển đổi và tối ưu glTF / GLB.

Tạo tile (Cộng đồng)

  • py3dtiles - Đọc và ghi pnts và b3dm qua API hoặc CLI.
  • Python 3DTiles Tilers - Công cụ và thư viện Python cho phép xây dựng bộ tile 3D Tiles từ nhiều định dạng hình học như OBJ, GeoJSON, IFC hoặc CityGML thông qua cơ sở dữ liệu 3dCityDB.
  • pg2b3dm - Chuyển đổi hình học 3D PostGIS thành tile b3dm.
  • mago 3DTiler - mago 3DTiler chuyển đổi nhiều dữ liệu không gian thành OGC 3D Tiles.
  • PLATEAU GIS Converter - Chuyển đổi mô hình thành phố 3D (CityGML) sang nhiều định dạng địa lý, bao gồm 3D Tiles, MVT và GeoPackage.
  • Obj2Tiles - Chuyển đổi tệp OBJ sang OGC 3D tiles.
  • 3dtiles - Chuyển đổi tệp Osgb(OpenSceneGraph Binary), Esri Shapefile và Fbx thành 3D Tiles.
  • gltf-to-3d-tiles - Chuyển đổi mô hình glTF sang định dạng Glb, b3dm hoặc 3d tiles.
  • citygml-to-3dtiles - Chuyển đổi từ CityGML sang Cesium 3D Tiles.
  • vts23dtiles - Chuyển đổi bộ tile VTS thành bộ tile 3D Tileset.
  • 3D City Database - Cơ sở dữ liệu địa lý để lưu trữ, biểu diễn và quản lý mô hình thành phố 3D ảo.
  • Cesium Point Cloud Generator - XYZ → 3D Point Cloud Tiles.
  • loaders.gl tile-converter - Chuyển đổi I3S sang 3D Tiles.
  • tyler - Chia nhỏ mô hình thành phố 3D được mã hóa bằng CityJSON

Công cụ glTF

  • glTF-Transform - glTF-Transform hỗ trợ đọc, chỉnh sửa và ghi mô hình 3D định dạng glTF 2.0.
  • Open Asset Import Library (assimp) - Thư viện và công cụ chuyển đổi hơn 40 định dạng tệp 3D sang glTF và nhiều hơn nữa (các định dạng tệp).
  • Blender exporter - Công cụ xuất glTF 2.0 của Blender.

Địa hình

Quantized Mesh là một định dạng mã hóa lưới địa hình cho kết xuất địa hình hiệu quả phía client. Các tệp này được hỗ trợ trong Cesium và deck.gl.
