Awesome 3D Tiles

Una lista curata di eccellente software open source, librerie, strumenti e risorse per 3D Tiles.

Contributi benvenuti. Aggiungi link tramite pull requests o crea una issue per avviare una discussione.

3D Tiles è una specifica aperta per la condivisione, visualizzazione, fusione e interazione con contenuti geospaziali 3D eterogenei e massivi su applicazioni desktop, web e mobile.

3D Tiles è uno standard di comunità OGC.

Visualizzatori

  • CesiumJS - Libreria JavaScript per creare globi 3D e mappe 2D in un browser web.
  • 3DCityDB-Web-Map-Client - Visualizzatore basato su Cesium per CityGML e 3D Tiles.
  • loaders.gl - Parser ed encoder per numerosi principali formati 3D, geospaziali e tabellari.
  • deck.gl - Framework WebGL per l'analisi esplorativa di grandi set di dati.
  • 3d-tiles-renderer - Renderer basato su Three.js per 3D Tiles.
  • three-loader-3dtiles - Modulo loader Three.js per gestire OGC 3D Tiles.
  • threedtiles - Visualizzatore 3DTiles per three.js.
  • mapbox-3dtiles - Layer personalizzato Mapbox GL JS per 3D Tiles.
  • A-Frame component - Componente A-Frame che utilizza 3D-Tiles.
  • iTowns - Framework JS/WebGL basato su Three.js.
  • giro3d - Framework JS/WebGL basato su Three.js (successore di iTown).
  • UD-Viz - Visualizzazione di dati urbani. Framework per creare applicazioni web per visualizzare e interagire con dati urbani geospaziali 3D (basato su iTowns/Tree.js).
  • vsgCs - Libreria per utilizzare 3D Tiles e altri contenuti geospaziali all'interno di una applicazione Vulkan Scene Graph (VSG).

Integrazioni con game engine, GIS desktop e strumenti di modellazione 3D (Open source)

  • QGIS - Sistema informativo geografico desktop multipiattaforma (dalla versione 3.34).
  • 3D Tiles for Godot - Estensione Godot 4 che integra le capacità Cesium 3D Tiles in Godot Engine.
  • Open 3D Engine (O3DE) - Cesium per O3DE.
  • blender-3d-tiler - Strumento Blender per la suddivisione in tile di modelli 3D.

Creazione tile (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - Libreria C++ per streaming 3D Tiles, elaborazione glTF.
  • 3D Tiles Tools - Strumenti e utilità per convertire, ottimizzare, processare e analizzare dati 3D Tiles.
  • glTF Pipeline - Strumenti JavaScript per conversione e ottimizzazione glTF / GLB.

Creazione tile (Comunità)

  • py3dtiles - Leggi e scrivi pnts e b3dm via API o CLI.
  • Python 3DTiles Tilers - Strumento e libreria Python che consente di creare tileset 3D Tiles da vari formati geometrici, ad esempio OBJ, GeoJSON, IFC o CityGML tramite database 3dCityDB.
  • pg2b3dm - Conversione di geometrie PostGIS 3D in tile b3dm.
  • mago 3DTiler - mago 3DTiler converte diversi dati spaziali in OGC 3D Tiles.
  • PLATEAU GIS Converter - Converte modelli di città 3D (CityGML) in vari formati geospaziali, inclusi 3D Tiles, MVT e GeoPackage.
  • Obj2Tiles - Converte file OBJ in OGC 3D tiles.
  • 3dtiles - Converte Osgb (OpenSceneGraph Binary), Esri Shapefile e file Fbx in 3D Tiles.
  • gltf-to-3d-tiles - Converte modelli glTF in formato Glb, b3dm o 3d tiles.
  • citygml-to-3dtiles - Conversione da CityGML a Cesium 3D Tiles.
  • vts23dtiles - Converte VTS tileset in tileset 3D Tiles.
  • 3D City Database - Geo database per archiviare, rappresentare e gestire modelli virtuali 3D di città.
  • Cesium Point Cloud Generator - XYZ → 3D Point Cloud Tiles.
  • loaders.gl tile-converter - Converte I3S in 3D Tiles.
  • tyler - Suddivisione in tile di modelli 3D di città codificati in CityJSON.

Strumenti glTF

  • glTF-Transform - glTF-Transform supporta lettura, modifica e scrittura di modelli 3D in formato glTF 2.0.
  • Open Asset Import Library (assimp) - Libreria e convertitore per oltre 40 formati di file 3D in glTF e altri (formati supportati).
  • Blender exporter - Esportatore Blender glTF 2.0.

Terreno

Quantized Mesh è un formato per codificare mesh di terreno per un efficiente rendering lato client. Tali file sono supportati in Cesium e deck.gl.

Consulta awesome-quantized-mesh-tiles per ulteriori risorse.
