بلاطات ثلاثية الأبعاد رائعة

قائمة منسقة من البرمجيات مفتوحة المصدر والمكتبات والأدوات والموارد الرائعة لبلاطات ثلاثية الأبعاد.

المساهمات مرحب بها. أضف الروابط من خلال طلبات السحب أو أنشئ إصدارًا لبدء النقاش.

بلاطات ثلاثية الأبعاد هي مواصفة مفتوحة لمشاركة وعرض ودمج والتفاعل مع محتوى جغرافي ثلاثي الأبعاد هائل غير متجانس عبر تطبيقات سطح المكتب والويب والهاتف المحمول.

بلاطات ثلاثية الأبعاد هي معيار مجتمع OGC.

العارضون

  • CesiumJS - مكتبة جافاسكريبت لإنشاء كرات أرضية ثلاثية الأبعاد وخرائط ثنائية الأبعاد في متصفح الويب.
  • 3DCityDB-Web-Map-Client - عارض قائم على Cesium لـ CityGML وبلاطات ثلاثية الأبعاد.
  • loaders.gl - محللات ومشفّرات للعديد من تنسيقات البيانات الجغرافية ثلاثية الأبعاد والجداولية الرئيسية.
  • deck.gl - إطار عمل WebGL لتحليل البيانات الاستكشافية البصرية لمجموعات البيانات الكبيرة.
  • 3d-tiles-renderer - عارض قائم على Three.js لبلاطات ثلاثية الأبعاد.
  • three-loader-3dtiles - وحدة تحميل لـ Three.js لمعالجة بلاطات OGC ثلاثية الأبعاد.
  • threedtiles - عارض بلاطات ثلاثية الأبعاد لـ three.js.
  • mapbox-3dtiles - طبقة مخصصة لـ Mapbox GL JS لبلاطات ثلاثية الأبعاد.
  • A-Frame component - مكون A-Frame باستخدام بلاطات ثلاثية الأبعاد.
  • iTowns - إطار عمل JS/WebGL قائم على Three.js.
  • giro3d - إطار عمل JS/WebGL قائم على Three.js (خليفة iTown).
  • UD-Viz - تصور بيانات المدن. إطار عمل لإنشاء تطبيقات ويب لتصور والتفاعل مع بيانات المدن الجغرافية ثلاثية الأبعاد (قائم على iTowns/Tree.js).
  • vsgCs - مكتبة لاستخدام بلاطات ثلاثية الأبعاد ومحتوى جغرافي آخر ضمن تطبيق Vulkan Scene Graph (VSG).

تكامل محركات الألعاب، نظم المعلومات الجغرافية المكتبية وأدوات النمذجة ثلاثية الأبعاد (مفتوحة المصدر)

  • QGIS - نظام معلومات جغرافية مكتبي متعدد المنصات (منذ الإصدار 3.34).
  • 3D Tiles for Godot - إضافة لمحرك Godot 4 تدمج إمكانيات Cesium لبلاطات ثلاثية الأبعاد في محرك Godot.
  • Open 3D Engine (O3DE) - Cesium لـ O3DE.
  • blender-3d-tiler - أداة Blender لتقطيع النماذج ثلاثية الأبعاد.

إنشاء البلاطات (Cesium GS Inc)

  • Cesium Ion (خدمة SaaS)
  • CDB to 3D Tiles - تحويل OGC CDB إلى بلاطات ثلاثية الأبعاد.
  • Cesium Native - مكتبة ++C لبث بلاطات ثلاثية الأبعاد ومعالجة glTF.
  • 3D Tiles Tools - أدوات لتحويل وتحسين ومعالجة وتحليل بيانات بلاطات ثلاثية الأبعاد.
  • glTF Pipeline - أدوات جافاسكريبت لتحويل وتحسين glTF / GLB.

إنشاء البلاطات (المجتمع)

  • py3dtiles - قراءة وكتابة pnts وb3dm عبر API أو CLI.
  • Python 3DTiles Tilers - أداة ومكتبة بايثون لبناء مجموعات بلاطات ثلاثية الأبعاد من تنسيقات هندسية متعددة مثل OBJ، GeoJSON، IFC أو CityGML عبر قواعد بيانات 3dCityDB.
  • pg2b3dm - تحويل الهندسة ثلاثية الأبعاد في PostGIS إلى بلاطات b3dm.
  • mago 3DTiler - mago 3DTiler يحول بيانات مكانية متنوعة إلى بلاطات OGC ثلاثية الأبعاد.
  • PLATEAU GIS Converter - تحويل نماذج المدن ثلاثية الأبعاد (CityGML) إلى صيغ مكانية متنوعة، بما فيها بلاطات ثلاثية الأبعاد وMVT وGeoPackage.
  • Obj2Tiles - تحويل ملفات OBJ إلى بلاطات OGC ثلاثية الأبعاد.
  • 3dtiles - تحويل ملفات Osgb (OpenSceneGraph Binary)، Esri Shapefile وFbx إلى بلاطات ثلاثية الأبعاد.
  • gltf-to-3d-tiles - تحويل نموذج glTF إلى Glb أو b3dm أو بلاطات ثلاثية الأبعاد.
  • citygml-to-3dtiles - تحويل من CityGML إلى بلاطات Cesium ثلاثية الأبعاد.
  • vts23dtiles - تحويل مجموعات بلاطات VTS إلى بلاطات ثلاثية الأبعاد.
  • 3D City Database - قاعدة بيانات جغرافية لتخزين وتمثيل وإدارة نماذج المدن الافتراضية ثلاثية الأبعاد.
  • Cesium Point Cloud Generator - XYZ إلى بلاطات سحابة نقطية ثلاثية الأبعاد.
  • loaders.gl tile-converter - تحويل I3S إلى بلاطات ثلاثية الأبعاد.
  • tyler - تقطيع نماذج المدن ثلاثية الأبعاد المشفرة في CityJSON.

أدوات glTF

  • glTF-Transform - glTF-Transform يدعم قراءة وتحرير وكتابة النماذج ثلاثية الأبعاد بصيغة glTF 2.0.
  • Open Asset Import Library (assimp) - مكتبة ومحول لأكثر من 40 تنسيق ملف ثلاثي الأبعاد إلى glTF وأكثر (تنسيقات الملفات).
  • Blender exporter - مصدر تصدير Blender glTF 2.0.

التضاريس

Quantized Mesh هو تنسيق لترميز شبكات التضاريس من أجل عرض التضاريس بكفاءة على جانب العميل. هذه الملفات مدعومة في Cesium وdeck.gl.

راجع awesome-quantized-mesh-tiles لمزيد من الموارد.
