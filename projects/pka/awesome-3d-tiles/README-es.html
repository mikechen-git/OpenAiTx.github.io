Awesome 3D Tiles

Una lista seleccionada de software de código abierto, bibliotecas, herramientas y recursos impresionantes para 3D tiles.

Contribuciones bienvenidas. Agrega enlaces a través de pull requests o crea un issue para iniciar una discusión.

3D Tiles es una especificación abierta para compartir, visualizar, fusionar e interactuar con contenido geoespacial 3D masivo y heterogéneo en aplicaciones de escritorio, web y móviles.

3D Tiles es un estándar comunitario de OGC.

Visualizadores

  • CesiumJS - Biblioteca JavaScript para crear globos terráqueos 3D y mapas 2D en un navegador web.
  • 3DCityDB-Web-Map-Client - Visualizador basado en Cesium para CityGML y 3D Tiles.
  • loaders.gl - Parsers y codificadores para muchos de los principales formatos 3D, geoespaciales y tabulares.
  • deck.gl - Framework potenciado por WebGL para análisis visual exploratorio de grandes conjuntos de datos.
  • 3d-tiles-renderer - Renderizador basado en Three.js para 3D Tiles.
  • three-loader-3dtiles - Módulo de carga para Three.js para manejar OGC 3D Tiles.
  • threedtiles - Visualizador de 3DTiles para three.js.
  • mapbox-3dtiles - Capa personalizada de Mapbox GL JS para 3D Tiles.
  • A-Frame component - Componente para A-Frame usando 3D-Tiles.
  • iTowns - Framework JS/WebGL basado en Three.js.
  • giro3d - Framework JS/WebGL basado en Three.js (sucesor de iTown).
  • UD-Viz - Visualización de Datos Urbanos. Framework para crear aplicaciones web para visualizar e interactuar con datos geoespaciales urbanos 3D (basado en iTowns/Three.js).
  • vsgCs - Biblioteca para usar 3D Tiles y otros contenidos geoespaciales dentro de una aplicación Vulkan Scene Graph (VSG).

Integraciones en motores de juegos, SIG de escritorio y herramientas de modelado 3D (Código abierto)

  • QGIS - Sistema de información geográfica de escritorio multiplataforma (desde la versión 3.34).
  • 3D Tiles for Godot - Extensión para Godot 4 que integra capacidades de Cesium 3D Tiles en el motor Godot.
  • Open 3D Engine (O3DE) - Cesium para O3DE.
  • blender-3d-tiler - Herramienta para Blender para dividir modelos 3D en tiles.

Creación de tiles (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - Biblioteca C++ para streaming de 3D Tiles y procesamiento de glTF.
  • 3D Tiles Tools - Herramientas y utilidades para convertir, optimizar, procesar y analizar datos de 3D Tiles.
  • glTF Pipeline - Herramientas en JavaScript para conversión y optimización de glTF / GLB.

Creación de tiles (Comunidad)

  • py3dtiles - Leer y escribir pnts y b3dm vía API o CLI.
  • Python 3DTiles Tilers - Herramienta y biblioteca Python que permite construir conjuntos de tiles 3D Tiles a partir de varios formatos geométricos, por ejemplo, OBJ, GeoJSON, IFC o CityGML a través de bases de datos 3dCityDB.
  • pg2b3dm - Conversión de geometrías 3D de PostGIS a tiles b3dm.
  • mago 3DTiler - mago 3DTiler convierte varios datos espaciales en OGC 3D Tiles.
  • PLATEAU GIS Converter - Convierte modelos de ciudad 3D (CityGML) en varios formatos geoespaciales, incluidos 3D Tiles, MVT y GeoPackage.
  • Obj2Tiles - Convierte archivos OBJ a OGC 3D tiles.
  • 3dtiles - Convierte archivos Osgb(OpenSceneGraph Binary), Esri Shapefile y Fbx en 3D Tiles.
  • gltf-to-3d-tiles - Convierte modelos glTF a formato Glb, b3dm o 3d tiles.
  • citygml-to-3dtiles - Convierte de CityGML a Cesium 3D Tiles.
  • vts23dtiles - Convierte un tileset VTS en un 3D Tileset.
  • 3D City Database - Base de datos geoespacial para almacenar, representar y gestionar modelos virtuales de ciudades 3D.
  • Cesium Point Cloud Generator - XYZ → Tiles de Nube de Puntos 3D.
  • loaders.gl tile-converter - Convierte I3S a 3D Tiles.
  • tyler - División en tiles de modelos de ciudad 3D codificados en CityJSON.

Herramientas glTF

  • glTF-Transform - glTF-Transform permite leer, editar y escribir modelos 3D en formato glTF 2.0.
  • Open Asset Import Library (assimp) - Biblioteca y conversor para más de 40 formatos de archivos 3D hacia glTF y otros (formatos de archivo).
  • Blender exporter - Exportador glTF 2.0 para Blender.

Terreno

Quantized Mesh es un formato para codificar mallas de terreno para renderizado eficiente de terreno en el cliente. Dichos archivos son soportados en Cesium y deck.gl.

Consulta awesome-quantized-mesh-tiles para más recursos.
