Awesome 3D Tiles

Une liste organisée de logiciels open source, bibliothèques, outils et ressources remarquables pour les 3D Tiles.

Contributions bienvenues. Ajoutez des liens via des pull requests ou créez une issue pour commencer une discussion.

3D Tiles est une spécification ouverte pour le partage, la visualisation, la fusion et l'interaction avec du contenu géospatial 3D massif et hétérogène sur des applications de bureau, web et mobiles.

3D Tiles est une norme communautaire de l’OGC.

Visionneuses

  • CesiumJS - Bibliothèque JavaScript pour créer des globes 3D et des cartes 2D dans un navigateur web.
  • 3DCityDB-Web-Map-Client - Visionneuse basée sur Cesium pour CityGML et 3D Tiles.
  • loaders.gl - Parseurs et encodeurs pour de nombreux formats majeurs 3D, géospatiaux et tabulaires.
  • deck.gl - Framework alimenté par WebGL pour l’analyse exploratoire visuelle de grands ensembles de données.
  • 3d-tiles-renderer - Rendu basé sur Three.js pour les 3D Tiles.
  • three-loader-3dtiles - Module de chargement Three.js pour gérer les OGC 3D Tiles.
  • threedtiles - Visionneuse 3DTiles pour three.js.
  • mapbox-3dtiles - Couche personnalisée Mapbox GL JS pour 3D Tiles.
  • A-Frame component - Composant A-Frame utilisant 3D-Tiles.
  • iTowns - Framework JS/WebGL basé sur Three.js.
  • giro3d - Framework JS/WebGL basé sur Three.js (successeur de iTown).
  • UD-Viz - Visualisation de données urbaines. Framework pour créer des applications web pour visualiser et interagir avec des données urbaines 3D géospatiales (basé sur iTowns/Tree.js).
  • vsgCs - Une bibliothèque pour utiliser les 3D Tiles et autres contenus géospatiaux dans une application Vulkan Scene Graph (VSG).

Moteurs de jeu, SIG de bureau et intégrations d’outils de modélisation 3D (Open source)

  • QGIS - Système d’information géographique de bureau multiplateforme (depuis la version 3.34).
  • 3D Tiles for Godot - Extension Godot 4 qui intègre les capacités Cesium 3D Tiles dans Godot Engine.
  • Open 3D Engine (O3DE) - Cesium pour O3DE.
  • blender-3d-tiler - Outil Blender pour tuiler les modèles 3D.

Création de tuiles (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - Bibliothèque C++ pour le streaming de 3D Tiles et le traitement glTF.
  • 3D Tiles Tools - Outils et utilitaires pour convertir, optimiser, traiter et analyser des données 3D Tiles.
  • glTF Pipeline - Outils JavaScript pour la conversion et l’optimisation glTF / GLB.

Création de tuiles (Communauté)

  • py3dtiles - Lire et écrire des fichiers pnts et b3dm via API ou CLI.
  • Python 3DTiles Tilers - Outil et bibliothèque Python permettant de générer des jeux de tuiles 3D Tiles à partir de divers formats géométriques (ex. OBJ, GeoJSON, IFC ou CityGML via des bases de données 3dCityDB).
  • pg2b3dm - Conversion des géométries 3D PostGIS vers des tuiles b3dm.
  • mago 3DTiler - mago 3DTiler convertit diverses données spatiales en OGC 3D Tiles.
  • PLATEAU GIS Converter - Conversion de modèles 3D urbains (CityGML) en divers formats géospatiaux, y compris 3D Tiles, MVT et GeoPackage.
  • Obj2Tiles - Convertit les fichiers OBJ en tuiles 3D OGC.
  • 3dtiles - Convertit les fichiers Osgb (OpenSceneGraph Binary), Esri Shapefile et Fbx en 3D Tiles.
  • gltf-to-3d-tiles - Convertit un modèle glTF en Glb, b3dm ou format 3d tiles.
  • citygml-to-3dtiles - Conversion de CityGML vers Cesium 3D Tiles.
  • vts23dtiles - Convertit un jeu de tuiles VTS en jeu de tuiles 3D.
  • 3D City Database - Base de données géo pour stocker, représenter et gérer des modèles de villes 3D virtuelles.
  • Cesium Point Cloud Generator - XYZ → Tuiles de nuage de points 3D.
  • loaders.gl tile-converter - Convertit I3S en 3D Tiles.
  • tyler - Tuilage de modèles 3D urbains encodés en CityJSON.

Outils glTF

  • glTF-Transform - glTF-Transform prend en charge la lecture, l’édition et l’écriture de modèles 3D au format glTF 2.0.
  • Open Asset Import Library (assimp) - Bibliothèque et convertisseur pour plus de 40 formats de fichiers 3D vers glTF et plus (formats de fichiers).
  • Blender exporter - Exportateur glTF 2.0 pour Blender.

Terrain

Quantized Mesh est un format pour encoder des maillages de terrain pour un rendu efficace côté client. Ces fichiers sont pris en charge dans Cesium et deck.gl.

Voir awesome-quantized-mesh-tiles pour plus de ressources.
