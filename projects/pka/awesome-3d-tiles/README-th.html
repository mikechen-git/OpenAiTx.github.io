Awesome 3D Tiles

รายการคัดสรรซอฟต์แวร์โอเพ่นซอร์ส ไลบรารี เครื่องมือ และทรัพยากรที่ยอดเยี่ยมสำหรับ 3D Tiles

ยินดีต้อนรับการมีส่วนร่วม เพิ่มลิงก์ผ่าน pull requests หรือสร้าง issue เพื่อเริ่มการสนทนา

3D Tiles เป็นข้อกำหนดแบบเปิดสำหรับการแบ่งปัน แสดงผล ผสาน และโต้ตอบกับข้อมูลภูมิสารสนเทศ 3 มิติขนาดใหญ่ที่มีความหลากหลายบนแอปพลิเคชันเดสก์ท็อป เว็บ และมือถือ

3D Tiles เป็น มาตรฐานชุมชน OGC

โปรแกรมแสดงผล (Viewers)

  • CesiumJS - ไลบรารี JavaScript สำหรับสร้างโลก 3 มิติและแผนที่ 2 มิติในเว็บเบราว์เซอร์
  • 3DCityDB-Web-Map-Client - Viewer สำหรับ CityGML และ 3D Tiles ที่ใช้ Cesium
  • loaders.gl - เครื่องมือแปลงและเข้ารหัสไฟล์ 3D, geospatial และตารางข้อมูลหลักๆ
  • deck.gl - กรอบงาน WebGL สำหรับวิเคราะห์ข้อมูลขนาดใหญ่แบบสำรวจเชิงภาพ
  • 3d-tiles-renderer - Renderer สำหรับ 3D Tiles ที่ใช้ Three.js
  • three-loader-3dtiles - โมดูลโหลด Three.js สำหรับจัดการ OGC 3D Tiles
  • threedtiles - Viewer 3DTiles สำหรับ three.js
  • mapbox-3dtiles - Mapbox GL JS custom layer สำหรับ 3D Tiles
  • A-Frame component - คอมโพเนนต์ A-Frame ที่ใช้ 3D-Tiles
  • iTowns - กรอบงาน JS/WebGL ที่ใช้ Three.js
  • giro3d - กรอบงาน JS/WebGL ที่ใช้ Three.js (สืบทอดจาก iTown)
  • UD-Viz - เฟรมเวิร์กสำหรับสร้างเว็บแอปฯ เพื่อแสดงและโต้ตอบกับข้อมูลเมือง 3 มิติ (อิงจาก iTowns/Tree.js)
  • vsgCs - ไลบรารีสำหรับใช้ 3D Tiles และเนื้อหาภูมิสารสนเทศอื่นๆ ภายในแอปพลิเคชัน Vulkan Scene Graph (VSG)

การเชื่อมต่อกับเอนจิ้นเกม, เดสก์ท็อป GIS และเครื่องมือโมเดล 3D (โอเพ่นซอร์ส)

  • QGIS - ระบบภูมิสารสนเทศเชิงพื้นที่บนเดสก์ท็อปแบบข้ามแพลตฟอร์ม (ตั้งแต่เวอร์ชัน 3.34)
  • 3D Tiles for Godot - ส่วนขยาย Godot 4 ที่ผสานความสามารถ Cesium 3D Tiles เข้ากับ Godot Engine
  • Open 3D Engine (O3DE) - Cesium สำหรับ O3DE
  • blender-3d-tiler - เครื่องมือ Blender สำหรับแบ่ง tiles โมเดล 3 มิติ

การสร้าง Tile (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles
  • Cesium Native - ไลบรารี C++ สำหรับสตรีม 3D Tiles และประมวลผล glTF
  • 3D Tiles Tools - เครื่องมือและยูทิลิตี้สำหรับแปลง ปรับปรุง ประมวลผล และวิเคราะห์ข้อมูล 3D Tiles
  • glTF Pipeline - เครื่องมือ Javascript สำหรับแปลงและเพิ่มประสิทธิภาพ glTF / GLB

การสร้าง Tile (ชุมชน)

  • py3dtiles - อ่านและเขียน pnts และ b3dm ผ่าน API หรือ CLI
  • Python 3DTiles Tilers - เครื่องมือและไลบรารี Python สำหรับสร้าง tileset 3D Tiles จากไฟล์รูปทรงต่างๆ เช่น OBJ, GeoJSON, IFC หรือ CityGML ผ่านฐานข้อมูล 3dCityDB
  • pg2b3dm - แปลง PostGIS 3D geometries เป็น b3dm tiles
  • mago 3DTiler - แปลงข้อมูลเชิงพื้นที่หลากหลายเป็น OGC 3D Tiles
  • PLATEAU GIS Converter - แปลงโมเดลเมือง 3 มิติ (CityGML) เป็นรูปแบบภูมิสารสนเทศต่างๆ รวมถึง 3D Tiles, MVT, และ GeoPackage
  • Obj2Tiles - แปลงไฟล์ OBJ เป็น OGC 3D Tiles
  • 3dtiles - แปลงไฟล์ Osgb (OpenSceneGraph Binary), Esri Shapefile และ Fbx เป็น 3D Tiles
  • gltf-to-3d-tiles - แปลงโมเดล glTF เป็น Glb, b3dm หรือ 3d tiles
  • citygml-to-3dtiles - แปลงจาก CityGML เป็น Cesium 3D Tiles
  • vts23dtiles - แปลง VTS tileset เป็น 3D Tileset
  • 3D City Database - ฐานข้อมูลภูมิสารสนเทศเพื่อจัดเก็บ แสดง และจัดการโมเดลเมือง 3 มิติ
  • Cesium Point Cloud Generator - XYZ → 3D Point Cloud Tiles
  • loaders.gl tile-converter - แปลง I3S เป็น 3D Tiles
  • tyler - แบ่ง tiles โมเดลเมือง 3 มิติที่เข้ารหัสด้วย CityJSON

เครื่องมือ glTF

  • glTF-Transform - รองรับการอ่าน แก้ไข และเขียนโมเดล 3 มิติในรูปแบบ glTF 2.0
  • Open Asset Import Library (assimp) - ไลบรารีและเครื่องมือแปลงไฟล์ 3D กว่า 40 รูปแบบเป็น glTF และอื่นๆ (รูปแบบไฟล์)
  • Blender exporter - Blender glTF 2.0 Exporter

ภูมิประเทศ (Terrain)

Quantized Mesh คือรูปแบบสำหรับเข้ารหัส mesh ภูมิประเทศเพื่อการแสดงผล terrain ฝั่งลูกข่ายอย่างมีประสิทธิภาพ ไฟล์เหล่านี้รองรับใน Cesium และ deck.gl

ดู awesome-quantized-mesh-tiles สำหรับทรัพยากรเพิ่มเติม
