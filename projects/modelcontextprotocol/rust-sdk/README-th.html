RMCP

รอการปล่อยเวอร์ชันแรก

เป็นการติดตั้ง SDK อย่างเป็นทางการของ Model Context Protocol ในภาษา Rust ที่ใช้ tokio async runtime

ที่เก็บนี้ประกอบด้วย crate ดังต่อไปนี้:

  • rmcp: crate หลักที่ให้การติดตั้งโปรโตคอล RMCP (หากต้องการข้อมูลเพิ่มเติม กรุณาเยี่ยมชม rmcp)
  • rmcp-macros: crate procedural macro สำหรับสร้างเครื่องมือ RMCP (หากต้องการข้อมูลเพิ่มเติม กรุณาเยี่ยมชม rmcp-macros)

การใช้งาน

นำเข้า crate

rmcp = { version = "0.1", features = ["server"] }
## หรือ dev channel
rmcp = { git = "https://github.com/modelcontextprotocol/rust-sdk", branch = "main" }

Third Dependencies

การขึ้นต่อพื้นฐาน:

  • tokio required
  • serde required

สร้าง Client

เริ่มต้น client 
use rmcp::{ServiceExt, transport::{TokioChildProcess, ConfigureCommandExt}};
use tokio::process::Command;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = ().serve(TokioChildProcess::new(Command::new("npx").configure(|cmd| {
        cmd.arg("-y").arg("@modelcontextprotocol/server-everything");
    }))?).await?;
    Ok(())
}

สร้าง Server

สร้าง transport 
use tokio::io::{stdin, stdout};
let transport = (stdin(), stdout());
สร้าง service

คุณสามารถสร้าง service ได้อย่างง่ายดายโดยใช้ ServerHandler หรือ ClientHandler

let service = common::counter::Counter::new();
เริ่มต้น server 
// การเรียกนี้จะจบกระบวนการ initialization
let server = service.serve(transport).await?;
โต้ตอบกับ server

เมื่อ server ถูก initialize แล้ว คุณสามารถส่ง request หรือ notification ได้:

// request
let roots = server.list_roots().await?;

// หรือส่ง notification
server.notify_cancelled(...).await?;
รอให้ service ปิดตัวลง 
let quit_reason = server.waiting().await?;
// หรือยกเลิกมัน
let quit_reason = server.cancel().await?;
