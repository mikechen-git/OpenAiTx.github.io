RMCP

Wacht op de eerste release.

Een officiële rust Model Context Protocol SDK-implementatie met tokio async runtime.

Deze repository bevat de volgende crates:

  • rmcp: De kerncrate die de RMCP-protocolimplementatie biedt (Als je meer informatie wilt, bezoek rmcp)
  • rmcp-macros: Een proc-macro crate voor het genereren van RMCP tool-implementaties (Als je meer informatie wilt, bezoek rmcp-macros)

Gebruik

Importeer de crate

rmcp = { version = "0.1", features = ["server"] }
## of dev channel
rmcp = { git = "https://github.com/modelcontextprotocol/rust-sdk", branch = "main" }

Derde afhankelijkheden

Basisafhankelijkheden:

  • tokio vereist
  • serde vereist

Bouw een Client

Start een client 
use rmcp::{ServiceExt, transport::{TokioChildProcess, ConfigureCommandExt}};
use tokio::process::Command;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = ().serve(TokioChildProcess::new(Command::new("npx").configure(|cmd| {
        cmd.arg("-y").arg("@modelcontextprotocol/server-everything");
    }))?).await?;
    Ok(())
}

Bouw een Server

Bouw een transport 
use tokio::io::{stdin, stdout};
let transport = (stdin(), stdout());
Bouw een service

Je kunt eenvoudig een service bouwen met ServerHandler of ClientHandler.

let service = common::counter::Counter::new();
Start de server 
// deze aanroep zal het initialisatieproces afronden
let server = service.serve(transport).await?;
Interactie met de server

Zodra de server is geïnitialiseerd, kun je verzoeken of notificaties sturen:

// verzoek
let roots = server.list_roots().await?;

// of stuur een notificatie
server.notify_cancelled(...).await?;
Wachten op het afsluiten van de service 
let quit_reason = server.waiting().await?;
// of annuleer het
let quit_reason = server.cancel().await?;
