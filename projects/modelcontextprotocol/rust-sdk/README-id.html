RMCP

Tunggu rilis pertama.

Implementasi resmi SDK Model Context Protocol untuk Rust dengan runtime async tokio.

Repositori ini berisi crate berikut:

  • rmcp: Crate inti yang menyediakan implementasi protokol RMCP (Untuk informasi lebih lanjut, silakan kunjungi rmcp)
  • rmcp-macros: Crate macro prosedural untuk menghasilkan implementasi alat RMCP (Untuk informasi lebih lanjut, silakan kunjungi rmcp-macros)

Penggunaan

Impor crate

rmcp = { version = "0.1", features = ["server"] }
## atau channel dev
rmcp = { git = "https://github.com/modelcontextprotocol/rust-sdk", branch = "main" }

Ketergantungan Pihak Ketiga

Ketergantungan dasar:

  • tokio diperlukan
  • serde diperlukan

Membangun Klien

Memulai klien 
use rmcp::{ServiceExt, transport::{TokioChildProcess, ConfigureCommandExt}};
use tokio::process::Command;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = ().serve(TokioChildProcess::new(Command::new("npx").configure(|cmd| {
        cmd.arg("-y").arg("@modelcontextprotocol/server-everything");
    }))?).await?;
    Ok(())
}

Membangun Server

Membangun transport 
use tokio::io::{stdin, stdout};
let transport = (stdin(), stdout());
Membangun service

Anda dapat dengan mudah membangun service menggunakan ServerHandler atau ClientHandler.

let service = common::counter::Counter::new();
Menjalankan server 
// panggilan ini akan menyelesaikan proses inisialisasi
let server = service.serve(transport).await?;
Berinteraksi dengan server

Setelah server diinisialisasi, Anda dapat mengirim permintaan atau notifikasi:

// request
let roots = server.list_roots().await?;

// atau kirim notifikasi
server.notify_cancelled(...).await?;
Menunggu penghentian service 
let quit_reason = server.waiting().await?;
// atau batalkan
let quit_reason = server.cancel().await?;
