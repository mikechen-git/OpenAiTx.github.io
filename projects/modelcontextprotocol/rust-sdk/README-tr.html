RMCP

İlk sürüm için bekleyin.

Tokio async çalışma zamanı ile resmi bir Rust Model Context Protocol SDK uygulaması.

Bu depo aşağıdaki paketleri içerir:

  • rmcp: RMCP protokol uygulamasını sağlayan çekirdek paket (Daha fazla bilgi almak için lütfen rmcp adresini ziyaret edin)
  • rmcp-macros: RMCP araç uygulamaları oluşturmak için prosedürel makro paketi (Daha fazla bilgi almak için lütfen rmcp-macros adresini ziyaret edin)

Kullanım

Paketi içe aktar

rmcp = { version = "0.1", features = ["server"] }
## veya geliştirme kanalı
rmcp = { git = "https://github.com/modelcontextprotocol/rust-sdk", branch = "main" }

Üçüncü Parti Bağımlılıklar

Temel bağımlılıklar:

  • tokio gereklidir
  • serde gereklidir

Bir İstemci Oluşturma

Bir istemci başlat 
use rmcp::{ServiceExt, transport::{TokioChildProcess, ConfigureCommandExt}};
use tokio::process::Command;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = ().serve(TokioChildProcess::new(Command::new("npx").configure(|cmd| {
        cmd.arg("-y").arg("@modelcontextprotocol/server-everything");
    }))?).await?;
    Ok(())
}

Bir Sunucu Oluşturma

Bir taşıma katmanı oluştur 
use tokio::io::{stdin, stdout};
let transport = (stdin(), stdout());
Bir servis oluştur

ServerHandler veya ClientHandler kullanarak kolayca bir servis oluşturabilirsiniz.

let service = common::counter::Counter::new();
Sunucuyu başlat 
// bu çağrı başlatma işlemini tamamlar
let server = service.serve(transport).await?;
Sunucu ile etkileşim

Sunucu başlatıldıktan sonra, istek veya bildirim gönderebilirsiniz:

// istek
let roots = server.list_roots().await?;

// veya bildirim gönder
server.notify_cancelled(...).await?;
Servisin kapanmasını bekleme 
let quit_reason = server.waiting().await?;
// veya iptal et
let quit_reason = server.cancel().await?;
