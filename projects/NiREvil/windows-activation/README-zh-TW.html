Windows 10 & 11 即時啟用：40 秒解決方案


本儲存庫提供一種能在 40 秒內啟用 Windows 10 和 11 的方法。兩種方法皆相容於 Windows 的家庭版、家庭 N、家庭單一語言版、家庭特定國家版、專業版、教育版及企業版。

方法一（推薦）

使用 PowerShell 即時啟用 Windows

[!TIP] Windows 10 & 11 有許多方式可啟動 PowerShell。[1]

在 Windows 中最快啟動 PowerShell 的方式之一是透過「開始」選單搜尋。請依照以下步驟操作：


步驟 1

只需點擊開始或搜尋圖示，在搜尋欄輸入「PowerShell」。

另一簡單方法：[2]



步驟 2

接著，請點擊 以系統管理員身分執行 來啟動 PowerShell，本操作需以管理員權限執行。



步驟 3

現在，稍等片刻以執行，然後請「複製」以下這行指令：

irm https://get.activated.win | iex

步驟 4

現在將其貼上（右鍵貼上），然後按下 Enter 鍵。在新開啟的視窗中，會有幾個選項，請選擇選項 1，並等待幾秒鐘完成。



恭喜您，

您的 Windows 已成功啟用。
現在您可以按 Enter 鍵離開 CMD 並關閉 PowerShell，並於 Windows 啟用選單中檢查授權狀態 [3]









方法二

使用 CMD（命令提示字元）即時啟用 Windows


[!NOTE] 請記得連接至網際網路

無需 VPN


步驟 1

首先，點擊開始或搜尋圖示，於搜尋欄輸入「CMD」。在看到命令提示字元選項後，以系統管理員身分執行

其他簡單方法：[2]

步驟 2

安裝 KMS 用戶端金鑰。

請使用以下指令：

slmgr /ipk yourlicensekey

[!NOTE] 請從上方清單選擇與您的 Windows 版本相符的授權金鑰，並將指令中的 yourlicencekey 替換成該金鑰。

[!TIP] 如何查詢 Windows 版本 [3]。




步驟 3

設定 KMS 主機位址。

請使用下列指令：

slmgr /skms kms8.msguides.com

以連線至我的 KMS 伺服器。




步驟 4

啟用您的 Windows。
最後一步，請使用以下指令啟用您的 Windows：

slmgr /ato




步驟 5

現在再次檢查啟用狀態 [6]。


完成 ✅

您的 Windows 已成功啟用。


保持好奇 🤍
