iText Core/Community (ранее известная как iTextSharp) — это высокопроизводительная, проверенная временем библиотека, которая позволяет создавать, изменять, просматривать и поддерживать PDF-документы, легко добавляя функциональность PDF в ваши программные проекты. Она также доступна для Java.

Ключевые возможности iText Core/Community:

  • Основная библиотека:
    • Создание PDF с использованием нашего движка разметки
    • Манипулирование PDF, например, объединение нескольких PDF в один, добавление нового контента и др.
    • Цифровая подпись PDF
    • Создание и изменение PDF-форм
    • Работа с документами PDF/A
    • Работа с документами PDF/UA
    • Криптография, соответствующая стандарту FIPS
    • Генерация штрих-кодов
    • Поддержка SVG
  • Дополнения:
    • Конвертация XML/HTML & CSS в PDF репозиторий, информация
    • Сокрытие конфиденциальной информации в PDF-документах репозиторий, информация
    • Поддержка международных наборов символов (например, арабский, китайский, иврит, тайский, ...) информация
    • Оптимизация PDF-документов для уменьшения размера файла и повышения производительности информация
    • Преобразование XFA-документов в статическую форму информация
    • Отладка PDF репозиторий, информация

Хотите узнать, на что способна библиотека? Загляните в наш Demo Lab! Здесь собрана коллекция демонстрационных приложений, готовых к использованию онлайн!

Начало работы

Самый простой способ начать — использовать NuGet, просто выполните следующую команду установки в папке вашего проекта:

dotnet add package itext --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>
dotnet add package itext.bouncy-castle-adapter --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>

Для более сложных сценариев использования ознакомьтесь с Руководством по установке. Вы также можете собрать iText Community из исходников.

Hello PDF!

Следующий пример демонстрирует, как просто создать простой PDF-документ:

using iText.Kernel.Pdf;
using iText.Layout;
using iText.Layout.Element;

namespace HelloPdf {
    class Program {
        static void Main(string[] args) {
            using var document = new Document(new PdfDocument(new PdfWriter("helloworld-pdf.pdf")));
            document.Add(new Paragraph("Hello World!"));
        }
    }
}

Лицензирование

iText распространяется по двойной лицензии AGPL/Коммерческое ПО.

AGPL — это бесплатная/открытая лицензия, однако это не означает, что программное обеспечение бесплатное!

AGPL — это лицензия copyleft, что означает, что любые производные работы также должны распространяться на тех же условиях. Если вы используете iText в программном обеспечении или сервисе, которые не могут соответствовать условиям AGPL, у нас есть коммерческая лицензия, освобождающая от таких обязательств.
