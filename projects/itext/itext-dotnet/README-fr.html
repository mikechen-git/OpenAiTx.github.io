iText Core/Community (anciennement connu sous le nom d'iTextSharp) est une bibliothèque haute performance, éprouvée sur le terrain, qui vous permet de créer, d’adapter, d’inspecter et de maintenir des documents PDF, ce qui vous permet d’ajouter facilement des fonctionnalités PDF à vos projets logiciels. Elle est également disponible pour Java.

Les principales fonctionnalités d’iText Core/Community sont :

  • Bibliothèque principale :
    • Création de PDF grâce à notre moteur de mise en page
    • Manipulation de PDF, par exemple fusion de plusieurs PDF en un seul, ajout de contenu, ...
    • Signature numérique de PDF
    • Création et manipulation de formulaires PDF
    • Travail avec des documents PDF/A
    • Travail avec des documents PDF/UA
    • Cryptographie conforme FIPS
    • Génération de codes-barres
    • Prise en charge du SVG
  • [Extensions :][all products]
    • Conversion XML/HTML & CSS en PDF repo, info
    • Rédaction d’informations sensibles dans des documents PDF repo, info
    • Prise en charge des jeux de caractères internationaux (ex. arabe, chinois, hébreu, thaï, ...) info
    • Optimisation des documents PDF pour réduire la taille des fichiers et augmenter les performances info
    • Aplatissement des documents XFA info
    • Débogage PDF repo, info

Vous souhaitez découvrir ce qui est possible ? Rendez-vous sur notre Demo Lab ! Il contient une collection d’applications de démonstration prêtes à être utilisées en ligne !

Démarrage rapide

Le moyen le plus simple de commencer est d’utiliser NuGet, il suffit d’exécuter la commande d’installation suivante dans le dossier de votre projet :

dotnet add package itext --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>
dotnet add package itext.bouncy-castle-adapter --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>

Pour des cas d’utilisation plus avancés, veuillez vous référer aux Instructions d’installation.
Vous pouvez également compiler iText Community à partir des sources.

Hello PDF !

L’exemple suivant montre à quel point il est facile de créer un document PDF simple :

using iText.Kernel.Pdf;
using iText.Layout;
using iText.Layout.Element;

namespace HelloPdf {
    class Program {
        static void Main(string[] args) {
            using var document = new Document(new PdfDocument(new PdfWriter("helloworld-pdf.pdf")));
            document.Add(new Paragraph("Hello World!"));
        }
    }
}
