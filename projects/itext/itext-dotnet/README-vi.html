iText Core/Community (trước đây gọi là iTextSharp) là một thư viện hiệu năng cao, đã được kiểm nghiệm thực tế, cho phép bạn tạo, chỉnh sửa, kiểm tra và duy trì các tài liệu PDF, giúp bạn dễ dàng thêm chức năng PDF vào các dự án phần mềm của mình. Thư viện này cũng có sẵn cho Java.

    • Mã hóa tuân thủ FIPS
    • Tạo mã vạch

  • [Các tiện ích bổ sung:][all products]
    • Chuyển đổi XML/HTML & CSS sang PDF repo, thông tin

Bắt đầu nhanh

Cách đơn giản nhất để bắt đầu là sử dụng NuGet, chỉ cần thực hiện lệnh cài đặt sau trong thư mục dự án của bạn:

dotnet add package itext --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>
dotnet add package itext.bouncy-castle-adapter --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>

Đối với các trường hợp sử dụng nâng cao, vui lòng tham khảo Hướng dẫn cài đặt. Bạn cũng có thể biên dịch iText Community từ mã nguồn.

Hello PDF!

Ví dụ sau đây cho thấy việc tạo một tài liệu PDF đơn giản dễ dàng như thế nào:

using iText.Kernel.Pdf;
using iText.Layout;
using iText.Layout.Element;

namespace HelloPdf {
    class Program {
        static void Main(string[] args) {
            using var document = new Document(new PdfDocument(new PdfWriter("helloworld-pdf.pdf")));
            document.Add(new Paragraph("Hello World!"));
        }
    }
}

Giấy phép

iText có hai loại giấy phép: AGPL/Phần mềm thương mại.

AGPL là giấy phép phần mềm miễn phí/mã nguồn mở, tuy nhiên, điều này không có nghĩa là phần mềm miễn phí!

AGPL là giấy phép copyleft, nghĩa là bất kỳ phần mềm phái sinh nào cũng phải được cấp phép dưới cùng điều khoản. Nếu bạn sử dụng iText trong phần mềm hoặc dịch vụ không thể tuân thủ điều khoản AGPL, chúng tôi có giấy phép thương mại giúp bạn miễn trừ các nghĩa vụ này.
