iText Core/Community (precedentemente noto come iTextSharp) è una libreria ad alte prestazioni, collaudata e affidabile che consente di creare, modificare, ispezionare e mantenere documenti PDF, permettendo di aggiungere facilmente funzionalità PDF ai tuoi progetti software. È disponibile anche per Java.

Le funzionalità principali di iText Core/Community sono:

  • Libreria Core:
    • Creazione di PDF tramite il nostro motore di layout
    • Manipolazione PDF, ad esempio unione di più PDF in uno, aggiunta di nuovo contenuto, ...
    • Firma digitale PDF
    • Creazione e manipolazione di moduli PDF
    • Lavorare con documenti PDF/A
    • Lavorare con documenti PDF/UA
    • Crittografia conforme a FIPS
    • Generazione di codici a barre
    • Supporto SVG
  • [Addons:][all products]
    • Conversione di XML/HTML & CSS in PDF repo, info
    • Redazione di informazioni sensibili nei documenti PDF repo, info
    • Supporto per set di caratteri internazionali (es. arabo, cinese, ebraico, tailandese, ...) info
    • Ottimizzazione dei documenti PDF per ridurre la dimensione del file e aumentare le prestazioni info
    • Appiattimento di documenti XFA info
    • Debug dei PDF repo, info

Vuoi scoprire cosa è possibile fare? Visita il nostro Demo Lab! Contiene una raccolta di applicazioni demo pronte per l'uso online!

Per iniziare

Il modo più semplice per iniziare è usare NuGet, basta eseguire il seguente comando di installazione nella cartella del tuo progetto:

dotnet add package itext --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>
dotnet add package itext.bouncy-castle-adapter --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>

Per casi d'uso più avanzati, fai riferimento alle linee guida per l'installazione. Puoi anche compilare iText Community dal sorgente.

Hello PDF!

Il seguente esempio mostra quanto sia facile creare un semplice documento PDF:

using iText.Kernel.Pdf;
using iText.Layout;
using iText.Layout.Element;

namespace HelloPdf {
    class Program {
        static void Main(string[] args) {
            using var document = new Document(new PdfDocument(new PdfWriter("helloworld-pdf.pdf")));
            document.Add(new Paragraph("Hello World!"));
        }
    }
}

Contribuire

Molte persone hanno contribuito a iText Core/Community nel corso degli anni. Se hai trovato un bug, un errore nella documentazione o hai una nuova funzionalità da implementare, accogliamo con piacere i tuoi contributi.

Piccoli cambiamenti o correzioni possono essere inviati come Pull Request, mentre per modifiche importanti ti chiediamo di contattarci a community@apryse.com così da poter coordinare meglio i nostri sforzi ed evitare duplicazioni di lavoro.

Leggi le nostre Linee guida per il contributo per dettagli su invio del codice, regole di codifica e altro.
