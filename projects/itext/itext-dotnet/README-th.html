iText Core/Community (เดิมชื่อ iTextSharp) เป็นไลบรารีประสิทธิภาพสูงที่ผ่านการทดสอบมาอย่างดี ช่วยให้คุณสามารถสร้าง ปรับแต่ง ตรวจสอบ และดูแลไฟล์ PDF ได้อย่างง่ายดาย เพิ่มความสามารถเกี่ยวกับ PDF ให้กับโครงการซอฟต์แวร์ของคุณได้อย่างง่ายดาย นอกจากนี้ยังมีให้ใช้งานสำหรับ Java ด้วย

คุณสมบัติหลักของ iText Core/Community ได้แก่:

  • ไลบรารีหลัก:
    • สร้างไฟล์ PDF ด้วย Layout Engine ของเรา
    • การปรับแต่ง PDF เช่น การรวมไฟล์ PDF หลายไฟล์ การเพิ่มเนื้อหาใหม่ ฯลฯ
    • การลงนามดิจิทัลใน PDF
    • การสร้างและจัดการฟอร์ม PDF
    • ทำงานกับเอกสาร PDF/A
    • ทำงานกับเอกสาร PDF/UA
    • การเข้ารหัสที่สอดคล้องกับ FIPS
    • การสร้างบาร์โค้ด
    • รองรับ SVG
  • ส่วนเสริม:
    • แปลง XML/HTML & CSS เป็น PDF repo, info
    • การเซ็นเซอร์ข้อมูลสำคัญในเอกสาร PDF repo, info
    • รองรับชุดอักขระนานาชาติ (เช่น อาหรับ, จีน, ฮิบรู, ไทย, ...) info
    • ปรับแต่งเอกสาร PDF ให้ขนาดไฟล์เล็กลงและประสิทธิภาพสูงขึ้น info
    • ทำให้เอกสาร XFA แบนราบ info
    • ดีบัก PDF repo, info

อยากค้นพบความสามารถต่าง ๆ เพิ่มเติม? ไปที่ Demo Lab! ที่นี่มีแอปสาธิตให้ทดลองใช้งานออนไลน์มากมาย!

เริ่มต้นใช้งาน

วิธีที่ง่ายที่สุดในการเริ่มต้นใช้งานคือใช้ NuGet เพียงรันคำสั่งติดตั้งด้านล่างในโฟลเดอร์ของโปรเจกต์คุณ:

dotnet add package itext --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>
dotnet add package itext.bouncy-castle-adapter --version <REPLACE_WITH_DESIRED_ITEXT_VERSION>

สำหรับกรณีการใช้งานขั้นสูง กรุณาดูที่ คู่มือการติดตั้ง คุณยังสามารถ คอมไพล์ iText Community จากซอร์ส ได้เช่นกัน

Hello PDF!

ตัวอย่างต่อไปนี้แสดงให้เห็นว่าการสร้างเอกสาร PDF อย่างง่ายนั้นง่ายเพียงใด:

using iText.Kernel.Pdf;
using iText.Layout;
using iText.Layout.Element;

namespace HelloPdf {
    class Program {
        static void Main(string[] args) {
            using var document = new Document(new PdfDocument(new PdfWriter("helloworld-pdf.pdf")));
            document.Add(new Paragraph("Hello World!"));
        }
    }
}

การร่วมพัฒนา

มีผู้คนมากมายที่ร่วมพัฒนา iText Core/Community ตลอดหลายปีที่ผ่านมา หากคุณพบข้อผิดพลาดในซอฟต์แวร์หรือเอกสาร หรือมีฟีเจอร์ใหม่ ๆ ที่อยากเสนอ เรายินดีต้อนรับการมีส่วนร่วมของคุณ

การแก้ไขหรือปรับปรุงเล็กน้อย สามารถส่งเป็น Pull Request ได้เลย ส่วนการเปลี่ยนแปลงใหญ่ ๆ กรุณาติดต่อเราที่ community@apryse.com เพื่อจะได้ประสานงานและป้องกันงานซ้ำซ้อน

กรุณาอ่าน Contribution Guidelines สำหรับรายละเอียดเกี่ยวกับการส่งโค้ด กฎการเขียนโค้ด ฯลฯ

การอนุญาตใช้งาน

iText มีการอนุญาตใช้งานแบบคู่ทั้ง AGPL/ซอฟต์แวร์เชิงพาณิชย์

AGPL เป็นซอฟต์แวร์โอเพ่นซอร์สฟรี แต่ไม่ได้หมายความว่าซอฟต์แวร์นี้จะ ฟรี!

AGPL เป็นไลเซนส์ copyleft หมายความว่างานที่ต่อยอดต้องได้รับอนุญาตภายใต้เงื่อนไขเดียวกัน หากคุณใช้ iText ในซอฟต์แวร์หรือบริการที่ไม่สามารถปฏิบัติตามเงื่อนไข AGPL ได้ เรายังมีไลเซนส์เชิงพาณิชย์ที่ยกเว้นข้อผูกมัดเหล่านี้ให้คุณ
