Início rápido

Várias opções de início rápido estão disponíveis:

  • Baixar a última versão
  • Clone o repositório: git clone https://github.com/twbs/bootstrap.git
  • Instale com npm: npm install bootstrap@v5.3.6
  • Instale com yarn: yarn add bootstrap@v5.3.6
  • Instale com Bun: bun add bootstrap@v5.3.6
  • Instale com Composer: composer require twbs/bootstrap:5.3.6
  • Instale com NuGet: CSS: Install-Package bootstrap Sass: Install-Package bootstrap.sass

Leia a página de Introdução para informações sobre o conteúdo do framework, modelos, exemplos e muito mais.

Bugs e solicitações de funcionalidades

Tem um bug ou uma solicitação de funcionalidade? Por favor, primeiro leia as diretrizes de issues e pesquise por issues existentes e fechadas. Se seu problema ou ideia ainda não foi abordado, abra uma nova issue.

Documentação

A documentação do Bootstrap, incluída neste repositório no diretório raiz, é construída com Astro e hospedada publicamente no GitHub Pages em https://getbootstrap.com/. A documentação também pode ser executada localmente.

A busca na documentação é alimentada pelo DocSearch da Algolia.

Executando a documentação localmente

  1. Execute npm install para instalar as dependências do Node.js, incluindo o Astro (o construtor do site).
  2. Execute npm run test (ou um script npm específico) para reconstruir os arquivos CSS e JavaScript distribuídos, bem como os ativos da documentação.
  3. A partir do diretório raiz /bootstrap, execute npm run docs-serve na linha de comando.
  4. Abra http://localhost:9001/ em seu navegador, e voilà.

Saiba mais sobre como usar o Astro lendo a documentação.

Contribuindo

Por favor, leia as nossas diretrizes de contribuição. Estão incluídas instruções para abertura de issues, padrões de codificação e notas sobre desenvolvimento.

Além disso, se sua pull request contiver correções ou funcionalidades em JavaScript, você deve incluir testes unitários relevantes. Todo HTML e CSS devem estar em conformidade com o Code Guide, mantido por Mark Otto.

Preferências de editor estão disponíveis no editor config para fácil utilização nos editores de texto mais comuns. Saiba mais e baixe plugins em https://editorconfig.org/.

Comunidade

Receba atualizações sobre o desenvolvimento do Bootstrap e converse com os mantenedores do projeto e membros da comunidade.

  • Siga @getbootstrap no X.
  • Leia e assine O Blog Oficial do Bootstrap.
  • Tire dúvidas e explore nossas Discussões no GitHub.
  • Discuta, tire dúvidas e mais no Discord da comunidade ou no subreddit do Bootstrap.
  • Converse com outros usuários do Bootstrap no IRC. No servidor irc.libera.chat, no canal #bootstrap.
  • Ajuda para implementação pode ser encontrada no Stack Overflow (com a tag bootstrap-5).
  • Desenvolvedores devem usar a palavra-chave bootstrap em pacotes que modificam ou adicionam funcionalidades ao Bootstrap ao distribuir via npm ou mecanismos similares para máxima descoberta.

Versionamento

Para transparência em nosso ciclo de lançamentos e na busca por manter compatibilidade retroativa, o Bootstrap é mantido sob as diretrizes de Versionamento Semântico. Às vezes cometemos erros, mas seguimos essas regras sempre que possível.

Veja a seção de Lançamentos do nosso projeto no GitHub para changelogs de cada versão do Bootstrap. As postagens de anúncio de lançamentos no blog oficial do Bootstrap contêm resumos das mudanças mais importantes de cada versão.
