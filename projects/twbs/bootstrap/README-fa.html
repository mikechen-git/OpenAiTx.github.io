شروع سریع

چندین گزینه برای شروع سریع موجود است:

  • آخرین نسخه را دانلود کنید
  • مخزن را کلون کنید: git clone https://github.com/twbs/bootstrap.git
  • نصب با npm: npm install bootstrap@v5.3.6
  • نصب با yarn: yarn add bootstrap@v5.3.6
  • نصب با Bun: bun add bootstrap@v5.3.6
  • نصب با Composer: composer require twbs/bootstrap:5.3.6
  • نصب با NuGet: CSS: Install-Package bootstrap Sass: Install-Package bootstrap.sass

برای اطلاعات بیشتر درباره محتویات چارچوب، قالب‌ها، نمونه‌ها و غیره، صفحه شروع کار را بخوانید.

آیا باگ یا درخواست ویژگی دارید؟ لطفاً ابتدا راهنمای مسائل را بخوانید و به دنبال مسائل باز و بسته‌شده بگردید. اگر مشکل یا ایده شما هنوز مطرح نشده است، لطفاً یک مسئله جدید باز کنید.

مستندات

مستندات بوت‌استرپ، که در ریشه این مخزن قرار دارد، با Astro ساخته شده و به صورت عمومی در GitHub Pages در https://getbootstrap.com/ میزبانی می‌شود. مستندات را می‌توانید به صورت محلی نیز اجرا کنید.

جستجوی مستندات توسط DocSearch الگولیا تامین می‌شود.

اجرای مستندات به صورت محلی

  1. دستور npm install را اجرا کنید تا وابستگی‌های Node.js، از جمله Astro (سایت‌ساز)، نصب شوند.
  2. دستور npm run test (یا اسکریپت npm خاص) را اجرا کنید تا فایل‌های CSS و جاوااسکریپت توزیع شده و همچنین دارایی‌های مستندات ما دوباره ساخته شوند.
  3. از دایرکتوری ریشه /bootstrap، دستور npm run docs-serve را در خط فرمان اجرا کنید.
  4. در مرورگر خود آدرس http://localhost:9001/ را باز کنید.

برای اطلاعات بیشتر درباره استفاده از Astro، مستندات آن را بخوانید.

مستندات نسخه‌های قبلی

تمام مستندات نسخه‌های قبلی را می‌توانید در https://getbootstrap.com/docs/versions/ بیابید.

نسخه‌های قبلی و مستندات آن‌ها نیز برای دانلود در دسترس هستند.

مشارکت

لطفاً راهنمای مشارکت ما را به طور کامل بخوانید. در این راهنما، دستورالعمل‌هایی برای باز کردن مسائل، استانداردهای کدنویسی و نکاتی درباره توسعه آورده شده است.

همچنین، اگر pull request شما شامل تغییرات یا ویژگی‌های JavaScript است، باید تست‌های واحد مرتبط را نیز اضافه کنید. تمام HTML و CSS باید مطابق راهنمای کد که توسط Mark Otto نگهداری می‌شود، باشد.

ترجیحات ویرایشگر در پیکربندی ویرایشگر برای استفاده آسان در ویرایشگرهای متن رایج موجود است. برای اطلاعات بیشتر و دانلود افزونه‌ها به https://editorconfig.org/ مراجعه کنید.

جامعه

اخبار توسعه بوت‌استرپ را دریافت کنید و با نگهدارندگان پروژه و اعضای جامعه گفتگو کنید.

  • دنبال کنید @getbootstrap در X.
  • بخوانید و مشترک شوید در وبلاگ رسمی بوت‌استرپ.
  • پرسش بپرسید و بحث‌های گیت‌هاب ما را مرور کنید.
  • بحث کنید، پرسش بپرسید و غیره در دیسکورد جامعه یا ساب‌ردیت بوت‌استرپ.
  • با سایر بوت‌استرپرها در IRC گفتگو کنید. در سرور irc.libera.chat، کانال #bootstrap.
  • کمک در پیاده‌سازی را در Stack Overflow بیابید (برچسب bootstrap-5).
  • توسعه‌دهندگان باید از کلیدواژه bootstrap برای بسته‌هایی که قابلیت‌های بوت‌استرپ را تغییر می‌دهند یا گسترش می‌دهند، هنگام توزیع از طریق npm یا مکانیزم‌های مشابه، جهت پیدا شدن بهتر استفاده کنند.

نسخه‌بندی

برای شفافیت در چرخه انتشار و تلاش برای حفظ سازگاری به عقب، بوت‌استرپ تحت راهنمای نسخه‌بندی معنایی نگهداری می‌شود. گاهی خطا می‌کنیم، اما تا حد امکان به این قوانین پایبندیم.

برای مشاهده تغییرات هر نسخه بوت‌استرپ به بخش Releases پروژه در گیت‌هاب مراجعه کنید. اطلاعیه‌های انتشار در وبلاگ رسمی بوت‌استرپ خلاصه‌ای از مهم‌ترین تغییرات هر نسخه را دارند.
