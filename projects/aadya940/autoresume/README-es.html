autoResume

Constructor de currículums de código abierto: pega tus enlaces, edita manualmente y deja que la IA te ayude a pulir, actualizar y adaptar tu currículum.

¿Cómo usar esto?

Enlace al Blog en Medium

El blog está desactualizado con respecto a la Interfaz de Usuario y la plantilla utilizada.

Características

  • Importación fácil de enlaces: Pega enlaces de sitios web y crea rápidamente un currículum en PDF.
  • Editor manual incluido: Ajusta o construye tu currículum desde cero con un editor de código integrado.
  • Sugerencias inteligentes: Comparte comentarios o nuevos enlaces, y las ediciones inteligentes te ayudarán a refinar y actualizar tu currículum.
  • Adaptado para empleos: Pega una descripción de puesto y obtén recomendaciones para alinear tu currículum.
  • Calidad LaTeX: Todos los currículums se generan usando LaTeX para un diseño limpio y profesional.
  • Vista previa instantánea: Visualiza tu currículum como PDF en tiempo real.
  • Reinicio fácil: Borra y reinicia tu currículum con un solo clic.
  • Local: Funciona completamente en tu máquina usando Docker.

---

Primeros pasos

Requisitos previos

  • Docker
  • Docker Compose

Configuración

  • Clona el repositorio:
``bash git clone https://github.com/aadya940/autoresume.git cd autoresume `
  • Construye e inicia la aplicación:
`bash docker compose up --build `
  • Accede a las aplicaciones:
_http://localhost:5173_

Uso

  • Abre autoResume en tu navegador.
  • Haz clic en el icono :gear:, pega tu CLAVE DE API DE GOOGLE y tu correo electrónico.
  • Pega enlaces a tu información profesional (por ejemplo, Github, sitio web personal, etc.).
  • Opcionalmente, añade comentarios o el enlace de una oferta de trabajo para una mayor personalización.
  • Haz clic en Generar Currículum y deja que la IA haga el resto.

Guía de contribución

Frontend

  • Implementa componentes de React en la carpeta frontend/src/components/
  • Usamos Chakra UI para implementar los componentes.
  • Importa y utiliza tus componentes en frontend/src/App.jsx

Backend

  • Implementa rutas API en el directorio backend/src/routes/
  • Añade funcionalidad de IA en el directorio backend/src/ai/
  • Usamos el formateador de código black para código Python
  • Mantén los controladores de rutas limpios y la lógica separada

Pautas generales

  • Crea una nueva rama para tu funcionalidad/arreglo: git checkout -b tu-nombre-de-rama`
  • Escribe mensajes de commit claros
  • Prueba tus cambios antes de enviar un PR

Envío de cambios

  • Crea un Pull Request
  • En la descripción de tu PR, incluye:
  • Qué cambios realizaste
  • Por qué los realizaste
  • Cualquier captura de pantalla o resultado de pruebas relevante
  • Cualquier cambio importante o pasos de migración necesarios
