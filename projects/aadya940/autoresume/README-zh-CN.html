autoResume

开源简历生成器，粘贴你的链接，手动编辑，让 AI 帮助你润色、更新并定制你的简历。

如何使用？

Medium 博客链接

该博客关于用户界面和模板部分内容已过时。

功能

  • 便捷链接导入： 从网站粘贴链接，快速生成 PDF 简历。
  • 内置手动编辑器： 使用嵌入式代码编辑器，从零开始或微调你的简历。
  • 智能建议： 提供反馈或新链接，智能编辑帮助你完善和更新简历。
  • 职位定制： 粘贴职位描述，获取简历匹配建议。
  • LaTeX 高质量： 所有简历均通过 LaTeX 生成，布局简洁专业。
  • 即时预览： 实时以 PDF 形式查看你的简历。
  • 一键重置： 一键清空并重新开始你的简历。
  • 本地运行： 通过 Docker 完全在本地机器上运行。

---

快速开始

前置条件

  • Docker
  • Docker Compose

安装

  • 克隆仓库：
``bash git clone https://github.com/aadya940/autoresume.git cd autoresume `
  • 构建并启动应用：
`bash docker compose up --build `
  • 访问应用：
_http://localhost:5173_

使用方法

  • 在浏览器中打开 autoResume。
  • 点击 :gear: 图标，粘贴你的 GOOGLE API KEY 和邮箱地址。
  • 粘贴你的职业信息链接（如 Github、个人网站等）。
  • 可选，添加反馈或职位信息链接以进一步定制。
  • 点击 生成简历，让 AI 完成剩下的工作！
