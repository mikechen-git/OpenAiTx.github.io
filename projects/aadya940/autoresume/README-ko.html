autoResume

오픈 소스 이력서 빌더, 링크를 붙여넣고, 직접 수정하며, AI가 이력서 다듬기, 업데이트, 맞춤화 작업을 도와줍니다.

어떻게 사용하나요?

미디엄 블로그 링크

블로그는 사용자 인터페이스와 사용된 템플릿 측면에서 최신 정보가 아닙니다.

주요 기능

  • 간편한 링크 가져오기: 웹사이트에서 링크를 붙여넣고, PDF 이력서를 빠르게 만드세요.
  • 수동 편집기 포함: 내장 코드 편집기를 통해 이력서를 세밀하게 다듬거나 처음부터 작성할 수 있습니다.
  • 스마트 제안: 피드백이나 새로운 링크를 공유하면, smartedits가 이력서를 개선 및 업데이트하는 데 도움을 줍니다.
  • 채용 맞춤화: 채용 공고를 붙여넣으면, 이력서에 맞는 추천 사항을 제공합니다.
  • LaTeX 품질: 모든 이력서는 LaTeX를 사용하여 깔끔하고 전문적인 레이아웃으로 생성됩니다.
  • 즉각적인 미리보기: 이력서를 실시간 PDF로 확인할 수 있습니다.
  • 간편 초기화: 클릭 한 번으로 이력서를 초기화하고 다시 시작할 수 있습니다.
  • 로컬 실행: Docker를 사용하여 내 컴퓨터에서 완전히 실행됩니다.

---

시작하기

사전 준비

  • Docker
  • Docker Compose

설치 방법

  • 저장소를 클론합니다:
``bash git clone https://github.com/aadya940/autoresume.git cd autoresume `
  • 앱을 빌드하고 시작합니다:
`bash docker compose up --build `
  • 앱에 접속합니다:
_http://localhost:5173_

사용법

  • 브라우저에서 autoResume을 엽니다.
  • :gear: 아이콘을 클릭하고, GOOGLE API KEY와 이메일을 입력합니다.
  • Github, 개인 웹사이트 등 본인 정보를 담은 링크를 붙여넣습니다.
  • 선택적으로, 추가 맞춤화를 위해 피드백이나 채용 공고 링크를 추가할 수 있습니다.
  • 이력서 생성을 클릭하면 AI가 나머지를 처리합니다!

라이선스

이 프로젝트는 Apache 2.0 라이선스 하에 배포됩니다.

기여 안내

프론트엔드

  • frontend/src/components/ 폴더에 React 컴포넌트를 구현합니다.
  • 컴포넌트 구현에 Chakra UI를 사용합니다.
  • frontend/src/App.jsx에서 컴포넌트를 임포트하고 사용합니다.

백엔드

  • backend/src/routes/ 디렉터리에 API 라우트를 구현합니다.
  • backend/src/ai/ 디렉터리에 AI 기능을 추가합니다.
  • Python 코드는 black 코드 포매터를 사용합니다.
  • 라우트 핸들러는 깔끔하게 유지하고, 로직은 분리합니다.
