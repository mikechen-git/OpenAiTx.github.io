autoResume

ओपन-सोर्स रिज़्यूमे बिल्डर, अपने लिंक पेस्ट करें, मैन्युअली एडिट करें, और AI की मदद से अपने रिज़्यूमे को पॉलिश, अपडेट और टेलर करें।

इसका उपयोग कैसे करें?

मीडियम ब्लॉग लिंक

यह ब्लॉग यूजर इंटरफेस और टेम्पलेट के संदर्भ में पुराना है।

विशेषताएँ

  • आसान लिंक इम्पोर्ट: वेबसाइटों से लिंक पेस्ट करें, और जल्दी से एक PDF रिज़्यूमे बनाएं।
  • मैन्युअल एडिटर शामिल: एम्बेडेड कोड एडिटर के साथ अपने रिज़्यूमे को फाइन-ट्यून करें या शुरू से बनाएं।
  • स्मार्ट सुझाव: फीडबैक या नए लिंक साझा करें, और स्मार्टएडिट्स आपके रिज़्यूमे को अपडेट और रिफाइन करने में मदद करेंगे।
  • नौकरियों के लिए टेलर: जॉब डिस्क्रिप्शन पेस्ट करें, और अपने रिज़्यूमे को संरेखित करने के लिए सिफारिशें प्राप्त करें।
  • LaTeX गुणवत्ता: सभी रिज़्यूमे LaTeX का उपयोग करके बनाए जाते हैं, ताकि क्लीन, प्रोफेशनल लेआउट मिले।
  • इंस्टैंट प्रीव्यू: अपने रिज़्यूमे को रियल टाइम में PDF के रूप में देखें।
  • आसान रिसेट: एक क्लिक में रिज़्यूमे क्लियर और रीस्टार्ट करें।
  • स्थानीय: Docker का उपयोग करके पूरी तरह आपकी मशीन पर चलता है।

शुरुआत कैसे करें

आवश्यकताएँ

  • Docker
  • Docker Compose

सेटअप

  1. रिपॉजिटरी क्लोन करें:

    git clone https://github.com/aadya940/autoresume.git
cd autoresume

  2. ऐप को बिल्ड और शुरू करें:

    docker compose up --build

  3. ऐप्स एक्सेस करें:

    http://localhost:5173

उपयोग

  1. अपने ब्राउज़र में autoResume खोलें।
  2. :gear: आइकन पर क्लिक करें, अपना GOOGLE API KEY और अपनी ईमेल पेस्ट करें।
  3. अपनी प्रोफेशनल जानकारी के लिंक पेस्ट करें (जैसे, Github, पर्सनल वेबसाइट, आदि)।
  4. वैकल्पिक रूप से, फीडबैक या जॉब पोस्टिंग लिंक और भी कस्टमाइज़ेशन के लिए जोड़ें।
  5. Generate Resume पर क्लिक करें और बाकी काम AI पर छोड़ दें!
