autoResume

سازنده رزومه متن‌باز، لینک‌های خود را جای‌گذاری کنید، به صورت دستی ویرایش نمایید و اجازه دهید هوش مصنوعی در صیقل دادن، به‌روزرسانی و سفارشی‌سازی رزومه به شما کمک کند.

چگونه از این استفاده کنیم؟

لینک بلاگ در Medium

این بلاگ نسبت به رابط کاربری و قالب فعلی منسوخ شده است.

ویژگی‌ها

  • وارد کردن آسان لینک: لینک‌ها را از وب‌سایت‌ها جای‌گذاری کنید و به سرعت یک رزومه PDF بسازید.
  • ویرایشگر دستی داخلی: رزومه خود را به صورت دقیق تنظیم کنید یا از ابتدا با یک ویرایشگر کد داخلی بسازید.
  • پیشنهادات هوشمند: بازخورد یا لینک جدید به اشتراک بگذارید و ویرایش‌های هوشمند به بهبود و به‌روزرسانی رزومه شما کمک خواهند کرد.
  • سفارشی‌شده برای شغل‌ها: یک شرح شغل جای‌گذاری کنید و پیشنهاداتی برای هماهنگی رزومه خود دریافت کنید.
  • کیفیت LaTeX: تمام رزومه‌ها با استفاده از LaTeX برای چیدمان تمیز و حرفه‌ای تولید می‌شوند.
  • پیش‌نمایش فوری: رزومه خود را به صورت PDF در لحظه مشاهده کنید.
  • بازنشانی آسان: رزومه خود را با یک کلیک پاک و مجدداً شروع کنید.
  • محلی: به طور کامل روی دستگاه شما با استفاده از Docker اجرا می‌شود.

شروع به کار

پیش‌نیازها

  • Docker
  • Docker Compose

راه‌اندازی

  1. مخزن را کلون کنید:

    git clone https://github.com/aadya940/autoresume.git
cd autoresume

  2. برنامه را بسازید و اجرا کنید:

    docker compose up --build

  3. به برنامه‌ها دسترسی پیدا کنید:

    http://localhost:5173

نحوه استفاده

  1. autoResume را در مرورگر خود باز کنید.
  2. روی آیکون :gear: کلیک کنید، کلید API گوگل و ایمیل خود را جای‌گذاری نمایید.
  3. لینک‌های اطلاعات حرفه‌ای خود را جای‌گذاری کنید (مثلاً Github، وب‌سایت شخصی و غیره).
  4. به صورت اختیاری، بازخورد یا لینک آگهی شغلی برای سفارشی‌سازی بیشتر اضافه کنید.
  5. روی ساخت رزومه کلیک کنید و اجازه دهید هوش مصنوعی باقی کارها را انجام دهد!

مجوز

این پروژه تحت مجوز Apache 2.0 منتشر شده است.

راهنمای مشارکت

فرانت‌اند

  • کامپوننت‌های React را در پوشه frontend/src/components/ پیاده‌سازی کنید.
  • ما از Chakra UI برای پیاده‌سازی کامپوننت‌ها استفاده می‌کنیم.
  • کامپوننت‌های خود را در frontend/src/App.jsx وارد و استفاده نمایید.

بک‌اند

  • مسیرهای API را در پوشه backend/src/routes/ پیاده‌سازی کنید.
  • قابلیت‌های هوش مصنوعی را در پوشه backend/src/ai/ اضافه نمایید.
  • ما از فرمت‌کننده کد black برای کدهای Python استفاده می‌کنیم.
  • هندلرهای مسیر را تمیز نگه دارید و منطق را جدا کنید.

دستورالعمل‌های عمومی
