autoResume

منشئ سيرة ذاتية مفتوح المصدر، الصق روابطك، حرّر يدويًا، ودع الذكاء الاصطناعي يساعدك في تحسين، وتحديث، وتخصيص سيرتك الذاتية.

رابط مدونة Medium

المدونة قديمة بالنسبة لواجهة المستخدم والقالب المستخدم.

الميزات

  • استيراد الروابط بسهولة: الصق الروابط من المواقع، واصنع سيرتك الذاتية بصيغة PDF بسرعة.
  • محرر يدوي مدمج: عدّل أو أنشئ سيرتك الذاتية من الصفر بواسطة محرر الكود المدمج.
  • اقتراحات ذكية: شارك التغذية الراجعة أو روابط جديدة، وستساعدك التعديلات الذكية في تحسين وتحديث سيرتك الذاتية.
  • مخصصة للوظائف: الصق وصف الوظيفة، واحصل على توصيات لمواءمة سيرتك الذاتية.
  • جودة LaTeX: جميع السير الذاتية يتم توليدها باستخدام LaTeX لتصميم نظيف واحترافي.
  • معاينة فورية: شاهد سيرتك الذاتية كملف PDF في الوقت الحقيقي.
  • إعادة تعيين بسهولة: امسح وابدأ سيرتك الذاتية بنقرة واحدة.
  • محلي: يعمل بالكامل على جهازك باستخدام Docker.

البدء

المتطلبات المسبقة

  • Docker
  • Docker Compose

الإعداد

  1. استنسخ المستودع:

    git clone https://github.com/aadya940/autoresume.git
cd autoresume

  2. قم ببناء وتشغيل التطبيق:

    docker compose up --build

  3. الوصول إلى التطبيقات:

    http://localhost:5173

طريقة الاستخدام

  1. افتح autoResume في متصفحك.
  2. انقر على أيقونة :gear:، ثم الصق مفتاح GOOGLE API الخاص بك والبريد الإلكتروني الخاص بك.
  3. الصق روابط معلوماتك المهنية (مثل Github، الموقع الشخصي، إلخ).
  4. اختياريًا، أضف تعليقات أو رابط إعلان وظيفة لمزيد من التخصيص.
  5. انقر على توليد السيرة الذاتية ودع الذكاء الاصطناعي يتولى الباقي!
