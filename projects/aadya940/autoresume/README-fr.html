autoResume

Générateur de CV open source : collez vos liens, éditez manuellement et laissez l’IA vous aider à améliorer, mettre à jour et adapter votre CV.

Comment l’utiliser ?

Lien vers l’article Medium

L’article n’est plus à jour concernant l’interface utilisateur et le modèle utilisé.

Fonctionnalités

  • Importation facile de liens : Collez des liens de sites web et créez rapidement un CV PDF.
  • Éditeur manuel inclus : Affinez ou créez votre CV de zéro grâce à un éditeur de code intégré.
  • Suggestions intelligentes : Partagez des retours ou de nouveaux liens, et les « smartedits » aideront à affiner et mettre à jour votre CV.
  • Adapté aux emplois : Collez une description de poste et obtenez des recommandations pour aligner votre CV.
  • Qualité LaTeX : Tous les CV sont générés en LaTeX pour une présentation propre et professionnelle.
  • Aperçu instantané : Visualisez votre CV en PDF en temps réel.
  • Réinitialisation facile : Effacez et recommencez votre CV en un clic.
  • Local : Fonctionne entièrement sur votre machine via Docker.

Démarrage

Prérequis

  • Docker
  • Docker Compose

Installation

  • Clonez le dépôt :
``bash git clone https://github.com/aadya940/autoresume.git cd autoresume `
  • Construisez et lancez l’application :
`bash docker compose up --build `
  • Accédez aux applications :
_http://localhost:5173_

Utilisation

  • Ouvrez autoResume dans votre navigateur.
  • Cliquez sur l’icône :gear:, collez votre CLÉ API GOOGLE et votre email.
  • Collez les liens vers vos informations professionnelles (ex. : Github, site personnel, etc.).
  • Optionnellement, ajoutez des retours ou le lien d’une offre d’emploi pour une personnalisation supplémentaire.
  • Cliquez sur Générer le CV et laissez l’IA faire le reste !

Guide de contribution

Frontend

  • Implémentez les composants React dans le dossier frontend/src/components/
  • Nous utilisons Chakra UI pour les composants.
  • Importez et utilisez vos composants dans frontend/src/App.jsx

Backend

  • Implémentez les routes API dans le dossier backend/src/routes/
  • Ajoutez la fonctionnalité IA dans le dossier backend/src/ai/
  • Nous utilisons le formateur de code black pour le code Python
  • Gardez les gestionnaires de routes propres et séparez la logique

Directives générales

  • Créez une nouvelle branche pour votre fonctionnalité/correctif : git checkout -b votre-nom-de-branche`
  • Rédigez des messages de commit clairs
  • Testez vos modifications avant de soumettre une PR

Soumission des modifications

  • Créez une Pull Request
  • Dans la description de votre PR, incluez :
  • Les modifications apportées
  • Les raisons de ces modifications
  • Toute capture d’écran ou résultat de test pertinent
  • Tout changement majeur ou étape de migration nécessaire
