Ursa

Ursa — это корпоративная библиотека пользовательского интерфейса для создания кроссплатформенных приложений на Avalonia UI.

.NET Foundation

Этот проект поддерживается .NET Foundation.

Начало работы

  1. Ursa

Добавьте пакет nuget:

dotnet add package Irihi.Ursa
  1. Ursa.Themes.Semi

Чтобы элементы управления Ursa появились в вашем приложении, необходимо добавить пакет темы, разработанный для Ursa. Ursa.Themes.Semi — это пакет темы для Ursa, вдохновленный Semi Design. Вы можете добавить его в свой проект, выполнив следующие шаги.

Добавьте пакет nuget:

dotnet add package Semi.Avalonia
dotnet add package Irihi.Ursa.Themes.Semi

Добавьте стили в приложение:

<Application...
    xmlns:semi="https://irihi.tech/semi"
    xmlns:u-semi="https://irihi.tech/ursa/themes/semi"
    ....>

    <Application.Styles>
        <semi:SemiTheme Locale="zh-CN" />
        <u-semi:SemiTheme Locale="zh-CN"/>
    </Application.Styles>

Теперь вы можете использовать элементы управления Ursa в вашем Avalonia-приложении.

<Window
    ...
    xmlns:u="https://irihi.tech/ursa"
    ...>
    <StackPanel Margin="20">
        <u:TagInput />
    </StackPanel>
</Window>

Кодекс поведения

В этом проекте принят кодекс поведения Contributor Covenant для разъяснения ожидаемого поведения в нашем сообществе. Для получения дополнительной информации см. .NET Foundation Code of Conduct.

Уведомление о совместимости

Ursa в настоящее время совместима с Avalonia 11.1.x и 11.2.x, но не поддерживает Avalonia 11.2.0.

Расширения

Prism Extension

Если вам необходимо интегрировать Ursa с Prism.Avalonia, вы можете использовать пакет Irihi.Ursa.PrismExtension. Этот пакет предоставляет сервисы, связанные с диалогами, для использования Ursa Dialogs в стиле Prism.

ReactiveUI Extension

Если вам нужно интегрировать Ursa с Avalonia.ReactiveUI, вы можете использовать пакет Irihi.Ursa.ReactiveUIExtension. Этот пакет реализует версии UrsaWindow и UrsaView для ReactiveUI. Подробности см. на wiki.
