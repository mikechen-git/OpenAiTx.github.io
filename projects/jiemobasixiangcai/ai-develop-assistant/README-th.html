🚀 MCP AI开发助手

เครื่องมือ MCP สำหรับช่วยนักพัฒนา AI ในการวิเคราะห์ความต้องการแบบอัจฉริยะและออกแบบสถาปัตยกรรม

✨ คุณสมบัติหลัก

  • การชี้แจงความต้องการอัจฉริยะ: ระบุประเภทโครงการโดยอัตโนมัติ สร้างคำถามที่ตรงประเด็น
  • การจัดการแบบรับรู้สาขา: ติดตามเป้าหมายโครงการ การออกแบบฟังก์ชัน ความชอบทางเทคนิค การออกแบบ UI ฯลฯ
  • การสร้างสถาปัตยกรรมอัตโนมัติ: สร้างข้อเสนอแนะสถาปัตยกรรมเทคนิคจากความต้องการที่ครบถ้วน
  • การจัดเก็บอย่างถาวร: บันทึกผลการวิเคราะห์โดยอัตโนมัติ รองรับการส่งออกเอกสาร

📁 การตั้งค่าอย่างรวดเร็ว

การตั้งค่ารุ่นเก่า

  • โคลนโค้ด
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • แนะนำให้ใช้ virtual environment
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • ติดตั้ง dependencies
`bash pip install -r requirements.txt `

  • ตำแหน่งไฟล์ config
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • เพิ่มการตั้งค่า
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • รีสตาร์ท Claude Desktop

การตั้งค่ารุ่นใหม่

#### 🔧 เครื่องมือหลัก
  • start_new_project - เริ่มโครงการใหม่
  • create_requirement_blueprint - สร้าง Blueprint ความต้องการ
  • requirement_clarifier - รับข้อเสนอแนะการชี้แจงความต้องการ
  • save_clarification_tasks - บันทึกงานชี้แจง
  • update_branch_status - อัปเดตสถานะสาขา
  • requirement_manager - ตัวจัดการเอกสารความต้องการ
  • check_architecture_prerequisites - ตรวจสอบเงื่อนไขเบื้องต้นของสถาปัตยกรรม
  • get_architecture_design_prompt - รับข้อเสนอแนะการออกแบบสถาปัตยกรรม
  • save_generated_architecture - บันทึกสถาปัตยกรรมที่สร้างขึ้น
  • export_final_document - ส่งออกเอกสารฉบับสมบูรณ์
  • view_requirements_status - ดูสถานะความต้องการ

#### การตั้งค่า (เชื่อมต่อโดยตรงแบบ remote ให้คัดลอกไปยังเครื่องมือของคุณ แล้วแทนที่ MCP_STORAGE_DIR ด้วยไดเรกทอรีในเครื่องของคุณ) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 ขั้นตอนการใช้งาน

ขั้นตอนพื้นฐาน

  • การชี้แจงความต้องการ
` requirement_clarifier("我要做一个在线教育平台") `

  • การจัดการความต้องการ
` requirement_manager("目标用户：学生和教师", "项目概述") `

  • ดูสถานะ
` view_requirements_status() `
  • ออกแบบสถาปัตยกรรม
` architecture_designer("สถาปัตยกรรมแพลตฟอร์มการศึกษาออนไลน์") `

  • ส่งออกเอกสาร
` export_final_document() `

🚀 เริ่มต้นใช้งาน

เริ่มต้นอย่างรวดเร็ว

  • กำหนดค่า Claude Desktop (ดูวิธีการกำหนดค่าด้านบน)
  • รีสตาร์ท Claude Desktop
  • เริ่มการวิเคราะห์ความต้องการอัจฉริยะ：
` requirement_clarifier("อธิบายแนวคิดโครงการของคุณ") `
  • ทำตามคำแนะนำอัจฉริยะของ AI เพื่อปรับแต่งแต่ละสาขาความต้องการอย่างเป็นขั้นตอน
  • ส่งออกเอกสารฉบับสมบูรณ์：
` export_final_document() `

แนวปฏิบัติที่ดีที่สุด

  • 💬 เชื่อมั่นในการจัดการสาขาของ AI: ให้ AI แนะนำคุณจนเสร็จสิ้นทุกสาขาความต้องการ
  • 🎯 แสดงความชอบอย่างชัดเจน: ระบุความชอบของคุณอย่างชัดเจนในเรื่องการเลือกเทคโนโลยี สไตล์ UI ฯลฯ
  • 📊 ตรวจสอบสถานะเป็นระยะ: ใช้ view_requirements_status` เพื่อติดตามความคืบหน้า
  • 🤖 อนุญาตให้ AI ตัดสินใจเมื่อเหมาะสม: สำหรับส่วนที่ไม่แน่ใจ สามารถบอกว่า "ใช้วิธีมาตรฐาน"
