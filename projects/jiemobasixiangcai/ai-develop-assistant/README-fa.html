🚀 MCP دستیار توسعه AI

ابزار MCP برای کمک به توسعه‌دهندگان AI در تحلیل نیازمندی‌ها و طراحی معماری هوشمند

✨ ویژگی‌های کلیدی

  • توضیح هوشمند نیازمندی‌ها: شناسایی خودکار نوع پروژه و تولید سوالات هدفمند
  • مدیریت شاخه آگاهانه: پیگیری اهداف پروژه، طراحی عملکرد، ترجیحات فنی، طراحی UI و سایر ابعاد
  • تولید خودکار معماری: تولید طرح معماری فنی بر اساس نیازمندی‌های کامل
  • ذخیره‌سازی پایدار: ذخیره خودکار نتایج تحلیل و پشتیبانی از خروجی گرفتن سند

📁 پیکربندی سریع

پیکربندی نسخه قدیمی

  • کلون کردن کد
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • پیشنهاد استفاده از محیط مجازی
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • نصب وابستگی‌ها
`bash pip install -r requirements.txt `

  • مکان فایل پیکربندی
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • افزودن پیکربندی
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • راه‌اندازی مجدد Claude Desktop

پیکربندی نسخه جدید

#### 🔧 ابزارهای کلیدی
  • start_new_project - شروع پروژه جدید
  • create_requirement_blueprint - ایجاد نقشه نیازمندی‌ها
  • requirement_clarifier - دریافت پیشنهادات توضیح نیازمندی
  • save_clarification_tasks - ذخیره وظایف توضیح نیازمندی‌ها
  • update_branch_status - به‌روزرسانی وضعیت شاخه‌ها
  • requirement_manager - مدیریت اسناد نیازمندی
  • check_architecture_prerequisites - بررسی پیش‌نیازهای معماری
  • get_architecture_design_prompt - دریافت پیشنهاد طراحی معماری
  • save_generated_architecture - ذخیره معماری تولیدشده
  • export_final_document - خروجی گرفتن سند کامل
  • view_requirements_status - مشاهده وضعیت نیازمندی‌ها

#### پیکربندی (اتصال مستقیم از راه دور، این تنظیم را در ابزار خود کپی کنید و MCP_STORAGE_DIR را با مسیر محلی خود جایگزین کنید) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 روند استفاده

مراحل پایه

  • توضیح نیازمندی
` requirement_clarifier("می‌خواهم یک پلتفرم آموزش آنلاین بسازم") `

  • مدیریت نیازمندی
` requirement_manager("کاربران هدف: دانش‌آموزان و معلمان", "خلاصه پروژه") `

  • مشاهده وضعیت
` view_requirements_status() `
  • طراحی معماری
` architecture_designer("معماری پلتفرم آموزش آنلاین") `

  • خروجی گرفتن از سند
` export_final_document() `

🚀 شروع استفاده

شروع سریع

  • پیکربندی Claude Desktop (طبق روش پیکربندی بالا)
  • راه‌اندازی مجدد Claude Desktop
  • شروع تحلیل هوشمند نیازمندی‌ها:
` requirement_clarifier("ایده پروژه خود را توصیف کنید") `
  • گام به گام زیر نظر راهنمایی هوشمند AI، هر شاخه نیازمندی را تکمیل کنید
  • خروجی گرفتن از سند کامل:
` export_final_document() `

بهترین شیوه‌ها

  • 💬 به مدیریت شاخه‌های AI اعتماد کنید: اجازه دهید AI شما را در تکمیل تمام شاخه‌های نیازمندی هدایت کند
  • 🎯 ترجیحات خود را شفاف بیان کنید: نسبت به انتخاب فناوری، سبک UI و غیره نظر خود را صریح بیان کنید
  • 📊 به‌طور منظم وضعیت را بررسی کنید: با استفاده از view_requirements_status` پیشرفت را مشاهده کنید
  • 🤖 در صورت نیاز اختیار را به AI بدهید: در بخش‌هایی که مطمئن نیستید بگویید "از راه‌حل متداول استفاده کن"
