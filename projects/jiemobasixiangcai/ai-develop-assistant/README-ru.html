🚀 MCP AI-разработчик-помощник

MCP-инструмент для поддержки AI-разработчиков в интеллектуальном анализе требований и проектировании архитектуры

✨ Ключевые особенности

  • Интеллектуальное уточнение требований: Автоматически определяет тип проекта, генерирует целевые вопросы
  • Ветвленное управление: Отслеживание целей проекта, проектирования функций, технических предпочтений, UI-дизайна и других аспектов
  • Автоматическая генерация архитектуры: Генерация технической архитектуры на основе полного набора требований
  • Персистентное хранение: Автоматическое сохранение результатов анализа, поддержка экспорта документов

📁 Быстрая настройка

Настройка старой версии

  • Клонирование репозитория
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • Рекомендуемое виртуальное окружение
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • Установка зависимостей
`bash pip install -r requirements.txt `

  • Расположение конфигурационного файла
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • Добавление конфигурации
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • Перезапуск Claude Desktop

Настройка новой версии

#### 🔧 Ключевые инструменты
  • start_new_project - начать новый проект
  • create_requirement_blueprint - создать план требований
  • requirement_clarifier - получить подсказки для уточнения требований
  • save_clarification_tasks - сохранить задачи по уточнению
  • update_branch_status - обновить статус ветки
  • requirement_manager - менеджер документации требований
  • check_architecture_prerequisites - проверить предварительные условия для архитектуры
  • get_architecture_design_prompt - получить подсказку по проектированию архитектуры
  • save_generated_architecture - сохранить сгенерированную архитектуру
  • export_final_document - экспортировать финальный документ
  • view_requirements_status - просмотреть статус требований

#### Конфигурация (скопируйте для прямого подключения, замените MCP_STORAGE_DIR на ваш локальный каталог) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 Процесс использования

Основные шаги

  • Уточнение требований
` requirement_clarifier("Я хочу создать онлайн-образовательную платформу") `

  • Управление требованиями
` requirement_manager("Целевая аудитория: студенты и преподаватели", "Обзор проекта") `

  • Просмотр статуса
` view_requirements_status() `
  • Архитектурное проектирование
` architecture_designer("Архитектура онлайн-образовательной платформы") `

  • Экспорт документа
` export_final_document() `

🚀 Начало работы

Быстрый старт

  • Настройте Claude Desktop (см. способ настройки выше)
  • Перезапустите Claude Desktop
  • Запустите интеллектуальный анализ требований:
` requirement_clarifier("Опишите вашу проектную идею") `
  • Следуйте умным подсказкам ИИ, поэтапно дорабатывайте каждую ветку требований
  • Экспортируйте полный документ:
` export_final_document() `

Лучшие практики

  • 💬 Доверьтесь управлению ветвями от ИИ: позвольте ИИ вести вас по всем веткам требований
  • 🎯 Четко выражайте предпочтения: по выбору технологий, стилю UI и т.д.
  • 📊 Регулярно проверяйте статус: используйте view_requirements_status` для отслеживания прогресса
  • 🤖 Давайте ИИ разумные полномочия: по неясным вопросам можно сказать "использовать стандартное решение"
