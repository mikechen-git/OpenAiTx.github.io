ai-develop-assistant

⭐ 103 stars Polish by jiemobasixiangcai

🚀 MCP AI开发助手

Narzędzie MCP wspomagające programistów AI w inteligentnej analizie wymagań i projektowaniu architektury

✨ Kluczowe funkcje

  • Inteligentne doprecyzowanie wymagań: Automatyczne rozpoznawanie typu projektu, generowanie ukierunkowanych pytań
  • Zarządzanie gałęziami z uwzględnieniem kontekstu: Śledzenie celów projektu, projektowania funkcjonalności, preferencji technologicznych, projektu UI i innych aspektów
  • Automatyczne generowanie architektury: Tworzenie rozwiązań architektury technicznej na podstawie kompletnych wymagań
  • Trwałe przechowywanie: Automatyczne zapisywanie wyników analizy, obsługa eksportu dokumentów

📁 Szybka konfiguracja

Konfiguracja starej wersji

  • Klonowanie kodu
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • Zalecane środowisko wirtualne
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • Instalacja zależności
`bash pip install -r requirements.txt `

  • Lokalizacja pliku konfiguracyjnego
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • Dodanie konfiguracji
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • Restart Claude Desktop

Konfiguracja nowej wersji

#### 🔧 Kluczowe narzędzia
  • start_new_project - Rozpocznij nowy projekt
  • create_requirement_blueprint - Utwórz szablon wymagań
  • requirement_clarifier - Uzyskaj podpowiedzi do doprecyzowania wymagań
  • save_clarification_tasks - Zapisz zadania doprecyzowujące
  • update_branch_status - Aktualizuj status gałęzi
  • requirement_manager - Menedżer dokumentów wymagań
  • check_architecture_prerequisites - Sprawdź warunki wstępne dla architektury
  • get_architecture_design_prompt - Uzyskaj podpowiedzi do projektowania architektury
  • save_generated_architecture - Zapisz wygenerowaną architekturę
  • export_final_document - Eksportuj kompletny dokument
  • view_requirements_status - Przeglądaj status wymagań

#### Konfiguracja (skopiuj bezpośrednio do swojego narzędzia, zamień MCP_STORAGE_DIR na swój lokalny katalog) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 Przebieg użytkowania

Podstawowe kroki

  • Doprecyzowanie wymagań
` requirement_clarifier("Chcę stworzyć platformę edukacji online") `

  • Zarządzanie wymaganiami
` requirement_manager("Docelowi użytkownicy: uczniowie i nauczyciele", "Opis projektu") `

  • Podgląd statusu
` view_requirements_status() `
  • Projektowanie architektury
` architecture_designer("Architektura platformy edukacji online") `

  • Eksportuj dokument
` export_final_document() `

🚀 Rozpocznij korzystanie

Szybki start

  • Skonfiguruj Claude Desktop (patrz metoda konfiguracji powyżej)
  • Zrestartuj Claude Desktop
  • Rozpocznij inteligentną analizę wymagań:
` requirement_clarifier("Opisz swój pomysł na projekt") `
  • Podążaj za inteligentnym prowadzeniem AI, stopniowo uzupełniając każdą gałąź wymagań
  • Eksportuj pełny dokument:
` export_final_document() `

Najlepsze praktyki

  • 💬 Zaufaj zarządzaniu gałęziami AI: Pozwól AI poprowadzić Cię przez wszystkie gałęzie wymagań
  • 🎯 Wyrażaj jasno swoje preferencje: Wyraźnie określ preferencje dotyczące wyboru technologii, stylu UI itp.
  • 📊 Regularnie sprawdzaj status: Używaj view_requirements_status`, aby monitorować postęp
  • 🤖 Odpowiednio upoważniaj AI: W przypadku niepewnych kwestii możesz powiedzieć "użyj standardowego rozwiązania"
