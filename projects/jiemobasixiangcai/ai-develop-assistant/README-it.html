🚀 MCP Assistente di Sviluppo AI

Strumento MCP per assistere gli sviluppatori AI nell'analisi intelligente dei requisiti e nella progettazione dell'architettura

✨ Caratteristiche principali

  • Chiarimento intelligente dei requisiti: Identificazione automatica del tipo di progetto, generazione di domande mirate
  • Gestione ramificata consapevole: Tracciamento degli obiettivi di progetto, progettazione delle funzionalità, preferenze tecniche, design UI e altro
  • Generazione automatica dell'architettura: Generazione di soluzioni architetturali tecniche basate su requisiti completi
  • Memorizzazione persistente: Salvataggio automatico dei risultati dell'analisi, supporto per l'esportazione dei documenti

📁 Configurazione rapida

Configurazione versione precedente

  • Clona il codice
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • Ambiente virtuale consigliato
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • Installa le dipendenze
`bash pip install -r requirements.txt `

  • Posizione del file di configurazione
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • Aggiungi configurazione
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • Riavvia Claude Desktop

Configurazione nuova versione

#### 🔧 Strumenti principali
  • start_new_project - Avvia un nuovo progetto
  • create_requirement_blueprint - Crea un blueprint dei requisiti
  • requirement_clarifier - Ottieni suggerimenti per il chiarimento dei requisiti
  • save_clarification_tasks - Salva i task di chiarimento
  • update_branch_status - Aggiorna lo stato della branch
  • requirement_manager - Gestore dei documenti dei requisiti
  • check_architecture_prerequisites - Verifica i prerequisiti dell'architettura
  • get_architecture_design_prompt - Ottieni suggerimenti per la progettazione dell'architettura
  • save_generated_architecture - Salva la progettazione architetturale generata
  • export_final_document - Esporta il documento completo
  • view_requirements_status - Visualizza lo stato dei requisiti

#### Configurazione (collega direttamente in remoto nel tuo strumento, sostituisci MCP_STORAGE_DIR con la tua directory locale) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 Flusso di utilizzo

Passaggi di base

  • Chiarimento dei requisiti
` requirement_clarifier("Voglio creare una piattaforma di educazione online") `

  • Gestione dei requisiti
` requirement_manager("Utenti target: studenti e insegnanti", "Panoramica del progetto") `

  • Visualizza stato
` view_requirements_status() `
  • Progettazione dell’architettura
` architecture_designer("Architettura della piattaforma di educazione online") `

  • Esporta documento
` export_final_document() `

🚀 Inizia a usare

Guida rapida

  • Configura Claude Desktop (consulta il metodo di configurazione sopra)
  • Riavvia Claude Desktop
  • Avvia l’analisi intelligente dei requisiti:
` requirement_clarifier("Descrivi la tua idea di progetto") `
  • Segui la guida intelligente dell’AI per perfezionare gradualmente ogni ramo dei requisiti
  • Esporta il documento completo:
` export_final_document() `

Best practice

  • 💬 Fidati della gestione dei rami da parte dell’AI: lascia che l’AI ti guidi nel completare tutti i rami dei requisiti
  • 🎯 Esprimi chiaramente le tue preferenze: specifica chiaramente le preferenze per la scelta tecnologica, lo stile UI, ecc.
  • 📊 Controlla regolarmente lo stato: usa view_requirements_status` per monitorare i progressi
  • 🤖 Autorizza opportunamente l’AI: per le parti incerte puoi dire "usa la soluzione convenzionale"
