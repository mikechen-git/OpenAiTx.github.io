🚀 MCP AI Assistente de Desenvolvimento

Ferramenta MCP para auxiliar desenvolvedores de IA na análise inteligente de requisitos e no design de arquitetura

✨ Recursos Principais

  • Esclarecimento Inteligente de Requisitos: Identifica automaticamente o tipo de projeto e gera perguntas direcionadas
  • Gestão de Ramo Sensível: Acompanha objetivos do projeto, design de funcionalidades, preferências técnicas, design UI, etc.
  • Geração Automática de Arquitetura: Gera soluções de arquitetura técnica com base nos requisitos completos
  • Armazenamento Persistente: Salva automaticamente os resultados da análise e suporta exportação de documentos

📁 Configuração Rápida

Configuração da Versão Antiga

  • Clonar o Código
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • Ambiente Virtual Recomendado
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • Instalar Dependências
`bash pip install -r requirements.txt `

  • Local do Arquivo de Configuração
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • Adicionar Configuração
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • Reinicie o Claude Desktop

Configuração da Nova Versão

#### 🔧 Ferramentas Principais
  • start_new_project - Iniciar novo projeto
  • create_requirement_blueprint - Criar blueprint de requisitos
  • requirement_clarifier - Obter dicas de esclarecimento de requisitos
  • save_clarification_tasks - Salvar tarefas de esclarecimento
  • update_branch_status - Atualizar status do ramo
  • requirement_manager - Gerenciador de documentação de requisitos
  • check_architecture_prerequisites - Verificar pré-requisitos de arquitetura
  • get_architecture_design_prompt - Obter dica de design de arquitetura
  • save_generated_architecture - Salvar design de arquitetura gerado
  • export_final_document - Exportar documento completo
  • view_requirements_status - Visualizar status dos requisitos

#### Configuração (copie para sua ferramenta, substitua MCP_STORAGE_DIR pelo seu diretório local) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 Fluxo de Utilização

Passos Básicos

  • Esclarecimento de Requisitos
` requirement_clarifier("Quero criar uma plataforma de educação online") `

  • Gestão de Requisitos
` requirement_manager("Usuários alvo: estudantes e professores", "Visão geral do projeto") `

  • Visualizar Status
` view_requirements_status() `
  • Design de Arquitetura
` architecture_designer("Arquitetura da Plataforma de Educação Online") `

  • Exportar Documento
` export_final_document() `

🚀 Comece a Usar

Guia Rápido

  • Configurar o Claude Desktop (consulte o método de configuração acima)
  • Reinicie o Claude Desktop
  • Comece a Análise Inteligente de Requisitos:
` requirement_clarifier("Descreva sua ideia de projeto") `
  • Siga a orientação inteligente da IA, aprimorando gradualmente cada ramificação dos requisitos
  • Exporte o documento completo:
` export_final_document() `

Melhores Práticas

  • 💬 Confie na gestão de ramificações da IA: deixe a IA guiá-lo em todos os ramos de requisitos
  • 🎯 Expresse preferências claramente: deixe claras suas preferências quanto à escolha de tecnologia, estilo de UI etc.
  • 📊 Verifique o status regularmente: use view_requirements_status` para acompanhar o progresso
  • 🤖 Autorize a IA quando necessário: para partes incertas, diga "use a solução padrão"
