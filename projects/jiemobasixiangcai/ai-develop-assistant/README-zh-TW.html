🚀 MCP AI開發助手

協助AI開發者進行智能化需求分析與架構設計的MCP工具

✨ 核心特性

  • 智能需求澄清: 自動識別專案類型，生成針對性問題
  • 分支感知管理: 追蹤專案目標、功能設計、技術偏好、UI設計等維度
  • 架構自動生成: 基於完整需求生成技術架構方案
  • 持久化儲存: 自動保存分析結果，支援匯出文件

📁 快速配置

舊版本配置

  • 克隆代碼
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • 推薦虛擬環境
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • 安裝依賴
`bash pip install -r requirements.txt `

  • 配置文件位置
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • 添加配置
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • 重啟Claude Desktop

新版本配置

#### 🔧 核心工具
  • start_new_project - 開始新專案
  • create_requirement_blueprint - 建立需求藍圖
  • requirement_clarifier - 獲取需求澄清提示
  • save_clarification_tasks - 保存澄清任務
  • update_branch_status - 更新分支狀態
  • requirement_manager - 需求文件管理器
  • check_architecture_prerequisites - 檢查架構前置條件
  • get_architecture_design_prompt - 獲取架構設計提示
  • save_generated_architecture - 保存生成的架構設計
  • export_final_document - 匯出完整文件
  • view_requirements_status - 查看需求狀態

#### 配置（遠端直連複製到你的工具中，將MCP_STORAGE_DIR替換為你的本地目錄） `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 使用流程

基本步驟

  • 需求澄清
` requirement_clarifier("我要做一個線上教育平台") `

  • 需求管理
` requirement_manager("目標用戶：學生和教師", "專案概述") `

  • 查看狀態
` view_requirements_status() `

  • 架構設計
` architecture_designer("線上教育平台架構") `

  • 導出文件
` export_final_document() `

🚀 開始使用

快速上手

  • 配置Claude Desktop（參考上面的配置方法）
  • 重啟Claude Desktop
  • 開始智慧需求分析：
` requirement_clarifier("描述你的專案想法") `
  • 跟隨AI的智慧引導，逐步完善各個需求分支
  • 導出完整文件：
` export_final_document() `

最佳實踐

  • 💬 信任AI的分支管理：讓AI引導你完成所有需求分支
  • 🎯 明確表達偏好：對技術選型、UI風格等明確表達偏好
  • 📊 定期查看狀態：使用 view_requirements_status` 了解進度
  • 🤖 適當授權AI：對不確定的部分可以說「用常規方案」
