🚀 MCP AI Pengembang Asisten

Alat MCP untuk membantu pengembang AI dalam analisis kebutuhan cerdas dan desain arsitektur

✨ Fitur Utama

  • Klarifikasi Kebutuhan Cerdas: Mengidentifikasi jenis proyek secara otomatis, menghasilkan pertanyaan yang relevan
  • Manajemen Cabang yang Sadar: Melacak tujuan proyek, desain fungsi, preferensi teknologi, desain UI, dan dimensi lainnya
  • Generasi Arsitektur Otomatis: Menghasilkan solusi arsitektur teknis berdasarkan kebutuhan yang lengkap
  • Penyimpanan Persisten: Secara otomatis menyimpan hasil analisis, mendukung ekspor dokumen

📁 Konfigurasi Cepat

Konfigurasi Versi Lama

  • Kloning kode
``bash git clone https://github.com/jiemobasixiangcai/ai-develop-assistant.git `
  • Rekomendasi lingkungan virtual
`bash python -m venv venv source venv/bin/activate # Unix/Linux/MacOS venv\Scripts\activate # Windows `
  • Instal dependensi
`bash pip install -r requirements.txt `

  • Lokasi file konfigurasi
` Windows: %APPDATA%\Claude\claude_desktop_config.json macOS: ~/Library/Application Support/Claude/claude_desktop_config.json `

  • Tambahkan konfigurasi
`json { "mcpServers": { "ai-develop-assistant": { "command": "python", "args": ["path/to/AIDevlopStudy.py"], "env": { "MCP_STORAGE_DIR": "./mcp_data" } } } } `

  • Restart Claude Desktop

Konfigurasi Versi Baru

#### 🔧 Alat Inti
  • start_new_project - Mulai proyek baru
  • create_requirement_blueprint - Buat blueprint kebutuhan
  • requirement_clarifier - Dapatkan prompt klarifikasi kebutuhan
  • save_clarification_tasks - Simpan tugas klarifikasi
  • update_branch_status - Perbarui status cabang
  • requirement_manager - Pengelola dokumen kebutuhan
  • check_architecture_prerequisites - Periksa prasyarat arsitektur
  • get_architecture_design_prompt - Dapatkan prompt desain arsitektur
  • save_generated_architecture - Simpan desain arsitektur yang dihasilkan
  • export_final_document - Ekspor dokumen lengkap
  • view_requirements_status - Lihat status kebutuhan

#### Konfigurasi (salin koneksi langsung jarak jauh ke alat Anda, ganti MCP_STORAGE_DIR dengan direktori lokal Anda) `json { "mcpServers": { "ai-develop-assistant": { "command": "uvx", "args": ["ai-develop-assistant@latest"], "env": { "MCP_STORAGE_DIR": "/path/to/your/storage" } } } } `

🎯 Alur Penggunaan

Langkah Dasar

  • Klarifikasi Kebutuhan
` requirement_clarifier("Saya ingin membuat sebuah platform pendidikan online") `

  • Manajemen Kebutuhan
` requirement_manager("Pengguna target: siswa dan guru", "Ringkasan proyek") `

  • Lihat Status
` view_requirements_status() `

  • Desain Arsitektur
` architecture_designer("Arsitektur Platform Pendidikan Online") `

  • Ekspor Dokumen
` export_final_document() `

🚀 Mulai Menggunakan

Cepat Mulai

  • Konfigurasi Claude Desktop (lihat metode konfigurasi di atas)
  • Restart Claude Desktop
  • Mulai Analisis Kebutuhan Cerdas:
` requirement_clarifier("Deskripsikan ide proyek Anda") `
  • Ikuti Panduan Cerdas AI untuk menyempurnakan setiap cabang kebutuhan secara bertahap
  • Ekspor Dokumen Lengkap:
` export_final_document() `

Praktik Terbaik

  • 💬 Percayakan Manajemen Cabang pada AI: Biarkan AI membimbing Anda menyelesaikan semua cabang kebutuhan
  • 🎯 Ekspresikan Preferensi secara Jelas: Ungkapkan preferensi Anda secara jelas untuk pemilihan teknologi, gaya UI, dll.
  • 📊 Pantau Status Secara Berkala: Gunakan view_requirements_status` untuk memantau progres
  • 🤖 Berikan Wewenang pada AI Secara Tepat: Untuk bagian yang tidak pasti, Anda bisa mengatakan "gunakan solusi konvensional"
