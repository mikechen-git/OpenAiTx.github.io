🗂️Program daftar file yang mendukung banyak penyimpanan, didukung oleh Gin dan Solidjs.

Fitur

  • Banyak penyimpanan
    • Penyimpanan lokal
    • Aliyundrive
    • OneDrive / Sharepoint (global, cn,de,us)
    • 189cloud (Personal, Family)
    • GoogleDrive
    • 123pan
    • FTP / SFTP
    • PikPak
    • S3
    • Seafile
    • UPYUN Storage Service
    • WebDav(Dukung OneDrive/SharePoint tanpa API)
    • Teambition(China,Internasional)
    • Mediatrack
    • 139yun (Pribadi, Keluarga, Grup)
    • YandexDisk
    • BaiduNetdisk
    • Terabox
    • UC
    • Quark
    • Thunder
    • Lanzou
    • ILanzou
    • Aliyundrive share
    • Google photo
    • Mega.nz
    • Baidu photo
    • SMB
    • 115
    • Cloudreve
    • Dropbox
    • FeijiPan
    • dogecloud
    • Azure Blob Storage
  • Mudah dipasang dan langsung digunakan
  • Pratinjau file (PDF, markdown, kode, teks biasa, ...)
  • Pratinjau gambar dalam mode galeri
  • Pratinjau video dan audio, dukung lirik dan subtitle
  • Pratinjau dokumen Office (docx, pptx, xlsx, ...)
  • Pratinjau rendering README.md
  • Salin permalink file dan unduh file langsung
  • Mode gelap
  • I18n
  • Rute terlindungi (perlindungan kata sandi dan autentikasi)
  • WebDav (lihat https://alist.nn.ci/guide/webdav.html untuk detail)
  • Deploy Docker
  • Proxy Cloudflare Workers
  • Unduh paket File/Folder
  • Unggah Web (Dapat mengizinkan pengunjung mengunggah), hapus, mkdir, rename, pindahkan dan salin
  • Unduh offline
  • Salin file antar dua penyimpanan
  • Akselerasi pengunduhan multi-thread untuk pengunduhan/stream thread tunggal
