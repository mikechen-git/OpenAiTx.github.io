GitHub Repository: AlistGo/alist

🗂️Un programa de listado de archivos que soporta múltiples almacenamientos, impulsado por Gin y Solidjs.

English | 中文 | 日本語 | Contributing | CODE_OF_CONDUCT

Características

  • Múltiples almacenamientos
    • Almacenamiento local
    • Aliyundrive
    • OneDrive / Sharepoint (global, cn,de,us)
    • 189cloud (Personal, Familiar)
    • GoogleDrive
    • 123pan
    • FTP / SFTP
    • PikPak
    • S3
    • Seafile
    • UPYUN Storage Service
    • WebDav (Soporta OneDrive/SharePoint sin API)
    • Teambition(China,Internacional)
    • Mediatrack
    • 139yun (Personal, Familiar, Grupo)
    • YandexDisk
    • BaiduNetdisk
    • Terabox
    • UC
    • Quark
    • Thunder
    • Lanzou
    • ILanzou
    • Aliyundrive share
    • Google photo
    • Mega.nz
    • Baidu photo
    • SMB
    • 115
    • Cloudreve
    • Dropbox
    • FeijiPan
    • dogecloud
    • Azure Blob Storage
  • Fácil de desplegar y listo para usar
  • Vista previa de archivos (PDF, markdown, código, texto plano, ...)
  • Vista previa de imágenes en modo galería
  • Vista previa de video y audio, soporta letras y subtítulos
  • Vista previa de documentos de Office (docx, pptx, xlsx, ...)
  • Renderizado de vista previa de README.md
  • Copia de enlace permanente de archivo y descarga directa de archivos
  • Modo oscuro
  • I18n
  • Rutas protegidas (protección por contraseña y autenticación)
  • WebDav (ver https://alist.nn.ci/guide/webdav.html para más detalles)
  • Despliegue en Docker
  • Proxy de Cloudflare Workers
  • Descarga de paquetes de archivos/carpetas
  • Subida web (puede permitir a los visitantes subir), eliminar, mkdir, renombrar, mover y copiar
  • Descarga offline
  • Copiar archivos entre dos almacenamientos
  • Aceleración de descarga multihilo para descarga/stream de un solo hilo
