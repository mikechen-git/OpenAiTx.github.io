  • Facile da distribuire e pronto all’uso
  • Anteprima file (PDF, markdown, codice, testo semplice, ...)
  • Anteprima immagini in modalità galleria
  • Anteprima video e audio, supporta testi e sottotitoli
  • Anteprima documenti Office (docx, pptx, xlsx, ...)
  • Rendering anteprima README.md
  • Copia link permanente file e download diretto file
  • Modalità scura
  • I18n
  • Percorsi protetti (protezione con password e autenticazione)
  • WebDav (vedi https://alist.nn.ci/guide/webdav.html per dettagli)
  • Deploy Docker
  • Proxy Cloudflare Workers
  • Download pacchetto file/cartella
  • Upload web (può consentire ai visitatori di caricare), elimina, mkdir, rinomina, sposta e copia
  • Download offline
  • Copia file tra due storage
  • Accelerazione download multi-thread per download/stream a thread singolo
