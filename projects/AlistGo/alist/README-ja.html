GitHub Repository: AlistGo/alist

🗂️GinとSolidjsを利用した、マルチストレージ対応のファイルリストプログラム。

English | 中文 | 日本語 | Contributing | CODE_OF_CONDUCT

特徴

  • 複数のストレージに対応
    • ローカルストレージ
    • Aliyundrive
    • OneDrive / Sharepoint (グローバル, 中国,de,us)
    • 189cloud（個人、ファミリー）
    • GoogleDrive
    • 123pan
    • FTP / SFTP
    • PikPak
    • S3
    • Seafile
    • UPYUN Storage Service
    • WebDav（APIなしでOneDrive/SharePointをサポート）
    • Teambition（中国、国際）
    • Mediatrack
    • 139yun（個人、ファミリー、グループ）
    • YandexDisk
    • BaiduNetdisk
    • Terabox
    • UC
    • Quark
    • Thunder
    • Lanzou
    • ILanzou
    • Aliyundrive share
    • Google photo
    • Mega.nz
    • Baidu photo
    • SMB
    • 115
    • Cloudreve
    • Dropbox
    • FeijiPan
    • dogecloud
    • Azure Blob Storage
  • 簡単なデプロイとすぐに使える設計
  • ファイルプレビュー（PDF、markdown、コード、テキストなど）
  • ギャラリーモードでの画像プレビュー
  • 動画・音声プレビュー、歌詞・字幕に対応
  • Officeドキュメントプレビュー（docx、pptx、xlsxなど）
  • README.md プレビュー表示
  • ファイルのパーマリンクコピーおよび直接ダウンロード
  • ダークモード
  • I18n対応
  • 保護ルート（パスワード保護・認証対応）
  • WebDav（詳細は https://alist.nn.ci/guide/webdav.html を参照）
  • Dockerによるデプロイ
  • Cloudflare Workers プロキシ
  • ファイル・フォルダのパッケージダウンロード
  • Webアップロード（訪問者アップロード可）、削除、mkdir、リネーム、移動、コピー
  • オフラインダウンロード
  • 2つのストレージ間でファイルコピー
  • シングルスレッドダウンロード/ストリーミング時のマルチスレッド高速化
