GitHub Repository: AlistGo/alist

🗂️一个支持多种存储的文件列表程序，由 Gin 和 Solidjs 驱动。

English | 中文 | 日本語 | Contributing | CODE_OF_CONDUCT

功能特性

  • 多存储支持
    • 本地存储
    • 阿里云盘
    • OneDrive / Sharepoint (全球, 中国,de,us)
    • 天翼云盘 (个人、家庭)
    • GoogleDrive
    • 123云盘
    • FTP / SFTP
    • PikPak
    • S3
    • Seafile
    • 又拍云存储服务
    • WebDav（支持无需 API 的 OneDrive/SharePoint）
    • Teambition（中国、国际）
    • Mediatrack
    • 139云盘（个人、家庭、群组）
    • YandexDisk
    • 百度网盘
    • Terabox
    • UC 网盘
    • 夸克网盘
    • 迅雷云盘
    • 蓝奏云
    • ILanzou
    • 阿里云盘分享
    • Google 相册
    • Mega.nz
    • 百度相册
    • SMB
    • 115
    • Cloudreve
    • Dropbox
    • 飞机盘
    • 多吉云
    • Azure Blob 存储
  • 部署简单，开箱即用
  • 文件预览（PDF、markdown、代码、纯文本等）
  • 图库模式下的图片预览
  • 视频音频预览，支持歌词和字幕
  • Office 文档预览（docx、pptx、xlsx 等）
  • README.md 预览渲染
  • 文件永久链接复制及直接下载
  • 暗黑模式
  • 多语言（I18n）
  • 路由保护（密码保护及认证）
  • WebDav（详情见 https://alist.nn.ci/guide/webdav.html）
  • Docker 部署
  • Cloudflare Workers 代理
  • 文件/文件夹打包下载
  • 网页上传（可允许访客上传）、删除、创建文件夹、重命名、移动和复制
  • 离线下载
  • 跨存储文件复制
  • 单线程下载/串流支持多线程加速
