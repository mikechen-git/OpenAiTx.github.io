GitHub Repository: AlistGo/alist

🗂️Um programa de listagem de arquivos que suporta múltiplos armazenamentos, desenvolvido com Gin e Solidjs.

English | 中文 | 日本語 | Contributing | CODE_OF_CONDUCT

Funcionalidades

  • Múltiplos armazenamentos
    • Armazenamento local
    • Aliyundrive
    • OneDrive / Sharepoint (global, cn,de,us)
    • 189cloud (Pessoal, Família)
    • GoogleDrive
    • 123pan
    • FTP / SFTP
    • PikPak
    • S3
    • Seafile
    • UPYUN Storage Service
    • WebDav (Suporte a OneDrive/SharePoint sem API)
    • Teambition (China,Internacional)
    • Mediatrack
    • 139yun (Pessoal, Família, Grupo)
    • YandexDisk
    • BaiduNetdisk
    • Terabox
    • UC
    • Quark
    • Thunder
    • Lanzou
    • ILanzou
    • Aliyundrive share
    • Google photo
    • Mega.nz
    • Baidu photo
    • SMB
    • 115
    • Cloudreve
    • Dropbox
    • FeijiPan
    • dogecloud
    • Azure Blob Storage
  • Fácil de implantar e pronto para uso
  • Pré-visualização de arquivos (PDF, markdown, código, texto simples, ...)
  • Pré-visualização de imagens no modo galeria
  • Pré-visualização de vídeo e áudio, suporte a letras e legendas
  • Pré-visualização de documentos Office (docx, pptx, xlsx, ...)
  • Renderização de visualização do README.md
  • Cópia de link permanente do arquivo e download direto do arquivo
  • Modo escuro
  • I18n
  • Rotas protegidas (proteção por senha e autenticação)
  • WebDav (veja https://alist.nn.ci/guide/webdav.html para detalhes)
  • Deploy com Docker
  • Proxy Cloudflare Workers
  • Download de pacotes de Arquivo/Pasta
  • Upload via web (Pode permitir visitantes a enviar arquivos), deletar, criar pasta, renomear, mover e copiar
  • Download offline
  • Copiar arquivos entre dois armazenamentos
  • Aceleração de download multithread para download/stream de thread única
