🗂️A file list program that supports multiple storages, powered by Gin and Solidjs.

English | 中文 | 日本語 | Contributing | CODE_OF_CONDUCT

Features

  • [x] Multiple storages
    • [x] Local storage
    • [x] Aliyundrive
    • [x] OneDrive / Sharepoint (global, cn,de,us)
    • [x] 189cloud (Personal, Family)
    • [x] GoogleDrive
    • [x] 123pan
    • [x] FTP / SFTP
    • [x] PikPak
    • [x] S3
    • [x] Seafile
    • [x] UPYUN Storage Service
    • [x] WebDav(Support OneDrive/SharePoint without API)
    • [x] Teambition(China,International)
    • [x] Mediatrack
    • [x] 139yun (Personal, Family, Group)
    • [x] YandexDisk
    • [x] BaiduNetdisk
    • [x] Terabox
    • [x] UC
    • [x] Quark
    • [x] Thunder
    • [x] Lanzou
    • [x] ILanzou
    • [x] Aliyundrive share
    • [x] Google photo
    • [x] Mega.nz
    • [x] Baidu photo
    • [x] SMB
    • [x] 115
    • [X] Cloudreve
    • [x] Dropbox
    • [x] FeijiPan
    • [x] dogecloud
    • [x] Azure Blob Storage
  • [x] Easy to deploy and out-of-the-box
  • [x] File preview (PDF, markdown, code, plain text, ...)
  • [x] Image preview in gallery mode
  • [x] Video and audio preview, support lyrics and subtitles
  • [x] Office documents preview (docx, pptx, xlsx, ...)
  • [x] README.md preview rendering
  • [x] File permalink copy and direct file download
  • [x] Dark mode
  • [x] I18n
  • [x] Protected routes (password protection and authentication)
  • [x] WebDav (see https://alist.nn.ci/guide/webdav.html for details)
  • [x] Docker Deploy
  • [x] Cloudflare Workers proxy
  • [x] File/Folder package download
  • [x] Web upload(Can allow visitors to upload), delete, mkdir, rename, move and copy
  • [x] Offline download
  • [x] Copy files between two storage
  • [x] Multi-thread downloading acceleration for single-thread download/stream
