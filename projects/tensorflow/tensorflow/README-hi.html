प्रलेखन | ------------------- | |

TensorFlow मशीन लर्निंग के लिए एक एंड-टू-एंड ओपन सोर्स प्लेटफॉर्म है। इसमें उपकरण, लाइब्रेरी, और समुदाय संसाधनों का एक व्यापक, लचीला ईकोसिस्टम है, जिससे शोधकर्ता एमएल में अत्याधुनिक शोध कर सकते हैं और डेवलपर्स आसानी से एमएल-सक्षम एप्लिकेशन बना और डिप्लॉय कर सकते हैं।

TensorFlow मूल रूप से Google Brain की मशीन इंटेलिजेंस टीम के शोधकर्ताओं और इंजीनियरों द्वारा मशीन लर्निंग और न्यूरल नेटवर्क में शोध के लिए विकसित किया गया था। हालांकि, यह फ्रेमवर्क अन्य क्षेत्रों में भी प्रयुक्त किया जा सकता है।

TensorFlow स्थिर Python और C++ API प्रदान करता है, साथ ही अन्य भाषाओं के लिए एक नॉन-गारंटीड बैकवर्ड कम्पेटिबल API भी उपलब्ध है।

रिलीज़ घोषणाओं और सुरक्षा अपडेट के लिए announce@tensorflow.org पर सदस्यता लें। सभी मेलिंग सूचियाँ देखें।

स्थापना करें

pip पैकेज, GPU समर्थन सक्षम करने, Docker कंटेनर का उपयोग करने, और सोर्स से बिल्ड करने के लिए TensorFlow स्थापना गाइड देखें।

वर्तमान रिलीज़ स्थापित करने के लिए, जिसमें CUDA-सक्षम GPU कार्ड (Ubuntu और Windows) का समर्थन शामिल है:

$ pip install tensorflow

अन्य डिवाइस (DirectX और MacOS-metal) डिवाइस प्लगइन्स के माध्यम से समर्थित हैं।

केवल CPU के लिए एक छोटा पैकेज भी उपलब्ध है:

$ pip install tensorflow-cpu

TensorFlow को नवीनतम संस्करण में अपडेट करने के लिए, ऊपर दिए गए कमांड में --upgrade फ्लैग जोड़ें।

रात्री (नाइटली) बायनरी परीक्षण के लिए tf-nightly और tf-nightly-cpu पैकेज PyPI पर उपलब्ध हैं।

अपना पहला TensorFlow प्रोग्राम आज़माएँ

$ python

>>> import tensorflow as tf
>>> tf.add(1, 2).numpy()
3
>>> hello = tf.constant('Hello, TensorFlow!')
>>> hello.numpy()
b'Hello, TensorFlow!'

अधिक उदाहरणों के लिए, TensorFlow ट्यूटोरियल देखें।

योगदान दिशानिर्देश

यदि आप TensorFlow में योगदान करना चाहते हैं, तो कृपया योगदान दिशानिर्देश अवश्य देखें। यह प्रोजेक्ट TensorFlow के आचार संहिता का पालन करता है। इसमें भाग लेकर, आप इस संहिता का पालन करने की अपेक्षा रखते हैं।

हम GitHub issues का उपयोग अनुरोधों और बग्स को ट्रैक करने के लिए करते हैं, कृपया सामान्य प्रश्नों और चर्चा के लिए TensorFlow Forum देखें, और विशिष्ट प्रश्नों के लिए Stack Overflow पर जाएँ।

TensorFlow प्रोजेक्ट ओपन-सोर्स सॉफ्टवेयर विकास में सामान्यतः स्वीकृत सर्वोत्तम प्रथाओं का पालन करने का प्रयास करता है।

पैचिंग दिशानिर्देश

TensorFlow के किसी विशिष्ट संस्करण को पैच करने के लिए, जैसे बग्स या सुरक्षा कमजोरियों को ठीक करने के लिए, इन चरणों का पालन करें:

  • TensorFlow रिपॉजिटरी क्लोन करें और अपनी इच्छित TensorFlow संस्करण की संबंधित शाखा पर स्विच करें, उदाहरण के लिए, संस्करण 2.8 के लिए r2.8 शाखा।
  • वांछित परिवर्तनों को लागू (चैरी-पिक) करें और किसी भी कोड संघर्ष को हल करें।
  • TensorFlow परीक्षण चलाएँ और सुनिश्चित करें कि वे पास हों।
  • सोर्स से TensorFlow pip पैकेज बनाएं।
