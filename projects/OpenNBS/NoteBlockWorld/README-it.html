ℹ Informazioni sul progetto

Note Block World è un sito web collaborativo e sociale dove gli utenti possono condividere le loro canzoni realizzate con i blocchi musicali usando Note Block Studio. Il nostro obiettivo è offrire alla community NBS una piattaforma per condividere le proprie creazioni con il mondo, scoprire nuova musica e connettersi con altri creatori!

💬 Problemi/Feedback

Se hai riscontrato un problema o desideri suggerire una nuova funzionalità, apri una nuova segnalazione sul repository. Assicurati di consultare la roadmap del progetto per vedere se la funzionalità a cui sei interessato è già pianificata. Cerchiamo sempre di migliorare il sito e ci piacerebbe conoscere la tua opinione!

🔧 Contribuire

Siamo aperti ai contributi sul progetto! Leggi la nostra guida per i contributori per configurare il progetto e iniziare.

Ti chiediamo di aprire una segnalazione per discutere le modifiche che vorresti apportare prima di inviare una pull request. In alternativa, puoi unirti al nostro server Discord per discutere le modifiche con il nostro team di sviluppo!

❤ Dona

Se desideri supportare il progetto, puoi donare tramite il nostro Open Collective o GitHub Sponsors! Le tue donazioni ci aiutano a coprire i costi di gestione del sito e ci permettono di dedicare più tempo allo sviluppo.

Grazie a tutti i nostri sponsor attuali e passati per averci aiutato a mantenere Note Block World attivo!
