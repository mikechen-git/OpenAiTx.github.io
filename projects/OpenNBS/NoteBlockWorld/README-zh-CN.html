ℹ 关于项目

Note Block World 是一个协作型、社交化的网站，用户可以在这里分享他们用 Note Block Studio 制作的音符方块乐曲。我们的目标是为 NBS 社区提供一个分享创作的平台，让大家可以向世界展示自己的作品，发现新音乐，并与其他创作者建立联系！

💬 问题/反馈

如果你发现了问题，或想建议新功能，请在本仓库新建 issue。在此之前请先查看项目路线图，看看你想要参与的功能是否已经在计划之中。我们始终致力于改进网站，欢迎你的想法和建议！

🔧 参与贡献

我们欢迎你为本项目贡献力量！请阅读我们的贡献者指南来了解项目搭建和入门方法。

在提交 Pull Request 之前，请先新建 issue讨论你想进行的更改。你也可以加入我们的 Discord 服务器与开发团队交流！

❤ 捐赠支持

感谢所有现有及曾经的赞助者，帮助我们让 Note Block World 持续运转！
