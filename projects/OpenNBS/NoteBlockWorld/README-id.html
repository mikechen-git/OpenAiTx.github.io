ℹ Tentang Proyek

Note Block World adalah situs web kolaboratif dan sosial di mana pengguna dapat berbagi lagu note block mereka yang dibuat dengan Note Block Studio. Tujuan kami adalah memberikan komunitas NBS sebuah platform untuk membagikan karya mereka ke dunia, menemukan musik baru, dan terhubung dengan kreator lainnya!
