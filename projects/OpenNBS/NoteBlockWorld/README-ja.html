ℹ プロジェクトについて

Note Block World は、ユーザーが Note Block Studio で作成したノートブロック曲を共有できる、協力型・ソーシャルウェブサイトです。NBSコミュニティが世界中に作品を発信し、新しい音楽を発見し、他のクリエイターと繋がるためのプラットフォームを提供することを目指しています！

💬 問題・フィードバック

問題を発見した場合や新機能の提案がある場合は、リポジトリの 新しいイシュー を作成してください。ご希望の機能が既に計画されているかどうかは プロジェクトのロードマップ をご確認ください。ウェブサイトの改善に努めておりますので、ぜひご意見をお寄せください！

🔧 コントリビューション

プロジェクトへの貢献を歓迎します！プロジェクトのセットアップや開始方法については コントリビューターガイド をご覧ください。

プルリクエストを送る前に、変更したい内容について イシューを新規作成 してください。または、Discordサーバー に参加し、開発チームと直接ご相談いただくことも可能です！

❤ 寄付

プロジェクトを支援したい方は、Open Collective または GitHub Sponsors からご寄付いただけます！皆様からのご支援は、ウェブサイト運営費用の補填や、開発により多くの時間を割くために役立てられます。

Note Block Worldの運営を支えてくださっている、全ての現・過去のスポンサーの皆様に感謝します！
