ℹ Sobre el proyecto

Note Block World es un sitio web colaborativo y social donde los usuarios pueden compartir sus canciones de bloques de notas creadas con Note Block Studio. ¡Nuestro objetivo es brindar a la comunidad de NBS una plataforma para compartir sus creaciones con el mundo, descubrir nueva música y conectar con otros creadores!

💬 Problemas/Comentarios

Si has encontrado un problema o deseas sugerir una función, por favor abre un nuevo issue en el repositorio. Asegúrate de consultar la hoja de ruta del proyecto para ver si la función en la que deseas trabajar ya está planificada. ¡Siempre estamos buscando mejorar el sitio web y nos encantaría conocer tu opinión!

🔧 Contribuir

¡Damos la bienvenida a contribuciones en el proyecto! Lee nuestra guía para colaboradores para configurar el proyecto y empezar.

Por favor, abre un issue para discutir los cambios que te gustaría realizar antes de enviar un pull request. Alternativamente, puedes unirte a nuestro servidor de Discord para discutir los cambios con nuestro equipo de desarrollo.

❤ Donar

Si deseas apoyar el proyecto, puedes donar a través de nuestro Open Collective o GitHub Sponsors. ¡Tus donaciones nos ayudan a cubrir los costos de operación del sitio web y nos permiten dedicar más tiempo a su desarrollo!

¡Gracias a todos nuestros patrocinadores actuales y pasados por ayudarnos a mantener Note Block World en funcionamiento!
