ℹ 關於本專案

Note Block World 是一個協作且具社交功能的網站，讓用戶能夠分享他們使用 Note Block Studio 創作的音符方塊歌曲。我們的目標是為 NBS 社群提供一個平台，讓大家能夠向世界分享自己的創作、發掘新音樂，並與其他創作者交流！

💬 問題/反饋

如果你發現了問題，或是想要提出新功能建議，請在本倉庫提交新問題。請務必查看專案開發規劃，確認你想參與的功能是否已經在規劃中。我們一直努力改進網站，非常樂意聽取你的建議！

🔧 貢獻

我們歡迎任何對本專案的貢獻！請閱讀我們的貢獻者指南，以設置專案環境並開始貢獻。

在提交 pull request 之前，請先提交一個 issue說明你想做的更改。或者，你也可以加入我們的 Discord 伺服器 與開發團隊討論你的想法！

❤ 捐款支持

如果你希望支持本專案，可以透過 Open Collective 或 GitHub Sponsors 進行捐款！你的支持將幫助我們維持網站運作，並讓我們能投入更多時間開發新功能。

感謝所有現有與曾經贊助我們的支持者，讓 Note Block World 能持續運轉！
