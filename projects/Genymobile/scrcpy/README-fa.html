این مخزن GitHub (https://github.com/Genymobile/scrcpy) تنها منبع رسمی برای این پروژه است. نسخه‌ها را از سایت‌های نامعتبر دانلود نکنید، حتی اگر نام آن‌ها شامل scrcpy باشد.

scrcpy (نسخه ۳.۲)

تلفظ: "screen copy"

این برنامه دستگاه‌های اندرویدی (تصویر و صدا) متصل شده از طریق USB یا TCP/IP را آینه می‌کند و امکان کنترل با استفاده از صفحه‌کلید و ماوس رایانه را فراهم می‌کند. نیاز به دسترسی root یا نصب هیچ اپی روی دستگاه ندارد. بر روی لینوکس، ویندوز و مک‌اواس کار می‌کند.

تمرکز این برنامه بر روی موارد زیر است:

  • سبکی: بومی، فقط نمایش صفحه دستگاه
  • کارایی: ۳۰ تا ۱۲۰ فریم بر ثانیه، بسته به دستگاه
  • کیفیت: ۱۹۲۰×۱۰۸۰ یا بالاتر
  • تاخیر پایین: ۳۵ تا ۷۰ میلی‌ثانیه
  • زمان راه‌اندازی پایین: حدود ۱ ثانیه برای نمایش اولین تصویر
  • عدم مزاحمت: هیچ چیزی روی دستگاه اندرویدی نصب نمی‌شود
  • مزایای کاربر: بدون حساب کاربری، بدون تبلیغات، بدون نیاز به اینترنت
  • آزادی: نرم‌افزار آزاد و متن‌باز

ویژگی‌های آن شامل:

  • ارسال صدا (اندروید ۱۱ به بالا)
  • ضبط
  • نمایشگر مجازی
  • آینه‌سازی با صفحه خاموش دستگاه اندروید
  • کپی-پیست در هر دو جهت
  • کیفیت قابل تنظیم
  • آینه‌سازی دوربین (اندروید ۱۲ به بالا)
  • آینه‌سازی به عنوان وبکم (V4L2) (فقط لینوکس)
  • شبیه‌سازی فیزیکی صفحه‌کلید و ماوس (HID)
  • پشتیبانی از گیم‌پد
  • حالت OTG
  • و امکانات بیشتر…

پیش‌نیازها

دستگاه اندرویدی حداقل باید API 21 (اندروید ۵.۰) داشته باشد.

ارسال صدا برای API >= 30 (اندروید ۱۱ به بالا) پشتیبانی می‌شود.

اطمینان حاصل کنید که اشکال‌زدایی USB را روی دستگاه خود فعال کرده‌اید.

در برخی دستگاه‌ها (به ویژه شیائومی)، ممکن است خطای زیر را دریافت کنید:

java.lang.SecurityException: Injecting input events requires the caller (or the source of the instrumentation, if any) to have the INJECT_EVENTS permission.

در این صورت باید یک گزینه اضافی به نام USB debugging (Security Settings) (این گزینه با USB debugging متفاوت است) را فعال کنید تا بتوانید با صفحه‌کلید و ماوس دستگاه را کنترل کنید. پس از تنظیم این گزینه، راه‌اندازی مجدد دستگاه ضروری است.

توجه داشته باشید که برای اجرای scrcpy در حالت OTG نیازی به اشکال‌زدایی USB نیست.

دریافت برنامه

  • لینوکس
  • ویندوز (راهنمای اجرای برنامه را بخوانید)
  • مک‌اواس

نکات ضروری

  • کاهش وضوح تصویر می‌تواند کارایی را به طور قابل توجهی افزایش دهد (scrcpy -m1024)
  • کلیک راست معادل دکمه BACK است
  • کلیک وسط معادل دکمه HOME است
  • Alt+f حالت تمام‌صفحه را فعال/غیرفعال می‌کند
  • سایر میانبرها نیز وجود دارد

مثال‌های استفاده

گزینه‌های زیادی وجود دارد که در صفحات مستندسازی شده جداگانه توضیح داده شده‌اند. در اینجا فقط چند مثال رایج آورده شده است.

  • ضبط صفحه با کدک H.265 (کیفیت بهتر)، محدود کردن اندازه به ۱۹۲۰، محدود کردن نرخ فریم به ۶۰fps، غیرفعال کردن صدا و کنترل دستگاه با شبیه‌سازی صفحه‌کلید فیزیکی:

    scrcpy --video-codec=h265 --max-size=1920 --max-fps=60 --no-audio --keyboard=uhid
scrcpy --video-codec=h265 -m1920 --max-fps=60 --no-audio -K  # نسخه کوتاه

  • اجرای VLC در یک نمایشگر مجازی جدید (جدا از نمایشگر دستگاه):

    scrcpy --new-display=1920x1080 --start-app=org.videolan.vlc

  • ضبط دوربین دستگاه با کیفیت H.265 و وضوح ۱۹۲۰x۱۰۸۰ (و میکروفون) در یک فایل MP4:

    scrcpy --video-source=camera --video-codec=h265 --camera-size=1920x1080 --record=file.mp4

  • ضبط دوربین جلوی دستگاه و ارائه آن به عنوان وبکم روی رایانه (در لینوکس):

    scrcpy --video-source=camera --camera-size=1920x1080 --camera-facing=front --v4l2-sink=/dev/video2 --no-playback

  • کنترل دستگاه بدون آینه‌سازی با شبیه‌سازی صفحه‌کلید و ماوس فیزیکی (بدون نیاز به اشکال‌زدایی USB):

    scrcpy --otg

  • کنترل دستگاه با استفاده از دسته‌های بازی متصل به رایانه:

    scrcpy --gamepad=uhid
scrcpy -G  # نسخه کوتاه
