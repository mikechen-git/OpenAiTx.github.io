scrcpy (v3.2)

diucapkan "screen copy"

Aplikasi ini melakukan mirror perangkat Android (video dan audio) yang terhubung melalui USB atau TCP/IP dan memungkinkan kontrol menggunakan keyboard dan mouse komputer. Tidak memerlukan akses root atau aplikasi yang diinstal di perangkat. Berjalan di Linux, Windows, dan macOS.

Fokus pada:

  • ringan: native, hanya menampilkan layar perangkat
  • performa: 30~120fps, tergantung perangkat
  • kualitas: 1920×1080 atau lebih
  • latensi rendah: 35~70ms
  • waktu mulai cepat: ~1 detik untuk menampilkan gambar pertama
  • tidak mengganggu: tidak ada yang tertinggal terpasang di perangkat Android
  • manfaat pengguna: tanpa akun, tanpa iklan, tanpa internet
  • kebebasan: perangkat lunak gratis dan sumber terbuka

Fitur-fiturnya meliputi:

  • penerusan audio (Android 11+)
  • perekaman
  • virtual display
  • mirror dengan layar perangkat Android mati
  • copy-paste dua arah
  • kualitas yang dapat dikonfigurasi
  • mirroring kamera (Android 12+)
  • mirror sebagai webcam (V4L2) (hanya Linux)
  • simulasi fisik keyboard dan mouse (HID)
  • dukungan gamepad
  • mode OTG
  • dan lainnya…

Prasyarat

Perangkat Android memerlukan minimal API 21 (Android 5.0).

Penerusan audio didukung untuk API >= 30 (Android 11+).

Pastikan Anda mengaktifkan USB debugging di perangkat Anda.

Pada beberapa perangkat (terutama Xiaomi), Anda mungkin mendapatkan error berikut:

java.lang.SecurityException: Injecting input events requires the caller (or the source of the instrumentation, if any) to have the INJECT_EVENTS permission.

Dalam kasus tersebut, Anda perlu mengaktifkan opsi tambahan USB debugging (Security Settings) (ini item yang berbeda dari USB debugging) untuk mengontrolnya menggunakan keyboard dan mouse. Setelah opsi ini diatur, perangkat perlu direboot.

Perlu dicatat bahwa USB debugging tidak diperlukan untuk menjalankan scrcpy dalam mode OTG.

Dapatkan aplikasinya

  • Linux
  • Windows (baca cara menjalankan)
  • macOS

Tips yang Wajib Diketahui

  • Mengurangi resolusi dapat sangat meningkatkan performa (scrcpy -m1024)
  • Klik kanan memicu BACK
  • Klik tengah memicu HOME
  • Alt+f mengaktifkan layar penuh
  • Terdapat banyak shortcut lainnya

Contoh penggunaan

Ada banyak opsi, didokumentasikan di halaman terpisah. Berikut beberapa contoh yang umum.

  • Tangkap layar dalam H.265 (kualitas lebih baik), batasi ukuran ke 1920, batasi frame rate ke 60fps, nonaktifkan audio, dan kontrol perangkat dengan mensimulasikan keyboard fisik:

    scrcpy --video-codec=h265 --max-size=1920 --max-fps=60 --no-audio --keyboard=uhid
scrcpy --video-codec=h265 -m1920 --max-fps=60 --no-audio -K  # versi singkat

  • Mulai VLC di virtual display baru (terpisah dari tampilan perangkat):

    scrcpy --new-display=1920x1080 --start-app=org.videolan.vlc

  • Rekam kamera perangkat dalam H.265 di 1920x1080 (dan mikrofon) ke file MP4:

    scrcpy --video-source=camera --video-codec=h265 --camera-size=1920x1080 --record=file.mp4

  • Tangkap kamera depan perangkat dan tampilkan sebagai webcam di komputer (di Linux):

    scrcpy --video-source=camera --camera-size=1920x1080 --camera-facing=front --v4l2-sink=/dev/video2 --no-playback

  • Kontrol perangkat tanpa mirror dengan mensimulasikan keyboard dan mouse fisik (USB debugging tidak diperlukan):

    scrcpy --otg

  • Kontrol perangkat menggunakan controller gamepad yang terpasang ke komputer:

    scrcpy --gamepad=uhid
scrcpy -G  # versi singkat
