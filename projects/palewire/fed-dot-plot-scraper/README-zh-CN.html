使用方法

克隆该仓库。

gh repo clone palewire/fed-dot-plot-scraper

进入目录。

cd fed-dot-plot-scraper

安装依赖包。

pipenv install --dev

运行爬虫程序。

pipenv run python -m src.scrape

该命令将在终端输出一个CSV文件。你可以通过如下方式将其写入文件：

pipenv run python -m src.scrape > output.csv
