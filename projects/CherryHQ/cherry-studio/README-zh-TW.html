🍒 Cherry Studio

Cherry Studio 是一款支援多種 LLM 供應商的桌面客戶端，適用於 Windows、Mac 和 Linux。

👏 加入 Telegram 群組｜Discord | QQ 群(575014769)

❤️ 喜歡 Cherry Studio？給我們一顆星星 🌟 或 贊助 支持開發！

🌠 截圖

🌟 主要功能

  1. 多元 LLM 供應商支援：
  • ☁️ 主流 LLM 雲端服務：OpenAI、Gemini、Anthropic 等
  • 🔗 AI 網頁服務整合：Claude、Peplexity、Poe 及其他
  • 💻 本地模型支援 Ollama、LM Studio
  1. AI 助理與對話：
  • 📚 300+ 預設配置的 AI 助理
  • 🤖 自訂助理建立
  • 💬 多模型同時對話
  1. 文件與資料處理：
  • 📄 支援文字、圖片、Office、PDF 等多種格式
  • ☁️ WebDAV 檔案管理與備份
  • 📊 Mermaid 圖表視覺化
  • 💻 程式碼語法高亮顯示
  1. 實用工具整合：
  • 🔍 全域搜尋功能
  • 📝 主題管理系統
  • 🔤 AI 驅動翻譯
  • 🎯 拖放排序
  • 🔌 小程式支援
  • ⚙️ MCP（模型上下文協議）伺服器
  1. 優化使用體驗：
  • 🖥️ 跨平台支援 Windows、Mac 和 Linux
  • 📦 開箱即用，無需環境配置
  • 🎨 明亮/深色主題及透明視窗
  • 📝 完整 Markdown 呈現
  • 🤲 輕鬆內容分享

貢獻者獎勵計劃

為了回饋我們的核心貢獻者並創造良性循環，我們制定了以下長期激勵方案。

本計劃的首個統計週期為 2025 年第三季（7 月、8 月、9 月），該週期的獎勵將於 10 月 1 日發放。

在任一統計週期內（例如第一輪為 7 月 1 日至 9 月 30 日），任何開發者只要對 Cherry Studio 的任一 GitHub 開源專案貢獻超過 30 次有意義的提交（commit），即可獲得以下福利：

  • Cursor 訂閱贊助：獲得 70 美元 Cursor 訂閱抵用金或報銷，讓 AI 成為您最高效的編程夥伴。
  • 模型無限使用權：獲得 DeepSeek 與 Qwen 模型的 API 無限次數調用權。
  • 前沿技術體驗：不定期獲得如 Claude、Gemini、OpenAI 等模型的 API 權益，持續站在技術前沿。

🏢 企業版

在社群版的基礎上，我們自豪地推出 Cherry Studio 企業版——一款可私有部署的 AI 生產力與管理平台，專為現代團隊與企業設計。

企業版聚焦團隊協作的核心挑戰，集中管理 AI 資源、知識與數據，幫助組織提升效率、激發創新、保障合規，同時確保所有資料 100% 由您自主掌控並於安全環境下運作。

核心優勢

  • 統一模型管理：集中整合與管理各類雲端 LLM（如 OpenAI、Anthropic、Google Gemini）與本地私有部署模型，員工無需個人配置即可開箱即用。
  • 企業級知識庫：建構、管理並團隊共享知識庫，確保知識沉澱與一致，團隊成員可基於統一且準確的資訊與 AI 互動。
  • 細緻權限管控：統一後台輕鬆管理員工帳號，為不同模型、知識庫及功能分配角色權限。
  • 全私有部署：可將整套後端服務部署於本地伺服器或私有雲，確保您的數據 100% 私有並自主掌控，滿足最嚴格的安全與合規要求。
  • 可靠後端服務：提供穩定的 API 服務、企業級數據備份與災難恢復機制，確保業務持續運作。
