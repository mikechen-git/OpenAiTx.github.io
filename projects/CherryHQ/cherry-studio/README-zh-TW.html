🍒 Cherry Studio

Cherry Studio 是一款支援多種 LLM 供應商的桌面客戶端，適用於 Windows、Mac 和 Linux。

👏 加入 Telegram 群組｜Discord | QQ 群(575014769)

❤️ 喜歡 Cherry Studio？給我們一顆星星 🌟 或 贊助 支持開發！

🌠 截圖

🌟 主要功能

  1. 多元 LLM 供應商支援：
  • ☁️ 主流 LLM 雲端服務：OpenAI、Gemini、Anthropic 等
  • 🔗 AI 網頁服務整合：Claude、Peplexity、Poe 及其他
  • 💻 本地模型支援 Ollama、LM Studio
  1. AI 助理與對話：
  • 📚 300+ 預設配置的 AI 助理
  • 🤖 自訂助理建立
  • 💬 多模型同時對話
  1. 文件與資料處理：
  • 📄 支援文字、圖片、Office、PDF 等多種格式
  • ☁️ WebDAV 檔案管理與備份
  • 📊 Mermaid 圖表視覺化
  • 💻 程式碼語法高亮
  1. 實用工具整合：
  • 🔍 全域搜尋功能
  • 📝 主題管理系統
  • 🔤 AI 驅動翻譯
  • 🎯 拖放排序
  • 🔌 小程式支援
  • ⚙️ MCP（模型上下文協議）伺服器
  1. 增強用戶體驗：
  • 🖥️ 跨平台支援 Windows、Mac、Linux
  • 📦 開箱即用，無需環境配置
  • 🎨 明暗主題與透明視窗
  • 📝 完整 Markdown 渲染
  • 🤲 內容輕鬆分享

📝 路線圖

我們正在積極開發以下功能和改進：

  1. 🎯 核心功能
  • 選擇助手－智慧內容選取增強
  • 深度研究－進階研究能力
  • 記憶系統－全局上下文感知
  • 文件預處理－提升文件處理效能
  • MCP 市集－模型上下文協議生態系
  1. 🗂 知識管理
  • 筆記與收藏
  • 動態畫布視覺化
  • OCR 功能
  • TTS（文字轉語音）支援
  1. 📱 平台支援
  • HarmonyOS 版本（PC）
  • Android App（第一階段）
  • iOS App（第一階段）
  • 多視窗支援
  • 視窗釘選功能
  1. 🔌 進階功能
  • 外掛系統
  • ASR（自動語音識別）
  • 助理與主題互動重構

🤝 參與貢獻

我們歡迎對 Cherry Studio 的貢獻！您可以透過以下方式參與：

  1. 貢獻程式碼：開發新功能或優化現有程式碼。
  2. 修復錯誤：提交您發現的錯誤修正。
  3. 維護議題：協助管理 GitHub 議題。
  4. 產品設計：參與設計討論。
  5. 撰寫文件：改進用戶手冊與指南。
  6. 社群互動：參與討論並協助用戶。
  7. 推廣應用：宣傳 Cherry Studio。

請參考 分支策略 以瞭解貢獻指南

開始貢獻

  1. Fork 儲存庫：Fork 並 Clone 到本地端。
  2. 建立分支：針對您的更改建立分支。
  3. 提交更改：Commit 並 Push 您的更改。
  4. 建立 Pull Request：描述您的更改及原因。

更多詳細說明請參閱我們的 貢獻指南。
