🍒 Cherry Studio

Cherry Studio 是一款桌面客户端，支持多种大语言模型（LLM）服务商，可在 Windows、Mac 和 Linux 上使用。

👏 加入 Telegram 群组｜Discord | QQ群(575014769)

❤️ 喜欢 Cherry Studio？欢迎点亮星标 🌟 或 赞助 支持开发！

🌠 截图

🌟 主要特性

  1. 多样化的 LLM 服务商支持：
  • ☁️ 主流大模型云服务：OpenAI、Gemini、Anthropic 等
  • 🔗 AI Web 服务接入：Claude、Peplexity、Poe 等
  • 💻 本地模型支持：Ollama、LM Studio
  1. AI 助手与对话：
  • 📚 300+ 预配置 AI 助手
  • 🤖 自定义助手创建
  • 💬 多模型同时对话
  1. 文档与数据处理：
  • 📄 支持文本、图片、Office、PDF 等多种格式
  • ☁️ WebDAV 文件管理与备份
  • 📊 Mermaid 图表可视化
  • 💻 代码语法高亮
  1. 实用工具集成：
  • 🔍 全局搜索功能
  • 📝 主题管理系统
  • 🔤 AI 驱动翻译
  • 🎯 拖拽排序
  • 🔌 小程序支持
  • ⚙️ MCP（模型上下文协议）服务器
  1. 增强用户体验：
  • 🖥️ 跨平台支持 Windows、Mac 和 Linux
  • 📦 开箱即用，无需环境配置
  • 🎨 明/暗主题与透明窗口
  • 📝 完整 Markdown 渲染
  • 🤲 轻松内容分享

📝 路线图

我们正在积极开发以下功能和改进：

  1. 🎯 核心功能
  • 选择助手——智能内容选择增强
  • 深度研究——高级研究能力
  • 记忆系统——全局上下文感知
  • 文档预处理——文档处理能力提升
  • MCP 市场——模型上下文协议生态系统
  1. 🗂 知识管理
  • 笔记与收藏
  • 动态画布可视化

  1. 📱 平台支持
  • HarmonyOS 版（PC）
  • 安卓 App（第一阶段）
  • iOS App（第一阶段）
  • 多窗口支持
  • 窗口置顶功能
  1. 🔌 高级功能
  • 插件系统
  • ASR（自动语音识别）

🤝 贡献指南

我们欢迎您为 Cherry Studio 做出贡献！以下是参与方式：

  1. 贡献代码：开发新功能或优化现有代码。
  2. 修复 Bug：提交你发现的问题修复。
  3. 维护问题：帮助管理 GitHub 问题。
  4. 产品设计：参与设计讨论。
  5. 撰写文档：改进用户手册和指南。
  6. 社区互动：参与讨论并帮助用户。
  7. 推广应用：传播 Cherry Studio。

  3. 提交更改：提交并推送你的更改。
