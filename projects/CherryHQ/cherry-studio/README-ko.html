🍒 Cherry Studio

Cherry Studio는 다양한 LLM 제공자를 지원하는 데스크탑 클라이언트로, Windows, Mac, Linux에서 사용할 수 있습니다.

👏 텔레그램 그룹｜디스코드 | QQ 그룹(575014769)에 참여하세요

❤️ Cherry Studio가 마음에 드시나요? 개발 지원을 위해 🌟 별을 남기거나 스폰서가 되어주세요!

🌠 스크린샷

🌟 주요 기능

  1. 다양한 LLM 제공자 지원:
  • ☁️ 주요 LLM 클라우드 서비스: OpenAI, Gemini, Anthropic 등
  • 🔗 AI 웹 서비스 통합: Claude, Peplexity, Poe 등
  • 💻 Ollama, LM Studio를 통한 로컬 모델 지원
  1. AI 어시스턴트 & 대화:
  • 📚 300개 이상의 사전 구성된 AI 어시스턴트
  • 🤖 커스텀 어시스턴트 생성
  • 💬 멀티 모델 동시 대화
  1. 문서 및 데이터 처리:
  • 📄 텍스트, 이미지, 오피스, PDF 등 지원
  • ☁️ WebDAV 파일 관리 및 백업
  • 📊 Mermaid 차트 시각화
  • 💻 코드 구문 하이라이팅
  1. 실용 도구 통합:
  • 🔍 글로벌 검색 기능
  • 📝 토픽 관리 시스템
  • 🔤 AI 기반 번역
  • 🎯 드래그 앤 드롭 정렬
  • 🔌 미니 프로그램 지원
  • ⚙️ MCP(Model Context Protocol) 서버
  1. 향상된 사용자 경험:
  • 🖥️ Windows, Mac, Linux 크로스플랫폼 지원
  • 📦 즉시 사용 가능, 환경 설정 불필요
  • 🎨 라이트/다크 테마 및 투명 창
  • 📝 완벽한 마크다운 렌더링
  • 🤲 손쉬운 콘텐츠 공유

🔧 개발자 공동 창작 프로그램

우리는 오픈소스 생태계 내에서 건강하고 긍정적인 피드백 루프를 조성하기 위해 Cherry Studio 개발자 공동창작 프로그램을 시작합니다. 훌륭한 소프트웨어는 협업을 통해 만들어지며, 병합된 모든 풀 리퀘스트는 프로젝트에 새로운 생명을 불어넣는다고 믿습니다.

여러분을 우리의 기여자 대열에 진심으로 초대하며, Cherry Studio의 미래를 함께 만들어 가길 바랍니다.

기여자 보상 프로그램

핵심 기여자에게 보답하고 선순환을 만들기 위해 다음과 같은 장기 인센티브 플랜을 마련했습니다.

이 프로그램의 첫 추적 기간은 2025년 3분기(7월, 8월, 9월)입니다. 이 기간의 보상은 10월 1일에 지급됩니다.

모든 추적 기간(예: 첫 사이클의 경우 7월 1일부터 9월 30일까지) 내에, Cherry Studio의 GitHub 오픈소스 프로젝트에 의미 있는 커밋 30개 이상을 기여한 개발자는 다음 혜택을 받을 수 있습니다:

  • Cursor 구독 지원: Cursor 구독에 대해 $70 USD 상당의 크레딧 또는 환급금을 받아, AI를 가장 효율적인 코딩 파트너로 활용할 수 있습니다.
  • 모델 무제한 이용권: DeepSeek 및 Qwen 모델에 대해 무제한 API 호출이 가능합니다.
  • 최첨단 기술 이용권: Claude, Gemini, OpenAI 등과 같은 모델의 API 접근 등 다양한 특전을 간헐적으로 제공하여, 기술의 최전선에 설 수 있도록 지원합니다.

함께 성장하는 미래 계획

활기찬 커뮤니티는 지속 가능한 오픈소스 프로젝트의 원동력입니다. Cherry Studio가 성장함에 따라 보상 프로그램도 함께 확장될 것입니다. 우리는 업계 최고 수준의 도구와 자원에 맞추어 혜택을 지속적으로 개선할 것을 약속합니다. 이를 통해 핵심 기여자들이 실질적인 지원을 받고, 개발자, 커뮤니티, 프로젝트가 함께 성장하는 선순환 구조를 만듭니다.

앞으로 프로젝트는 더욱 개방적인 자세를 취해 전체 오픈소스 커뮤니티에 환원할 예정입니다.

🏢 엔터프라이즈 에디션

커뮤니티 에디션을 기반으로, 저희는 Cherry Studio 엔터프라이즈 에디션을 새롭게 선보입니다. 이는 현대 팀과 기업을 위해 설계된 프라이빗 배포형 AI 생산성 및 관리 플랫폼입니다.

엔터프라이즈 에디션은 AI 리소스, 지식, 데이터를 중앙에서 관리함으로써 팀 협업의 핵심 과제를 해결합니다. 조직이 효율성을 높이고, 혁신을 촉진하며, 컴플라이언스를 보장할 수 있도록 하면서도, 100% 데이터 통제를 안전한 환경에서 유지할 수 있도록 지원합니다.

주요 장점

  • 통합 모델 관리: 다양한 클라우드 기반 LLM(OpenAI, Anthropic, Google Gemini 등) 및 로컬에 배포된 프라이빗 모델을 중앙에서 통합 관리합니다. 임직원은 별도 설정 없이 즉시 사용할 수 있습니다.
  • 엔터프라이즈급 지식 베이스: 팀 전체가 공유하는 지식 베이스를 구축·관리·공유합니다. 지식을 체계적으로 관리하여 일관성을 유지하고, 팀원이 통합된 정확한 정보에 기반해 AI와 상호작용할 수 있게 합니다.
  • 세분화된 접근 제어: 통합 관리 백엔드에서 임직원 계정을 쉽게 관리하고, 모델·지식 베이스·기능별로 역할 기반 권한을 부여할 수 있습니다.
  • 완전 프라이빗 배포: 전체 백엔드 서비스를 온프레미스 서버 또는 프라이빗 클라우드에 배포하여, 데이터가 100% 사내에 남고, 가장 엄격한 보안 및 컴플라이언스 기준을 충족합니다.
  • 신뢰성 높은 백엔드 서비스: 안정적인 API 서비스, 엔터프라이즈급 데이터 백업 및 복구 메커니즘을 제공하여 비즈니스 연속성을 보장합니다.

✨ 온라인 데모

🚧 퍼블릭 베타 안내

엔터프라이즈 에디션은 현재 초기 퍼블릭 베타 단계로, 기능을 적극적으로 개선 중입니다. 아직 완전히 안정적이지 않을 수 있습니다. 체험 중 문제를 발견하거나 소중한 의견이 있으시다면 이메일을 통해 피드백을 보내주시면 진심으로 감사하겠습니다.
