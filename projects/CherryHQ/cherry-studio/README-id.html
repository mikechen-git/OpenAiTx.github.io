🍒 Cherry Studio

Cherry Studio adalah klien desktop yang mendukung berbagai penyedia LLM, tersedia di Windows, Mac, dan Linux.

👏 Bergabunglah dengan Grup Telegram｜Discord | Grup QQ (575014769)

❤️ Suka dengan Cherry Studio? Beri bintang 🌟 atau Sponsor untuk mendukung pengembangan!

🌠 Tangkapan Layar

🌟 Fitur Utama

  1. Dukungan Beragam Penyedia LLM:
  • ☁️ Layanan Cloud LLM Utama: OpenAI, Gemini, Anthropic, dan lainnya
  • 🔗 Integrasi Layanan Web AI: Claude, Peplexity, Poe, dan lainnya
  • 💻 Dukungan Model Lokal dengan Ollama, LM Studio
  1. Asisten AI & Percakapan:
  • 📚 300+ Asisten AI yang telah dikonfigurasi sebelumnya
  • 🤖 Pembuatan Asisten Kustom
  • 💬 Percakapan Multi-model Secara Simultan
  1. Pemrosesan Dokumen & Data:
  • 📄 Dukungan untuk Teks, Gambar, Office, PDF, dan lainnya
  • ☁️ Manajemen dan Cadangan File WebDAV
  • 📊 Visualisasi Diagram Mermaid
  • 💻 Penyorotan Sintaks Kode
  1. Integrasi Alat Praktis:
  • 🔍 Fungsionalitas Pencarian Global
  • 📝 Sistem Manajemen Topik
  • 🔤 Penerjemahan Berbasis AI
  • 🎯 Pengurutan Drag-and-drop
  • 🔌 Dukungan Mini Program
  • ⚙️ Server MCP (Model Context Protocol)
  1. Pengalaman Pengguna yang Ditingkatkan:
  • 🖥️ Dukungan lintas platform untuk Windows, Mac, dan Linux
  • 📦 Siap Pakai, Tidak Memerlukan Pengaturan Lingkungan
  • 🎨 Tema Terang/Gelap dan Jendela Transparan
  • 📝 Rendering Markdown Lengkap
  • 🤲 Berbagi Konten dengan Mudah

🔧 Program Kolaborasi Pengembang

Kami meluncurkan Program Ko-Kreasi Pengembang Cherry Studio untuk mendorong terciptanya ekosistem umpan balik yang sehat dan positif di lingkungan open-source. Kami percaya bahwa perangkat lunak hebat dibangun secara kolaboratif, dan setiap pull request yang berhasil digabungkan memberikan kehidupan baru pada proyek ini.

Kami dengan tulus mengundang Anda untuk bergabung menjadi kontributor dan membentuk masa depan Cherry Studio bersama kami.

Program Penghargaan Kontributor

Untuk memberikan apresiasi kepada kontributor inti kami dan menciptakan siklus yang baik, kami telah menetapkan rencana insentif jangka panjang berikut.

Periode pelacakan perdana untuk program ini adalah Q3 2025 (Juli, Agustus, September). Penghargaan untuk siklus ini akan didistribusikan pada 1 Oktober.

Dalam setiap periode pelacakan (misalnya, 1 Juli hingga 30 September untuk siklus pertama), setiap pengembang yang berkontribusi lebih dari 30 commit bermakna ke salah satu proyek open-source Cherry Studio di GitHub berhak mendapatkan manfaat berikut:

  • Sponsorship Langganan Cursor: Mendapatkan kredit atau penggantian biaya sebesar $70 USD untuk langganan Cursor Anda, menjadikan AI sebagai mitra coding Anda yang paling efisien.
  • Akses Model Tanpa Batas: Mendapatkan akses API tanpa batas untuk model DeepSeek dan Qwen.
  • Akses Teknologi Terkini: Nikmati manfaat eksklusif, termasuk akses API ke model seperti Claude, Gemini, dan OpenAI, sehingga Anda tetap berada di garis depan teknologi.

Bertumbuh Bersama & Rencana Masa Depan

Komunitas yang dinamis adalah kekuatan pendorong di balik proyek open-source yang berkelanjutan. Seiring pertumbuhan Cherry Studio, program penghargaan kami juga akan berkembang. 🏢 Enterprise Edition

Berdasarkan Community Edition, kami dengan bangga memperkenalkan Cherry Studio Enterprise Edition—platform AI produktivitas dan manajemen yang dapat di-deploy secara privat, dirancang untuk tim dan perusahaan modern.

Enterprise Edition menangani tantangan utama kolaborasi tim dengan memusatkan manajemen sumber daya AI, pengetahuan, dan data. Solusi ini memberdayakan organisasi untuk meningkatkan efisiensi, mendorong inovasi, dan memastikan kepatuhan, sekaligus menjaga kontrol penuh atas data Anda dalam lingkungan yang aman.

Keunggulan Inti

  • Manajemen Model Terpadu: Integrasi dan manajemen terpusat untuk berbagai LLM berbasis cloud (misal OpenAI, Anthropic, Google Gemini) serta model privat yang di-deploy secara lokal. Karyawan dapat langsung menggunakannya tanpa konfigurasi individu.
  • Basis Pengetahuan Kelas Perusahaan: Bangun, kelola, dan bagikan basis pengetahuan untuk seluruh tim. Pastikan pengetahuan tetap tersimpan dan konsisten, sehingga anggota tim dapat berinteraksi dengan AI berdasarkan informasi yang terintegrasi dan akurat.
  • Kontrol Akses Granular: Kelola akun karyawan dan tetapkan hak akses berbasis peran untuk setiap model, basis pengetahuan, dan fitur melalui backend admin yang terpusat.
  • Deploy Secara Privat Sepenuhnya: Deploy seluruh layanan backend di server on-premises atau private cloud Anda, memastikan data Anda tetap 100% privat dan di bawah kendali Anda demi memenuhi standar keamanan dan kepatuhan yang paling ketat.
  • Layanan Backend Andal: Menyediakan layanan API yang stabil, mekanisme backup dan pemulihan data kelas perusahaan untuk menjamin kelangsungan bisnis.
