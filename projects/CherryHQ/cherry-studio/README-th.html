🍒 Cherry Studio

Cherry Studio เป็นไคลเอนต์เดสก์ท็อปที่รองรับผู้ให้บริการ LLM หลายราย ใช้งานได้บน Windows, Mac และ Linux

👏 เข้าร่วม กลุ่ม Telegram｜Discord | กลุ่ม QQ(575014769)

❤️ ชอบ Cherry Studio ไหม? ให้ดาว 🌟 หรือ สนับสนุน เพื่อช่วยในการพัฒนา!

🌠 ภาพหน้าจอ

🌟 คุณสมบัติเด่น

  1. รองรับผู้ให้บริการ LLM หลากหลาย:
  • ☁️ บริการคลาวด์ LLM หลัก: OpenAI, Gemini, Anthropic และอื่นๆ
  • 🔗 การเชื่อมต่อบริการ AI บนเว็บ: Claude, Peplexity, Poe และอื่นๆ
  • 💻 รองรับโมเดลโลคอลด้วย Ollama, LM Studio
  1. ผู้ช่วย AI & การสนทนา:
  • 📚 ผู้ช่วย AI ที่ตั้งค่ามาก่อนกว่า 300+ รายการ
  • 🤖 การสร้างผู้ช่วยแบบกำหนดเอง
  • 💬 การสนทนาแบบหลายโมเดลพร้อมกัน
  1. การประมวลผลเอกสารและข้อมูล:
  • 📄 รองรับข้อความ รูปภาพ เอกสาร Office, PDF และอื่น ๆ
  • ☁️ การจัดการไฟล์และสำรองข้อมูลผ่าน WebDAV
  • 📊 การแสดงผลแผนภูมิ Mermaid
  • 💻 ไฮไลท์โค้ดตามรูปแบบไวยากรณ์
  1. การผสานเครื่องมือที่ใช้งานได้จริง:
  • 🔍 ฟังก์ชันการค้นหาทั่วโลก
  • 📝 ระบบจัดการหัวข้อ
  • 🔤 การแปลภาษาด้วย AI
  • 🎯 การจัดเรียงด้วยการลากและวาง
  • 🔌 รองรับ Mini Program
  • ⚙️ เซิร์ฟเวอร์ MCP (Model Context Protocol)
  1. ประสบการณ์ผู้ใช้ที่ดียิ่งขึ้น:
  • 🖥️ รองรับข้ามแพลตฟอร์มสำหรับ Windows, Mac และ Linux
  • 📦 พร้อมใช้งานทันที ไม่ต้องตั้งค่าสภาพแวดล้อม
  • 🎨 ธีมสว่าง/มืด และหน้าต่างโปร่งใส
  • 📝 การแสดงผล Markdown อย่างสมบูรณ์
  • 🤲 แชร์เนื้อหาได้อย่างง่ายดาย

📝 Roadmap

เรากำลังพัฒนาและปรับปรุงฟีเจอร์ต่อไปนี้อย่างต่อเนื่อง:

  1. 🎯 คุณสมบัติหลัก
  • ผู้ช่วยเลือกเนื้อหา – เพิ่มประสิทธิภาพการเลือกเนื้อหาอัจฉริยะ
  • การวิจัยเชิงลึก – ความสามารถในการวิจัยขั้นสูง
  • ระบบหน่วยความจำ – การรับรู้บริบททั่วทั้งระบบ
  • การประมวลผลเอกสารล่วงหน้า – ปรับปรุงการจัดการเอกสาร
  • ตลาด MCP – ระบบนิเวศ Model Context Protocol
  1. 🗂 การจัดการความรู้
  • โน้ตและคอลเลกชัน
  • การแสดงผล Canvas แบบไดนามิก
  • ความสามารถ OCR
  • รองรับ TTS (Text-to-Speech)
  1. 📱 รองรับแพลตฟอร์ม
  • HarmonyOS Edition (PC)
  • แอป Android (เฟส 1)
  • แอป iOS (เฟส 1)
  • รองรับ Multi-Window
  • ฟังก์ชันปักหมุดหน้าต่าง
  1. 🔌 ฟีเจอร์ขั้นสูง
  • ระบบปลั๊กอิน
  • ASR (การรู้จำเสียงพูดอัตโนมัติ)
  • ปรับโครงสร้างการโต้ตอบระหว่างผู้ช่วยและหัวข้อ

🤝 การมีส่วนร่วม

เรายินดีต้อนรับทุกการมีส่วนร่วมกับ Cherry Studio! คุณสามารถมีส่วนร่วมได้ดังนี้:

  1. ส่งโค้ด: พัฒนาฟีเจอร์ใหม่หรือปรับปรุงโค้ดเดิม
  2. แก้ไขบั๊ก: ส่งการแก้ไขสำหรับบั๊กที่พบ
  3. ดูแลปัญหา: ช่วยบริหารจัดการ issue บน GitHub
  4. ออกแบบผลิตภัณฑ์: เข้าร่วมอภิปรายด้านการออกแบบ
  5. เขียนเอกสาร: ปรับปรุงคู่มือและเอกสารการใช้งาน
  6. มีส่วนร่วมกับชุมชน: ร่วมพูดคุยและช่วยเหลือผู้ใช้
  7. โปรโมตการใช้งาน: เผยแพร่และประชาสัมพันธ์ Cherry Studio

ดูแนวทางการมีส่วนร่วมได้ที่ Branching Strategy

เริ่มต้น

  1. Fork Repository: Fork และ clone ไปยังเครื่องของคุณ
  2. สร้าง Branch: สำหรับการเปลี่ยนแปลงของคุณ
  3. ส่งการเปลี่ยนแปลง: Commit และ push การเปลี่ยนแปลงของคุณ
  4. เปิด Pull Request: อธิบายการเปลี่ยนแปลงและเหตุผล

เราขอเชิญชวนคุณมาร่วมเป็นหนึ่งในผู้มีส่วนร่วมและร่วมสร้างอนาคตของ Cherry Studio ไปกับเรา

โปรแกรมรางวัลสำหรับผู้มีส่วนร่วม

เพื่อเป็นการตอบแทนผู้มีส่วนร่วมหลักและสร้างวงจรที่ดีงาม เราได้จัดตั้งแผนจูงใจระยะยาวดังต่อไปนี้

รอบการติดตามครั้งแรกของโปรแกรมนี้จะอยู่ในไตรมาสที่ 3 ปี 2025 (กรกฎาคม สิงหาคม กันยายน) โดยรางวัลของรอบนี้จะถูกแจกจ่ายในวันที่ 1 ตุลาคม

ในช่วงระยะเวลาติดตามใด ๆ (เช่น 1 กรกฎาคม ถึง 30 กันยายน สำหรับรอบแรก) นักพัฒนาคนใดที่มีส่วนร่วมมากกว่า 30 คอมมิตที่มีความหมาย ในโปรเจกต์โอเพนซอร์สของ Cherry Studio บน GitHub จะมีสิทธิ์ได้รับสิทธิประโยชน์ดังนี้:

  • สนับสนุนการสมัครสมาชิก Cursor: รับเครดิตหรือเงินคืนมูลค่า $70 USD สำหรับการสมัครสมาชิก Cursor ให้ AI กลายเป็นคู่หูเขียนโค้ดที่มีประสิทธิภาพสูงสุดของคุณ
  • เข้าถึงโมเดลแบบไม่จำกัด: ได้รับ API calls ไม่จำกัด สำหรับโมเดล DeepSeek และ Qwen
  • เข้าถึงเทคโนโลยีล้ำสมัย: รับสิทธิพิเศษเป็นระยะ เช่น การเข้าถึง API สำหรับโมเดล Claude, Gemini และ OpenAI เพื่อให้คุณอยู่ในแนวหน้าของเทคโนโลยี

เติบโตไปด้วยกัน & แผนในอนาคต

ชุมชนที่มีชีวิตชีวาคือแรงขับเคลื่อนสำคัญของโปรเจกต์โอเพนซอร์สที่ยั่งยืน เมื่อ Cherry Studio เติบโตขึ้น โปรแกรมรางวัลของเราก็จะพัฒนาตามไปด้วย เรามุ่งมั่นที่จะปรับปรุงสิทธิประโยชน์ของเราให้สอดคล้องกับเครื่องมือและทรัพยากรที่ดีที่สุดในอุตสาหกรรม เพื่อให้ผู้มีส่วนร่วมหลักได้รับการสนับสนุนที่มีความหมาย สร้างวงจรที่ดีที่นักพัฒนา ชุมชน และโปรเจกต์เติบโตไปด้วยกัน

ต่อจากนี้ โปรเจกต์จะเปิดกว้างมากขึ้นเรื่อย ๆ เพื่อให้กลับคืนแก่ชุมชนโอเพนซอร์สทั้งหมด

🏢 Enterprise Edition

ต่อยอดจาก Community Edition เราภูมิใจนำเสนอ Cherry Studio Enterprise Edition — แพลตฟอร์ม AI ด้านการเพิ่มประสิทธิภาพและการจัดการสำหรับทีมและองค์กรยุคใหม่ ที่สามารถติดตั้งใช้งานแบบส่วนตัวได้

Enterprise Edition แก้ไขปัญหาหลักในการทำงานร่วมกันของทีม โดยการรวมศูนย์การจัดการทรัพยากร AI องค์ความรู้ และข้อมูล ช่วยให้องค์กรเพิ่มประสิทธิภาพ ส่งเสริมนวัตกรรม และรับรองการปฏิบัติตามข้อกำหนด ทั้งยังควบคุมข้อมูลได้ 100% ในสภาพแวดล้อมที่ปลอดภัย

ข้อได้เปรียบหลัก

  • การจัดการโมเดลแบบรวมศูนย์: รวมและจัดการ LLM ที่หลากหลายทั้งแบบคลาวด์ (เช่น OpenAI, Anthropic, Google Gemini) และโมเดลส่วนตัวที่ติดตั้งใช้งานในองค์กร พนักงานสามารถใช้งานได้ทันทีโดยไม่ต้องตั้งค่ารายบุคคล
  • ฐานความรู้ระดับองค์กร: สร้าง จัดการ และแบ่งปันฐานความรู้สำหรับทั้งทีม มั่นใจได้ว่าองค์ความรู้ยังคงอยู่และมีความสอดคล้อง ให้สมาชิกในทีมโต้ตอบกับ AI ได้โดยอิงจากข้อมูลที่ถูกต้องและเป็นหนึ่งเดียว
  • ควบคุมสิทธิ์การเข้าถึงอย่างละเอียด: จัดการบัญชีพนักงานและกำหนดสิทธิ์ตามบทบาทสำหรับโมเดล ฐานความรู้ และฟีเจอร์ต่าง ๆ ได้อย่างง่ายดายผ่านแอดมินแบ็คเอนด์แบบรวมศูนย์
  • ติดตั้งใช้งานแบบส่วนตัวเต็มรูปแบบ: ติดตั้งบริการแบ็คเอนด์ทั้งหมดลงบนเซิร์ฟเวอร์ของคุณเองหรือคลาวด์ส่วนตัว เพื่อให้มั่นใจว่าข้อมูลของคุณเป็นส่วนตัว 100% และอยู่ภายใต้การควบคุม เพื่อให้เป็นไปตามมาตรฐานความปลอดภัยและข้อกำหนดที่เข้มงวดที่สุด
  • บริการแบ็คเอนด์ที่เชื่อถือได้: ให้บริการ API ที่เสถียร ระบบสำรองและกู้คืนข้อมูลระดับองค์กร เพื่อความต่อเนื่องทางธุรกิจ
