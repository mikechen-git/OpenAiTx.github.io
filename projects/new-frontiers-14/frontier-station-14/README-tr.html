Derleme

  1. Bu repoyu klonlayın:
git clone https://github.com/new-frontiers-14/frontier-station-14.git
  1. Proje klasörüne gidin ve alt modülleri başlatmak ve motoru yüklemek için RUN_THIS.py dosyasını çalıştırın:
cd frontier-station-14
python RUN_THIS.py
  1. Çözümü derleyin:

Sunucuyu dotnet build komutuyla derleyin.

Projeyi derleme hakkında daha ayrıntılı talimatlar.
