FanImeTsf

이것은 FanImeServer의 TSF 엔드입니다.

알림: 현재 64비트 앱만 지원합니다.

빌드 방법

필수 조건

  • Visual Studio 2022
  • CMake
  • vcpkg
  • Python3
  • gsudo

vcpkg와 gsudo가 Scoop으로 설치되어 있는지 확인하십시오.

빌드 단계

빌드

먼저, 저장소를 클론합니다.

git clone --recursive https://github.com/fanlumaster/FanImeTsf.git

그 다음, 환경을 준비합니다.

cd FanImeTsf
python .\scripts\prepare_env.py

그리고, 빌드합니다.

.\scripts\lcompile.ps1

설치

파워셸을 관리자 권한으로 실행하고, 시스템의 Enable sudo 옵션이 활성화되어 있는지 확인하십시오.

그런 다음, C:\Program Files\에 FanImeTsf라는 폴더를 만들고, FanImeTsf.dll 파일을 복사합니다.

gsudo
Copy-Item -Path ".\FanImeTsf\build64\Debug\FanImeTsf.dll" -Destination "C:\Program Files\FanImeTsf"

그런 다음, 설치합니다.

cd "C:\Program Files\FanImeTsf"
sudo regsvr32 .\FanImeTsf.dll

제거

cd "C:\Program Files\FanImeTsf"
sudo regsvr32 /u .\FanImeTsf.dll

스크린샷

로드맵

현재 샤오허 쌍병만 지원합니다.

중국어

  • 샤오허 쌍병
  • 전음(Quanpin)
  • 한자 부수 사용 시 도움 코드
  • 사용자 정의 가능한 사전
  • 사용자 정의 IME 엔진
  • 사용자 정의 스킨
  • 간체/번체 전환
  • 영어 자동 완성
  • 오픈소스 클라우드 IME API
  • 후보창 UI를 세로/가로 모드로 전환
  • 기능 스위치: 대부분의 기능은 사용자가 자유롭게 전환하거나 사용자 정의할 수 있어야 합니다.
