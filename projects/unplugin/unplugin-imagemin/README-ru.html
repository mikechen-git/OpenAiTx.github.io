Squoosh && Svgo

Unplugin-imagemin поддерживает два инструмента сжатия

Squoosh — это веб-приложение для сжатия изображений, уменьшающее их размер за счет различных форматов. Squoosh на rust & wasm

Svgo Поддержка сжатия изображений в формате svg

📦 Установка

pnpm add unplugin-imagemin@latest -D

#### поддержка vite и rollup

Базовый пример

import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';
import imagemin from 'unplugin-imagemin/vite';
// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue(), imagemin()],
});
