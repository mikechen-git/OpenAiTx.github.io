Squoosh && Svgo

Unplugin-imagemin mendukung dua alat kompresi

Squoosh adalah aplikasi web kompresi gambar yang mengurangi ukuran gambar melalui berbagai format. Squoosh dengan rust & wasm

Svgo Mendukung kompresi gambar dalam format svg

📦 Instalasi

pnpm add unplugin-imagemin@latest -D

#### mendukung vite dan rollup

Dasar

import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';
import imagemin from 'unplugin-imagemin/vite';
// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue(), imagemin()],
});
