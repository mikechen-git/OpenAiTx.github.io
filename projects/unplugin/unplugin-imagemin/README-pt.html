Squoosh && Svgo

Unplugin-imagemin suporta duas ferramentas de compressão

Squoosh é um aplicativo web de compressão de imagens que reduz o tamanho das imagens em vários formatos. Squoosh com rust & wasm

Svgo Suporte à compressão de imagens no formato svg

📦 Instalação

pnpm add unplugin-imagemin@latest -D

#### suporte a vite e rollup

Básico

import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';
import imagemin from 'unplugin-imagemin/vite';
// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue(), imagemin()],
});
