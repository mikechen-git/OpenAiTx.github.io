Squoosh && Svgo

افزونه unplugin-imagemin از دو ابزار فشرده‌سازی پشتیبانی می‌کند

Squoosh یک برنامه وب برای فشرده‌سازی تصویر است که اندازه تصاویر را در فرمت‌های مختلف کاهش می‌دهد. Squoosh با rust و wasm

Svgo پشتیبانی از فشرده‌سازی تصاویر با فرمت svg

📦 نصب

pnpm add unplugin-imagemin@latest -D

import { defineConfig } from 'vite';
