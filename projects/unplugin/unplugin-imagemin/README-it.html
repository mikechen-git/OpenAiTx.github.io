Squoosh && Svgo

Unplugin-imagemin supporta due strumenti di compressione

Squoosh è un'app web per la compressione delle immagini che riduce le dimensioni tramite numerosi formati. Squoosh con rust & wasm

Svgo Supporta la compressione delle immagini in formato svg

📦 Installazione

pnpm add unplugin-imagemin@latest -D

#### supporta vite e rollup

Base

import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';
import imagemin from 'unplugin-imagemin/vite';
// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue(), imagemin()],
});
