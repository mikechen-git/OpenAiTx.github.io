Squoosh && Svgo

Unplugin-imagemin ondersteunt twee compressietools

Squoosh is een webapp voor beeldcompressie die afbeeldingsgroottes reduceert via diverse formaten. Squoosh met rust & wasm

Svgo Ondersteunt compressie van afbeeldingen in svg-formaat

📦 Installatie

pnpm add unplugin-imagemin@latest -D

#### ondersteuning voor vite en rollup

Basis

import { defineConfig } from 'vite';
import vue from '@vitejs/plugin-vue';
import imagemin from 'unplugin-imagemin/vite';
// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue(), imagemin()],
});
