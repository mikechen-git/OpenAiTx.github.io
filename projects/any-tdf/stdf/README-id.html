STDF

简体中文 • Website

Pendahuluan

Pustaka komponen web mobile berbasis Svelte dan Tailwind.

Simple • Tiny • Design • Fast

Fitur

  • 🔥 Dukungan penuh untuk Svelte v5, Tailwind CSS v4, dan TypeScript.
  • 🚀 Tanpa runtime, tanpa virtual DOM — berjalan lebih cepat secara online.
  • 🧰 API kaya — mudah mengatur gaya komponen sesuai kebutuhan Anda.
  • 🍭 Mendukung mode gelap dan konfigurasi tema ganda.
  • 📖 Menyediakan dokumentasi lengkap dalam Bahasa Mandarin dan Inggris serta contoh komponen.
  • 🌍 Mendukung internasionalisasi, dengan lebih dari 60 paket bahasa bawaan.
  • 🫰 Interaksi ramah pengguna, parameter animasi dapat dikonfigurasi, serta mendukung tema kustom.
  • 🤝 Mendukung pustaka Tailwind seperti UnoCSS.
  • 🫡 Tanpa dependensi pihak ketiga — tidak perlu khawatir soal versi pustaka atau masalah keamanan.
  • 📦 Mendukung impor sesuai kebutuhan untuk mengurangi ukuran bundle.
  • 🪜 Dilengkapi alat scaffolding untuk pembuatan proyek dengan cepat.
  • 🔗 Dilengkapi plugin penggabungan SVG untuk menggabungkan file SVG secara cepat, mendukung Iconify.
  • 🍺 Plugin IDE untuk melihat API komponen dengan cepat.

Memulai

pnpm create stdf@latest

npm create stdf@latest
# atau
npm init stdf@latest
# atau
npx create-stdf@latest

bun create stdf@latest

yarn create stdf@latest
