परिचय

Svelte और Tailwind पर आधारित मोबाइल वेब घटक लाइब्रेरी।

Simple • Tiny • Design • Fast

विशेषताएँ

  • 🔥 Svelte v5, Tailwind CSS v4, और TypeScript के लिए पूर्ण समर्थन।
  • 🚀 कोई रनटाइम नहीं, कोई वर्चुअल DOM नहीं — ऑनलाइन तेज़ चलता है।
  • 🧰 समृद्ध API — अपनी आवश्यकताओं के अनुसार घटकों की शैलियाँ आसानी से कॉन्फ़िगर करें।
  • 🍭 डार्क मोड और मल्टीपल थीम कॉन्फ़िगरेशन का समर्थन करता है।
  • 📖 व्यापक चीनी और अंग्रेजी दस्तावेज़ और घटक उदाहरण प्रदान करता है।
  • 🌍 अंतर्राष्ट्रीयकरण का समर्थन करता है, 60+ अंतर्निहित भाषा पैकेजों के साथ।
  • 🫰 यूज़र-फ्रेंडली इंटरैक्शन, एनीमेशन पैरामीटर कॉन्फ़िगर किए जा सकते हैं, और कस्टम थीम का समर्थन करता है।
  • 🤝 UnoCSS जैसी Tailwind लाइब्रेरीज़ का समर्थन करता है।
  • 🫡 कोई थर्ड-पार्टी डिपेंडेंसी नहीं — लाइब्रेरी वर्शन या सुरक्षा मुद्दों की चिंता नहीं।
  • 📦 ऑन-डिमांड इंपोर्ट का समर्थन करता है जिससे बंडल साइज़ कम होता है।
  • 🪜 जल्दी प्रोजेक्ट बनाने के लिए स्कैफोल्डिंग टूल के साथ आता है।
  • 🔗 तेज़ी से SVG फाइलों को मर्ज करने के लिए SVG मर्जिंग प्लगइन के साथ आता है, Iconify का समर्थन।
  • 🍺 घटक API जल्दी देखने के लिए IDE प्लगइन।
