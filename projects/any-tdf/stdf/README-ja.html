STDF

简体中文 • Website

はじめに

Svelte と Tailwind に基づいたモバイルWebコンポーネントライブラリ。

Simple（シンプル） • Tiny（小型） • Design（デザイン） • Fast（高速）

特徴

  • 🔥 Svelte v5、Tailwind CSS v4、TypeScript を完全サポート。
  • 🚀 ランタイムなし、仮想DOMなし — オンラインでより高速に動作。
  • 🧰 豊富なAPI — コンポーネントのスタイルを簡単に設定可能。
  • 🍭 ダークモードおよび複数テーマの設定をサポート。
  • 📖 豊富な中国語および英語ドキュメントとコンポーネント例を提供。
  • 🌍 60以上の言語パッケージを内蔵し、国際化対応。
  • 🫰 使いやすいインタラクション、アニメーションパラメータの設定やカスタムテーマに対応。
  • 🤝 UnoCSSなどのTailwind系ライブラリをサポート。
  • 🫡 サードパーティ依存なし — ライブラリのバージョンやセキュリティ問題の心配不要。
  • 📦 必要なものだけの個別インポートでバンドルサイズ削減。
  • 🪜 プロジェクト作成を迅速化するスキャフォールディングツール同梱。
  • 🔗 SVG統合プラグイン同梱でSVGファイルを素早く統合、Iconify対応。
  • 🍺 コンポーネントAPIをすばやく確認できるIDEプラグイン。

はじめ方

pnpm create stdf@latest

npm create stdf@latest
# or
npm init stdf@latest
# or
npx create-stdf@latest

bun create stdf@latest
