STDF

简体中文 • Website

แนะนำ

ไลบรารีคอมโพเนนต์เว็บสำหรับมือถือที่พัฒนาบนพื้นฐานของ Svelte และ Tailwind

Simple • Tiny • Design • Fast

คุณสมบัติ

  • 🔥 รองรับ Svelte v5, Tailwind CSS v4 และ TypeScript อย่างเต็มรูปแบบ
  • 🚀 ไม่มี runtime ไม่มี virtual DOM — ทำงานได้เร็วกว่าออนไลน์
  • 🧰 API ที่หลากหลาย — กำหนดสไตล์คอมโพเนนต์ได้ง่ายตามต้องการ
  • 🍭 รองรับโหมดมืดและตั้งค่าธีมได้หลายแบบ
  • 📖 มีเอกสารภาษาอังกฤษและจีนที่ครอบคลุม พร้อมตัวอย่างคอมโพเนนต์
  • 🌍 รองรับการแปลภาษาสากล พร้อมแพ็กเกจภาษาในตัวกว่า 60 ภาษา
  • 🫰 อินเทอร์เฟซที่ใช้งานง่าย กำหนดพารามิเตอร์แอนิเมชันได้ รองรับธีมแบบกำหนดเอง
  • 🤝 รองรับไลบรารี Tailwind เช่น UnoCSS
  • 🫡 ไม่มีการพึ่งพาไลบรารีภายนอก — ไม่ต้องกังวลเรื่องเวอร์ชันหรือความปลอดภัย
  • 📦 รองรับการนำเข้าเฉพาะที่ใช้งาน เพื่อลดขนาด bundle
  • 🪜 มีเครื่องมือ scaffolding สำหรับสร้างโปรเจกต์อย่างรวดเร็ว
  • 🔗 มาพร้อมปลั๊กอินรวมไฟล์ SVG เพื่อรวมไฟล์ SVG ได้อย่างรวดเร็ว รองรับ Iconify
  • 🍺 ปลั๊กอินสำหรับ IDE เพื่อดู API ของคอมโพเนนต์ได้อย่างรวดเร็ว
