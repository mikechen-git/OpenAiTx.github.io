STDF

简体中文 • الموقع الإلكتروني

المقدمة

مكتبة مكونات ويب للهواتف المحمولة مبنية على Svelte و Tailwind .

Simple • Tiny • Design • Fast

الميزات

  • 🔥 دعم كامل لـ Svelte v5 وTailwind CSS v4 وTypeScript.
  • 🚀 بدون وقت تشغيل، بدون DOM افتراضي — أداء أسرع على الإنترنت.
  • 🧰 واجهة برمجة تطبيقات غنية — يمكنك بسهولة ضبط أنماط المكونات لتلبية احتياجاتك.
  • 🍭 يدعم الوضع الداكن وتكوين السمات المتعددة.
  • 📖 يوفر وثائق شاملة باللغتين الصينية والإنجليزية وأمثلة للمكونات.
  • 🌍 يدعم التدويل، مع أكثر من 60 حزمة لغة مدمجة.
  • 🫰 تفاعل سهل الاستخدام، يمكن ضبط معلمات الحركة ويدعم السمات المخصصة.
  • 🤝 يدعم مكتبات Tailwind مثل UnoCSS.
  • 🫡 بدون تبعيات من طرف ثالث — لا داعي للقلق بشأن إصدارات المكتبات أو مشكلات الأمان.
  • 📦 يدعم الاستيراد عند الطلب لتقليل حجم الحزمة.
  • 🪜 يأتي مع أداة توليد المشاريع السريعة.
  • 🔗 يأتي مع إضافة دمج SVG لدمج ملفات SVG بسرعة، يدعم Iconify.
  • 🍺 إضافة لـ IDE لعرض واجهات برمجة تطبيقات المكونات بسرعة.
