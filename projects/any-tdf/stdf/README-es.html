STDF

简体中文 • Sitio Web

Introducción

Biblioteca de componentes web móviles basada en Svelte y Tailwind .

Simple • Tiny • Design • Fast

Características

  • 🔥 Soporte completo para Svelte v5, Tailwind CSS v4 y TypeScript.
  • 🚀 Sin runtime, sin DOM virtual — ejecución más rápida en línea.
  • 🧰 API rica — configura fácilmente los estilos de los componentes para adaptarlos a tus necesidades.
  • 🍭 Soporta modo oscuro y configuración de múltiples temas.
  • 📖 Documentación extensa en chino e inglés y ejemplos de componentes.
  • 🌍 Soporta internacionalización, con más de 60 paquetes de idiomas integrados.
  • 🫰 Interacción fácil de usar, los parámetros de animación son configurables y se admiten temas personalizados.
  • 🤝 Soporta bibliotecas de Tailwind como UnoCSS.
  • 🫡 Sin dependencias de terceros — sin preocuparse por versiones de librerías o problemas de seguridad.
  • 📦 Soporta importación bajo demanda para reducir el tamaño del bundle.
  • 🪜 Incluye una herramienta de scaffolding para la creación rápida de proyectos.
  • 🔗 Incluye un plugin para la fusión de SVGs, soporta Iconify.
  • 🍺 Plugin para IDE para ver rápidamente las APIs de los componentes.
