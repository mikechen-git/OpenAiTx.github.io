简体中文 • Sito web

Introduzione

Libreria di componenti web mobile basata su Svelte e Tailwind .

Semplice • Tiny • Design • Fast

Caratteristiche

  • 🔥 Supporto completo per Svelte v5, Tailwind CSS v4 e TypeScript.
  • 🚀 Nessun runtime, nessun virtual DOM — esecuzione più veloce online.
  • 🧰 API ricca — configura facilmente gli stili dei componenti per soddisfare le tue esigenze.
  • 🍭 Supporta la modalità scura e la configurazione di più temi.
  • 📖 Fornisce ampia documentazione in cinese e inglese ed esempi di componenti.
  • 🌍 Supporta l'internazionalizzazione, con oltre 60 pacchetti lingua integrati.
  • 🫰 Interazione user-friendly, parametri di animazione configurabili e supporto per temi personalizzati.
  • 🤝 Supporta librerie Tailwind come UnoCSS.
  • 🫡 Nessuna dipendenza di terze parti — nessuna preoccupazione per le versioni delle librerie o problemi di sicurezza.
  • 📦 Supporta l'importazione su richiesta per ridurre la dimensione del bundle.
  • 🪜 Include uno strumento scaffolding per la creazione rapida di progetti.
  • 🔗 Include un plugin per unire file SVG rapidamente, supporta Iconify.
  • 🍺 Plugin IDE per visualizzare rapidamente le API dei componenti.

Per iniziare

pnpm create stdf@latest

npm create stdf@latest
# oppure
npm init stdf@latest
# oppure
npx create-stdf@latest
