Giới thiệu

Thư viện thành phần web di động dựa trên Svelte và Tailwind.

Simple • Tiny • Design • Fast

Tính năng

  • 🔥 Hỗ trợ đầy đủ Svelte v5, Tailwind CSS v4 và TypeScript.
  • 🚀 Không có runtime, không có virtual DOM — chạy nhanh hơn trên môi trường online.
  • 🧰 API phong phú — dễ dàng cấu hình kiểu dáng thành phần để phù hợp nhu cầu của bạn.
  • 🍭 Hỗ trợ chế độ tối và nhiều cấu hình chủ đề khác nhau.
  • 📖 Cung cấp tài liệu và ví dụ thành phần phong phú bằng tiếng Trung và tiếng Anh.
  • 🌍 Hỗ trợ quốc tế hóa, với hơn 60 gói ngôn ngữ tích hợp sẵn.
  • 🫰 Tương tác thân thiện với người dùng, tham số hoạt ảnh có thể cấu hình, hỗ trợ chủ đề tùy chỉnh.
  • 🤝 Hỗ trợ các thư viện Tailwind như UnoCSS.
  • 🫡 Không phụ thuộc vào thư viện bên thứ ba — không lo về phiên bản hoặc các vấn đề bảo mật.
  • 📦 Hỗ trợ nhập theo nhu cầu giúp giảm kích thước bundle.
  • 🪜 Tích hợp công cụ scaffolding để khởi tạo dự án nhanh chóng.
  • 🔗 Tích hợp plugin gộp SVG giúp gộp nhanh các tệp SVG, hỗ trợ Iconify.
  • 🍺 Plugin IDE để xem nhanh API thành phần.
