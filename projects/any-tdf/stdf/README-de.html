STDF









简体中文 • Website

Einführung

Mobile Web-Komponentenbibliothek basierend auf Svelte und Tailwind.

Simple • Tiny • Design • Fast

Merkmale

  • 🔥 Volle Unterstützung für Svelte v5, Tailwind CSS v4 und TypeScript.
  • 🚀 Kein Runtime, kein virtuelles DOM — läuft schneller online.
  • 🧰 Umfangreiche API — Komponentenstile einfach konfigurieren, um Ihre Anforderungen zu erfüllen.
  • 🍭 Unterstützt Dark Mode und mehrere Theme-Konfigurationen.
  • 📖 Bietet umfangreiche chinesische und englische Dokumentation sowie Komponentenbeispiele.
  • 🌍 Unterstützt Internationalisierung, mit über 60 integrierten Sprachpaketen.
  • 🫰 Benutzerfreundliche Interaktion, Animationsparameter sind konfigurierbar, benutzerdefinierte Themes werden unterstützt.
  • 🤝 Unterstützt Tailwind-Bibliotheken wie UnoCSS.
  • 🫡 Keine Drittanbieter-Abhängigkeiten — keine Sorgen um Bibliotheksversionen oder Sicherheitsprobleme.
  • 📦 Unterstützt bedarfsgerechten Import zur Reduzierung der Bundle-Größe.
  • 🪜 Kommt mit einem Scaffolding-Tool für die schnelle Projekterstellung.
  • 🔗 Mitgeliefertes SVG-Merge-Plugin zum schnellen Zusammenführen von SVG-Dateien, unterstützt Iconify.
  • 🍺 IDE-Plugin für die schnelle Ansicht von Komponenten-APIs.

Schnellstart

pnpm create stdf@latest

npm create stdf@latest
# oder
npm init stdf@latest
# oder
npx create-stdf@latest

bun create stdf@latest

yarn create stdf@latest
