STDF

简体中文 • Website

Giriş

Svelte ve Tailwind tabanlı mobil web bileşen kütüphanesi.

Simple • Tiny • Design • Fast

Özellikler

  • 🔥 Svelte v5, Tailwind CSS v4 ve TypeScript için tam destek.
  • 🚀 Çalışma zamanı yok, sanal DOM yok — çevrimiçi daha hızlı çalışır.
  • 🧰 Zengin API — bileşen stillerini ihtiyacınıza göre kolayca yapılandırın.
  • 🍭 Karanlık mod ve çoklu tema yapılandırmasını destekler.
  • 📖 Kapsamlı Çince ve İngilizce dokümantasyon ve bileşen örnekleri sunar.
  • 🌍 Uluslararasılaştırmayı destekler, 60'tan fazla yerleşik dil paketiyle birlikte gelir.
  • 🫰 Kullanıcı dostu etkileşim, animasyon parametreleri yapılandırılabilir ve özel temalar desteklenir.
  • 🤝 UnoCSS gibi Tailwind kütüphanelerini destekler.
  • 🫡 Üçüncü parti bağımlılık yoktur — kütüphane sürümleri veya güvenlik sorunları için endişelenmenize gerek yoktur.
  • 📦 Paket boyutunu azaltmak için isteğe bağlı ithalatı destekler.
  • 🪜 Hızlı proje oluşturmak için hazır iskelet aracı ile birlikte gelir.
  • 🔗 SVG dosyalarını hızlıca birleştirmek için SVG birleştirme eklentisi ile birlikte gelir, Iconify desteği mevcut.
  • 🍺 Bileşen API'lerini hızlıca görüntülemek için IDE eklentisi.
