Wat is NPOI


Dit project is de .NET-versie van het Apache POI-project. Met NPOI kun je eenvoudig Office 2003/2007-bestanden lezen/schrijven.

Donatie

Als je profiteert/voordeel hebt van NPOI, sponsor mij dan via Github Sponsor of doneer aan dit project. Dank je wel!

Voordelen van NPOI

a. Het is volledig gratis te gebruiken

b. Dekt de meeste functies van Excel (celstijl, dataformaat, formule, enz.)

c. Ondersteunde formaten: xls, xlsx, docx.

d. Ontworpen om interface-georiënteerd te zijn (bekijk de NPOI.SS namespace)

e. Ondersteunt niet alleen exporteren maar ook importeren

f. Echte succesvolle cases wereldwijd

g. Werkt op zowel Windows als Linux

Systeemeisen

.NET Standard 2.1 (.NET Core 3.x, .NET 5, .NET 6)

.NET Standard 2.0 (.NET Core 2.x)

.NET Framework 4.0 en hoger
