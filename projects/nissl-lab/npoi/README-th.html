What's NPOI


โปรเจกต์นี้เป็นเวอร์ชัน .NET ของโปรเจกต์ Apache POI ด้วย NPOI คุณสามารถอ่าน/เขียนไฟล์ Office 2003/2007 ได้อย่างง่ายดาย

Donation

Advantage of NPOI

a. ฟรีทั้งหมดในการใช้งาน

b. ครอบคลุมฟีเจอร์ส่วนใหญ่ของ Excel (รูปแบบเซลล์, รูปแบบข้อมูล, สูตร และอื่น ๆ)

c. รูปแบบไฟล์ที่รองรับ: xls, xlsx, docx

d. ออกแบบให้ใช้งานแบบเน้นอินเทอร์เฟซ (ดูที่ namespace NPOI.SS)

e. รองรับทั้งการส่งออกและนำเข้า

f. มีกรณีศึกษาความสำเร็จจริงทั่วโลก

g. ทำงานได้ทั้งบน Windows และ Linux

System Requirement

.NET Standard 2.1 (.NET Core 3.x, .NET 5, .NET 6)

.NET Standard 2.0 (.NET Core 2.x)

.NET Framework 4.0 ขึ้นไป
