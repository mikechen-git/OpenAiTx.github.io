Refined GitHub

Браузерное расширение, которое упрощает интерфейс GitHub и добавляет полезные функции

Мы много работаем с GitHub и замечаем множество неудобств, которые хотим исправить. Так что берегитесь, здесь могут быть «драконы».

Мы надеемся, что GitHub обратит внимание и внедрит некоторые из этих столь необходимых улучшений. Если вам нравятся какие-либо из этих улучшений, пожалуйста, откройте обсуждение на GitHub feedback или свяжитесь с поддержкой GitHub, чтобы предложить их реализацию.

Также поддерживается GitHub Enterprise: Как включить.

Торговые марки GITHUB и REFINED GITHUB принадлежат GitHub, Inc. и используются по лицензии.

Репозитории

  • 🔥 Добавляет значок статуса сборки/CI рядом с именем репозитория.
  • Добавляет полезные ссылки в выпадающее меню навигации по репозиторию
  • Добавляет ссылку для обмена веток в режиме сравнения веток.
  • Показывает возраст репозитория в боковой панели.
  • В ваших форках показывает количество открытых вами PR в оригинальный репозиторий.
  • Делает некоторые кнопки в списках файлов репозитория более компактными, чтобы освободить место для других функций.
  • Автоматически отключает проекты и вики при создании репозитория.
  • Делает боковые панели фиксированными в репозиториях, задачах и PR, если они помещаются во viewport.
  • Добавляет ссылку на github.io сайт пользователя из его репозитория.
  • В боковой панели workflows показывает индикатор возможности ручного запуска workflow и время следующего запланированного запуска, если это актуально.
  • Упрощает удаление репозиториев. Добавляет кнопку "Delete fork" для репозиториев с 0 звёздами. Включает удаление репозиториев двумя кликами с помощью shift alt.
  • Помогает находить форки в архивированных репозиториях.
  • Перемещает вкладки "Security" и "Insights" в выпадающее меню навигации по репозиторию. Также перемещает вкладки "Projects", "Actions" и "Wiki", если они пустые/не используются.
  • Добавляет аватар профиля в заголовок публичных репозиториев.
  • Добавляет ссылку для создания задачи из любой точки репозитория.
  • Показывает маленький аватар рядом с именем пользователя в списках задач/PR и упоминаниях.
  • 🔥 Показывает, смотрите ли вы на последнюю версию репозитория или есть какие-то невыпущенные коммиты.
  • 🔥 Добавляет ссылку на PR, который запустил workflow.
  • Делает вкладки более компактными на мобильных устройствах, чтобы больше вкладок помещалось на экране.
  • Показывает, является ли репозиторий форком, и добавляет количество звёзд в заголовок.
  • При просмотре файлов репозитория по определённому тегу добавляет ссылку на сам релиз/тег.

  • Добавляет ссылку для предпросмотра HTML-файлов.
  • Подсвечивает наиболее недавно изменённые элементы в списках файлов.
  • Позволяет просмотреть предыдущую версию файла в один клик.

Код

  • Улучшает горячую клавишу y: теперь она также копирует постоянную ссылку.
  • Делает кликабельными символические ссылки на файлы.

  • Добавляет горячую клавишу для снятия выделения с текущей строки: esc.
  • Показывает front matter Markdown в виде вертикальной таблицы.
  • В списках коммитов ветки показывает PR, который затрагивает текущую ветку.

Написание комментариев

  • 🔥 Включает tab и shift tab для отступов в полях комментариев.
  • Добавляет кнопку в редакторе текста для вставки сворачиваемого контента (через <details>).
  • 🔥 Автоматически изменяет размер полей комментариев по содержимому, убирая полосы прокрутки.
  • Позволяет редактировать любой комментарий одним кликом, без необходимости открывать выпадающее меню.
  • Оборачивает выделенный текст при нажатии одной из Markdown-символов вместо замены его: [ ` ' " * ~ _
  • Скрывает ненужные всплывающие подсказки и элементы панели инструментов поля комментариев (на каждую есть горячая клавиша.)
  • Добавляет кнопку для @упоминания пользователя в задачах и PR.
  • Добавляет кнопку в редакторе текста для быстрой вставки упрощённой HTML-таблицы.
  • Уведомляет пользователя о незавершённых комментариях в скрытых вкладках.
  • Добавляет кнопку для удаления комментариев к обзору одним кликом при их редактировании.
  • Отключает отправку с помощью enter в некоторых полях заголовка коммита/PR/issue, чтобы избежать случайной отправки. Используйте ctrl enter вместо этого.

Чтение комментариев

  • 🔥 Добавляет аватары реакций, показывающие кто отреагировал на комментарий.
  • Встраивает короткие gist при ссылке на них в комментариях отдельной строкой.
  • Добавляет ссылки для просмотра репозитория и связанных файлов на момент каждого комментария.
  • Добавляет настоящее имя пользователя рядом с его ником.
  • Укорачивает URL и ссылки на репозитории до читаемых ссылок вида "_user/repo/.file@d71718d".
  • 🔥 Показывает скрытые комментарии прямо в обсуждении.
  • 🔥 Подсвечивает самый полезный комментарий в задачах.
  • Скрывает комментарии с реакциями ("+1", "👍", …) (кроме комментариев мейнтейнеров) но их всё равно можно показать.
  • Ограничивает высоту больших блоков кода и цитат.
  • Упрощает интерфейс для скрытия комментариев.
  • Делает иконку "комментария" в списках задач ссылкой на последний комментарий в задаче.
  • В длинных обсуждениях, где GitHub скрывает комментарии под "N скрытых элементов. Загрузить ещё...", alt-клик загружает до 200 комментариев сразу вместо 60.
  • Добавляет горячие клавиши для навигации по задачам, PR и спискам файлов в PR: j — фокус на комментарий/файл ниже; k — выше.
  • Сообщает о скрытых комментариях в заголовке длинных задач. Также прокручивает к скрытым комментариям при нажатии Cmd+F или Ctrl+F.

  • Удаляет дублирующуюся информацию в заголовке issues и PR ("User wants to merge X commits from Y into Z")
  • Делает коммиты и PR от ботов менее заметными, чтобы уменьшить шум.
  • Добавляет ярлык для отмены редактирования заголовка issue/PR: esc.

Просмотр pull request-ов

  • Добавляет фильтры Checks и Draft PR в выпадающем списке фильтров PR.
  • Показывает цветные счетчики ревью в списках PR.
  • Показывает базовую ветку в списках PR, если это не ветка по умолчанию.
  • Скрывает неактивные деплои в PR.
  • Добавляет дублирующие кнопки навигации по коммитам внизу страницы вкладки Commits.
  • Добавляет индикаторы комментариев, когда они скрыты в ревью PR.
  • Показывает, в каких PR есть конфликты, в списках PR.
  • Добавляет статистику diff по коммитам в PR.
  • Добавляет перечеркивание для удаленных веток в PR.
  • Позволяет отметить/снять отметку "Просмотрено" сразу с нескольких файлов на вкладке Files в PR. Кликните по первому чекбоксу, затем с shift по другому — все файлы между ними будут отмечены/сняты.
  • 🔥 Показывает первую Git-метку, в которую был включен смерженный PR, или предлагает создать релиз, если еще не был выпущен.
  • Переходит к первому непросмотренному файлу в PR при клике по прогресс-бару.
  • Добавляет ссылку для перехода к последнему комментарию с запросом изменений.
  • Добавляет ссылку для открытия последнего деплоя из заголовка PR.
  • Всегда использует объединенный diff для файлов, где split-diff не имеет смысла.
  • Делает черновые PR более заметными в списках.

Редактирование pull request-ов

  • 🔥 Использует заголовок PR как заголовок squash-коммита по умолчанию и обновляет заголовок PR при изменении заголовка коммита.
  • Добавляет кнопку "Update branch" для каждого PR. В GitHub эта функция тоже есть, но ее нужно настраивать вручную для защищенных веток.
  • Упрощает форму ревью в PR: подтверждайте или отклоняйте ревью быстрее с помощью однокликовых кнопок.
  • Добавляет горячие клавиши для переключения вкладок PR: g ← и g →, или g 1, g 2, g 3, g 4.
  • Автоматически удаляет ветку сразу после слияния PR, если возможно. Исключаются ветки с часто используемыми именами (develop, release/* и др.), которые обычно существуют долго.
  • Позволяет создавать черновики PR и публичные gist в один клик.
  • Очищает сообщение merge-коммита PR от лишнего, оставляя только уникальных co-author-ов.
  • Добавляет быстрые кнопки ревью в боковую панель PR, автоматически фокусирует textarea для ревью и добавляет горячую клавишу для открытия окна ревью: v.
  • Использует первый коммит в качестве заголовка и описания нового PR.

Коммиты

  • Добавляет ссылки для просмотра raw-версии, blame и истории файлов в PR и коммитах.
  • Добавляет однокликовые кнопки для изменения вида diff и игнорирования пробелов, а также горячую клавишу для игнорирования пробелов: d w.
  • Расширяет кнопку Expand diff, чтобы она была кликабельна по всей ширине экрана.
  • Скрывает значки diff, так как различия уже выделены цветом.
  • Советует ограничивать заголовки коммитов и PR 72 символами.
  • Отображает соответствующие теги рядом с коммитами.
  • Помечает merge-коммиты в списках коммитов.
  • При просмотре blame, Alt-клик по кнопке “Reblame” сначала извлекает коммиты из соответствующего PR, а не рассматривает коммит как одно изменение.
  • Иконкой показывает, добавляются или удаляются файлы в коммитах и PR.
  • Позволяет разворачивать/сворачивать диффы файлов по клику на их заголовке.
  • Сохраняет текущую ветку и путь при просмотре всех коммитов автора.
  • Позволяет сворачивать/разворачивать сообщения коммитов по клику на коробке коммита.
  • Делает надпись "X files changed" на страницах сравнения ссылкой на diff.
  • Показывает тип conventional commit в виде метки перед сообщением коммита.
