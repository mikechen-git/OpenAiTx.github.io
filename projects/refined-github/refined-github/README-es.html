Refined GitHub

Extensión de navegador que simplifica la interfaz de GitHub y añade funciones útiles

Usamos GitHub mucho y notamos muchas molestias que nos gustaría corregir. Así que aquí hay dragones.

Nuestra esperanza es que GitHub note e implemente algunas de estas mejoras tan necesarias. Así que si te gusta alguna de estas mejoras, por favor abre una discusión en GitHub feedback o contacta con GitHub support para que lo hagan.

GitHub Enterprise también es compatible: Cómo activarlo.

Las marcas registradas GITHUB y REFINED GITHUB son propiedad de GitHub, Inc. y se utilizan bajo licencia.

Netiqueta

  • Agrega recordatorios de netiqueta poco intrusivos (issues cerrados antiguos, issues altamente activos, PRs en borrador, …).

Correciones para deficiencias de GitHub

  • Elimina el efecto hover de archivos en el navegador de archivos del repo.
  • Elimina información innecesaria o redundante de la barra lateral del repositorio.
  • Enlaza referencias de ramas en las páginas de "Quick PR".
  • Elimina el autocompletado en los campos de búsqueda.
  • Hace que "View file" en PRs apunte a la rama en vez del commit, para que los botones de Editar/Eliminar estén habilitados en la página "View file".
  • Reintenta descargar imágenes que fallaron debido al proxy limitado de GitHub.
  • Convierte algunos menús desplegables de 2 clics en 1 solo clic donde no era necesario.
  • Sugiere corregir enlaces que GitHub acorta erróneamente.
  • Enlaza el nombre de usuario en el popup de historial de ediciones.
  • Evita crear PRs duplicados al hacer clic por error más de una vez en "Create pull request".
  • Mueve el botón "Convert PR to Draft" a la caja de mergeabilidad y agrega feedback visual a su botón de confirmación.
  • Previene que el panel de merge se abra automáticamente en cada carga de página después de haber sido abierto una vez.
  • Agrega atajos de teclado para seleccionar elementos en la paleta de comandos usando ctrl n y ctrl p (solo macOS).
  • Mientras escribes/editas comentarios, abre los enlaces de previsualización en nueva pestaña en vez de navegar fuera de la página.
  • Redirige a la lista de issues del repo cuando la búsqueda no incluye is:pr.
  • Elimina el parámetro redundante "readme-ov-file" de las URLs de repos.
  • Cierra la lista de checks al hacer clic fuera del modal.
  • Enlaza los números de línea donde GitHub olvidó agregar enlaces.
  • Desplaza el árbol de archivos hasta el archivo actual.
  • Desactiva modales que reducen la experiencia de usuario en vez de mejorarla.
  • Mantiene los enlaces en la misma pestaña cuando no deberían abrirse en una nueva. "Abrir en nueva pestaña" debe ser elección del usuario, no el comportamiento por defecto.

Anteriormente parte de Refined GitHub

GitHub se inspiró en Refined GitHub e implementó de forma nativa docenas de sus funciones, 🎉 por lo que han sido eliminadas de esta extensión. También puedes ver todas las funciones pasadas de Refined GitHub en una sola lista.

Personalización

La mayoría de las funciones pueden desactivarse si están basadas en JavaScript y puedes sobrescribir nuestro CSS con el tuyo propio en las opciones de la extensión.

Nos complace recibir sugerencias y contribuciones, pero ten en cuenta que este es un proyecto altamente opinativo. Hay un estándar alto para agregar funciones. Siempre habrá desacuerdos entre usuarios. Dicho esto, estamos abiertos a debatir ideas.
