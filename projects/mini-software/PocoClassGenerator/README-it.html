PocoClassGenerator è iniziato con il generatore POCO di Necroskillz Generate C# POCOs from SQL statement in LINQPad

Caratteristiche

  • Supporta la generazione del codice della classe POCO per tutte le tabelle e viste dell’attuale DataBase
  • Supporta Dapper.Contrib
  • Supporta diversi RDBMS: sqlserver,oracle,mysql,postgresql
  • Mini e veloce (genera il codice di 100 tabelle in soli 5 secondi)
  • Utilizza la query SQL dello schema della tabella con il dialetto appropriato per ogni database

  • POCOGenerator Generate Class By Dynamic SQL | .NET Fiddle
  • POCO Class Generator GenerateAllTables | .NET Fiddle
  • DataTable POCO Class Generator | .NET Fiddle

Inizia subito

👇Prima : Copia e incolla il codice da PocoClassGenerator.cs nel tuo progetto o in LINQPad.
oppure installa da NuGet

PM> install-package PocoClassGenerator

👇Secondo : Usa la connessione per chiamare GenerateAllTables e stampare il risultato.

using (var connection = Connection)
{
	Console.WriteLine(connection.GenerateAllTables());
}

Supporto per la classe POCO Dapper Contrib

  • Basta chiamare il metodo con GeneratorBehavior.DapperContrib
using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.DapperContrib);
    Console.WriteLine(result);
}

Demo online : POCO Dapper Contrib Class Generator GenerateAllTables | .NET Fiddle

Genera Commenti

using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.Comment);
    Console.WriteLine(result);
}

Genera Vista

using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.View);
    Console.WriteLine(result);
}

Genera Vista, Commento e Dapper.Contrib

using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.View | GeneratorBehavior.Comment | GeneratorBehavior.DapperContrib);
    Console.WriteLine(result);
}

Genera una singola classe tramite SQL

  1. Genera una singola classe
using (var connection = Connection)
{
	var classCode = connection.GenerateClass("select * from Table");
	Console.WriteLine(classCode);
}
  1. Specifica il nome della classe
using (var connection = Connection)
{
	var classCode = connection.GenerateClass("with EMP as (select 1 ID,'WeiHan' Name,25 Age) select * from EMP", className: "EMP");
	Console.WriteLine(classCode);
}

Codice in DataTablePocoClassGenerator.cs

var dt = new DataTable();
dt.TableName = "TestTable";
dt.Columns.Add(new DataColumn() { ColumnName = "ID", DataType = typeof(string) });

var result = dt.GenerateClass();
var expect =
@"public class TestTable
{
public string ID { get; set; }
}";
Assert.Equal(expect, result);
