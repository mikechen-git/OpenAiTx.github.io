PocoClassGenerator được bắt đầu từ POCO Generator của Necroskillz Generate C# POCOs from SQL statement in LINQPad

Tính năng

  • Hỗ trợ sinh code lớp POCO cho tất cả bảng và view hiện tại trong Cơ sở dữ liệu
  • Hỗ trợ Dapper.Contrib
  • Hỗ trợ nhiều RDBMS: sqlserver,oracle,mysql,postgresql
  • Nhẹ và nhanh (chỉ mất 5 giây để sinh code cho 100 bảng)
  • Sử dụng SQL truy vấn bảng schema phù hợp với từng loại cơ sở dữ liệu

Bắt đầu sử dụng

👇Bước đầu tiên: Sao chép và dán mã PocoClassGenerator.cs vào dự án của bạn hoặc LINQPad.
hoặc cài đặt từ NuGet

PM> install-package PocoClassGenerator

👇Bước hai: Sử dụng Connection để gọi GenerateAllTables rồi in ra kết quả.

using (var connection = Connection)
{
	Console.WriteLine(connection.GenerateAllTables());
}

Hỗ trợ Dapper Contrib POCO Class

  • Chỉ cần gọi phương thức với GeneratorBehavior.DapperContrib
using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.DapperContrib);
    Console.WriteLine(result);
}

Demo trực tuyến: POCO Dapper Contrib Class Generator GenerateAllTables | .NET Fiddle

Sinh chú thích (Comment)

using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.Comment);
    Console.WriteLine(result);
}

Sinh lớp cho View

using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.View);
    Console.WriteLine(result);
}

Sinh lớp cho View, Comment và Dapper.Contrib

using (var conn = GetConnection())
{
    var result = conn.GenerateAllTables(GeneratorBehavior.View | GeneratorBehavior.Comment | GeneratorBehavior.DapperContrib);
    Console.WriteLine(result);
}

Sinh một lớp từ sql

  1. Sinh một lớp
using (var connection = Connection)
{
	var classCode = connection.GenerateClass("select * from Table");
	Console.WriteLine(classCode);
}
  1. Chỉ định tên lớp
using (var connection = Connection)
{
	var classCode = connection.GenerateClass("with EMP as (select 1 ID,'WeiHan' Name,25 Age) select * from EMP", className: "EMP");
	Console.WriteLine(classCode);
}

DataTablePocoClass

Mã nguồn tại DataTablePocoClassGenerator.cs

var dt = new DataTable();
dt.TableName = "TestTable";
dt.Columns.Add(new DataColumn() { ColumnName = "ID", DataType = typeof(string) });

var result = dt.GenerateClass();
var expect =
@"public class TestTable
{
public string ID { get; set; }
}";
Assert.Equal(expect, result);
