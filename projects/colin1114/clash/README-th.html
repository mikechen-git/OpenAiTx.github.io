🚀 Clash 订阅转换器

เครื่องมือออนไลน์ที่สวยงามและทรงพลัง สำหรับแปลงลิงก์สมัครสมาชิกพร็อกซีหลากหลายรูปแบบเป็นไฟล์คอนฟิก Clash รองรับการดีพลอยอัตโนมัติผ่าน Cloudflare Workers & Pages ให้บริการแปลงลิงก์สมัครสมาชิกที่รวดเร็วและเชื่อถือได้

✨ ฟีเจอร์เด่น

  • 🎨 อินเทอร์เฟซสวยงามทันสมัย - ใช้พื้นหลังไล่สีและดีไซน์แบบการ์ด
  • 🔄 รองรับหลายโปรโตคอล - รองรับ V2Ray, VLESS, Shadowsocks, Trojan และโปรโตคอลยอดนิยมอื่นๆ
  • ☁️ ดีพลอยบนคลาวด์ - ใช้ Cloudflare Workers เพื่อ CDN acceleration ทั่วโลก
  • 📱 ออกแบบ Responsive - เหมาะกับเดสก์ท็อปและมือถือทุกขนาดหน้าจอ
  • ⚡ แปลงลิงก์ได้รวดเร็ว - แปลงลิงก์สมัครสมาชิกเป็นไฟล์คอนฟิก Clash แบบเรียลไทม์
  • 📋 คัดลอกอัจฉริยะ - รองรับการคัดลอก YAML ที่สร้างขึ้นด้วยคลิกเดียว รองรับทุกเบราว์เซอร์
  • 📡 สร้างลิงก์สมัครสมาชิก - สร้างลิงก์สมัครสมาชิกสำหรับนำเข้า Clash ได้โดยอัตโนมัติ
  • 🌐 เข้าถึง YAML ออนไลน์ - มีหลายวิธีในการเข้าถึงไฟล์ YAML:
  • 📡 ลิงก์สมัครสมาชิก: รองรับอัปเดตอัตโนมัติใน Clash client
  • 🔗 ดูออนไลน์: เปิดดูคอนฟิกในเบราว์เซอร์โดยตรง
  • 💾 ดาวน์โหลดไฟล์: ดาวน์โหลดไฟล์ .yaml ลงเครื่อง
  • ⚡ เข้าถึงรวดเร็ว: ลิงก์ดาวน์โหลดแบบย่อ
  • 🔧 รองรับ UTF-8 - รองรับชื่อโหนดและคอนฟิกภาษาจีนอย่างสมบูรณ์แบบ
  • 🆓 ใช้งานฟรี 100% - ใช้บริการฟรีของ Cloudflare ไม่มีข้อจำกัด

  • OPTIONS /* - คำขอพรีไฟลท์ CORS

🔧 การแก้ไขปัญหาการดีพลอย
ปัญหาที่พบบ่อย
#### 1. Wrangler หา entry point ไม่เจอ
ข้อความผิดพลาด: Missing entry-point to Worker script
วิธีแก้ไข: 

  • ตรวจสอบให้มีไฟล์ wrangler.jsonc อยู่ที่ไดเรกทอรีรากของโปรเจกต์
    • 

  • ตรวจสอบว่าใน wrangler.jsonc ฟิลด์ main ชี้ไปที่ worker.js
#### 2. GitHub Actions ดีพลอยไม่สำเร็จ
วิธีแก้ไข:

  • เพิ่ม Secrets ด้านล่างนี้ใน Settings ของคลัง GitHub:
    • 
  
  • CLOUDFLARE_API_TOKEN: โทเคน API ของ Cloudflare
    • 
  
  • CLOUDFLARE_ACCOUNT_ID: Cloudflare Account ID
#### 3. Pages สร้างไม่สำเร็จ
แนวทางแก้ไข:

  • ตั้งค่าคำสั่งสร้างเป็นค่าว่างหรือ echo "No build required"
    • 

  • ตรวจสอบให้แน่ใจว่าโฟลเดอร์ผลลัพธ์ของการสร้างถูกตั้งค่าเป็น /
คำอธิบายไฟล์กำหนดค่าการปรับใช้
  • wrangler.jsonc: การกำหนดค่า Wrangler CLI สำหรับการปรับใช้ Worker
    • 

  • _headers: การกำหนดค่า HTTP Header สำหรับ Cloudflare Pages
    • 

  • .github/workflows/deploy.yml: การกำหนดค่า GitHub Actions สำหรับการปรับใช้อัตโนมัติ
🔧 การกำหนดค่าแบบกำหนดเอง
แก้ไขกลุ่ม Proxy
สามารถแก้ไขฟังก์ชัน convertToClash ใน worker.js เพื่อปรับแต่งกลุ่ม Proxy ได้ดังนี้:
javascript 'proxy-groups': [ { name: '🚀 เลือกโหนด', type: 'select', proxies: ['♻️ เลือกอัตโนมัติ', '🎯 ตรงทั่วโลก', ...proxyNames] }, // เพิ่มกลุ่ม Proxy เพิ่มเติม... ] 


กำหนดกฎเส้นทางเอง
แก้ไขอาร์เรย์ rules เพื่อเพิ่มกฎเส้นทางที่กำหนดเอง:
javascript rules: [ 'DOMAIN-SUFFIX,youtube.com,🚀 เลือกโหนด', 'DOMAIN-SUFFIX,local,DIRECT', // เพิ่มกฎเพิ่มเติม... ] `

📊 คุณสมบัติด้านประสิทธิภาพ

  • CDN ทั่วโลก - Cloudflare มี Edge Node ครอบคลุมทั่วโลก
  • ตอบสนองรวดเร็ว - เวลาตอบสนองเฉลี่ย < 100ms
  • ความพร้อมใช้งานสูง - ความพร้อมให้บริการ 99.9%
  • โควต้าฟรี - 100,000 คำขอต่อวัน

🔒 หมายเหตุด้านความปลอดภัย

  • ความเป็นส่วนตัวของข้อมูล - ข้อมูลการสมัครสมาชิกจะใช้เฉพาะในกระบวนการแปลงและจะไม่ถูกจัดเก็บ
  • การเข้ารหัส HTTPS - การสื่อสารทั้งหมดถูกเข้ารหัสด้วย HTTPS
  • โอเพ่นซอร์สโปร่งใส - ซอร์สโค้ดทั้งหมดเปิดเผย สามารถตรวจสอบได้ด้วยตนเอง
