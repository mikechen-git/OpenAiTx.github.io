เริ่มต้นใช้งาน

กรุณาติดตามเอกสารที่ vuejs.org!

สปอนเซอร์

Vue.js เป็นโครงการโอเพ่นซอร์สที่ได้รับอนุญาตภายใต้ MIT ซึ่งการพัฒนาของโครงการนี้เป็นไปได้อย่างสมบูรณ์ด้วยการสนับสนุนจาก ผู้สนับสนุน ที่ยอดเยี่ยมเหล่านี้ หากคุณต้องการเข้าร่วม กรุณาพิจารณา สนับสนุนการพัฒนา Vue ด้วย

สปอนเซอร์พิเศษ

คำถาม

สำหรับคำถามและการสนับสนุน กรุณาใช้ ฟอรั่มอย่างเป็นทางการ หรือ แชทชุมชน รายการ issue ของ repo นี้ใช้ เฉพาะ สำหรับรายงานบั๊กและคำขอฟีเจอร์เท่านั้น

ปัญหา

กรุณาตรวจสอบให้แน่ใจว่าคุณปฏิบัติตามข้อกำหนดของ issue และใช้ ตัวช่วยสร้าง issue ใหม่ เมื่อเปิด issue ใหม่ หาก issue ไม่เป็นไปตามแนวทาง อาจถูกปิดทันที

การร่วมพัฒนา

กรุณาอ่าน คู่มือการร่วมพัฒนา ก่อนที่จะส่ง pull request หากคุณมีโปรเจกต์/คอมโพเนนต์/เครื่องมือที่เกี่ยวข้องกับ Vue สามารถเพิ่มเข้ามาได้ด้วยการ pull request ไปที่ รายการที่คัดสรรนี้!

ขอขอบคุณทุกคนที่ได้ร่วมพัฒนา Vue มาแล้ว!
