Vue 2 ได้สิ้นสุดการสนับสนุนแล้ว

คุณกำลังดูที่ repository ที่ไม่ใช้งานแล้วของ Vue 2 ขณะนี้ repository ที่ได้รับการดูแลและอัปเดตสำหรับ Vue เวอร์ชันล่าสุดคือ vuejs/core

Vue ได้สิ้นสุดการสนับสนุนเมื่อวันที่ 31 ธันวาคม 2023 โดยจะไม่มีการเพิ่มฟีเจอร์ใหม่ อัปเดต หรือแก้ไขใด ๆ อีกต่อไป อย่างไรก็ตาม คุณยังสามารถใช้งานได้ผ่านช่องทางจัดจำหน่ายต่าง ๆ ที่มีอยู่ (CDNs, ตัวจัดการแพ็กเกจ, Github ฯลฯ)

หากคุณกำลังจะเริ่มโปรเจกต์ใหม่ กรุณาเริ่มต้นด้วย Vue เวอร์ชันล่าสุด (3.x) เราแนะนำให้ผู้ใช้ Vue 2 ปัจจุบันอัปเกรด (คู่มือ) แต่เราก็เข้าใจว่าผู้ใช้บางรายอาจไม่มีเวลาหรือแรงจูงใจเพียงพอ หากคุณจำเป็นต้องใช้ Vue 2 ต่อไป แต่ต้องมีข้อกำหนดด้านความปลอดภัยหรือการปฏิบัติตามข้อกำหนดสำหรับซอฟต์แวร์ที่ไม่มีการดูแล กรุณาดู Vue 2 NES

ผู้สนับสนุน

Vue.js เป็นโครงการโอเพ่นซอร์สที่ได้รับอนุญาตภายใต้ MIT โดยการพัฒนาต่อเนื่องเป็นไปได้เพราะการสนับสนุนจาก ผู้สนับสนุน เหล่านี้ หากคุณต้องการเข้าร่วม กรุณาพิจารณา สนับสนุนการพัฒนา Vue

ผู้สนับสนุนพิเศษ

บทนำ

Vue (อ่านว่า /vjuː/ คล้ายกับ "view") คือ เฟรมเวิร์กแบบก้าวหน้า สำหรับการสร้างส่วนติดต่อผู้ใช้ ถูกออกแบบมาให้สามารถนำไปใช้แบบค่อยเป็นค่อยไป และสามารถขยายขนาดได้ตั้งแต่ไลบรารีขนาดเล็กจนถึงเฟรมเวิร์กเต็มรูปแบบตามแต่กรณีการใช้งาน ประกอบด้วยไลบรารีหลักที่เน้นเฉพาะเลเยอร์ของ view และระบบนิเวศของไลบรารีเสริมที่ช่วยจัดการความซับซ้อนใน Single-Page Applications ขนาดใหญ่

ความเข้ากันได้ของเบราว์เซอร์

Vue.js รองรับเบราว์เซอร์ทั้งหมดที่ รองรับ ES5 (IE8 และต่ำกว่าไม่รองรับ)

ระบบนิเวศ

| โครงการ | สถานะ | คำอธิบาย | | --------------------- | ------------------------------------------------------------ | -------------------------------------------------------- | | vue-router | | ระบบกำหนดเส้นทางสำหรับ single-page application | | vuex | | การจัดการ state ขนาดใหญ่ | | vue-cli | | การสร้างโครงสร้างเริ่มต้นสำหรับโปรเจกต์ | | vue-loader | | Loader สำหรับ Single File Component (*.vue file) บน webpack | | vue-server-renderer | | รองรับการเรนเดอร์ฝั่งเซิร์ฟเวอร์ | | vue-class-component | | TypeScript decorator สำหรับ API แบบ class-based | | vue-rx | | การใช้งานร่วมกับ RxJS | | vue-devtools | | ส่วนขยาย DevTools สำหรับเบราว์เซอร์ |
