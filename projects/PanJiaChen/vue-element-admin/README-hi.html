परिचय

vue-element-admin एक प्रोडक्शन-रेडी फ्रंट-एंड समाधान है जो एडमिन इंटरफेस के लिए बनाया गया है। यह vue पर आधारित है और UI टूलकिट के रूप में element-ui का उपयोग करता है।

vue-element-admin vue के नवीनतम डेवलपमेंट स्टैक पर आधारित है और इसमें i18n समाधान, एंटरप्राइज एप्लिकेशन के लिए टेम्प्लेट्स तथा कई बेहतरीन फीचर्स बिल्ट-इन हैं। यह आपको बड़े और जटिल सिंगल-पेज एप्लिकेशन बनाने में मदद करता है। मुझे विश्वास है कि आपकी जो भी जरूरतें हों, यह प्रोजेक्ट आपकी मदद करेगा।

v4.1.0+ वर्शन के बाद, डिफ़ॉल्ट मास्टर ब्रांच में i18n सपोर्ट नहीं है। कृपया i18n ब्रांच का उपयोग करें, यह मास्टर ब्रांच के साथ अपडेट होती रहेगी।

वर्तमान वर्शन v4.0+ है, जो vue-cli पर आधारित है। यदि आपको कोई समस्या मिले, कृपया issue डालें। यदि आप पुराना वर्शन उपयोग करना चाहते हैं, तो ब्रांच बदलकर tag/3.11.0 पर जाएँ, यह vue-cli पर निर्भर नहीं करता।

यह प्रोजेक्ट पुराने ब्राउज़र (जैसे IE) को सपोर्ट नहीं करता। कृपया स्वयं polyfill जोड़ें।

तैयारी

आपको अपने लोकल सिस्टम पर node और git इंस्टॉल करना होगा। यह प्रोजेक्ट ES2015+, vue, vuex, vue-router, vue-cli, axios और element-ui पर आधारित है, तथा सभी डेटा अनुरोध Mock.js का उपयोग करके सिम्युलेट किए गए हैं। इनकी पूर्व जानकारी और समझ होना इस प्रोजेक्ट के उपयोग के लिए अत्यंत सहायक होगी।

प्रारंभ करें

# प्रोजेक्ट क्लोन करें
git clone https://github.com/PanJiaChen/vue-element-admin.git

# प्रोजेक्ट डायरेक्टरी में जाएँ
cd vue-element-admin

# निर्भरता इंस्टॉल करें
npm install

# विकास मोड
npm run dev

यह स्वतः http://localhost:9527 खोल देगा

बिल्ड

# टेस्ट वातावरण के लिए बिल्ड करें
npm run build:stage

# प्रोडक्शन वातावरण के लिए बिल्ड करें
npm run build:prod

उन्नत

# रिलीज वातावरण का पूर्वावलोकन करें
npm run preview

# रिलीज वातावरण का पूर्वावलोकन + स्थैतिक संसाधन विश्लेषण
npm run preview -- --report

# कोड फॉर्मेट चेक
npm run lint

# कोड फॉर्मेट चेक और स्वतः सुधार
npm run lint -- --fix
