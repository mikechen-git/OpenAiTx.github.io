Introdução

vue-element-admin é uma solução front-end pronta para produção para interfaces administrativas. Baseia-se em vue e utiliza o kit de UI element-ui.

vue-element-admin é baseado no mais novo stack de desenvolvimento do vue e possui uma solução i18n incorporada, modelos típicos para aplicações empresariais e muitos recursos excelentes. Ele ajuda você a construir aplicações de página única grandes e complexas. Acredito que, quaisquer que sejam suas necessidades, este projeto irá lhe ajudar.

Após a versão v4.1.0+, o branch master padrão não suportará i18n. Por favor, use o Branch i18n, ele acompanhará as atualizações do master

A versão atual é v4.0+ construída com vue-cli. Se encontrar um problema, por favor abra um issue. Se quiser usar a versão antiga, você pode trocar para o branch tag/3.11.0, ela não depende do vue-cli

Este projeto não suporta navegadores de versões antigas (ex: IE). Por favor, adicione polyfill por conta própria.

Preparação

Você precisa instalar node e git localmente. O projeto é baseado em ES2015+, vue, vuex, vue-router, vue-cli, axios e element-ui. Todos os dados das requisições são simulados usando Mock.js. Compreender e aprender esses conhecimentos previamente irá ajudar muito no uso deste projeto.

Começando

# clone o projeto
git clone https://github.com/PanJiaChen/vue-element-admin.git

# entre no diretório do projeto
cd vue-element-admin

# instale as dependências
npm install

# desenvolvimento
npm run dev

Isso abrirá automaticamente http://localhost:9527

Build

# build para ambiente de teste
npm run build:stage

# build para ambiente de produção
npm run build:prod

Avançado

# pré-visualizar o efeito do ambiente de release
npm run preview

# pré-visualizar o efeito do ambiente de release + análise de recursos estáticos
npm run preview -- --report

# checagem de formatação de código
npm run lint

# checagem de formatação de código e correção automática
npm run lint -- --fix
