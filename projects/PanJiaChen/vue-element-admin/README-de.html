Einführung

vue-element-admin ist eine produktionsreife Frontend-Lösung für Admin-Oberflächen. Sie basiert auf vue und verwendet das UI-Toolkit element-ui.

vue-element-admin basiert auf dem neuesten Entwicklungsstack von Vue und besitzt eine integrierte i18n-Lösung, typische Templates für Unternehmensanwendungen und viele großartige Funktionen. Es hilft Ihnen dabei, große und komplexe Single-Page-Anwendungen zu erstellen. Ich glaube, unabhängig von Ihren Anforderungen wird dieses Projekt Ihnen weiterhelfen.

Ab Version v4.1.0+ unterstützt der Standard-Master-Branch kein i18n mehr. Bitte nutzen Sie den i18n-Branch, dieser bleibt mit dem Master-Branch aktuell.

Die aktuelle Version ist v4.0+ und basiert auf vue-cli. Bei Problemen bitte ein Issue erstellen. Möchten Sie die alte Version nutzen, wechseln Sie bitte zum Branch tag/3.11.0, diese Version benötigt kein vue-cli.

Dieses Projekt unterstützt keine älteren Browser-Versionen (z.B. IE). Bitte fügen Sie bei Bedarf selbst Polyfills hinzu.

Vorbereitung

Sie müssen node und git lokal installieren. Das Projekt basiert auf ES2015+, vue, vuex, vue-router, vue-cli, axios und element-ui. Alle Anfragedaten werden mit Mock.js simuliert. Das Vorwissen und Lernen dieser Technologien ist für die Nutzung dieses Projekts von großem Vorteil.

Einstieg

# Projekt klonen
git clone https://github.com/PanJiaChen/vue-element-admin.git

# In das Projektverzeichnis wechseln
cd vue-element-admin

# Abhängigkeiten installieren
npm install

# Entwicklung starten
npm run dev

Dies öffnet automatisch http://localhost:9527

Build

# Build für Testumgebung
npm run build:stage

# Build für Produktionsumgebung
npm run build:prod

Erweiterte Nutzung

# Vorschau der Release-Umgebung
npm run preview

# Vorschau + Analyse der statischen Ressourcen
npm run preview -- --report

# Code-Formatprüfung
npm run lint

# Code-Formatprüfung und automatische Korrektur
npm run lint -- --fix
