المقدمة

vue-element-admin هو حل جاهز للإنتاج للواجهات الخلفية للإدارة. يعتمد على vue ويستخدم أداة الواجهات element-ui.

vue-element-admin مبني على أحدث حزمة تطوير لـ vue ويدعم الترجمة الدولية (i18n) ويحتوي على قوالب نموذجية لتطبيقات الشركات والعديد من الميزات الرائعة. يساعدك في بناء تطبيقات صفحة واحدة كبيرة ومعقدة. أؤمن بأن هذا المشروع سيساعدك مهما كانت احتياجاتك.

  • تايب سكريبت: vue-typescript-admin-template (الشكر لـ: @Armour)

  • مشاريع رائعة

بعد الإصدار v4.1.0+، لن يدعم الفرع الرئيسي الترجمة الدولية (i18n) افتراضيًا. يرجى استخدام فرع i18n، وسيتم تحديثه بالتوازي مع الفرع الرئيسي

الإصدار الحالي هو v4.0+ مبني على vue-cli. إذا وجدت مشكلة، يرجى فتح issue. إذا أردت استخدام الإصدار القديم، يمكنك التبديل إلى tag/3.11.0، وهو لا يعتمد على vue-cli

هذا المشروع لا يدعم المتصفحات ذات الإصدارات المنخفضة (مثل IE). يرجى إضافة polyfill بنفسك.

التحضير

تحتاج إلى تثبيت node و git محليًا. المشروع يعتمد على ES2015+، vue، vuex، vue-router، vue-cli، axios و element-ui، جميع بيانات الطلبات يتم محاكاتها باستخدام Mock.js. فهمك لهذه التقنيات مسبقًا سيساعدك كثيرًا في استخدام هذا المشروع.

البدء

# استنساخ المشروع
git clone https://github.com/PanJiaChen/vue-element-admin.git

# ادخل إلى مجلد المشروع
cd vue-element-admin

# تثبيت التبعيات
npm install

# تطوير
npm run dev

سيتم فتح http://localhost:9527 تلقائيًا

البناء

# بناء لبيئة الاختبار
npm run build:stage

# بناء لبيئة الإنتاج
npm run build:prod

متقدم

# معاينة تأثير بيئة الإصدار
npm run preview

# معاينة تأثير بيئة الإصدار + تحليل الموارد الثابتة
npm run preview -- --report

# فحص تنسيق الكود
npm run lint

# فحص تنسيق الكود مع تصحيح تلقائي
npm run lint -- --fix
