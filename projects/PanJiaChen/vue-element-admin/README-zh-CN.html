简介

vue-element-admin 是一个生产环境就绪的后台前端解决方案。它基于 vue，使用了 UI 组件库 element-ui。

vue-element-admin 基于最新版的 vue 开发栈，内置 i18n 国际化解决方案，包含企业级应用常用模板和诸多优秀特性，助力你快速搭建大型复杂的单页应用。无论你的需求是什么，这个项目都能帮到你。

v4.1.0+ 版本起，默认 master 分支不再支持 i18n，请使用 i18n 分支，该分支会保持与 master 分支同步更新。

当前版本为 v4.0+，基于 vue-cli 构建。如果发现问题请提 issue。如果你需要旧版本，可以切换到 tag/3.11.0 分支，该版本不依赖 vue-cli。

本项目不支持低版本浏览器（如 IE），如需兼容请自行添加 polyfill。

准备工作

你需要在本地安装 node 和 git。本项目基于 ES2015+、vue、vuex、vue-router、vue-cli、axios 以及 element-ui，所有请求数据使用 Mock.js 模拟。提前了解并学习这些知识将大大有助于你使用本项目。

快速上手

# 克隆项目
git clone https://github.com/PanJiaChen/vue-element-admin.git

# 进入项目目录
cd vue-element-admin

# 安装依赖
npm install

# 启动开发环境
npm run dev

这将自动打开 http://localhost:9527

构建

# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod

进阶

# 预览发布环境效果
npm run preview

# 预览发布环境效果 + 静态资源分析
npm run preview -- --report

# 代码格式检查
npm run lint

# 代码格式检查并自动修复
npm run lint -- --fix
