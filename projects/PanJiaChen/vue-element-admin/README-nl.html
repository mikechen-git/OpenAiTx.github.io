Introductie

vue-element-admin is een productieklare front-end oplossing voor beheerdersinterfaces. Het is gebaseerd op vue en gebruikt de UI Toolkit element-ui.

vue-element-admin is gebaseerd op de nieuwste ontwikkelstack van vue en heeft een ingebouwde i18n-oplossing, typische sjablonen voor bedrijfsapplicaties en veel geweldige functies. Het helpt je om grote en complexe Single-Page Applicaties te bouwen. Wat je behoeften ook zijn, ik geloof dat dit project je zal helpen.

Vanaf versie v4.1.0+ ondersteunt de standaard master branch geen i18n. Gebruik de i18n Branch, deze blijft synchroon met master.

De huidige versie is v4.0+ gebouwd op vue-cli. Als je een probleem vindt, plaats dan een issue. Wil je de oude versie gebruiken, schakel dan over naar tag/3.11.0, deze is niet afhankelijk van vue-cli.

Dit project ondersteunt geen oudere browsers (zoals IE). Voeg zelf een polyfill toe indien nodig.

Voorbereiding

Je moet lokaal node en git installeren. Het project is gebaseerd op ES2015+, vue, vuex, vue-router, vue-cli, axios en element-ui. Alle aanvraagdata wordt gesimuleerd met Mock.js. Kennis van en ervaring met deze onderdelen helpt aanzienlijk bij het gebruik van dit project.

Aan de slag

# kloon het project
git clone https://github.com/PanJiaChen/vue-element-admin.git

# ga naar de projectmap
cd vue-element-admin

# installeer afhankelijkheden
npm install

# ontwikkelen
npm run dev

Dit opent automatisch http://localhost:9527

Build

# build voor testomgeving
npm run build:stage

# build voor productieomgeving
npm run build:prod

Geavanceerd

# preview van de releaseomgeving
npm run preview

# preview van de releaseomgeving + statische bronanalyse
npm run preview -- --report

# code opmaakcontrole
npm run lint

# code opmaakcontrole en automatisch repareren
npm run lint -- --fix
