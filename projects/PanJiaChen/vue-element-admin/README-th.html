บทนำ

vue-element-admin คือโซลูชันส่วนหน้าสำหรับอินเทอร์เฟซผู้ดูแลระบบที่พร้อมสำหรับการใช้งานในสายการผลิต พัฒนาขึ้นโดยใช้ vue และใช้ UI Toolkit element-ui

vue-element-admin สร้างขึ้นจากสแต็กการพัฒนาล่าสุดของ vue และมีระบบ i18n ในตัว มีเทมเพลตตัวอย่างสำหรับแอปพลิเคชันองค์กร และมีฟีเจอร์ที่ยอดเยี่ยมมากมาย ช่วยให้คุณสร้าง Single-Page Applications ที่มีขนาดใหญ่และซับซ้อนได้ เชื่อว่าไม่ว่าคุณจะมีความต้องการแบบใด โปรเจกต์นี้จะช่วยคุณได้

  • ดูตัวอย่าง

  • เอกสารประกอบ

  • Gitter

  • บริจาค

  • Wiki

  • Gitee สำหรับผู้ใช้ในประเทศจีนสามารถเข้าชมตัวอย่างออนไลน์ที่นี่

  • เทมเพลตพื้นฐานแนะนำให้ใช้: vue-admin-template

  • เดสก์ท็อป: electron-vue-admin

  • Typescript: vue-typescript-admin-template (เครดิต: @Armour)

  • awesome-project

หลังจากเวอร์ชัน v4.1.0+ สาขา master เริ่มต้นจะไม่รองรับ i18n โปรดใช้ i18n Branch ซึ่งจะอัปเดตตาม master เสมอ

เวอร์ชันปัจจุบันคือ v4.0+ สร้างด้วย vue-cli หากพบปัญหา กรุณาส่ง issue หากต้องการใช้เวอร์ชันเก่า สามารถสลับไปที่ tag/3.11.0 ซึ่งไม่ขึ้นกับ vue-cli

โปรเจกต์นี้ไม่รองรับเบราว์เซอร์เวอร์ชันต่ำ (เช่น IE) กรุณาเพิ่ม polyfill เอง

การเตรียมตัว

คุณต้องติดตั้ง node และ git ในเครื่อง โปรเจกต์นี้พัฒนาบนพื้นฐานของ ES2015+, vue, vuex, vue-router, vue-cli, axios และ element-ui ข้อมูลทุกคำขอถูกจำลองด้วย Mock.js การเข้าใจและเรียนรู้ความรู้เหล่านี้ล่วงหน้าจะช่วยให้ใช้โปรเจกต์นี้ได้อย่างมาก

เริ่มต้นใช้งาน

# โคลนโปรเจกต์
git clone https://github.com/PanJiaChen/vue-element-admin.git

# เข้าสู่ไดเรกทอรีโปรเจกต์
cd vue-element-admin

# ติดตั้ง dependencies
npm install

# เริ่มพัฒนา
npm run dev

ระบบจะเปิด http://localhost:9527 โดยอัตโนมัติ

การ build

# build สำหรับสภาพแวดล้อมทดสอบ
npm run build:stage

# build สำหรับสภาพแวดล้อมผลิต
npm run build:prod

ขั้นสูง

# แสดงตัวอย่างสภาพแวดล้อม release
npm run preview

# แสดงตัวอย่าง release พร้อมวิเคราะห์ resource แบบ static
npm run preview -- --report

# ตรวจสอบรูปแบบโค้ด
npm run lint

# ตรวจสอบรูปแบบโค้ดและแก้ไขอัตโนมัติ
npm run lint -- --fix
