Material UI

Material UI is een uitgebreide bibliotheek van React-componenten die onze onafhankelijke implementatie van Google's Material Design systeem bevat. Het wordt vertrouwd door enkele van 's werelds beste productteams omdat het grondig getest is in de praktijk door meer dan een decennium aan ontwikkeling door duizenden open-source bijdragers.

De kernfunctionaliteit van Material UI wordt uitgebreid door MUI X, een suite van complexe componenten voor geavanceerde gebruikssituaties. Toolpad bouwt voort op Material UI om full-stack componenten en een low-code interne toolbouwer te bieden.

Joy UI

Deze repository bevat ook Joy UI, een experimentele componentenbibliotheek die ons eigen Joy Design implementeert. Joy UI is in bèta en de ontwikkeling is momenteel gepauzeerd. Wanneer je een nieuw project helemaal opnieuw start, raden wij Material UI aan boven Joy UI omdat we voortdurende ondersteuning kunnen garanderen.

Houd er rekening mee dat de beheerders zich vooral op andere projecten richten en mogelijk niet tijdig kunnen reageren op issues of pull requests met betrekking tot Joy UI.

Bekijk de Joy UI documentatie.
