Material UI

Material UI è una libreria completa di componenti React che presenta la nostra implementazione indipendente del sistema Material Design di Google. È scelta da alcuni dei migliori team di prodotto al mondo perché è stata ampiamente collaudata in più di un decennio di sviluppo da migliaia di contributori open-source.

La funzionalità principale di Material UI è estesa da MUI X, una suite di componenti complessi per casi d'uso avanzati. Toolpad si basa su Material UI per offrire componenti full-stack e un builder low-code per strumenti interni.

Joy UI

Questo repository contiene anche Joy UI, una libreria sperimentale di componenti che implementa il nostro design Joy proprietario. Joy UI è in beta e lo sviluppo è attualmente in pausa. Quando inizi un nuovo progetto da zero, consigliamo Material UI rispetto a Joy UI perché possiamo garantire supporto continuo.

Tieni presente che i manutentori sono principalmente concentrati su altri progetti e potrebbero non essere in grado di rispondere tempestivamente a problemi o pull request relativi a Joy UI.

Consulta la documentazione di Joy UI.
