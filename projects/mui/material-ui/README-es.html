Material UI

Material UI es una biblioteca integral de componentes React que presenta nuestra implementación independiente del sistema Material Design de Google. Es confiable para algunos de los mejores equipos de producto del mundo porque ha sido rigurosamente probada en batalla durante más de una década de desarrollo por miles de colaboradores de código abierto.

La funcionalidad principal de Material UI se amplía mediante MUI X, un conjunto de componentes complejos para casos de uso avanzados. Toolpad se basa en Material UI para proporcionar componentes full-stack y un constructor de herramientas internas low-code.

Joy UI

Este repositorio también contiene Joy UI, una biblioteca experimental de componentes que implementa nuestro propio Joy Design. Joy UI está en beta y el desarrollo está actualmente en pausa. Al comenzar un nuevo proyecto desde cero, recomendamos Material UI sobre Joy UI porque podemos garantizar soporte continuo.

Ten en cuenta que los mantenedores se centran principalmente en otros proyectos y puede que no puedan responder a tiempo a incidencias o pull requests relacionados con Joy UI.

Consulta la documentación de Joy UI.
