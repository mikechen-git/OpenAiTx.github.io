Material UI

Material UI는 Google의 Material Design 시스템을 독립적으로 구현한 React 컴포넌트의 종합 라이브러리입니다.
수천 명의 오픈소스 기여자들이 10년이 넘는 개발 기간 동안 철저히 검증해왔기 때문에, 세계 유수의 제품팀들이 신뢰하고 있습니다.

Material UI의 핵심 기능은 고급 사용 사례를 위한 복잡한 컴포넌트 모음인 MUI X로 확장할 수 있습니다.
Toolpad는 Material UI 위에 구축되어 풀스택 컴포넌트와 로우코드 내부 도구 빌더를 제공합니다.

Joy UI

이 저장소에는 자체 Joy Design을 구현한 실험적 컴포넌트 라이브러리인 Joy UI도 포함되어 있습니다.
Joy UI는 베타 상태이며, _현재 개발이 보류 중_입니다.
새 프로젝트를 처음부터 시작하는 경우에는 지속적인 지원을 보장할 수 있기 때문에 Joy UI보다 Material UI 사용을 권장합니다.

메인테이너들은 현재 다른 프로젝트에 주력하고 있으므로, Joy UI 관련 이슈나 풀 리퀘스트에 신속하게 응답하지 못할 수 있습니다.

Joy UI 문서를 확인하세요.
