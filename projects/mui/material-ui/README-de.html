Material UI

Material UI ist eine umfassende Bibliothek von React-Komponenten, die unsere eigenständige Implementierung des Material Design Systems von Google enthält. Sie wird von einigen der weltweit führenden Produktteams verwendet, da sie durch mehr als ein Jahrzehnt Entwicklung und Tausende von Open-Source-Beiträgen ausgiebig erprobt wurde.

Die Kernfunktionalität von Material UI wird durch MUI X erweitert, einer Suite komplexer Komponenten für fortgeschrittene Anwendungsfälle. Toolpad baut auf Material UI auf und bietet Full-Stack-Komponenten sowie einen Low-Code-Builder für interne Tools.

Joy UI

Dieses Repository enthält auch Joy UI, eine experimentelle Komponentenbibliothek, die unser eigenes Joy Design implementiert. Joy UI befindet sich im Beta-Status und die Entwicklung ist derzeit ausgesetzt. Wenn Sie ein neues Projekt von Grund auf beginnen, empfehlen wir Material UI gegenüber Joy UI, da wir nur so eine fortlaufende Unterstützung garantieren können.

Beachten Sie, dass sich die Betreuer hauptsächlich auf andere Projekte konzentrieren und möglicherweise nicht zeitnah auf Probleme oder Pull Requests im Zusammenhang mit Joy UI reagieren können.
