Material UI

Material UI คือไลบรารีคอมโพเนนต์ React ที่ครอบคลุม โดยมีการพัฒนาและใช้งานระบบ Material Design ของ Google ขึ้นใหม่อย่างอิสระ ได้รับความไว้วางใจจากทีมผลิตภัณฑ์ชั้นนำของโลก เนื่องจากผ่านการทดสอบอย่างเข้มข้นตลอดเวลากว่าทศวรรษจากผู้ร่วมพัฒนาโอเพ่นซอร์สหลายพันคน

ฟังก์ชันหลักของ Material UI สามารถขยายเพิ่มเติมได้ด้วย MUI X ชุดคอมโพเนนต์ที่ซับซ้อนสำหรับการใช้งานขั้นสูง Toolpad สร้างขึ้นบน Material UI เพื่อมอบคอมโพเนนต์แบบฟูลสแตกและเครื่องมือสร้างแอปพลิเคชันภายในแบบ low-code

Joy UI

รีโพซิทอรีนี้ยังมี Joy UI ซึ่งเป็นไลบรารีคอมโพเนนต์เชิงทดลองที่พัฒนาตาม Joy Design ของเราเอง Joy UI อยู่ในช่วงเบต้าและ ขณะนี้หยุดการพัฒนาไว้ชั่วคราว หากเริ่มโปรเจกต์ใหม่ แนะนำให้เลือก Material UI แทน Joy UI เนื่องจากเราสามารถรับประกันการซัพพอร์ตอย่างต่อเนื่อง

โปรดทราบว่าผู้ดูแลหลักเน้นงานในโปรเจกต์อื่นเป็นหลัก และอาจไม่สามารถตอบสนองต่อปัญหาหรือ pull request ที่เกี่ยวข้องกับ Joy UI ได้ทันท่วงที

การมีส่วนร่วม

อ่าน แนวทางการมีส่วนร่วม เพื่อเรียนรู้เกี่ยวกับกระบวนการพัฒนา วิธีเสนอการแก้ไขบั๊กและปรับปรุง และวิธี build และทดสอบการเปลี่ยนแปลงของคุณ

การมีส่วนร่วมไม่ได้หมายถึงแค่การแจ้งปัญหาหรือ pull request เท่านั้น! ยังมีวิธีอื่น ๆ ในการ สนับสนุน Material UI นอกเหนือจากการพัฒนาโค้ด
