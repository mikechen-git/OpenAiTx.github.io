Éditeur Vidéo Open-Source

Pas une application d’édition vidéo ordinaire, utilisant React, Remotion & TypeScript.


[!NOTE]
L’application est en cours de développement actif. Ceci est une version MVP précoce. Rejoignez le serveur Discord si vous souhaitez l’exécuter.


Une alternative open-source à Capcut, Canva et RVE.

✨Fonctionnalités

  • 🎬Édition vidéo non linéaire
  • 🔀Prise en charge multi-pistes
  • 👀Aperçu en direct
  • 📤Exportation vidéo
  • 📜Sous licence MIT

🐋Déploiement

git clone https://github.com/robinroy03/videoeditor.git
cd videoeditor
docker compose up

🧑‍💻Développement

pnpm i
pnpm run dev (frontend)
pnpm dlx tsx app/videorender/videorender.ts (backend)
uv run backend/main.py
mettez `isProduction` sur `false` dans `/app/utils/api.ts`

Vous aurez également besoin d’une GEMINI_API_KEY si vous souhaitez utiliser l’IA.

📃À FAIRE

Nous avons beaucoup de travail ! Pour commencer, nous prévoyons d’intégrer toutes les API Remotion. J’ajouterai bientôt une feuille de route appropriée. Rejoignez le serveur Discord pour les mises à jour et le support.
