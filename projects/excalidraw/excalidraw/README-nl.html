Een open source virtueel whiteboard in handgetekende stijl.
Samenwerkend en end-to-end versleuteld.


Functies

De Excalidraw editor (npm package) ondersteunt:

  • 💯 Gratis & open-source.
  • 🎨 Oneindig, canvas-gebaseerd whiteboard.
  • ✍️ Handgetekende stijl.
  • 🌓 Donkere modus.
  • 🏗️ Aanpasbaar.
  • 📷 Afbeeldingsondersteuning.
  • 😀 Ondersteuning voor vormbibliotheken.
  • 🌐 Lokalisatie (i18n) ondersteuning.
  • 🖼️ Exporteren naar PNG, SVG & klembord.
  • 💾 Open formaat - exporteer tekeningen als een .excalidraw json-bestand.
  • ⚒️ Breed scala aan tools - rechthoek, cirkel, ruit, pijl, lijn, vrij tekenen, gum...
  • ➡️ Pijl-binding & gelabelde pijlen.
  • 🔙 Ongedaan maken / Opnieuw uitvoeren.
  • 🔍 In- en uitzoomen en pannen.

Excalidraw.com

De app gehost op excalidraw.com is een minimale showcase van wat je kunt bouwen met Excalidraw. De broncode hiervan is ook onderdeel van deze repository, en de app beschikt over:

  • 📡 PWA-ondersteuning (werkt offline).
  • 🤼 Realtime samenwerking.
  • 🔒 End-to-end encryptie.
  • 💾 Local-first ondersteuning (autoslaat in de browser).
  • 🔗 Deelbare links (exporteer naar een alleen-lezen link die je kunt delen met anderen).

We zullen deze functies in de toekomst als plug-and-play plugins toevoegen voor de npm package.

Snel starten

Opmerking: onderstaande instructies zijn voor het installeren van het Excalidraw npm package wanneer je Excalidraw in je eigen app wilt integreren. Om de repository lokaal te draaien voor ontwikkeling, zie onze Development Guide.

Gebruik npm of yarn om het pakket te installeren.

npm install react react-dom @excalidraw/excalidraw
# of
yarn add react react-dom @excalidraw/excalidraw
