Funkcje

Edytor Excalidraw (pakiet npm) obsługuje:

  • 💯 Darmowy i otwartoźródłowy.
  • 🎨 Nieskończony whiteboard oparty na canvasie.
  • ✍️ Styl odręcznych rysunków.
  • 🌓 Tryb ciemny.
  • 🏗️ Możliwość dostosowania.
  • 📷 Wsparcie dla obrazów.
  • 😀 Wsparcie dla bibliotek kształtów.
  • 🌐 Obsługa lokalizacji (i18n).
  • 🖼️ Eksport do PNG, SVG i schowka.
  • 💾 Otwarty format – eksport rysunków jako plik .excalidraw (JSON).
  • ⚒️ Szeroka gama narzędzi – prostokąt, koło, diament, strzałka, linia, odręczne rysowanie, gumka...
  • ➡️ Więzanie strzałek i strzałki z etykietami.
  • 🔙 Cofnij / ponów.
  • 🔍 Obsługa zoomu i przesuwania.

Excalidraw.com

Aplikacja dostępna na excalidraw.com to minimalistyczna prezentacja tego, co możesz zbudować z Excalidraw. Jej kod źródłowy jest również częścią tego repozytorium, a aplikacja oferuje:

  • 📡 Wsparcie PWA (działa offline).
  • 🤼 Współpraca w czasie rzeczywistym.
  • 🔒 Szyfrowanie end-to-end.
  • 💾 Wsparcie lokalne (autosave w przeglądarce).
  • 🔗 Udostępnianie linków (eksport do linku tylko do odczytu, którym możesz się podzielić).

Te funkcjonalności będziemy w przyszłości dodawać jako gotowe pluginy do pakietu npm.

Szybki start

Uwaga: poniższe instrukcje dotyczą instalacji pakietu npm Excalidraw podczas integracji Excalidraw z Twoją aplikacją. Aby uruchomić repozytorium lokalnie do celów deweloperskich, sprawdź nasz Przewodnik deweloperski.

Użyj npm lub yarn, aby zainstalować pakiet.

npm install react react-dom @excalidraw/excalidraw
# lub
yarn add react react-dom @excalidraw/excalidraw

Zajrzyj do naszej dokumentacji, aby uzyskać więcej szczegółów!
