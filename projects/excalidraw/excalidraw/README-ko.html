Excalidraw 에디터 | 블로그 | 문서 | Excalidraw+

오픈 소스 가상 손그림 스타일 화이트보드.
협업 및 종단 간 암호화 지원.


주요 기능

Excalidraw 에디터(npm 패키지)는 다음을 지원합니다:

  • 💯 무료 & 오픈 소스
  • 🎨 무한 캔버스 기반 화이트보드
  • ✍️ 손그림과 유사한 스타일
  • 🌓 다크 모드
  • 🏗️ 사용자 정의 가능
  • 📷 이미지 지원
  • 😀 도형 라이브러리 지원
  • 🌐 다국어(i18n) 지원
  • 🖼️ PNG, SVG, 클립보드로 내보내기
  • 💾 오픈 포맷 - .excalidraw json 파일로 내보내기
  • ⚒️ 다양한 도구 - 사각형, 원, 다이아몬드, 화살표, 선, 자유 그리기, 지우개 등
  • ➡️ 화살표 바인딩 및 라벨 화살표
  • 🔙 실행 취소/다시 실행(Undo/Redo)
  • 🔍 확대/축소 및 패닝 지원

Excalidraw.com

excalidraw.com에 호스팅된 앱은 Excalidraw로 무엇을 만들 수 있는지 보여주는 최소한의 예시입니다. 이 앱의 소스 코드도 이 저장소에 포함되어 있으며, 주요 기능은 다음과 같습니다:

  • 📡 PWA 지원(오프라인 동작)
  • 🤼 실시간 협업
  • 🔒 종단 간 암호화
  • 💾 로컬 우선 지원(브라우저에 자동 저장)
  • 🔗 공유 가능한 링크(읽기 전용 링크로 내보내어 다른 사람과 공유 가능)

이 기능들은 앞으로 npm 패키지를 위한 플러그인 형태로 추가될 예정입니다.

빠른 시작

참고: 아래 안내는 Excalidraw를 여러분의 앱에 통합할 때 사용하는 npm 패키지 설치 방법입니다. 저장소를 로컬에서 개발용으로 실행하려면 개발 가이드를 참고하세요.

npm 또는 yarn을 사용해 패키지를 설치하세요.

npm install react react-dom @excalidraw/excalidraw
# 또는
yarn add react react-dom @excalidraw/excalidraw

자세한 내용은 문서를 참고하세요!
