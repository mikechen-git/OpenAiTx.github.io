سبورة افتراضية مفتوحة المصدر بأسلوب مرسوم باليد.
تعاونية ومشفرة من الطرف إلى الطرف.


الميزات

يدعم محرر Excalidraw (حزمة npm):

  • 💯 مجاني ومفتوح المصدر.
  • 🎨 سبورة لا نهائية قائمة على القماش.
  • ✍️ أسلوب مرسوم باليد.
  • 🌓 الوضع الليلي.
  • 🏗️ قابل للتخصيص.
  • 📷 دعم الصور.
  • 😀 دعم مكتبات الأشكال.
  • 🌐 دعم الترجمة (i18n).
  • 🖼️ تصدير إلى PNG، SVG والحافظة.
  • 💾 تنسيق مفتوح - تصدير الرسومات كملف .excalidraw بصيغة json.
  • ⚒️ مجموعة واسعة من الأدوات - مستطيل، دائرة، ماسة، سهم، خط، رسم حر، ممحاة...
  • ➡️ ربط الأسهم والسهم مع تسمية.
  • 🔙 تراجع / إعادة.
  • 🔍 دعم التكبير والتحريك.

Excalidraw.com

التطبيق المستضاف على excalidraw.com هو عرض بسيط لما يمكنك بناؤه باستخدام Excalidraw. شفرة المصدر الخاصة به جزء من هذا المستودع أيضاً، ويحتوي التطبيق على الميزات التالية:

  • 📡 دعم تطبيقات الويب التقدمية (يعمل دون اتصال).
  • 🤼 تعاون في الوقت الحقيقي.
  • 🔒 تشفير من الطرف إلى الطرف.
  • 💾 دعم العمل المحلي أولاً (حفظ تلقائي في المتصفح).
  • 🔗 روابط قابلة للمشاركة (تصدير إلى رابط للعرض فقط يمكنك مشاركته مع الآخرين).

سنضيف هذه الميزات كمكونات إضافية لحزمة npm في المستقبل.

بدء سريع

ملاحظة: التعليمات التالية لتثبيت حزمة Excalidraw على npm عند دمج Excalidraw في تطبيقك الخاص. لتشغيل المستودع محليًا لأغراض التطوير، يرجى الرجوع إلى دليل التطوير.

استخدم npm أو yarn لتثبيت الحزمة.

npm install react react-dom @excalidraw/excalidraw
# أو
yarn add react react-dom @excalidraw/excalidraw

اطلع على التوثيق الخاص بنا لمزيد من التفاصيل!
