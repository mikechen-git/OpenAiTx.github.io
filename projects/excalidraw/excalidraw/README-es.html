Colaborativa y cifrada de extremo a extremo.


Características

El editor Excalidraw (paquete npm) soporta:

  • 💯 Gratis y de código abierto.
  • 🎨 Pizarra infinita basada en canvas.
  • ✍️ Estilo similar al dibujo a mano.
  • 🌓 Modo oscuro.
  • 🏗️ Personalizable.
  • 📷 Soporte de imágenes.
  • 😀 Soporte para bibliotecas de formas.
  • 🌐 Soporte de localización (i18n).
  • 🖼️ Exportar a PNG, SVG y portapapeles.
  • 💾 Formato abierto: exporta dibujos como archivo JSON .excalidraw.
  • ⚒️ Amplia gama de herramientas: rectángulo, círculo, diamante, flecha, línea, dibujo libre, borrador...
  • ➡️ Unión de flechas y flechas etiquetadas.
  • 🔙 Deshacer / Rehacer.
  • 🔍 Soporte de zoom y desplazamiento.

Excalidraw.com

La aplicación alojada en excalidraw.com es una demostración mínima de lo que puedes construir con Excalidraw. Su código fuente también es parte de este repositorio, y la aplicación incluye:

  • 📡 Soporte PWA (funciona sin conexión).
  • 🤼 Colaboración en tiempo real.
  • 🔒 Cifrado de extremo a extremo.
  • 💾 Soporte local-first (guardado automático en el navegador).
  • 🔗 Enlaces compartibles (exporta a un enlace de solo lectura que puedes compartir con otros).

En el futuro, agregaremos estas características como plugins integrables para el paquete npm.

Inicio rápido

Nota: las siguientes instrucciones son para instalar el paquete npm de Excalidraw al integrar Excalidraw en tu propia aplicación. Para ejecutar el repositorio localmente para desarrollo, consulta nuestra Guía de desarrollo.

Usa npm o yarn para instalar el paquete.

npm install react react-dom @excalidraw/excalidraw
# o
yarn add react react-dom @excalidraw/excalidraw
