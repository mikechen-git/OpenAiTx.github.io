Tính năng

Trình biên tập Excalidraw (gói npm) hỗ trợ:

  • 💯 Miễn phí & mã nguồn mở.
  • 🎨 Bảng trắng vô hạn dựa trên canvas.
  • ✍️ Phong cách vẽ tay.
  • 🌓 Chế độ tối.
  • 🏗️ Tùy biến linh hoạt.
  • 📷 Hỗ trợ hình ảnh.
  • 😀 Hỗ trợ thư viện hình dạng.
  • 🌐 Hỗ trợ đa ngôn ngữ (i18n).
  • 🖼️ Xuất ra PNG, SVG & clipboard.
  • 💾 Định dạng mở - xuất bản vẽ dưới dạng tệp json .excalidraw.
  • ⚒️ Đa dạng công cụ - hình chữ nhật, hình tròn, hình thoi, mũi tên, đường thẳng, vẽ tự do, tẩy...
  • ➡️ Mũi tên liên kết & mũi tên có nhãn.
  • 🔙 Hoàn tác / Làm lại.
  • 🔍 Hỗ trợ phóng to và di chuyển.

Excalidraw.com

Ứng dụng được lưu trữ tại excalidraw.com là một bản trình diễn tối giản những gì bạn có thể xây dựng với Excalidraw. Mã nguồn của ứng dụng này cũng là một phần trong kho lưu trữ này, với các tính năng:

  • 📡 Hỗ trợ PWA (hoạt động ngoại tuyến).
  • 🤼 Cộng tác thời gian thực.
  • 🔒 Mã hóa đầu-cuối.
  • 💾 Ưu tiên lưu trữ cục bộ (tự động lưu vào trình duyệt).
  • 🔗 Liên kết chia sẻ (xuất ra liên kết chỉ đọc có thể chia sẻ cho người khác).

Chúng tôi sẽ bổ sung các tính năng này dưới dạng plugin cho gói npm trong tương lai.

Bắt đầu nhanh

Lưu ý: hướng dẫn sau dành cho việc cài đặt gói npm Excalidraw khi tích hợp Excalidraw vào ứng dụng của bạn. Để chạy kho lưu trữ này trên máy để phát triển, vui lòng tham khảo Hướng dẫn phát triển của chúng tôi.

Sử dụng npm hoặc yarn để cài đặt gói.

npm install react react-dom @excalidraw/excalidraw
# hoặc
yarn add react react-dom @excalidraw/excalidraw
