ویژگی‌ها

ویرایشگر Excalidraw (پکیج npm) از امکانات زیر پشتیبانی می‌کند:

  • 💯 رایگان و متن‌باز.
  • 🎨 وایت‌برد بی‌نهایت مبتنی بر بوم (canvas).
  • ✍️ سبک شبیه دست‌نویس.
  • 🌓 حالت تیره.
  • 🏗️ قابل سفارشی‌سازی.
  • 📷 پشتیبانی از تصویر.
  • 😀 پشتیبانی از کتابخانه‌های اشکال.
  • 🌐 پشتیبانی از بومی‌سازی (i18n).
  • 🖼️ خروجی به PNG، SVG و کلیپ‌بورد.
  • 💾 فرمت باز - خروجی نقاشی به صورت فایل json با پسوند .excalidraw.
  • ⚒️ ابزارهای متنوع - مستطیل، دایره، لوزی، پیکان، خط، رسم آزاد، پاک‌کن و...
  • ➡️ اتصال پیکان‌ها و پیکان‌های دارای برچسب.
  • 🔙 واگرد (undo) / تکرار (redo).
  • 🔍 پشتیبانی از بزرگ‌نمایی و جابه‌جایی (pan).

Excalidraw.com

برنامه‌ی میزبانی‌شده در excalidraw.com نمونه‌ای ساده از آنچه با Excalidraw می‌توانید بسازید است. کد منبع این برنامه نیز بخشی از این مخزن است و این ویژگی‌ها را دارد:

  • 📡 پشتیبانی از PWA (قابل استفاده آفلاین).
  • 🤼 همکاری همزمان (Real-time).
  • 🔒 رمزگذاری سرتاسری.
  • 💾 ذخیره‌سازی محلی (به طور خودکار در مرورگر ذخیره می‌شود).
  • 🔗 پیوندهای قابل اشتراک‌گذاری (خروجی به پیوند فقط خواندنی برای به اشتراک‌گذاری با دیگران).

در آینده این ویژگی‌ها را به صورت افزونه‌های قابل نصب برای پکیج npm ارائه خواهیم داد.

شروع سریع

نکته: دستورالعمل‌های زیر برای نصب پکیج npm Excalidraw هنگام یکپارچه‌سازی با برنامه‌ی شما است. برای اجرای مخزن به صورت محلی جهت توسعه، لطفاً به راهنمای توسعه مراجعه کنید.

برای نصب پکیج از npm یا yarn استفاده کنید.

npm install react react-dom @excalidraw/excalidraw
# یا
yarn add react react-dom @excalidraw/excalidraw

برای اطلاعات بیشتر به مستندات ما مراجعه کنید!
