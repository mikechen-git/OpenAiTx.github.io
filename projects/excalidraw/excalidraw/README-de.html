Funktionen

Der Excalidraw Editor (npm-Paket) unterstützt:

  • 💯 Kostenlos & Open-Source.
  • 🎨 Unendliches, canvas-basiertes Whiteboard.
  • ✍️ Handgezeichneter Stil.
  • 🌓 Dark Mode.
  • 🏗️ Anpassbar.
  • 📷 Bildunterstützung.
  • 😀 Unterstützung für Formenbibliotheken.
  • 🌐 Unterstützung für Lokalisierung (i18n).
  • 🖼️ Export als PNG, SVG & Zwischenablage.
  • 💾 Offenes Format – exportiere Zeichnungen als .excalidraw-JSON-Datei.
  • ⚒️ Große Auswahl an Werkzeugen – Rechteck, Kreis, Raute, Pfeil, Linie, Freihand, Radierer...
  • ➡️ Pfeilbindung & beschriftete Pfeile.
  • 🔙 Rückgängig / Wiederherstellen.
  • 🔍 Zoom- und Verschiebungsunterstützung.

Excalidraw.com

Die unter excalidraw.com gehostete App ist eine minimalistische Demonstration dessen, was Sie mit Excalidraw erstellen können. Ihr Quellcode ist ebenfalls Teil dieses Repositories und bietet folgende Funktionen:

  • 📡 PWA-Unterstützung (funktioniert offline).
  • 🤼 Echtzeit-Kollaboration.
  • 🔒 Ende-zu-Ende-Verschlüsselung.
  • 💾 Local-first-Unterstützung (automatisches Speichern im Browser).
  • 🔗 Teilbare Links (Export zu einem schreibgeschützten Link, den Sie mit anderen teilen können).

Wir werden diese Funktionen zukünftig als Plug-and-Play-Plugins für das npm-Paket bereitstellen.

Schnellstart

Hinweis: Die folgenden Anweisungen gelten für die Installation des Excalidraw npm-Pakets, wenn Sie Excalidraw in Ihre eigene App integrieren möchten. Um das Repository lokal für die Entwicklung auszuführen, beachten Sie bitte unseren Entwicklungsleitfaden.

Verwenden Sie npm oder yarn, um das Paket zu installieren.

npm install react react-dom @excalidraw/excalidraw
# oder
yarn add react react-dom @excalidraw/excalidraw
