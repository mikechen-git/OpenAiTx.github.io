Fitur

Editor Excalidraw (paket npm) mendukung:

  • 💯 Gratis & sumber terbuka.
  • 🎨 Papan tulis berbasis kanvas tanpa batas.
  • ✍️ Gaya mirip coretan tangan.
  • 🌓 Mode gelap.
  • 🏗️ Dapat dikustomisasi.
  • 📷 Dukungan gambar.
  • 😀 Dukungan pustaka bentuk.
  • 🌐 Dukungan lokalisasi (i18n).
  • 🖼️ Ekspor ke PNG, SVG & clipboard.
  • 💾 Format terbuka - ekspor gambar sebagai file json .excalidraw.
  • ⚒️ Beragam alat - persegi panjang, lingkaran, belah ketupat, panah, garis, free-draw, penghapus...
  • ➡️ Pengikatan panah & panah berlabel.
  • 🔙 Undo / Redo.
  • 🔍 Dukungan zoom dan panning.

Excalidraw.com

Aplikasi yang dihosting di excalidraw.com adalah contoh minimal dari apa yang dapat Anda bangun dengan Excalidraw. Kode sumbernya juga merupakan bagian dari repositori ini, dan aplikasi ini memiliki fitur:

  • 📡 Dukungan PWA (bisa digunakan offline).
  • 🤼 Kolaborasi waktu nyata.
  • 🔒 Enkripsi end-to-end.
  • 💾 Dukungan local-first (penyimpanan otomatis di browser).
  • 🔗 Tautan yang dapat dibagikan (ekspor ke tautan readonly yang dapat Anda bagikan ke orang lain).

Kami akan menambahkan fitur-fitur ini sebagai plugin drop-in untuk paket npm di masa mendatang.

Mulai Cepat

Catatan: Instruksi berikut untuk menginstal paket npm Excalidraw saat mengintegrasikan Excalidraw ke aplikasi Anda sendiri. Untuk menjalankan repositori secara lokal untuk pengembangan, silakan lihat Panduan Pengembangan kami.

Gunakan npm atau yarn untuk menginstal paketnya.

npm install react react-dom @excalidraw/excalidraw
# atau
yarn add react react-dom @excalidraw/excalidraw
