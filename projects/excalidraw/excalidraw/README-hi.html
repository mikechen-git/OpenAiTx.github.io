  • 🌓 डार्क मोड।

  • 🌐 स्थानीयकरण (i18n) समर्थन।
  • 🖼️ PNG, SVG और क्लिपबोर्ड में निर्यात।

  • ⚒️ विस्तृत टूल्स - आयत, वृत्त, हीरा, तीर, रेखा, मुक्त-चित्रण, इरेज़र...

  • 🔍 ज़ूम और पैनिंग समर्थन।

Excalidraw.com

excalidraw.com पर होस्ट किया गया ऐप Excalidraw के साथ आप क्या बना सकते हैं उसका एक न्यूनतम प्रदर्शन है। इसका स्रोत कोड भी इस रिपॉजिटरी का हिस्सा है, और ऐप में ये विशेषताएँ हैं:

  • 📡 PWA समर्थन (ऑफ़लाइन काम करता है)।
  • 🤼 रीयल-टाइम सहयोग।
  • 🔒 एंड-टू-एंड एन्क्रिप्शन।
  • 💾 लोकल-फर्स्ट समर्थन (ब्राउज़र में ऑटोसेव होता है)।
  • 🔗 साझा करने योग्य लिंक (एक केवल-पढ़ने योग्य लिंक में निर्यात करें जिसे आप दूसरों के साथ साझा कर सकते हैं)।

हम भविष्य में इन सुविधाओं को npm पैकेज के लिए ड्रॉप-इन प्लगइन्स के रूप में जोड़ेंगे।

त्वरित प्रारंभ

नोट: निम्न निर्देश Excalidraw npm पैकेज को अपने ऐप में एकीकृत करते समय इंस्टॉलेशन के लिए हैं। डेवलपमेंट के लिए रिपॉजिटरी को स्थानीय रूप से चलाने हेतु कृपया हमारे डेवलपमेंट गाइड देखें।

पैकेज इंस्टॉल करने के लिए npm या yarn का उपयोग करें।

npm install react react-dom @excalidraw/excalidraw
# या
yarn add react react-dom @excalidraw/excalidraw

अधिक जानकारी के लिए हमारा दस्तावेज़ देखें!
