React Native

Học một lần, viết ở mọi nơi:
Xây dựng ứng dụng di động với React.

Bắt đầu · Học những điều cơ bản · Trình diễn · Đóng góp · Cộng đồng · Hỗ trợ

React Native mang React's - khung giao diện khai báo UI đến iOS và Android. Với React Native, bạn sử dụng các điều khiển giao diện gốc và có quyền truy cập đầy đủ vào nền tảng gốc.

  • Khai báo. React giúp việc tạo giao diện người dùng tương tác trở nên dễ dàng. Các view khai báo làm cho mã của bạn dễ dự đoán hơn và dễ debug hơn.
  • Dựa trên thành phần. Xây dựng các thành phần được đóng gói quản lý trạng thái của chính chúng, sau đó kết hợp chúng để tạo ra các giao diện phức tạp.
  • Tốc độ phát triển. Thấy các thay đổi cục bộ trong vài giây. Thay đổi mã JavaScript có thể được tải lại trực tiếp mà không cần xây dựng lại ứng dụng gốc.
  • Tính di động. Tái sử dụng mã trên iOS, Android và nền tảng khác.

React Native được phát triển và hỗ trợ bởi nhiều công ty và các cộng tác viên lõi cá nhân. Tìm hiểu thêm trong tổng quan hệ sinh thái của chúng tôi.

📋 Yêu cầu

Các ứng dụng React Native có thể hướng đến iOS 15.1 và Android 7.0 (API 24) hoặc mới hơn. Bạn có thể sử dụng Windows, macOS hoặc Linux làm hệ điều hành phát triển, mặc dù việc xây dựng và chạy ứng dụng iOS chỉ giới hạn trên macOS. Các công cụ như Expo có thể được sử dụng để khắc phục điều này.

🎉 Xây dựng ứng dụng React Native đầu tiên của bạn

Làm theo hướng dẫn Bắt đầu. Cách cài đặt React Native được khuyến nghị phụ thuộc vào dự án của bạn. Dưới đây là hướng dẫn nhanh cho các kịch bản phổ biến nhất:

  • Trải nghiệm React Native
  • Tạo ứng dụng mới
  • Thêm React Native vào ứng dụng hiện có

📖 Tài liệu

Tài liệu đầy đủ cho React Native có thể được tìm thấy trên website của chúng tôi.

Tài liệu React Native trình bày về các thành phần, API và các chủ đề riêng biệt cho React Native. Để biết thêm tài liệu về API React dùng chung giữa React Native và React DOM, hãy tham khảo tài liệu React.

Nguồn tài liệu và website React Native được lưu trữ trong một kho riêng, @facebook/react-native-website.

🚀 Nâng cấp

Nâng cấp lên các phiên bản mới của React Native có thể mang đến cho bạn nhiều API, view, công cụ phát triển và các tính năng khác. Xem Hướng dẫn Nâng cấp để biết hướng dẫn chi tiết.

Các bản phát hành React Native được thảo luận tại kho thảo luận này.

👏 Cách đóng góp

Mục đích chính của kho lưu trữ này là tiếp tục phát triển lõi React Native. Chúng tôi muốn việc đóng góp cho dự án này trở nên dễ dàng và minh bạch nhất có thể, và chúng tôi biết ơn cộng đồng đã đóng góp sửa lỗi và cải tiến. Đọc bên dưới để biết cách bạn có thể tham gia cải thiện React Native.

Quy tắc Ứng xử

Facebook đã áp dụng Quy tắc Ứng xử mà chúng tôi mong đợi các thành viên dự án tuân thủ. Vui lòng đọc toàn văn để hiểu những hành động nào sẽ và không được chấp nhận.

Hướng dẫn Đóng góp

Đọc Hướng dẫn Đóng góp của chúng tôi để tìm hiểu về quy trình phát triển, cách đề xuất sửa lỗi và cải tiến, cũng như cách xây dựng và kiểm thử các thay đổi với React Native.

Lộ trình mã nguồn mở

Bạn có thể tìm hiểu thêm về tầm nhìn của chúng tôi cho React Native trong Lộ trình.

Các vấn đề dễ cho người mới

Chúng tôi có một danh sách good first issues gồm các lỗi có phạm vi khá hạn chế. Đây là nơi tuyệt vời để bắt đầu, tích lũy kinh nghiệm và làm quen với quy trình đóng góp của chúng tôi.
