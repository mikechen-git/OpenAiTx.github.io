React Native

एक बार सीखें, कहीं भी लिखें:
React के साथ मोबाइल ऐप्स बनाएं।

शुरुआत करें · बुनियादी बातें सीखें · प्रदर्शनी · योगदान करें · समुदाय · सहायता

React Native React's घोषणात्मक UI फ्रेमवर्क को iOS और Android पर लाता है। React Native के साथ, आप नेटिव UI नियंत्रणों का उपयोग करते हैं और नेटिव प्लेटफ़ॉर्म तक पूरी पहुंच रखते हैं।

  • घोषणात्मक। React के साथ इंटरैक्टिव UI बनाना आसान है। घोषणात्मक दृश्य आपके कोड को अधिक अनुमानित और डिबग करने में आसान बनाते हैं।
  • कंपोनेंट-आधारित। ऐसे संलग्नित कंपोनेंट्स बनाएं जो अपनी स्थिति को संभालते हैं, फिर उन्हें मिलाकर जटिल UI बनाएं।
  • डेवलपर गति। सेकंडों में स्थानीय परिवर्तन देखें। जावास्क्रिप्ट कोड में बदलाव बिना नेटिव ऐप को फिर से बनाए लाइव रीलोड किए जा सकते हैं।
  • पोर्टेबिलिटी। iOS, Android, और अन्य प्लेटफ़ॉर्म में कोड का पुन: उपयोग करें।

React Native को कई कंपनियों और व्यक्तिगत कोर योगदानकर्ताओं द्वारा विकसित और समर्थित किया गया है। हमारे इकोसिस्टम अवलोकन में और जानें।

📋 आवश्यकताएँ

React Native ऐप्स iOS 15.1 और Android 7.0 (API 24) या नए को लक्षित कर सकते हैं। आप विंडोज़, मैकओएस, या लिनक्स को अपने विकास ऑपरेटिंग सिस्टम के रूप में उपयोग कर सकते हैं, हालांकि iOS ऐप्स बनाना और चलाना केवल मैकओएस तक सीमित है। Expo जैसे टूल्स का उपयोग इस सीमा को दूर करने के लिए किया जा सकता है।

🎉 अपना पहला React Native ऐप बनाना

शुरुआत गाइड का पालन करें। React Native को स्थापित करने का अनुशंसित तरीका आपके प्रोजेक्ट पर निर्भर करता है। यहाँ सबसे सामान्य परिस्थितियों के लिए छोटे-छोटे गाइड्स दिए गए हैं:

  • React Native आज़माएं
  • एक नया एप्लिकेशन बनाना
  • मौजूदा एप्लिकेशन में React Native जोड़ना

📖 दस्तावेज़ीकरण

React Native के लिए पूरी दस्तावेज़ीकरण हमारे वेबसाइट पर उपलब्ध है।

React Native दस्तावेज़ीकरण उन कंपोनेंट्स, API, और विषयों पर चर्चा करता है जो React Native के लिए विशिष्ट हैं। React Native और React DOM के बीच साझा किए गए React API के लिए आगे की दस्तावेज़ीकरण हेतु React दस्तावेज़ीकरण देखें।

React Native दस्तावेज़ीकरण और वेबसाइट का स्रोत एक अलग भंडार में होस्ट किया गया है, @facebook/react-native-website।

🚀 अपग्रेडिंग

React Native के नए संस्करणों में अपग्रेड करने से आपको अधिक API, दृश्य, डेवलपर टूल्स, और अन्य सुविधाओं तक पहुँच मिल सकती है। निर्देशों के लिए अपग्रेडिंग गाइड देखें।

React Native रिलीज़ के बारे में इस चर्चा रिपॉज़िटरी में चर्चा की जाती है।

👏 कैसे करें योगदान

इस रिपॉज़िटरी का मुख्य उद्देश्य React Native कोर का निरंतर विकास है। हम चाहते हैं कि इस प्रोजेक्ट में योगदान करना यथासंभव आसान और पारदर्शी हो, और हम बग फिक्स और सुधारों के लिए समुदाय के आभारी हैं। जानने के लिए पढ़ें कि आप React Native को बेहतर बनाने में कैसे भाग ले सकते हैं।

आचार संहिता

Facebook ने एक आचार संहिता अपनाई है जिसकी हम अपेक्षा करते हैं कि प्रोजेक्ट प्रतिभागी पालन करेंगे। कृपया पूरा पाठ पढ़ें ताकि आप समझ सकें कि कौन से कार्य स्वीकार्य हैं और कौन से नहीं।

योगदान गाइड

हमारे विकास प्रक्रिया, बग फिक्स और सुधारों का प्रस्ताव कैसे करें, और अपने बदलावों का निर्माण और परीक्षण कैसे करें, जानने के लिए हमारा योगदान गाइड पढ़ें।

ओपन सोर्स रोडमैप

React Native के लिए हमारी दृष्टि के बारे में आप रोडमैप में और जान सकते हैं।

अच्छी पहली समस्याएँ

हमारे पास अच्छी पहली समस्याओं की एक सूची है जिनमें सीमित दायरे की बग्स होती हैं। यह आरंभ करने, अनुभव प्राप्त करने और हमारे योगदान प्रक्रिया से परिचित होने के लिए शानदार जगह है।
