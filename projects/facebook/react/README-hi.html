React ·

React एक जावास्क्रिप्ट लाइब्रेरी है जो यूजर इंटरफेस बनाने के लिए उपयोग की जाती है।

  • डिक्लेरेटिव: React इंटरएक्टिव यूआई बनाना आसान बनाता है। अपने एप्लिकेशन की प्रत्येक स्थिति के लिए सरल दृश्य डिज़ाइन करें, और जब आपके डेटा में बदलाव हो, तो React कुशलतापूर्वक केवल सही कंपोनेंट्स को अपडेट और रेंडर करेगा। डिक्लेरेटिव व्यूज़ आपके कोड को अधिक पूर्वानुमानित, समझने में आसान और डिबग करने में सरल बनाते हैं।
  • कंपोनेंट-आधारित: इनकैप्सुलेटेड कंपोनेंट्स बनाएं जो अपनी खुद की स्थिति प्रबंधित करते हैं, फिर उन्हें जोड़कर जटिल यूआई बनाएं। चूँकि कंपोनेंट लॉजिक टेम्पलेट्स की बजाय जावास्क्रिप्ट में लिखा जाता है, इसलिए आप आसानी से समृद्ध डेटा अपने ऐप के माध्यम से पास कर सकते हैं और स्थिति को DOM से बाहर रख सकते हैं।
  • एक बार सीखें, कहीं भी लिखें: हम आपके तकनीकी स्टैक के बाकी हिस्सों के बारे में कोई पूर्वानुमान नहीं लगाते, इसलिए आप React में नई सुविधाएँ विकसित कर सकते हैं बिना मौजूदा कोड को दोबारा लिखे। React सर्वर पर Node का उपयोग करके भी रेंडर कर सकता है और React Native के ज़रिए मोबाइल ऐप्स को भी शक्ति दे सकता है।

जानें कि अपनी परियोजना में React का उपयोग कैसे करें।

इंस्टॉलेशन

React को शुरू से ही क्रमिक रूप से अपनाने के लिए डिज़ाइन किया गया है, और आप जितना चाहें उतना या जितना ज़रूरत हो उतना React इस्तेमाल कर सकते हैं:

  • त्वरित प्रारंभ का उपयोग करके React का अनुभव लें।
  • मौजूदा परियोजना में React जोड़ें ताकि आप जितना चाहें उतना React इस्तेमाल कर सकें।
  • नया React ऐप बनाएं अगर आप एक शक्तिशाली जावास्क्रिप्ट टूलचेन खोज रहे हैं।

डाक्यूमेंटेशन

आप React का डाक्यूमेंटेशन वेबसाइट पर देख सकते हैं।

त्वरित अवलोकन के लिए Getting Started पृष्ठ देखें।

डाक्यूमेंटेशन कई भागों में विभाजित है:

  • त्वरित प्रारंभ
  • ट्यूटोरियल
  • React में सोचना
  • इंस्टॉलेशन
  • यूआई का वर्णन करना
  • इंटरएक्टिविटी जोड़ना
  • स्टेट प्रबंधन
  • एडवांस्ड गाइड्स
  • API संदर्भ
  • सहायता कहाँ प्राप्त करें
  • योगदान गाइड

आप इस रिपॉजिटरी में पुल रिक्वेस्ट भेजकर इसे बेहतर बना सकते हैं।

उदाहरण

हमारे पास कई उदाहरण वेबसाइट पर उपलब्ध हैं। यहाँ पहला उदाहरण है जिससे आप शुरुआत कर सकते हैं:

import { createRoot } from 'react-dom/client';

function HelloMessage({ name }) {
  return <div>Hello {name}</div>;
}

const root = createRoot(document.getElementById('container'));
root.render(<HelloMessage name="Taylor" />);

यह उदाहरण पेज पर एक कंटेनर में "Hello Taylor" को रेंडर करेगा।

आप देखेंगे कि हमने एक HTML-जैसी सिंटैक्स का उपयोग किया है; इसे हम JSX कहते हैं। React का उपयोग करने के लिए JSX आवश्यक नहीं है, लेकिन इससे कोड अधिक पठनीय हो जाता है, और इसे लिखना HTML जैसा महसूस होता है।

योगदान

इस रिपॉजिटरी का मुख्य उद्देश्य React कोर का सतत विकास करना है, जिससे यह तेज और उपयोग में आसान बन सके। React का विकास ओपनली GitHub पर होता है, और हम बग फिक्स और सुधारों में योगदान के लिए समुदाय के आभारी हैं। नीचे पढ़ें कि आप React को बेहतर बनाने में कैसे भाग ले सकते हैं।

आचार संहिता

Facebook ने एक आचार संहिता अपनाई है, जिसकी हम अपेक्षा करते हैं कि परियोजना प्रतिभागी उसका पालन करें। कृपया पूरा पाठ पढ़ें ताकि आप समझ सकें कि कौन से कार्य स्वीकार्य हैं और कौन से नहीं।

योगदान गाइड

हमारी योगदान गाइड पढ़ें ताकि आप हमारे विकास प्रक्रिया, बग फिक्स और सुधार प्रस्तावित करने, तथा React में अपने परिवर्तनों को बिल्ड और टेस्ट करने के बारे में जान सकें।

शुरुआती-अनुकूल समस्याएँ

आपकी सहायता के लिए और आपको हमारे योगदान प्रक्रिया से परिचित कराने के लिए, हमारे पास शुरुआती-अनुकूल समस्याएँ की सूची है जिनका दायरा सीमित है। यह शुरू करने के लिए शानदार जगह है।
