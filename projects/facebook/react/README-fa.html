ری‌اکت (React) ·

ری‌اکت یک کتابخانه جاوااسکریپت برای ساخت رابط‌های کاربری است.

  • اعلانی (Declarative): ری‌اکت ساخت رابط‌های کاربری تعاملی را بدون دردسر می‌کند. نماهای ساده‌ای برای هر وضعیت در برنامه خود طراحی کنید و ری‌اکت به طور مؤثر فقط اجزای صحیح را هنگام تغییر داده به‌روزرسانی و رندر می‌کند. نماهای اعلانی کد شما را قابل پیش‌بینی‌تر، ساده‌تر برای فهم و راحت‌تر برای رفع اشکال می‌نماید.
  • مبتنی بر کامپوننت: کامپوننت‌های کپسوله‌شده‌ای بسازید که وضعیت خود را مدیریت می‌کنند، سپس آن‌ها را برای ساخت رابط‌های کاربری پیچیده ترکیب نمایید. از آنجا که منطق کامپوننت به جای قالب‌ها در جاوااسکریپت نوشته می‌شود، می‌توانید به راحتی داده‌های غنی را در سراسر برنامه خود عبور دهید و وضعیت را خارج از DOM نگه دارید.
  • یک بار بیاموزید، همه جا بنویسید: ما در مورد بقیه پشته فناوری شما فرضی نمی‌گیریم، بنابراین می‌توانید ویژگی‌های جدید را در ری‌اکت توسعه دهید بدون اینکه نیاز به بازنویسی کدهای موجود داشته باشید. ری‌اکت همچنین می‌تواند با استفاده از Node روی سرور رندر شود و برنامه‌های موبایل را با استفاده از React Native پشتیبانی کند.

یاد بگیرید چگونه ری‌اکت را در پروژه خود استفاده کنید.

نصب

ری‌اکت از ابتدا برای پذیرش تدریجی طراحی شده است و می‌توانید به اندازه نیاز خود از ری‌اکت استفاده کنید:

  • از شروع سریع برای تجربه اولیه ری‌اکت بهره بگیرید.
  • افزودن ری‌اکت به پروژه موجود تا به اندازه نیاز خود از ری‌اکت استفاده کنید.
  • ایجاد یک برنامه جدید ری‌اکت اگر به دنبال یک ابزار قدرتمند جاوااسکریپت هستید.

مستندات

شما می‌توانید مستندات ری‌اکت را در وبسایت بیابید.

صفحه شروع سریع را برای یک مرور سریع بررسی کنید.

مستندات به چند بخش تقسیم شده‌اند:

  • شروع سریع
  • آموزش
  • تفکر در ری‌اکت
  • نصب
  • توصیف رابط کاربری
  • افزودن تعامل
  • مدیریت وضعیت
  • راهنماهای پیشرفته
  • مرجع API
  • کجا پشتیبانی بگیرید
  • راهنمای مشارکت

شما می‌توانید با ارسال پول ریکوئست به این مخزن آن را بهبود دهید.

مثال‌ها

ما چندین مثال در وبسایت داریم. در اینجا اولین مثال برای شروع آورده شده است:

import { createRoot } from 'react-dom/client';

function HelloMessage({ name }) {
  return <div>Hello {name}</div>;
}

const root = createRoot(document.getElementById('container'));
root.render(<HelloMessage name="Taylor" />);

این مثال عبارت "Hello Taylor" را در یک کانتینر در صفحه رندر خواهد کرد.

متوجه خواهید شد که ما از نگارشی شبیه HTML استفاده کردیم؛ ما آن را JSX می‌نامیم. استفاده از JSX برای کار با ری‌اکت الزامی نیست، اما کد را خواناتر می‌کند و نوشتن آن شبیه نوشتن HTML است.

مشارکت

هدف اصلی این مخزن، ادامه توسعه هسته ری‌اکت، سریع‌تر و آسان‌تر کردن آن است. توسعه ری‌اکت به صورت عمومی در گیت‌هاب انجام می‌شود و ما از جامعه برای رفع اشکال‌ها و بهبودها سپاسگزاریم. برای یادگیری نحوه مشارکت در بهبود ری‌اکت، ادامه را بخوانید.

کد رفتار

فیسبوک یک کد رفتار را پذیرفته است که انتظار داریم شرکت‌کنندگان پروژه به آن پایبند باشند. لطفاً متن کامل را بخوانید تا بدانید چه اقداماتی پذیرفته و چه اقداماتی غیرقابل قبول هستند.

راهنمای مشارکت

راهنمای مشارکت ما را بخوانید تا با فرآیند توسعه، نحوه پیشنهاد رفع اشکال و بهبودها و نحوه ساخت و آزمایش تغییرات خود در ری‌اکت آشنا شوید.
