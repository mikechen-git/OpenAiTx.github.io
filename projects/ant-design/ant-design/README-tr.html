⌨️ Geliştirme

Ücretsiz, çevrimiçi, saf ön uç geliştirme ortamı olan opensumi.run kullanın.

Veya yerel olarak klonlayın:

$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start

Tarayıcınızı açın ve http://127.0.0.1:8001 adresini ziyaret edin, daha fazla bilgi için Geliştirme bölümüne bakın.
