📦 安装

npm install antd

yarn add antd

pnpm add antd

bun add antd

🔨 使用

import { Button, DatePicker } from 'antd';

export default () => (
  <>
    <Button type="primary">按我</Button>
    <DatePicker placeholder="请选择日期" />
  </>
);

⌨️ 开发

使用 opensumi.run，一个免费的在线前端开发环境。

或本地克隆：

$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start

打开浏览器访问 http://127.0.0.1:8001，更多内容请见 开发者指引。
