  • 🌈 UI kelas enterprise yang dirancang untuk aplikasi web.

⌨️ Pengembangan

Gunakan opensumi.run, lingkungan pengembangan front-end murni gratis secara online.

Atau kloning secara lokal:

$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start

Buka browser Anda dan kunjungi http://127.0.0.1:8001, selengkapnya lihat di Pengembangan.

🤝 Kontribusi

Mari membangun antd yang lebih baik bersama-sama.

Kami sangat mengundang kontribusi dari semua orang. Sebelum memulai, silakan luangkan waktu untuk membaca Panduan Kontribusi. Jangan ragu untuk berbagi ide Anda melalui Pull Request atau GitHub Issues. Jika Anda tertarik meningkatkan basis kode kami, jelajahi Instruksi Pengembangan dan nikmati perjalanan coding Anda! :)

Untuk kolaborator, patuhi Prinsip Pull Request kami dan gunakan Template Pull Request saat membuat Pull Request.
