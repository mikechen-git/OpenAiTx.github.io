⌨️ การพัฒนา

ใช้ opensumi.run สภาพแวดล้อมการพัฒนาแบบ front-end ออนไลน์ฟรี

หรือโคลนมาพัฒนาในเครื่อง:

$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start

เปิดเบราว์เซอร์ของคุณที่ http://127.0.0.1:8001 ดูรายละเอียดเพิ่มเติมที่ Development

🤝 การมีส่วนร่วม

มาร่วมกันสร้าง antd ให้ดียิ่งขึ้น

เรายินดีต้อนรับการมีส่วนร่วมจากทุกคน ก่อนเริ่มต้นขอให้คุณอ่าน คู่มือการมีส่วนร่วม หากคุณมีไอเดียหรือต้องการเสนอแนะ ส่งผ่าน Pull Requests หรือ GitHub Issues ได้เลย หากสนใจพัฒนาระบบ ลองอ่าน Development Instructions แล้วสนุกกับการเขียนโค้ด! :)

สำหรับผู้ร่วมพัฒนา กรุณายึดตาม หลักการ Pull Request และใช้ เทมเพลต Pull Request ทุกครั้งที่สร้าง Pull Request
