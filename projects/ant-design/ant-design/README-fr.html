Ant Design

Un langage de conception d'interface utilisateur de classe entreprise et une bibliothèque UI React.

Changelog · Signaler un bug · Demander une fonctionnalité · English · 中文

❤️ Sponsors et Soutiens

✨ Fonctionnalités

  • 🌈 Interface utilisateur de classe entreprise conçue pour les applications web.
  • 📦 Un ensemble de composants React de haute qualité prêts à l'emploi.
  • 🛡 Écrit en TypeScript avec des types statiques prévisibles.
  • ⚙️ Ensemble complet de ressources de conception et d'outils de développement.
  • 🌍 Prise en charge de l'internationalisation pour des dizaines de langues.
  • 🎨 Personnalisation puissante des thèmes basée sur CSS-in-JS.

⌨️ Développement

Utilisez opensumi.run, un environnement de développement front-end pur, gratuit et en ligne.

Ou clonez localement :

$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start

Ouvrez votre navigateur et visitez http://127.0.0.1:8001, voir plus de détails dans Développement.
