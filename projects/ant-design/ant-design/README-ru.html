✨ Особенности

  • 🌈 Корпоративный UI, спроектированный для веб-приложений.
  • 📦 Набор высококачественных React-компонентов из коробки.
  • 🛡 Написан на TypeScript с предсказуемой статической типизацией.
  • ⚙️ Полный пакет дизайнерских ресурсов и инструментов разработки.
  • 🌍 Поддержка интернационализации для десятков языков.
  • 🎨 Мощная настройка тем на основе CSS-in-JS.

⌨️ Разработка

Используйте opensumi.run, бесплатную онлайн-среду для фронтенд-разработки.

Или клонируйте локально:

$ git clone git@github.com:ant-design/ant-design.git
$ cd ant-design
$ npm install
$ npm start

Откройте браузер и перейдите по адресу http://127.0.0.1:8001, подробнее смотрите в Development.
