Hallo hackers. Hallo maldevs. Hallo reversers. Leuk dat je hier bent om de duistere kracht van Zig te ontdekken!

Onderhouden en beheerd door @CX330Blake

Intro

[!BELANGRIJK] Dit project wordt continu bijgewerkt!

Dit project biedt veel implementaties van malwaretechnieken met Zig, aangezien ik er een grote fan van ben. Je kunt deze repo gebruiken om Zig te bewapenen. Black-Hat-Zig wordt voortdurend bijgewerkt om ervoor te zorgen dat het zoveel mogelijk inhoud bevat. Het zou perfect zijn als je een PR voor dit project wilt maken.

Oké, laten we de planeet hacken!

Waarom Zig?

  • 🤝 Makkelijk te integreren met C/C++ broncode
  • 🔎 Het is nieuwer, dus moeilijker te detecteren
  • 💪 Zeer laag niveau controle, zelfs lager dan C
  • 😱 Moeilijker te reverse engineeren door de moderne compiler
  • 🛡️ Minimale runtime afhankelijkheden - Geen grote standaardbibliotheek footprint
  • 🎯 Geen ongedefinieerd gedrag - Expliciete afhandeling van edge cases voorkomt crashes
