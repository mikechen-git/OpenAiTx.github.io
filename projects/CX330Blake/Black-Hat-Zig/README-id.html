Intro

[!PENTING] Proyek ini terus diperbarui!

Proyek ini menyediakan banyak implementasi teknik malware menggunakan Zig karena saya penggemar beratnya. Anda dapat menggunakan repo ini untuk mempersenjatai Zig. Black-Hat-Zig terus diperbarui untuk memastikan berisi sebanyak mungkin konten. Akan sangat sempurna jika Anda ingin membuat PR untuk proyek ini.

Oke, mari kita hack planet ini!

Kenapa Zig?

  • 🤝 Mudah berinteraksi dengan source C/C++
  • 🔎 Ini lebih baru, jadi lebih sulit terdeteksi
  • 💪 Kontrol tingkat rendah yang kuat, bahkan lebih rendah dari C
  • 😱 Lebih sulit untuk RE karena kompilernya yang modern
  • 🛡️ Ketergantungan runtime minimal - Tidak ada jejak pustaka standar yang besar
  • 🎯 Tidak ada perilaku tak terdefinisi - Penanganan eksplisit untuk edge case mencegah crash
