快速開始

要運行並與 Llama 3.2 聊天：

ollama run llama3.2

自訂模型

從 GGUF 匯入

Ollama 支援於 Modelfile 匯入 GGUF 模型：

  1. 建立名為 Modelfile 的檔案，內容包含 FROM 指令，指定欲匯入模型的本機檔案路徑。

    FROM ./vicuna-33b.Q4_0.gguf

  2. 在 Ollama 中建立模型

    ollama create example -f Modelfile

  3. 運行該模型

    ollama run example

從 Safetensors 匯入

詳情請參閱匯入模型指南。

自訂提示詞

Ollama 庫中的模型可透過提示詞進行自訂。例如要自訂 llama3.2 模型：

ollama pull llama3.2

建立一個 Modelfile：

FROM llama3.2

# 設定溫度為 1 [數值越高越有創意，越低越一致]
PARAMETER temperature 1

# 設定系統訊息
SYSTEM """
你是超級瑪利歐兄弟的瑪利歐。僅以瑪利歐（助理）身份回答。
"""

然後建立並運行模型：

ollama create mario -f ./Modelfile
ollama run mario
>>> hi
Hello! It's your friend Mario.

更多 Modelfile 操作資訊，請參閱 Modelfile 文件。

CLI 參考

建立模型

ollama create 用於根據 Modelfile 建立模型。

ollama create mymodel -f ./Modelfile

拉取模型

ollama pull llama3.2

此指令亦可用於更新本地模型，只會拉取差異部分。

移除模型

ollama rm llama3.2

複製模型

ollama cp llama3.2 my-model

多行輸入

多行輸入可用 """ 包住文字：

>>> """Hello,
... world!
... """
I'm a basic program that prints the famous "Hello, world!" message to the console.

多模態模型

ollama run llava "What's in this image? /Users/jmorgan/Desktop/smile.png"

輸出：這張圖像以黃色笑臉為主題，可能是畫面的焦點。

以參數傳遞提示詞

ollama run llama3.2 "Summarize this file: $(cat README.md)"

輸出：Ollama 是一個輕量、可擴展的本地語言模型框架，提供簡單 API 以便建立、運行與管理模型，並有豐富的內建模型庫可於多種應用中輕易使用。

顯示模型資訊

ollama show llama3.2

列出本機所有模型

ollama list

列出當前載入的模型

ollama ps

停止當前運行的模型

ollama stop llama3.2

啟動 Ollama

ollama serve 用於在不啟動桌面應用時啟動 ollama 服務。

編譯建置

詳見 開發人員指南

運行本地編譯版本

啟動伺服器：

./ollama serve

另開一個終端視窗，運行模型：

./ollama run llama3.2

REST API

Ollama 提供 REST API 以運行及管理模型。

產生回應

curl http://localhost:11434/api/generate -d '{
  "model": "llama3.2",
  "prompt":"Why is the sky blue?"
}'

與模型對話

curl http://localhost:11434/api/chat -d '{
  "model": "llama3.2",
  "messages": [
    { "role": "user", "content": "why is the sky blue?" }
  ]
}'

所有端點請參閱 API 文件。

  • Lunary 領先的開源 LLM 可觀測平台，提供即時分析、提示模板管理、PII 遮蔽、代理追蹤等企業級功能。
  • OpenLIT 是 OpenTelemetry 原生工具，用於 Ollama 應用與 GPU 監控（追蹤與度量）。
  • HoneyHive 是 AI 代理的可觀測/評估平台，用於評估代理效能、追蹤錯誤、監控生產品質。
  • Langfuse 開源 LLM 可觀測性平台，協助團隊協作監控、評估、除錯 AI 應用。
  • MLflow Tracing 開源 LLM 可觀測工具，API 便於記錄與視覺化追蹤，助於 GenAI 應用除錯與評估。
