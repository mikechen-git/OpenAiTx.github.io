Kursor untuk Desainer – Editor Kode Visual-First Open-Source

Buat situs web, prototipe, dan desain dengan AI di Next.js + TailwindCSS. Lakukan edit langsung di DOM browser dengan editor visual. Desain secara realtime menggunakan kode. Alternatif open-source untuk Bolt.new, Lovable, V0, Replit Agent, Figma Make, Webflow, dll.

Onlook untuk Desktop (alias Onlook Alpha)

Kami sedang dalam tahap awal pratinjau untuk Onlook Web. Jika Anda mencari aplikasi electron desktop yang dapat diunduh, kini telah dipindahkan ke Onlook Desktop.

Mengapa kami beralih ke web? Baca tentang keputusan kami: Migrasi dari Electron ke Web

Memulai

Segera tersedia dengan aplikasi hosting atau menjalankan secara lokal.

Penggunaan

Onlook dapat berjalan pada proyek Next.js + TailwindCSS mana pun, impor proyek Anda ke dalam Onlook atau mulai dari awal di editor.

Gunakan chat AI untuk membuat atau mengedit proyek yang sedang Anda kerjakan. Kapan saja, Anda selalu dapat klik kanan pada elemen untuk membuka lokasi persis elemen tersebut di kode.


Gambar div baru dan atur ulang di dalam kontainer induknya dengan drag-and-drop.


Pratinjau kode berdampingan dengan desain situs Anda.


Gunakan toolbar editor Onlook untuk menyesuaikan gaya Tailwind, memanipulasi objek secara langsung, dan bereksperimen dengan tata letak.

Cara Kerja

  1. Saat Anda membuat aplikasi, kami memuat kode ke dalam web container
  2. Container menjalankan dan melayani kode tersebut
  3. Editor kami menerima tautan pratinjau dan menampilkannya dalam iFrame
  4. Editor kami membaca dan mengindeks kode dari container
  5. Kami menginstrumentasi kode agar dapat memetakan elemen ke tempatnya di kode
  6. Saat elemen diedit, kami mengedit elemen di iFrame kami, lalu di kode
  7. Chat AI kami juga memiliki akses kode dan alat untuk memahami serta mengedit kode

Arsitektur ini secara teori dapat diskalakan untuk bahasa atau framework apa pun yang menampilkan elemen DOM secara deklaratif (misal: jsx/tsx/html). Saat ini kami fokus untuk membuatnya bekerja dengan baik di Next.js dan TailwindCSS.
