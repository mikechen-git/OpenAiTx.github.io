Onlook for Desktop (หรือ Onlook Alpha)

Onlook Web ยังอยู่ในช่วงพรีวิว หากคุณกำลังมองหาแอป electron เวอร์ชันดาวน์โหลด ย้ายไปที่ Onlook Desktop

ทำไมเราย้ายไปเว็บ? อ่านเหตุผลของเราได้ที่: Migrating from Electron to Web

เริ่มต้นใช้งาน

จะพร้อมใช้งานเร็วๆ นี้ทั้งแบบ แอปบนโฮสต์ หรือ รันในเครื่อง

วิธีใช้งาน

Onlook จะทำงานกับโปรเจกต์ Next.js + TailwindCSS ใดก็ได้ นำเข้าโปรเจกต์ของคุณ เข้าสู่ Onlook หรือเริ่มจากศูนย์ภายในตัวแก้ไข

ใช้ AI chat เพื่อสร้างหรือแก้ไขโปรเจกต์ที่คุณกำลังทำงานอยู่ ทุกเมื่อที่ต้องการ คุณสามารถคลิกขวาที่อีลีเมนต์เพื่อเปิดตำแหน่งของอีลีเมนต์ในโค้ดได้ทันที


วาด div ใหม่และจัดเรียงใหม่ภายใน parent container ได้ด้วยการลากและวาง


ดูตัวอย่างโค้ดแบบเคียงข้างกับดีไซน์ของเว็บไซต์คุณ


ใช้แถบเครื่องมือของ Onlook เพื่อปรับแต่งสไตล์ Tailwind จัดการออบเจ็กต์โดยตรง และทดลองกับเลย์เอาต์ต่างๆ

เอกสารประกอบ

สำหรับเอกสารฉบับเต็ม เยี่ยมชม docs.onlook.com

ดูวิธีการ Contribute ได้ที่ Contributing to Onlook ในเอกสารของเรา

วิธีการทำงาน

  1. เมื่อคุณสร้างแอป เราจะโหลดโค้ดเข้าไปใน web container
  2. คอนเทนเนอร์จะรันและให้บริการโค้ด
  3. ตัวแก้ไขของเราจะรับลิงก์พรีวิวและแสดงใน iFrame
  4. ตัวแก้ไขของเราจะอ่านและจัดทำดัชนีโค้ดจากคอนเทนเนอร์
  5. เราติดตั้งเครื่องมือในโค้ดเพื่อจับคู่องค์ประกอบกับตำแหน่งในโค้ด
  6. เมื่อองค์ประกอบถูกแก้ไข เราจะแก้ไขใน iFrame แล้วจึงแก้ไขในโค้ด
  7. AI chat ของเราก็เข้าถึงโค้ดและมีเครื่องมือในการเข้าใจและแก้ไขโค้ด

สถาปัตยกรรมนี้ในทางทฤษฎีสามารถขยายไปยังภาษา หรือเฟรมเวิร์กใดๆ ที่ แสดง DOM แบบ declarative (เช่น jsx/tsx/html) เรามุ่งเน้นให้ทำงานได้ดี กับ Next.js และ TailwindCSS ในตอนนี้

สำหรับการแนะนำแบบละเอียด ดูได้ที่ Architecture Docs

เทคโนโลยีที่เราใช้

ด้านหน้า (Front-end)

  • Next.js - Full stack
  • TailwindCSS - สไตล์
  • tRPC - Server interface

ฐานข้อมูล (Database)

  • Supabase - Auth, Database, Storage
  • Drizzle - ORM

AI

  • AI SDK - LLM client
  • Anthropic - ผู้ให้บริการ LLM model
  • Morph Fast Apply - ผู้ให้บริการ fast apply model
  • Relace - ผู้ให้บริการ fast apply model

Sandbox และโฮสติ้ง

  • CodeSandboxSDK - Dev sandbox
  • Freestyle - โฮสติ้ง

Runtime

  • Bun - Monorepo, runtime, bundler
  • Docker - การจัดการคอนเทนเนอร์
