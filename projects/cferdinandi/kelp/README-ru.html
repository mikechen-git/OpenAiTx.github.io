Kelp

Библиотека пользовательского интерфейса для тех, кто любит HTML, основанная на современных CSS и веб-компонентах.

Читать документацию →

Примечание: Kelp находится на этапе альфа-разработки. Вы можете использовать его, экспериментировать и сообщать об ошибках, но имейте в виду, что со временем многое может измениться.

Быстрый старт

Kelp работает без необходимости сборки.

CDN — самый быстрый и простой способ начать работу, но вы также можете скачать файлы с GitHub, если предпочитаете.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/kelpui@0/css/kelp.css">

Kelp использует семантическое версионирование. Вы можете выбрать основную, минорную или патч-версию из CDN с помощью синтаксиса @1.2.3. Все доступные версии можно найти в разделе релизов.

Демонстрация

Включённый файл index.html — это демонстрация Kelp по принципу "всё и сразу". В нём собраны все функции и компоненты в одном большом файле.

Пока Kelp находится в бета-версии, этапа компиляции нет. Веб-компоненты используют ES-импорты и требуют локального сервера для запуска.

Используйте любой предпочитаемый сервер или воспользуйтесь встроенным http-server, запустив команду npm run dev.

npm install
npm run dev

По мере приближения Kelp к версии v1 будет добавлен этап компиляции, чтобы это стало ненужным.

Тесты

Kelp использует...

  • Playwright для тестирования
  • Biome для линтинга и форматирования
  • Процесс непрерывной интеграции при развертывании и pull-запросах
# Запустить тесты
npm run test

# Запустить линтер
npm run lint

Лицензия

Бесплатно для использования по лицензии Kelp Commons. Также доступны коммерческие варианты лицензии.
