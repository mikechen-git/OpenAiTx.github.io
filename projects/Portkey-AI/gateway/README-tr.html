Hızlı Başlangıç (2 dakika)

1. AI Gateway'inizi Kurun

# Gateway'i yerelde çalıştırın (Node.js ve npm gerektirir)
npx @portkey-ai/gateway

Gateway şu adreste çalışıyor: http://localhost:8787/v1

Gateway Konsolu şu adreste çalışıyor: http://localhost:8787/public/

Dağıtım rehberleri:   Portkey Cloud (Önerilir)   Docker   Node.js   Cloudflare   Replit   Diğerleri...

2. İlk isteğinizi yapın

# pip install -qU portkey-ai

from portkey_ai import Portkey

# OpenAI uyumlu istemci
client = Portkey(
    provider="openai", # veya 'anthropic', 'bedrock', 'groq', vb.
    Authorization="sk-***" # sağlayıcı API anahtarı
)

# AI Gateway'iniz üzerinden bir istek gönderin
client.chat.completions.create(
    messages=[{"role": "user", "content": "Hava nasıl?"}],
    model="gpt-4o-mini"
)

Desteklenen Kütüphaneler:   JS   Python   REST   OpenAI SDK'ları   Langchain   LlamaIndex   Autogen   CrewAI   Daha fazla..

Gateway Konsolu'nda (http://localhost:8787/public/) tüm yerel loglarınızı tek bir yerde görebilirsiniz.

3. Yönlendirme & Guardrails

LLM gateway'deki Configler, yönlendirme kuralları oluşturmanıza, güvenilirlik eklemenize ve guardrails ayarlamanıza olanak tanır.

config = {
  "retry": {"attempts": 5},

  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}

# Konfigürasyonu istemciye ekleyin
client = client.with_options(config=config)

client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Rastgele Apple veya Bat yanıtla"}]
)

# Bu, guardrail "Apple" içeren tüm yanıtları reddettiği için daima "Bat" olarak yanıt döner. Retry konfigürasyonu ise 5 kez tekrar dener ve sonra vazgeçer.

Raporu Edinin

Temel Özellikler

Güvenilir Yönlendirme

  • Yedek Sağlayıcılar (Fallbacks): LLM gateway kullanarak başarısız isteklerde başka bir sağlayıcıya veya modele otomatik geçiş yapın. Yedekleme tetiklenecek hata türlerini belirleyebilirsiniz. Uygulamanızın güvenilirliğini artırır.
  • Otomatik Yeniden Denemeler: Başarısız istekleri otomatik olarak 5 defaya kadar yeniden deneyin. Üstel bekleme stratejisi, ağ aşırı yüklenmesini önlemek için deneme aralıklarını ayarlar.
  • Yük Dengeleme: Yüksek erişilebilirlik ve en iyi performans için LLM isteklerini birden fazla API anahtarı veya AI sağlayıcısına ağırlıklarla dağıtın.
  • İstek Zaman Aşımı: Ayrıntılı istek zaman aşımları belirleyerek düzensiz LLM’leri ve gecikmeleri yönetin, belirlenen süreyi aşan isteklerin otomatik olarak sonlandırılmasını sağlayın.
  • Çok Modlu LLM Gateway: Birden fazla sağlayıcıdan görsel, ses (metinden konuşmaya & konuşmadan metne) ve görüntü üretim modellerini, tanıdık OpenAI imzası ile çağırın.
  • Gerçek Zamanlı API’ler: OpenAI tarafından sunulan gerçek zamanlı API’leri entegre websocket sunucusu üzerinden çağırın.

Güvenlik & Doğruluk

  • Korumalar (Guardrails): LLM giriş ve çıkışlarınızı belirlediğiniz kontrollerle doğrulayın. Güvenlik ve doğruluk standartlarına uyum sağlamak için 40’tan fazla önceden tanımlı koruma seçebilirsiniz. Kendi korumalarınızı getirebilir ya da birçok partnerimizden seçim yapabilirsiniz.
  • Güvenli Anahtar Yönetimi: Kendi anahtarlarınızı kullanın veya anında sanal anahtarlar oluşturun.
  • Rol Tabanlı Erişim Kontrolü: Kullanıcılarınız, çalışma alanlarınız ve API anahtarlarınız için ayrıntılı erişim kontrolü.
  • Uyumluluk & Veri Gizliliği: AI gateway, SOC2, HIPAA, GDPR ve CCPA uyumludur.

Maliyet Yönetimi

  • Akıllı önbellekleme: LLM’lerden gelen yanıtları önbelleğe alarak maliyetleri azaltın ve gecikmeyi iyileştirin. Basit ve anlamsal* önbellekleme desteklenir.
  • Kullanım analizleri: Yapay zeka ve LLM kullanımınızı izleyin ve analiz edin; istek hacmi, gecikme, maliyetler ve hata oranları dahil.
  • Sağlayıcı optimizasyonu*: Kullanım eğilimleri ve fiyatlandırma modellerine göre otomatik olarak en uygun maliyetli sağlayıcıya geçiş yapar.

İşbirliği & İş Akışları

  • Agent Desteği: Popüler agent framework’leri ile sorunsuz entegre olarak karmaşık AI uygulamaları oluşturun. Gateway, Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow ve hatta Özel Agentlar ile entegre çalışır.
  • Prompt Şablon Yönetimi*: Prompt şablonlarınızı evrensel bir prompt playground üzerinden işbirliğiyle oluşturun, yönetin ve versiyonlayın.

* Sadece barındırılan ve kurumsal sürümlerde mevcuttur
