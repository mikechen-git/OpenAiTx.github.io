AI Gateway

通过一个快速且友好的 API 路由到 250+ 大语言模型

文档 | 企业版 | 托管网关 | 更新日志 | API 参考


AI Gateway 旨在实现对 1600+ 语言、视觉、音频和图像模型的快速、可靠与安全路由。它是一款轻量级、开源、企业级就绪的解决方案，可让你在 2 分钟内集成任何语言模型。

  • [x] 极致快速（<1ms 延迟），体积极小（122kb）
  • [x] 生产验证，每天处理超过 100 亿 token
  • [x] 企业级就绪，提升安全、可扩展性和支持自定义部署

使用 AI Gateway 能做什么？

  • 2 分钟内集成任何大语言模型 - 快速开始
  • 通过 自动重试 和 故障转移 防止停机
  • 使用 负载均衡 和 条件路由 扩展 AI 应用
  • 利用 防护栏 保护你的 AI 部署
  • 借助 多模态能力 实现文本之外的更多功能
  • 深入探索 智能体工作流 集成

快速开始（2 分钟完成）

1. 安装并启动你的 AI Gateway

# 本地运行网关（需 Node.js 和 npm 支持）
npx @portkey-ai/gateway

网关运行在 http://localhost:8787/v1

网关控制台运行在 http://localhost:8787/public/

部署指南：   Portkey Cloud（推荐）   Docker   Node.js   Cloudflare   Replit   其他...

2. 发起你的第一个请求

# pip install -qU portkey-ai

from portkey_ai import Portkey

# OpenAI 兼容客户端
client = Portkey(
    provider="openai", # 可选 'anthropic'、'bedrock'、'groq' 等
    Authorization="sk-***" # 你的模型 API 密钥
)

# 通过 AI Gateway 发起请求
client.chat.completions.create(
    messages=[{"role": "user", "content": "What's the weather like?"}],
    model="gpt-4o-mini"
)

支持的库：   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   更多..

在 Gateway 控制台（http://localhost:8787/public/）中，你可以查看所有本地日志。

3. 路由与防护栏

LLM 网关中的 Configs 可让你创建路由规则，增强可靠性并设置防护栏。

config = {
  "retry": {"attempts": 5},

  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}

# 将 config 附加到客户端
client = client.with_options(config=config)

client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Reply randomly with Apple or Bat"}]
)

# 所有包含 "Apple" 的回复都会被防护栏拒绝，因此总是返回 "Bat"。重试配置会在放弃前最多重试 5 次。

核心特性

可靠路由

  • 故障转移：当请求失败时，使用 LLM 网关自动切换到另一个提供商或模型。您可以指定在哪些错误情况下触发故障转移，提高应用的可靠性。
  • 自动重试：自动重试失败的请求，最多可达 5 次。采用指数退避策略，间隔重试以防止网络过载。
  • 负载均衡：通过加权分配，将 LLM 请求分散到多个 API 密钥或 AI 提供商上，确保高可用性和最佳性能。
  • 请求超时：通过设置精细的请求超时时间，管理不稳定的 LLM 和延迟，自动终止超时请求。
  • 多模态 LLM 网关：以熟悉的 OpenAI 接口调用来自多个供应商的视觉、音频（文本转语音 & 语音转文本）及图像生成模型。
  • 实时 API：通过集成 websockets 服务器，调用 OpenAI 推出的实时 API。

安全与准确性

  • 安全防护措施：验证您的 LLM 输入输出，确保符合您指定的检测标准。从 40 多种预构建防护措施中选择，以确保符合安全与准确性标准。您可以自定义防护措施或选择我们的众多合作伙伴。
  • 安全密钥管理：使用您自己的密钥，或即时生成虚拟密钥。
  • 基于角色的访问控制：为您的用户、工作空间和 API 密钥提供精细化访问控制。
  • 合规性与数据隐私：AI 网关符合 SOC2、HIPAA、GDPR 和 CCPA 标准。

成本管理

  • 智能缓存：缓存 LLM 的响应以降低成本并提升延迟表现。支持简单与语义*缓存。
  • 使用分析：监控和分析您的 AI 与 LLM 用量，包括请求量、延迟、成本和错误率。
  • 供应商优化*：根据使用模式和价格模型，自动切换到性价比最高的供应商。

协作与工作流

  • 代理支持：无缝集成流行的代理框架，构建复杂的 AI 应用。网关可与 Autogen、CrewAI、LangChain、LlamaIndex、Phidata、Control Flow 及 自定义代理 深度集成。
  • 提示模板管理*：通过通用提示广场，协作创建、管理和版本化您的提示模板。

* 功能在托管版和企业版中提供
