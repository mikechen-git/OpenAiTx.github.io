Wrappers

Wrappers — это фреймворк для разработки Postgres Foreign Data Wrappers (FDW), написанный на Rust. Его цель — упростить разработку Postgres FDW, сохраняя современные возможности языка Rust, такие как высокая производительность, строгая типизация и безопасность.

Wrappers также представляет собой коллекцию FDW, созданных Supabase. В настоящее время поддерживаются следующие FDW, и ведётся разработка новых:

| FDW | Описание | Чтение | Изменение | | ------------------------------------------------ | ---------------------------------------------------------------------------- | ------ | --------- | | HelloWorld | Демонстрационный FDW, показывающий, как создать базовый FDW. | | | | BigQuery | FDW для Google BigQuery | ✅ | ✅ | | Clickhouse | FDW для ClickHouse | ✅ | ✅ | | Stripe | FDW для API Stripe | ✅ | ✅ | | Firebase | FDW для Google Firebase | ✅ | ❌ | | Airtable | FDW для API Airtable | ✅ | ❌ | | S3 | FDW для AWS S3 | ✅ | ❌ | | Logflare | FDW для Logflare | ✅ | ❌ | | Auth0 | FDW для Auth0 | ✅ | ❌ | | SQL Server | FDW для Microsoft SQL Server | ✅ | ❌ | | Redis | FDW для Redis | ✅ | ❌ | | AWS Cognito | FDW для AWS Cognito | ✅ | ❌ | | Notion | Wasm FDW для Notion | ✅ | ❌ | | Snowflake | Wasm FDW для Snowflake | ✅ | ✅ | | Paddle | Wasm FDW для Paddle | ✅ | ✅ | | Calendly | Wasm FDW для Calendly | ✅ | ❌ | | Cal.com | Wasm FDW для Cal.com | ✅ | ❌ | | Clerk | Wasm FDW для Clerk | ✅ | ❌ | | Cloudflare D1 | Wasm FDW для Cloudflare D1 | ✅ | ✅ | | Orb | Wasm FDW для Orb | ✅ | ❌ | | HubSpot | Wasm FDW для HubSpot | ✅ | ❌ | | Slack | Wasm FDW для Slack | ✅ | ❌ | | Apache Iceberg| FDW для Apache Iceberg | ✅ | ❌ |

Внимание

Восстановление логического бэкапа базы данных с материализованным представлением, использующим внешнюю таблицу, может завершиться неудачей. По этой причине либо не используйте внешние таблицы в материализованных представлениях, либо используйте их только в базах данных с включёнными физическими бэкапами.

Возможности

  • Минимальный интерфейс и простота реализации.
  • Поддержка богатых типов данных.
  • Поддержка синхронных и асинхронных источников данных, таких как СУБД, RESTful API, плоские файлы и др.
  • Построен на основе pgrx, предоставляя более высокоуровневый интерфейс без сокрытия низкоуровневых C API.
  • Поддерживается pushdown для WHERE, ORDER BY, LIMIT.

Установка

Wrappers — это расширение pgrx, вы можете следовать инструкциям по установке pgrx для установки Wrappers.

В основном, выполните следующую команду для установки FDW после установки pgrx. Например,

cargo pgrx install --pg-config [path_to_pg_config] --features stripe_fdw
