Wrappers

Wrappers คือเฟรมเวิร์กสำหรับการพัฒนา Postgres Foreign Data Wrappers (FDW) ที่เขียนด้วยภาษา Rust โดยมีเป้าหมายเพื่อให้งานพัฒนา Postgres FDW ง่ายขึ้น ในขณะที่ยังคงความสามารถสมัยใหม่ของภาษา Rust เช่น ประสิทธิภาพสูง ระบบประเภทข้อมูลที่แข็งแกร่ง และความปลอดภัย

Wrappers ยังเป็นชุดรวมของ FDW ที่สร้างโดย Supabase ขณะนี้เรารองรับ FDW ต่อไปนี้ และกำลังพัฒนาเพิ่มเติม:

คำเตือน

การกู้คืนข้อมูลฐานข้อมูลแบบ logical backup ที่มี materialized view โดยใช้ foreign table อาจล้มเหลว ด้วยเหตุผลนี้ ไม่ควรใช้ foreign table ใน materialized view หรือควรใช้เฉพาะในฐานข้อมูลที่เปิดใช้งานการสำรองข้อมูลแบบ physical เท่านั้น

คุณสมบัติ

  • อินเทอร์เฟซน้อยและง่ายต่อการนำไปใช้
  • รองรับชนิดข้อมูลที่หลากหลาย
  • รองรับทั้ง backend แบบ synchronous และ asynchronous เช่น RDBMS, RESTful API, ไฟล์ flat file ฯลฯ
  • สร้างบน pgrx มอบอินเทอร์เฟซระดับสูงโดยไม่ซ่อน API ระดับล่างของภาษา C
  • รองรับ WHERE, ORDER BY, LIMIT pushdown

เอกสารประกอบ

  • คู่มือการใช้งาน
  • คู่มือสำหรับนักพัฒนา (docs.rs)

การติดตั้ง

Wrappers เป็นส่วนขยายของ pgrx คุณสามารถทำตาม ขั้นตอนการติดตั้ง pgrx เพื่อทำการติดตั้ง Wrappers

โดยพื้นฐาน ให้รันคำสั่งด้านล่างเพื่อทำการติดตั้ง FDW หลังติดตั้ง pgrx ตัวอย่างเช่น

cargo pgrx install --pg-config [path_to_pg_config] --features stripe_fdw
