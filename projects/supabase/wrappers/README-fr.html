Wrappers

Wrappers est un framework de développement pour les Foreign Data Wrappers (FDW) de Postgres (FDW), écrit en Rust. Son objectif est de faciliter le développement de FDW pour Postgres tout en conservant les capacités modernes du langage Rust, telles que la haute performance, la robustesse des types et la sécurité.

Wrappers est également une collection de FDW développés par Supabase. Nous prenons actuellement en charge les FDW suivants, d'autres étant en cours de développement :

Avertissement

La restauration d’une sauvegarde logique d’une base de données contenant une vue matérialisée utilisant une table étrangère peut échouer. Pour cette raison, n’utilisez pas de tables étrangères dans les vues matérialisées ou utilisez-les uniquement dans des bases de données pour lesquelles les sauvegardes physiques sont activées.

Fonctionnalités

  • Interface minimale et facile à implémenter.
  • Prise en charge de types de données riches.
  • Prise en charge des backends synchrones et asynchrones, tels que les SGBDR, APIs RESTful, fichiers plats, etc.
  • Construit sur pgrx, offrant des interfaces de haut niveau sans masquer les API C de bas niveau.
  • WHERE, ORDER BY, LIMIT sont pris en charge en pushdown.

Installation

Wrappers est une extension pgrx. Vous pouvez suivre les étapes d’installation de pgrx pour installer Wrappers.

En résumé, exécutez la commande suivante pour installer un FDW après avoir installé pgrx. Par exemple :

cargo pgrx install --pg-config [path_to_pg_config] --features stripe_fdw
