سوپ‌بیس

سوپ‌بیس یک جایگزین متن‌باز برای Firebase است. ما در حال ساخت ویژگی‌های Firebase با استفاده از ابزارهای متن‌باز در سطح سازمانی هستیم.

  • [x] پایگاه داده میزبانی‌شده Postgres. مستندات
  • [x] احراز هویت و مجوزدهی. مستندات
  • [x] APIهای خودکار تولید شده.
    • [x] REST. مستندات
    • [x] GraphQL. مستندات
    • [x] اشتراک‌های بلادرنگ. مستندات
  • [x] توابع.
    • [x] توابع پایگاه داده. مستندات
    • [x] توابع Edge مستندات
  • [x] ذخیره‌سازی فایل. مستندات
  • [x] جعبه‌ابزار هوش مصنوعی + وکتور/جاسازی. مستندات
  • [x] داشبورد

برای دریافت اعلان به‌روزرسانی‌های مهم، "releases" این مخزن را دنبال کنید.

مستندات

برای مشاهده مستندات کامل، به supabase.com/docs مراجعه کنید.

برای مشاهده نحوه مشارکت، به شروع کار مراجعه کنید.

جامعه و پشتیبانی

  • فروم جامعه. بهترین برای: کمک در توسعه، گفتگو درباره بهترین شیوه‌های پایگاه داده.
  • مسائل گیت‌هاب. بهترین برای: خطاها و باگ‌هایی که هنگام استفاده از سوپ‌بیس با آن مواجه می‌شوید.
  • پشتیبانی ایمیلی. بهترین برای: مشکلات مربوط به پایگاه داده یا زیرساخت شما.
  • دیسکورد. بهترین برای: اشتراک‌گذاری برنامه‌های خود و تعامل با جامعه.

چگونه کار می‌کند

سوپ‌بیس ترکیبی از ابزارهای متن‌باز است. ما ویژگی‌های Firebase را با محصولات متن‌باز در سطح سازمانی می‌سازیم. اگر ابزارها و جوامع مربوطه با مجوز MIT، Apache 2 یا مشابه وجود داشته باشند، آن ابزار را استفاده و پشتیبانی می‌کنیم. اگر ابزار وجود نداشته باشد، خودمان آن را می‌سازیم و متن‌باز می‌کنیم. سوپ‌بیس معادل یک‌به‌یک Firebase نیست. هدف ما ارائه تجربه توسعه‌ای مشابه Firebase با استفاده از ابزارهای متن‌باز است.

معماری

سوپ‌بیس یک پلتفرم میزبانی‌شده است. شما می‌توانید ثبت‌نام کنید و بدون نیاز به نصب چیزی، از سوپ‌بیس استفاده کنید. همچنین می‌توانید خود میزبانی کنید و به‌صورت محلی توسعه دهید.

  • Postgres یک سیستم پایگاه داده شی‌ء-رابطه‌ای است با بیش از ۳۰ سال توسعه فعال که اعتبار زیادی در پایداری، غنای ویژگی‌ها و عملکرد کسب کرده است.
  • Realtime یک سرور Elixir است که به شما امکان می‌دهد با استفاده از websockets به درج، به‌روزرسانی و حذف‌های PostgreSQL گوش دهید. Realtime از قابلیت replication داخلی Postgres برای پیگیری تغییرات پایگاه داده استفاده می‌کند، تغییرات را به JSON تبدیل کرده و آن را از طریق websockets به مشتریان مجاز ارسال می‌کند.
  • PostgREST یک سرور وب است که پایگاه داده PostgreSQL شما را مستقیماً به یک API RESTful تبدیل می‌کند.
  • GoTrue یک API احراز هویت مبتنی بر JWT است که ثبت‌نام، ورود و مدیریت نشست کاربران در برنامه‌های شما را ساده می‌کند.
  • Storage یک API RESTful برای مدیریت فایل‌ها در S3 است که مجوزها را Postgres مدیریت می‌کند.
  • pg_graphql یک افزونه PostgreSQL که یک API GraphQL ارائه می‌دهد.
  • postgres-meta یک API RESTful برای مدیریت Postgres است که امکان دریافت جداول، افزودن نقش‌ها، اجرای کوئری و غیره را فراهم می‌کند.
  • Kong یک دروازه API بومی ابری است.
