Hoe het werkt

Supabase is een combinatie van open source tools. Wij bouwen de functionaliteiten van Firebase met open source producten van ondernemingsniveau. Als de tools en communities bestaan met een MIT-, Apache 2- of een gelijkwaardige open licentie, gebruiken en ondersteunen we die tool. Als de tool niet bestaat, bouwen en open-sourcen we deze zelf. Supabase is geen 1-op-1 kopie van Firebase. Ons doel is ontwikkelaars een Firebase-achtige ervaring te bieden met open source tools.

Architectuur

Supabase is een gehost platform. Je kunt je aanmelden en Supabase direct gebruiken zonder iets te installeren. Je kunt ook zelf hosten en lokaal ontwikkelen.

  • Postgres is een object-relationeel databasesysteem met meer dan 30 jaar actieve ontwikkeling, bekend om zijn betrouwbaarheid, uitgebreide functionaliteit en prestaties.
  • Realtime is een Elixir-server waarmee je kunt luisteren naar PostgreSQL-inserts, -updates en -deletes via websockets. Realtime gebruikt de ingebouwde replicatiefunctie van Postgres om databasewijzigingen te volgen, converteert deze naar JSON en verstuurt de JSON via websockets naar geautoriseerde clients.
  • PostgREST is een webserver die je PostgreSQL-database direct omzet in een RESTful API.
  • GoTrue is een op JWT gebaseerde authenticatie-API die het aanmelden, inloggen en sessiebeheer in je applicaties vereenvoudigt.
  • Storage is een RESTful API voor het beheren van bestanden in S3, waarbij Postgres de permissies beheert.
  • pg_graphql is een PostgreSQL-extensie die een GraphQL API biedt.
  • postgres-meta is een RESTful API voor het beheren van je Postgres, waarmee je tabellen kunt ophalen, rollen toevoegen, queries uitvoeren, enz.
  • Kong is een cloud-native API-gateway.
