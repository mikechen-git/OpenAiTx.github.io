كيف يعمل

سوبابيز عبارة عن مجموعة من الأدوات مفتوحة المصدر. نقوم ببناء ميزات Firebase باستخدام منتجات مفتوحة المصدر على مستوى المؤسسات. إذا كانت الأدوات والمجتمعات موجودة بترخيص مفتوح مثل MIT أو Apache 2 أو ما يعادلهما، فسوف نستخدم وندعم تلك الأداة. إذا لم تكن الأداة موجودة، نقوم ببنائها وفتح مصدرها بأنفسنا. سوبابيز ليس مطابقًا تمامًا لـ Firebase. هدفنا هو توفير تجربة تطوير مماثلة لـ Firebase باستخدام الأدوات مفتوحة المصدر.

العمارة

سوبابيز عبارة عن منصة مستضافة. يمكنك التسجيل والبدء في استخدام سوبابيز دون تثبيت أي شيء. يمكنك أيضًا استضافتها ذاتيًا والتطوير محليًا.

  • Postgres هو نظام قاعدة بيانات علائقية كائنية مع أكثر من 30 عامًا من التطوير النشط مما أكسبه سمعة قوية من حيث الاعتمادية وغنى الميزات والأداء.
  • Realtime هو خادم Elixir يسمح لك بالاستماع إلى الإدخالات والتحديثات والحذف في PostgreSQL باستخدام Websockets. يقوم Realtime باستطلاع وظيفة النسخ المدمجة في Postgres للتغييرات في قاعدة البيانات، ويحوّل التغييرات إلى JSON، ثم يبث JSON عبر Websockets للعملاء المصرح لهم.
  • PostgREST هو خادم ويب يحوّل قاعدة بيانات PostgreSQL مباشرة إلى واجهة RESTful API.
  • GoTrue هو واجهة برمجة تطبيقات للمصادقة تعتمد على JWT تسهل عمليات التسجيل وتسجيل الدخول وإدارة الجلسات في تطبيقاتك.
  • Storage واجهة RESTful API لإدارة الملفات في S3، مع إدارة الأذونات بواسطة Postgres.
  • pg_graphql إضافة لـ PostgreSQL توفر واجهة GraphQL API.
  • postgres-meta هو RESTful API لإدارة Postgres الخاص بك، يتيح لك جلب الجداول، إضافة الأدوار، وتشغيل الاستعلامات، إلخ.
  • Kong هو بوابة API سحابية.
