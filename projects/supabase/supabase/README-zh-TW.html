運作原理

Supabase 是由多個開源工具組成。我們正在使用企業級、開源產品打造 Firebase 的各項功能。如果有現成的工具與社群，且採用 MIT、Apache 2 或同等開源授權，我們會直接使用並支援該工具。若沒有現成工具，我們會自行開發並開源。Supabase 並非 Firebase 的一對一對應，我們的目標是讓開發者能夠以開源工具體驗類似 Firebase 的開發者體驗。

架構說明

Supabase 是一個 託管平台。您可以直接註冊並開始使用 Supabase，無需安裝任何軟體。 您也可以選擇 自我託管 或 本地開發。

  • Postgres 是一個具有 30 多年活躍開發歷史的物件關聯式資料庫系統，以其穩定性、功能強大和高效能著稱。
  • Realtime 是一個 Elixir 伺服器，可透過 websockets 監聽 PostgreSQL 的新增、更新和刪除。Realtime 輪詢 Postgres 內建的複製功能以偵測資料庫變更，將變更轉換為 JSON，然後透過 websockets 廣播給授權用戶端。
  • PostgREST 是一個網頁伺服器，能將您的 PostgreSQL 資料庫直接轉換為 RESTful API。
  • GoTrue 是一個基於 JWT 的認證 API，可簡化應用中的用戶註冊、登入和會話管理。
  • Storage 是一個 RESTful API，用於管理 S3 檔案，並由 Postgres 負責權限管理。
  • pg_graphql 是一個 PostgreSQL 擴充套件，可提供 GraphQL API。
  • postgres-meta 是一個 RESTful API，用於管理您的 Postgres，支援查詢資料表、新增角色、執行查詢等功能。
  • Kong 是一個雲原生 API Gateway。
