Formatação

O conteúdo das notas é formatado de acordo com o Guia de Tipografia para Chinês para garantir a legibilidade do conteúdo.

Não se utiliza ![]() para referenciar imagens, mas sim a tag <img>. Por um lado, isso permite controlar o tamanho exibido da imagem, por outro lado, porque o GFM não suporta <center> ![]() </center> para centralizar imagens, só é possível centralizar usando <div align="center"> <img src=""/> </div>.
