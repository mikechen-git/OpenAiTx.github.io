ASP.NET SignalR

重要：本產品不會再新增新功能。我們目前處於維護模式，只修復關鍵性問題。詳情請參閱我們的部落格文章。 本儲存庫主機承載 ASP.NET SignalR 的程式碼與專案管理，適用於使用 System.Web 或 Katana 的 .NET Framework 應用程式。如果您正在尋找 ASP.NET Core SignalR 的相關資訊，請參閱 https://github.com/aspnet/AspNetCore/tree/main/src/SignalR 儲存庫。

ASP.NET SignalR 是一個為 ASP.NET 開發人員設計的函式庫，可讓您非常輕鬆地將即時網頁功能加入您的應用程式。什麼是「即時網頁」功能？這是指您的伺服器端程式碼能夠在事件發生時，立即將內容推送到已連線的用戶端。

可以用來做什麼？

從伺服器推送資料到用戶端（不僅僅是瀏覽器用戶端）一直是一個困難的問題。SignalR 讓這一切變得非常簡單，並且幫您處理所有繁瑣的細節。
