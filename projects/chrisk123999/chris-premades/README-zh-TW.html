Readme

一套自動化項目集合，包括法術、職業特性、怪物特性等，並提供支援其運作的機制，以及基於高自動化 Midi-QOL 的 D&D5e 環境下的生活品質擴充功能。本模組內含的彙編資料並不包含物品描述。本模組依賴數個不同作者的模組，請勿就本模組相關的錯誤或問題騷擾 tposney、Wasp 或其他模組作者。錯誤回報與大型模組需求可於 GitHub 提出。如需更快速的支援與功能建議，歡迎加入Discord 伺服器。

主要功能:

  • 自動化法術、物品、職業特性等彙編資料。
  • 標題列按鈕，提供介面以套用與配置內建自動化功能。
  • 部分自動化項目搭配 Eskiemoh 製作的精美動畫，使用 Patreon JB2A 及 Animated Spell Effects: Cartoon。
  • 為面對面遊戲設計的手動擲骰自訂結果解析器。
  • 從 Midi-QOL 工作流程擴展的自訂 API，實現精確事件時序與重疊法術效果自動化。
  • 提供公開 API 可執行自訂巨集。
  • 多種可選的生活品質 UI 擴充功能。
  • 效果變化包括自動描述、狀態效果圖示更換，以及為狀態效果自動套用相關 Midi-QOL 旗標。
  • 自訂效果介面，用於儲存與套用自訂效果。
  • 可選及自訂規則機制，包括 DMG 連斬、爆炸治療、BG3 武器動作等。
  • 自動角色備份。
