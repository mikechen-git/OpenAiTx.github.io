Readme

자동화된 아이템(주문, 클래스 기능, 몬스터 기능 등)과 이를 가능하게 하는 메커니즘, 고도의 자동화를 지원하는 Midi-QOL 기반 D&D5e 환경을 위한 품질 향상 확장 기능 모음입니다. 이 모듈에 포함된 컴펜디움에는 아이템 설명이 포함되어 있지 않습니다. 본 모듈은 여러 저자의 모듈에 의존하지만, tposney, Wasp 또는 기타 모듈 저자에게 본 모듈 관련 버그나 이슈로 문의하지 마십시오. 버그 신고 및 대규모 모듈 구현 요청은 GitHub에서 할 수 있습니다. 보다 빠른 지원 및 기능 요청은 Discord 서버에서 가능합니다.

  • Eskiemoh가 제작한 일부 자동화를 위한 화려한 애니메이션 지원(Patreon JB2A 및 Animated Spell Effects: Cartoon 활용)
  • 대면 플레이용 수동 주사위 굴림을 위한 맞춤 롤 리졸버 제공
  • Midi-QOL 워크플로우를 확장한 커스텀 API로 정밀한 이벤트 타이밍과 중첩 주문 효과 자동화 지원
  • 커스텀 매크로 실행을 위한 공개 API 제공
  • 다양한 선택적 품질 향상 UI 확장 기능
  • 자동 설명, 상태 효과 아이콘 변경, Midi-QOL 플래그 적용 등 효과 변화 기능
  • 커스텀 효과 저장 및 적용을 위한 전용 효과 인터페이스
  • DMG Cleave, 폭발 치유, BG3 무기 액션 등 선택적 및 홈브루 메커니즘 지원
  • 자동 캐릭터 백업 기능
