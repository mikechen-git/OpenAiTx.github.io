Readme

Una colección de elementos automatizados, incluyendo conjuros, rasgos de clase, rasgos de monstruos, etc., mecánicas para hacerlos posibles y extensiones de calidad de vida para un entorno D&D5e de alta automatización basado en Midi-QOL. Los compendios incluidos en este módulo no contienen descripciones de los objetos. Aunque este módulo tiene varias dependencias de módulos de diferentes autores, no moleste a tposney, Wasp ni a ningún otro autor de módulos con errores o problemas relacionados con este módulo. Los informes de errores y las solicitudes de implementación de grandes módulos pueden hacerse en GitHub. Para soporte más rápido y solicitudes de funciones, puede acceder al servidor de Discord.

Características Principales:

  • Compendios de conjuros, objetos, rasgos de clase automatizados y más.
  • Botón en la barra de título que proporciona una interfaz para aplicar y configurar las automatizaciones incluidas.
  • Animaciones atractivas creadas por Eskiemoh para algunas automatizaciones, usando Patreon JB2A y Animated Spell Effects: Cartoon.
  • Resolución de tiradas personalizada para tiradas manuales, diseñada para juegos presenciales.
  • API personalizada que extiende el flujo de trabajo de Midi-QOL y permite temporización precisa de eventos y automatización de efectos superpuestos de conjuros.
  • API pública para ejecutar macros personalizadas.
  • Varias extensiones opcionales de la interfaz de usuario orientadas a la calidad de vida.
  • Cambios en los efectos, incluyendo descripciones automáticas, cambios de iconos de efectos de estado y aplicación de flags relevantes de Midi-QOL a los efectos de estado.
  • Interfaz personalizada de efectos para almacenar y aplicar efectos personalizados.
  • Mecánicas opcionales y caseras, incluyendo DMG Cleave, Exploding Heals y BG3 Weapon Actions.
  • Copias de seguridad automáticas de personajes.
