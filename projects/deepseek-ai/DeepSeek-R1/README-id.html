Tautan Makalah👁️

1. Pendahuluan

Kami memperkenalkan model penalaran generasi pertama kami, DeepSeek-R1-Zero dan DeepSeek-R1. DeepSeek-R1-Zero, sebuah model yang dilatih melalui reinforcement learning (RL) skala besar tanpa supervised fine-tuning (SFT) sebagai langkah awal, menunjukkan performa luar biasa dalam penalaran. Dengan RL, DeepSeek-R1-Zero secara alami memunculkan banyak perilaku penalaran yang kuat dan menarik. Namun, DeepSeek-R1-Zero menghadapi tantangan seperti pengulangan tak berujung, keterbacaan yang buruk, dan pencampuran bahasa. Untuk mengatasi masalah ini dan meningkatkan performa penalaran, kami memperkenalkan DeepSeek-R1, yang memasukkan data cold-start sebelum RL. DeepSeek-R1 mencapai performa yang sebanding dengan OpenAI-o1 pada tugas matematika, kode, dan penalaran. Untuk mendukung komunitas riset, kami telah open-source DeepSeek-R1-Zero, DeepSeek-R1, dan enam model dense yang didistilasi dari DeepSeek-R1 berbasis Llama dan Qwen. DeepSeek-R1-Distill-Qwen-32B melampaui OpenAI-o1-mini pada berbagai benchmark, mencapai hasil state-of-the-art baru untuk model dense.

CATATAN: Sebelum menjalankan model seri DeepSeek-R1 secara lokal, kami sangat menyarankan untuk meninjau bagian Rekomendasi Penggunaan.

2. Ringkasan Model

Post-Training: Reinforcement Learning Skala Besar pada Model Dasar

  • Kami langsung menerapkan reinforcement learning (RL) pada model dasar tanpa bergantung pada supervised fine-tuning (SFT) sebagai langkah awal. Pendekatan ini memungkinkan model untuk mengeksplorasi chain-of-thought (CoT) dalam menyelesaikan masalah kompleks, menghasilkan DeepSeek-R1-Zero. DeepSeek-R1-Zero menunjukkan kemampuan seperti verifikasi mandiri, refleksi, dan menghasilkan CoT panjang, menjadi tonggak penting bagi komunitas riset. Secara khusus, ini adalah riset terbuka pertama yang memvalidasi bahwa kemampuan penalaran LLM dapat didorong murni melalui RL, tanpa membutuhkan SFT. Terobosan ini membuka jalan untuk kemajuan masa depan di bidang ini.

  • Kami memperkenalkan pipeline kami untuk mengembangkan DeepSeek-R1. Pipeline ini menggabungkan dua tahap RL yang bertujuan untuk menemukan pola penalaran yang lebih baik dan menyelaraskan dengan preferensi manusia, serta dua tahap SFT yang berfungsi sebagai benih untuk kemampuan penalaran dan non-penalaran model. Kami percaya pipeline ini akan bermanfaat bagi industri dengan menciptakan model yang lebih baik.

Distilasi: Model Kecil Juga Bisa Kuat

  • Kami menunjukkan bahwa pola penalaran dari model besar dapat didistilasi ke model yang lebih kecil, menghasilkan performa yang lebih baik dibandingkan pola penalaran yang ditemukan melalui RL pada model kecil. DeepSeek-R1 open source, serta API-nya, akan bermanfaat bagi komunitas riset untuk mendistilasi model kecil yang lebih baik di masa depan.
  • Menggunakan data penalaran yang dihasilkan oleh DeepSeek-R1, kami melakukan fine-tuning pada beberapa model dense yang banyak digunakan di komunitas riset. Hasil evaluasi menunjukkan bahwa model dense kecil hasil distilasi memiliki performa sangat baik pada benchmark. Kami open-source checkpoint distilasi 1.5B, 7B, 8B, 14B, 32B, dan 70B berbasis Qwen2.5 dan seri Llama3 untuk komunitas.

6. Cara Menjalankan Secara Lokal

Model DeepSeek-R1

Silakan kunjungi repo DeepSeek-V3 untuk informasi lebih lanjut tentang menjalankan DeepSeek-R1 secara lokal.

CATATAN: Transformers dari Hugging Face belum didukung secara langsung.

Model DeepSeek-R1-Distill

Model DeepSeek-R1-Distill dapat digunakan dengan cara yang sama seperti model Qwen atau Llama.

Sebagai contoh, Anda dapat dengan mudah memulai layanan menggunakan vLLM:

vllm serve deepseek-ai/DeepSeek-R1-Distill-Qwen-32B --tensor-parallel-size 2 --max-model-len 32768 --enforce-eager

Anda juga dapat dengan mudah memulai layanan menggunakan SGLang

python3 -m sglang.launch_server --model deepseek-ai/DeepSeek-R1-Distill-Qwen-32B --trust-remote-code --tp 2

Rekomendasi Penggunaan

Kami merekomendasikan mengikuti konfigurasi berikut saat menggunakan model seri DeepSeek-R1, termasuk benchmarking, untuk mencapai performa yang diharapkan:

  1. Atur suhu dalam rentang 0.5-0.7 (0.6 direkomendasikan) untuk mencegah pengulangan tanpa akhir atau output yang tidak koheren.
  2. Hindari menambahkan prompt sistem; semua instruksi harus berada dalam prompt pengguna.
  3. Untuk masalah matematika, disarankan menyertakan perintah dalam prompt seperti: "Silakan lakukan penalaran langkah demi langkah, dan letakkan jawaban akhir Anda dalam \boxed{}."
  4. Saat mengevaluasi performa model, disarankan untuk melakukan beberapa pengujian dan merata-ratakan hasilnya.

Selain itu, kami telah mengamati bahwa model seri DeepSeek-R1 cenderung melewati pola berpikir (yaitu, mengeluarkan "<think>\n\n</think>") saat merespon beberapa kueri, yang dapat berdampak buruk pada performa model. Untuk memastikan model melakukan penalaran secara menyeluruh, kami merekomendasikan untuk memaksa model memulai responsnya dengan "<think>\n" di awal setiap output.

Prompt Resmi

Di web/app resmi DeepSeek, kami tidak menggunakan prompt sistem tetapi merancang dua prompt khusus untuk unggah file dan pencarian web demi pengalaman pengguna yang lebih baik. Selain itu, suhu di web/app adalah 0.6.

Untuk unggah file, silakan ikuti template berikut untuk membuat prompt, di mana {file_name}, {file_content} dan {question} adalah argumen.

file_template = \
"""[file name]: {file_name}
[file content begin]
{file_content}
[file content end]
{question}"""

Untuk Pencarian Web, {search_results}, {cur_date}, dan {question} adalah argumen.

Untuk kueri dalam bahasa Tionghoa, kami menggunakan prompt:

search_answer_zh_template = \
'''# 以下内容是基于用户发送的消息的搜索结果:
{search_results}
在我给你的搜索结果中，每个结果都是[webpage X begin]...[webpage X end]格式的，X代表每篇文章的数字索引。请在适当的情况下在句子末尾引用上下文。请按照引用编号[citation:X]的格式在答案中对应部分引用上下文。如果一句话源自多个上下文，请列出所有相关的引用编号，例如[citation:3][citation:5]，切记不要将引用集中在最后返回引用编号，而是在答案对应部分列出。
在回答时，请注意以下几点：
- 今天是{cur_date}。
- 并非搜索结果的所有内容都与用户的问题密切相关，你需要结合问题，对搜索结果进行甄别、筛选。
- 对于列举类的问题（如列举所有航班信息），尽量将答案控制在10个要点以内，并告诉用户可以查看搜索来源、获得完整信息。优先提供信息完整、最相关的列举项；如非必要，不要主动告诉用户搜索结果未提供的内容。
- 对于创作类的问题（如写论文），请务必在正文的段落中引用对应的参考编号，例如[citation:3][citation:5]，不能只在文章末尾引用。你需要解读并概括用户的题目要求，选择合适的格式，充分利用搜索结果并抽取重要信息，生成符合用户要求、极具思想深度、富有创造力与专业性的答案。你的创作篇幅需要尽可能延长，对于每一个要点的论述要推测用户的意图，给出尽可能多角度的回答要点，且务必信息量大、论述详尽。
- 如果回答很长，请尽量结构化、分段落总结。如果需要分点作答，尽量控制在5个点以内，并合并相关的内容。
- 对于客观类的问答，如果问题的答案非常简短，可以适当补充一到两句相关信息，以丰富内容。
- 你需要根据用户要求和回答内容选择合适、美观的回答格式，确保可读性强。
- 你的回答应该综合多个相关网页来回答，不能重复引用一个网页。
- 除非用户要求，否则你回答的语言需要和用户提问的语言保持一致。

# 用户消息为：
{question}'''

7. Lisensi

Repositori kode ini dan bobot model dilisensikan di bawah MIT License. Seri DeepSeek-R1 mendukung penggunaan komersial, mengizinkan segala modifikasi dan karya turunan, termasuk, namun tidak terbatas pada, distilasi untuk melatih LLM lain. Perlu dicatat bahwa:

  • DeepSeek-R1-Distill-Qwen-1.5B, DeepSeek-R1-Distill-Qwen-7B, DeepSeek-R1-Distill-Qwen-14B dan DeepSeek-R1-Distill-Qwen-32B diturunkan dari Qwen-2.5 series, yang awalnya dilisensikan di bawah Apache 2.0 License, dan kini di-finetune dengan 800k sampel yang dikurasi dengan DeepSeek-R1.
  • DeepSeek-R1-Distill-Llama-8B diturunkan dari Llama3.1-8B-Base dan awalnya dilisensikan di bawah Llama3.1 license.
  • DeepSeek-R1-Distill-Llama-70B diturunkan dari Llama3.3-70B-Instruct dan awalnya dilisensikan di bawah Llama3.3 license.
