เอกสารประกอบ • ส่งรายงานปัญหา • ติดต่อเรา

Editor คือส่วนขยายเครื่องมือแก้ไขข้อความ (rich text editor) พร้อมด้วยคอมโพเนนต์ Taiga UI ที่เกี่ยวข้อง

การมีส่วนร่วม

หากคุณมีข้อเสนอแนะในการปรับปรุง Editor หรือพบข้อผิดพลาด กรุณาเปิด issue! เราต้องการการมีส่วนร่วมทุกรูปแบบ

ดูรายละเอียดเพิ่มเติมได้ที่ แนวทางการมีส่วนร่วม

การดูแลรักษา

Editor เป็นส่วนหนึ่งของชุดไลบรารี Taiga UI ซึ่งได้รับการสนับสนุนและใช้งานโดยองค์กรขนาดใหญ่ คุณจึงมั่นใจได้ในเรื่องการดูแลและการพัฒนาอย่างต่อเนื่อง
