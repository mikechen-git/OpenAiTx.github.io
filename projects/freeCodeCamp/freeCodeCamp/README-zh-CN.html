freeCodeCamp.org 的开源代码库与课程体系

freeCodeCamp.org 是一个友好的社区，你可以在这里免费学习编程。该网站由一个由捐赠者支持的 501(c)(3) 慈善机构运营，旨在帮助数百万忙碌的成年人转型进入科技领域。我们的社区已经帮助了超过 40,000 人获得了他们的第一份开发者工作。

我们的全栈 Web 开发和机器学习课程完全免费，且自定进度。我们有成千上万个交互式编程挑战，帮助你拓展技能。

认证证书

freeCodeCamp.org 提供多项免费的开发者认证证书。每项认证都需要你构建 5 个必做的 Web 应用项目，并配有数百个可选编程挑战，帮助你为项目做好准备。我们估计每个认证证书对初学者来说大约需要 300 小时完成。

freeCodeCamp.org 课程体系中的这 50 个项目，每一个都有独立的敏捷用户故事和自动化测试。这些有助于你逐步完善项目，并确保在提交前已经满足所有用户故事。

你可以通过 freeCodeCamp 的 CDN 引入这些测试套件。这意味着你可以在 CodePen、Replit 或你本地的开发环境中构建这些项目。

一旦你获得了认证证书，它将永久属于你。你可以随时将其链接到你的 LinkedIn 或简历。当你的潜在雇主或自由客户点击该链接时，他们会看到专属于你的验证认证。

唯一的例外情况是我们发现有违反我们学术诚信政策的行为。当我们发现有人明确抄袭（提交他人的代码或项目且未注明），我们会像所有严谨的学习机构那样——撤销他们的证书并封禁这些人。
